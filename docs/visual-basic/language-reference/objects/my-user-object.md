 Informace o metody a vlastnosti `My.User` objektu, najdete v části <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>.  
  
 Další informace najdete v tématu [přístup k datům uživatele](../../../visual-basic/developing-apps/programming/accessing-user-data.md).
