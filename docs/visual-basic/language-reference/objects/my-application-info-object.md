# <a name="myapplicationinfo-object"></a>My.Application.Info – objekt
Poskytuje vlastnosti pro získání informací o aplikaci, například číslo verze, popis, načíst sestavení a tak dále.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Application.Info` objektu, najdete v části <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Můžete použít vlastnosti <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> třídy ke získání informací o souboru na disku.  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Třída:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Sestavení:** jazyka Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
