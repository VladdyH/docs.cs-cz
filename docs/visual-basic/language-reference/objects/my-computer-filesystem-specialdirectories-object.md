## <a name="remarks"></a>Poznámky  
 Informace o tom, metody a vlastnosti `My.Computer.FileSystem.SpecialDirectories` objektu, najdete v článku <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Další informace najdete v tématu [postupy: načtení obsahu adresáře Moje dokumenty](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).
