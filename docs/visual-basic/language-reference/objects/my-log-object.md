# <a name="mylog-object"></a>My.Log – objekt
Poskytuje vlastnosti a metody pro zápis do aplikace naslouchající protokolům událostí a výjimek informace.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Log` objektu, najdete v části <xref:Microsoft.VisualBasic.Logging.AspLog>.  
  
 `My.Log` Je k dispozici pro aplikace ASP.NET pouze objekt. Pro klientské aplikace, použijte [My.Application.Log – objekt](../../../visual-basic/language-reference/objects/my-application-log-object.md).
