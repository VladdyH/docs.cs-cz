# <a name="mycomputermouse-object"></a>My.Computer.Mouse – objekt
Poskytuje vlastnosti pro získání informací o formátu a konfiguraci myši v místním počítači nainstalována.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Computer.Mouse` objektu, najdete v části <xref:Microsoft.VisualBasic.Devices.Mouse>.  
  
 Další informace najdete v tématu [přístup k myši](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md).  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Třída:** <xref:Microsoft.VisualBasic.Devices.Mouse>  
  
 **Sestavení:** jazyka Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
