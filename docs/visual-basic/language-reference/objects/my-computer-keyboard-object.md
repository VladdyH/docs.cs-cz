# <a name="mycomputerkeyboard-object"></a>My.Computer.Keyboard – objekt
Poskytuje vlastnosti pro přístup k aktuální stav klávesnice, jako je například co klíče jsou aktuálně stisknutí a poskytuje metody k odeslání stisknutí kláves do aktivního okna.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Computer.Keyword` objektu, najdete v části <xref:Microsoft.VisualBasic.Devices.Keyboard>.  
  
 Další informace najdete v tématu [přístup ke klávesnici](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-keyboard.md).  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:**<xref:Microsoft.VisualBasic.Devices>  
  
 **Třída:**<xref:Microsoft.VisualBasic.Devices.Keyboard>  
  
 **Sestavení:** jazyka Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
