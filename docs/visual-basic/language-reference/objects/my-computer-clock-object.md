# <a name="mycomputerclock-object"></a>My.Computer.Clock – objekt
Poskytuje vlastnosti pro přístup k aktuálním místním časem a světového koordinovaného času (greenwichský střední čas ekvivalent) z systémových hodin.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Computer.Clock` objektu, najdete v části <xref:Microsoft.VisualBasic.Devices.Clock>.  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:**<xref:Microsoft.VisualBasic.Devices>  
  
 **Třída:**<xref:Microsoft.VisualBasic.Devices.Clock>  
  
 **Sestavení:** jazyka Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
