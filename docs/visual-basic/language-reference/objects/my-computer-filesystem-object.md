# <a name="mycomputerfilesystem-object"></a>My.Computer.FileSystem – objekt
Poskytuje vlastnosti a metody pro práci s disky, souborů a adresářů.  
  
## <a name="remarks"></a>Poznámky  
 Informace o metody a vlastnosti `My.Computer.FileSystem` objektu, najdete v části <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
 Další informace najdete v tématu [přístup k souborům v jazyce Visual Basic](../../../visual-basic/developing-apps/programming/drives-directories-files/file-access.md).  
  
> [!NOTE]
>  Můžete také použít třídy v <xref:System.IO?displayProperty=nameWithType> obor názvů pro práci s disky, souborů a adresářů.  
  
## <a name="requirements"></a>Požadavky  
 **Namespace:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Třída:** <xref:Microsoft.VisualBasic.MyServices.FileSystemProxy> (poskytuje přístup k <xref:Microsoft.VisualBasic.FileIO.FileSystem>)  
  
 **Sestavení:** jazyka Visual Basic Runtime Library (v souboru Microsoft.VisualBasic.dll)
