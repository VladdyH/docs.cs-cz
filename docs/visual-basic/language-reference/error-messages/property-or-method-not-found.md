# <a name="property-or-method-not-found"></a>Vlastnost nebo metoda nebyla nalezena.
Metody odkazovaného objektu nebo objekt vlastnosti není definován.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Možná jste zadali nesprávný název objektu. Pokud chcete zobrazit, jaké vlastnosti a metody jsou definovány pro objekt, zobrazí prohlížeč objektů. Vyberte příslušné objektu knihovny, chcete-li zobrazit seznam dostupných vlastností a metod.
