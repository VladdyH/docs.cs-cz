# <a name="string-constants-must-end-with-a-double-quote"></a>Řetězcové konstanty musí končit dvojitými uvozovkami.
Řetězcové konstanty musí začínat a končit znaky uvozovek.  
  
 **ID chyby:** BC30648  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že řetězcový literál končí uvozovky ("). Pokud můžete vložit hodnoty z jiné textové editory, ujistěte se, vložených znak je platný uvozovky a mezi znaky, které se podobají, jako je například "smart" nebo "složené" uvozovky ("nebo") nebo dva jednoduché uvozovky (").
