# <a name="type-mismatch-visual-basic"></a>Neshoda typu (Visual Basic)
Pokusili jste se převést hodnotu na jiný typ způsobem, který není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte přiřazení a ujistěte se, že je platný.  
  
2.  Zajistěte, aby že nebyla úspěšná objekt na procedury, která očekává jednu vlastnost nebo hodnotu.  
  
3.  Ujistěte se, že jste nepoužili název modulu nebo produktu project kde byl očekáván výraz.
