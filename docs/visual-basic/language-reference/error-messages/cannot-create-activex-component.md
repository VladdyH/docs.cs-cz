# <a name="cannot-create-activex-component"></a>Součást prvku ActiveX nelze vytvořit.
Pokusili jste se umístit ovládacího prvku ActiveX ve formuláři v době návrhu nebo přidání formuláře do projektu s ovládacím prvkem ActiveX v něm, ale přidružené informace v registru nebyla nalezena.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Informace v registru byla odstraněna nebo poškozený. Přeinstalujte ovládacího prvku ActiveX nebo kontaktujte dodavatele ovládacího prvku.
