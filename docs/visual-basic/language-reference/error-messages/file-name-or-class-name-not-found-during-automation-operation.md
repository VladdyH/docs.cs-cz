# <a name="file-name-or-class-name-not-found-during-automation-operation-visual-basic"></a>Název souboru nebo třídy nebyl v průběhu operace Automation nalezen. (Visual Basic)
Název zadaný pro název souboru nebo třídy v volání `GetObject` funkce nebyla nalezena.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte názvy a zkuste to znovu. Zajistěte, aby název použitý k `class` parametr se shoduje s zaregistrována v systému.
