# <a name="overflow-visual-basic-error"></a>Přetečení (chyba jazyka Visual Basic)
Literál reprezentuje hodnotu mimo hranice datového typu, který je právě přiřazován.  
  
 **ID chyby:** BC30036  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Cílový datový typ a přepište literál tak, aby odpovídala rozsahu najdete rozsah hodnot.
