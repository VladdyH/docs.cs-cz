# <a name="argument-not-optional-visual-basic"></a>Argument není nepovinný (Visual Basic).
Počet a typy argumentů musí shodovat s hodnotami očekává. Buď je nesprávný počet argumentů nebo vynechání argument není nepovinný. Argument lze vynechat jenom volání procedury definované uživatelem, pokud byla deklarována `Optional` v definici postupu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte všechny potřebné argumenty.  
  
2.  Zajistěte, aby vynechání argumenty jsou volitelné. Pokud tomu tak není, zadejte argument ve volání buď, nebo parametr deklarovat `Optional` v definici.
