# <a name="array-subscript-expression-missing"></a>Výraz dolního indexu pole chybí.
Inicializace pole zanechává na jeden nebo více dolní indexy, které definují meze pole. Příkaz může například obsahovat výraz `myArray (5,5,,10)`, což ponechá na třetí dolní index.  
  
 **ID chyby:** BC30306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte chybějící dolní index.
