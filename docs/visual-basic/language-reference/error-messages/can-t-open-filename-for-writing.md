# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Může & č. 39; otevřete t & č. 39; &lt;filename&gt;& č. 39; pro zápis.
Zadaný soubor nelze otevřít pro zápis, případně vzhledem k tomu, že již byla otevřena.  
  
 **ID chyby:** BC2012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zavřete soubor a znovu ho otevřete.  
  
2.  Zkontrolujte v souboru oprávnění.
