# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Můžete&#39;t otevřete &#39; &lt;filename&gt; &#39; pro zápis.
Zadaný soubor nelze otevřít pro zápis, případně vzhledem k tomu, že již byla otevřena.  
  
 **ID chyby:** BC2012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zavřete soubor a znovu ho otevřete.  
  
2.  Zkontrolujte v souboru oprávnění.
