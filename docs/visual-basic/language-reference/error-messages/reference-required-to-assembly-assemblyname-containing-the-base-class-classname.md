# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-base-class-39ltclassnamegt39"></a>Odkaz vyžadoval sestavení &#39; &lt;assemblyname&gt; &#39; obsahující základní třídu &#39; &lt;elementem classname&gt;&#39;
Odkaz vyžadoval sestavení '\<assemblyname >' obsahující základní třídu\<classname >'. Přidáte do projektu.  
  
 Je třída definovaná v dynamické knihovně (DLL) nebo sestavení, který není přímo odkazuje ve vašem projektu. Visual Basic – kompilátor vyžaduje odkaz, aby se zabránilo nejednoznačnosti v případě, že je třída definovaná v více než jeden knihovny DLL nebo sestavení.  
  
 **ID chyby:** BC30007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrňte název neregistrované knihovny DLL nebo sestavení odkazy projektu.
