# <a name="39sub-main39-was-not-found-in-39ltnamegt39"></a>&#39;Sub Main&#39; nebyl nalezen v &#39; &lt;název&gt;&#39;
`Sub Main` je pro něj byla zadána chybějící nebo nesprávné umístění.  
  
 **ID chyby:** BC30420  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte chybějící `Sub Main` prohlášení, nebo pokud existuje, ho přesunout do příslušného umístění v kódu. Další informace o `Sub Main`, najdete v části [hlavní procedura v jazyce Visual Basic](../../../visual-basic/programming-guide/program-structure/main-procedure.md).  
  
2.  Zadejte umístění objektu spuštění projektu v **formulář spuštění** pole **Návrhář projektu**.
