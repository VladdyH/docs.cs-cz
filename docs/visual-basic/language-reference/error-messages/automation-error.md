# <a name="automation-error"></a>Chyba automatizace
Došlo k chybě při spuštění metody nebo získání nebo nastavení vlastností proměnné objektu. Chyba byla hlášena aplikace, která vytvoří objekt.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte vlastnosti `Err` objektem pro určení zdroj a povaze chyby.  
  
2.  Použití `On Error Resume Next` bezprostředně před přístupem příkaz a potom vyhledejte chyby okamžitě po příkazu přístupem.
