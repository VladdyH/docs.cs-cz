# <a name="file-not-found-visual-basic-run-time-error"></a>Soubor nebyl nalezen (chyba za běhu jazyka Visual Basic).
Soubor nebyl nalezen, kde je zadán. Chyba má následující možné příčiny:  
  
-   Příkaz odkazuje na soubor, který neexistuje.  
  
-   Došlo pokusu o volání procedury v dynamické knihovně (DLL), ale knihovny zadaný v `Lib` klauzuli `Declare` příkaz nebyl nalezen.  
  
-   Jste se pokusili otevřít projekt nebo načíst textový soubor, který neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název souboru a specifikace cesty.
