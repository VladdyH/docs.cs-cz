# <a name="39addressof39-operand-must-be-the-name-of-a-method-without-parentheses"></a>&#39;AddressOf&#39; operand musí být název metody (bez závorek).
`AddressOf` Operátor vytvoří instanci postup delegáta, který odkazuje na zvláštní postup. Syntaxe je následující.  
  
 `AddressOf``procedurename`  
  
 Vložit v závorkách následující argument `AddressOf`, kde není potřeba.  
  
 **ID chyby:** BC30577  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat závorkách následující argument `AddressOf`.  
  
2.  Zkontrolujte, zda že je název metody.
