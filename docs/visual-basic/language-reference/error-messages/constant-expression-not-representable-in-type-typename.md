# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>Konstantní výraz nelze reprezentovat v typu č. 39; &lt;typename&gt;& č. 39;
Pokoušíte se vyhodnotit konstanta, který se nevejde do typu cíle obvykle, protože ho je přetečení rozsahu.  
  
 **ID chyby:** BC30439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte typ cíle na takový, který dokáže zpracovat konstanta.
