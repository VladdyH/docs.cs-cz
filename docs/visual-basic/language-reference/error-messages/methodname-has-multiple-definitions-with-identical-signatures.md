# <a name="39ltmethodnamegt39-has-multiple-definitions-with-identical-signatures"></a>& č. 39; &lt;methodname&gt;& č. 39; má několik definic s identickými podpisy
A `Function` nebo `Sub` deklarace postup používá seznam název a argument identické postupu jako předchozí deklarace. Možnou příčinou je pokus o přetížení původní procedury. Přetížené procedury musí mít jiný argument seznamy.  
  
 **ID chyby:** BC30269  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název procedury nebo argument seznamu, nebo odeberte duplicitní deklarace.
