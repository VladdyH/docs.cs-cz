# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>Identifikátor URI oboru názvů XML & č. 39; http://www.w3.org/XML/1998/namespace & č. 39; může mít pouze vazby & č. 39; xmlns & č. 39;
Identifikátor URI http://www.w3.org/XML/1998/namespace se používá v deklarace oboru názvů XML. Tento identifikátor URI je vyhrazený obor názvů a nemůže být součástí deklarace oboru názvů XML.  
  
 **ID chyby:** BC31183  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat deklaraci oboru názvů XML nebo nahradit identifikátoru URI http://www.w3.org/XML/1998/namespace platný identifikátor URI oboru názvů.
