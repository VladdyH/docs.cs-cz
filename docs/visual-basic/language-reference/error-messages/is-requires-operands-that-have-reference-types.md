# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>& č. 39; je & č. 39; vyžaduje operandy mají odkazové typy, ale tento operand je typ hodnoty & č. 39; &lt;typename&gt;& č. 39;
`Is` Relační operátor určuje, zda dva objektové proměnné odkazují na stejnou instanci. Toto porovnání není definován pro typy hodnot.  
  
 **ID chyby:** BC30020  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použít příslušné aritmetické relační operátor nebo `Like` operátor k porovnání dvou typů hodnotu.
