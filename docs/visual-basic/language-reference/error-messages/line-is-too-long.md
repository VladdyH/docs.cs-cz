# <a name="line-is-too-long"></a>Řádek je příliš dlouhý.
Řádky textu zdroje nesmí překročit 65 535 znaků.  
  
 **ID chyby:** BC30494  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkraťte délku řádek pro 65 535 znaků nebo méně.
