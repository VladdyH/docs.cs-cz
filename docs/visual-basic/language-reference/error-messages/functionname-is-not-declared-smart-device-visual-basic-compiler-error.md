# <a name="39ltfunctionnamegt39-is-not-declared-smart-devicevisual-basic-compiler-error"></a>& č. 39; &lt;%{FunctionName/&gt;& č. 39; není deklarovaný (inteligentní zařízení nebo Visual Basic chyba kompilátoru)
<`functionname`> není deklarovaný. Funkce souborových vstupně-výstupních operací je dostupné v `Microsoft.VisualBasic` obor názvů, ale cílové verze rozhraní .NET Compact Framework ji nepodporuje.  
  
 **ID chyby:** BC30766  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Provedení operace se soubory s funkcí definovaných v `System.IO` oboru názvů.
