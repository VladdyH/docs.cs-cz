# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>Volání procedury nebo argument není platné (Visual Basic).
Některé části volání nelze dokončit.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte povolené rozsahy argumenty, abyste měli jistotu, že žádné uspořádání překračuje povolené hodnoty.
