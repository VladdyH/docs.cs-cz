# <a name="39optional39-expected"></a>&#39;Volitelné&#39; očekávání
Požadovaný argument následuje za volitelným argumentem v deklaraci postupu. Každý argument za volitelným argumentem musí být také volitelné.  
  
 **ID chyby:** BC30202  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Pokud je argument bude vyžadovat, přesuňte jej na atributy stála před první nepovinný argument v seznamu argumentů.  
  
2.  Pokud argument má být volitelné, použijte `Optional` – klíčové slovo.
