# <a name="attributes-visual-basic"></a>Atributy (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]poskytuje několik atributů, které umožňují, že objekty spolupráce s nespravovaným kódem a jeden atribut, který umožňuje členům modul bez názvu modulu získat přístup. Následující tabulka uvádí atributy používané [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Dá pokyn kompilátoru přidat metadata, která umožňuje třídy mají být exponovány jako objekt COM.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Umožňuje členům modul přístup k použití pouze kvalifikace potřebné pro modul.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Označuje, že má být použit ve struktuře nebo jiné než místní proměnné pole jako pole pevnou délkou.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Označuje, že řetězec má být použit jako by se měla pevná délka.|
