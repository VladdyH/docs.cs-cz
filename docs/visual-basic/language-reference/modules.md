# <a name="modules-visual-basic"></a>Moduly (Visual Basic)
Visual Basic poskytuje několik modulů, které umožňují zjednodušit běžné úlohy v kódu, včetně manipulace s řetězci, provádění výpočty, získávání informací o systému, provádění operací se soubory a adresáře a tak dále. V následující tabulce jsou uvedeny v modulech jazyka Visual Basic.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants>|Obsahuje různé konstanty. Tyto konstanty lze použít kdekoli v kódu.|  
|<xref:Microsoft.VisualBasic.ControlChars>|Obsahuje konstantní řídicí znaky pro tisk a zobrazení textu.|  
|<xref:Microsoft.VisualBasic.Conversion>|Obsahuje členy převodu desetinného čísla na jiných základů čísla řetězce, řetězců na čísla a jeden datový typ, který má jiné.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|Obsahuje členy získat aktuální datum nebo čas, provádět výpočty data, vrátí datum nebo čas, nastavení data a času nebo času dobu trvání procesu.|  
|<xref:Microsoft.VisualBasic.ErrObject>|Obsahuje informace o běhové chyby a metody vyvolání nebo vymazat chybu.|  
|<xref:Microsoft.VisualBasic.FileSystem>|Obsahuje členy, které provádějí operace souboru, adresáře nebo složky a systému.|  
|<xref:Microsoft.VisualBasic.Financial>|Obsahuje postupy, které jsou používány k provádění finanční výpočty.|  
|<xref:Microsoft.VisualBasic.Globals>|Obsahuje informace o aktuální verze skriptovací stroje.|  
|<xref:Microsoft.VisualBasic.Information>|Obsahuje členy, kteří vrátit, testovat nebo ověřit informace, například velikost pole, názvy typů a tak dále.|  
|<xref:Microsoft.VisualBasic.Interaction>|Obsahuje členy interakci s objekty, aplikace a systémy.|  
|<xref:Microsoft.VisualBasic.Strings>|Obsahuje členy provést operace s řetězci například přeformátování řetězce, hledání řetězce, získávání délky řetězce a tak dále.|  
|<xref:Microsoft.VisualBasic.VBMath>|Obsahuje členy provádět matematické operace.|
