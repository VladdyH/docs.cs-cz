## <a name="reserved-keywords"></a>Vyhrazená slova  
 Následující klíčová slova jsou *vyhrazené*, což znamená, že je nelze použít jako názvy pro programování prvky jako jsou například postupy nebo proměnných. Toto omezení můžete obejít, zadejte název do hranatých závorek (`[]`). Další informace najdete v tématu "Uvozené názvy" v [deklarované názvy elementů](../../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
> [!NOTE]
>  Nedoporučujeme používat řídicí názvy, protože váš kód mohl zajistit těžko čitelný, a může vést k jemně chyb, které může být obtížné vyhledat.
