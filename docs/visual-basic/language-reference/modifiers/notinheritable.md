# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Určuje, že třídy nelze použít jako základní třída.  
  
## <a name="remarks"></a>Poznámky  
 **Alternativní podmínky**. Třída, která nemůže být zděděna se někdy nazývá *zapečetěné* třídy.  
  
 `NotInheritable` Modifikátor můžete v tomto kontextu použít:
