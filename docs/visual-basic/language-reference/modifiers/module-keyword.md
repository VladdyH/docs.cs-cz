# <a name="module-ltkeywordgt-visual-basic"></a>Modul &lt;– klíčové slovo&gt; (Visual Basic)
Určuje, že atribut na začátku zdrojového souboru platí pro aktuální modul sestavení.  
  
## <a name="remarks"></a>Poznámky  
 Počet atributů se týkají jednotlivých programovací element, jako je například třída nebo vlastnost. Použít takového atributu připojením atribut bloku, v rámci lomené závorky (`< >`), přímo k příkazu deklarace.  
  
 Pokud atribut se vztahují pouze na následující element, ale aktuální sestavení modulu, můžete umístit atribut bloku na začátku zdrojový soubor a identifikovat atribut s `Module` – klíčové slovo. Pokud se vztahuje na celý sestavení, můžete použít [sestavení](../../../visual-basic/language-reference/modifiers/assembly.md) – klíčové slovo.  
  
 `Module` Modifikátor není stejný jako [Module – příkaz](../../../visual-basic/language-reference/statements/module-statement.md).
