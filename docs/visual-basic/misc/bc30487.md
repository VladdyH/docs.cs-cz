# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>Operátor &#39; &lt;operatorname&gt; &#39; není definován pro typ &#39; &lt;typename&gt;&#39;
Pokusili jste se použít unární operátor (`+`, `-`, nebo `Not`) podle typu, například `String`, pro který není definován.  
  
 **ID chyby:** BC30487  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Unární operátor nahraďte příslušné operátor pro výraz.
