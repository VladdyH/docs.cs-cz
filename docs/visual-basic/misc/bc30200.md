# <a name="39new39-is-not-valid-in-this-context"></a>& č. 39; Nové & č. 39; v tomto kontextu není platné
`New` – Klíčové slovo dojde, kdy je potřeba název typu. Kontext volání pro typ hodnoty, které nelze kombinovat s `New`.  
  
 **ID chyby:** BC30200  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `New` – klíčové slovo.
