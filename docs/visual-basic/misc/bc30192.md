# <a name="end-of-parameter-list-expected"></a>Konec seznam parametrů očekávání
A `ParamArray` specifikace následuje jeden nebo více dalších argument specifikacích v deklaraci postupu.  
  
 **ID chyby:** BC30192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `ParamArray` na konec seznamu argumentů.
