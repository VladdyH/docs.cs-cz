# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Metoda nemůže mít ParamArray i volitelné parametry
A `ParamArray` argument je automaticky volitelná a musí být pouze nepovinný argument v postupu deklaraci. Je nutné zadat všechny předchozí argumenty.  
  
 **ID chyby:** BC30046  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte volitelné argumenty v deklaraci.
