# <a name="39structure39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;Struktura&#39; omezení nelze zadat vícekrát pro stejný parametr typu
Seznam omezení obsahuje [struktury (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) omezení více než jednou.  
  
 Seznam omezení pro parametr typu můžete určit, že argument typu předaný parametr typu musí být typ hodnoty (se `Structure` omezení) nebo musí být odkazovým typem (s [třídy (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) omezení). Nelze zadat obě omezení na stejný parametr typu, a některé z nich nelze zadat více než jednou.  
  
 ID chyby: BC32102  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny nadbytečné `Structure` klíčová slova. Měli byste mít pouze jednu v seznamu omezení.
