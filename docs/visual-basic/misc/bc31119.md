# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>& č. 39; Ukončení RemoveHandler & č. 39; musí být první příkaz na řádek
`End RemoveHandler` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End RemoveHandler`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31119  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
