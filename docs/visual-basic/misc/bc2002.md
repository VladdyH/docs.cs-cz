# <a name="source-file-ltfilenamegt-specified-multiple-times"></a>Zdrojový soubor &lt;filename&gt; zadán více než jednou.
Soubor byl zadán vícekrát.  
  
 **ID chyby:** BC2002  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátory redundantní souboru.
