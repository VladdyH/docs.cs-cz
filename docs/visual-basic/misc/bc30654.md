# <a name="39return39-statement-in-a-function-get-or-operator-must-return-a-value"></a>&#39;Vrátí&#39; příkaz v funkce, Get nebo operátor musí vrátit hodnotu
`Return` příkazy musí být použít k vrácení hodnoty volání procedury. Nemůžete použít `Return` programu příkazy samy o sobě k řízení toku.  
  
 **ID chyby:** BC30654  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte hodnotu, která může vrátit funkce nebo procedury.  
  
2.  Použití `End` příkaz způsobí program, který chcete ukončit aktuální procedury.
