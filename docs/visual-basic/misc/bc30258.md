# <a name="classes-can-inherit-only-from-other-classes"></a>Třídy lze dědit pouze z jiné třídy
`Inherits` Příkaz určuje jako jeho základní typ typu než třídu.  
  
 **ID chyby:** BC30258  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte základní typ pro třídu, která je již definován.
