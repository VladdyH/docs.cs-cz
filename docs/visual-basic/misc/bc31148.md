# <a name="xml-namespace-prefix-39prefix39-is-not-defined"></a>Předpona oboru názvů XML &#39;předponu&#39; není definován
Literál XML nebo ve vlastnosti osy XML nebyl zadán nedefinované předpony oboru názvů XML.  
  
 **ID chyby:** BC31148  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definovat předponu oboru názvů XML pomocí `Imports` příkaz.  
  
 -nebo-  
  
-   Změňte předponu oboru názvů XML pro literál XML nebo vlastnost osy XML na dříve definované předpony oboru názvů XML.
