# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>Nelze převést počáteční hodnotu typu &#39; &lt;type1&gt;&#39;, omezit hodnota &#39; &lt;type2&gt; &#39; a hodnotu z kroku &#39; &lt;type3&gt; &#39; do společný typ.
Zadaný `For...Next` hodnoty ovládacích prvků mají různé typy a proto nejsou platné vzhledem k tomu, že nelze převést na společný typ.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte typ jednoho nebo více hodnot, tak všechny lze převést na společný typ.
