# <a name="relational-operator-expected"></a>Byl očekáván relační operátor.
A `Case` obsahuje příkaz `Is` klauzule, ale žádné relační operátor jako `=` nebo `>`. Pokud `Case` příkaz nezahrnuje operátor, `=` se předpokládá, že a `Is` se nepoužívá.  
  
 **ID chyby:** BC30239  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Is` – klíčové slovo nebo po ní následuje operátor porovnání.
