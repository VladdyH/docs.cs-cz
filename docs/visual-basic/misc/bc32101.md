# <a name="39class39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>& č. 39; Třída & č. 39; omezení nelze zadat více než jednou pro stejný parametr typu
Obsahuje seznam omezení [třídy (Visual Basic)](http://msdn.microsoft.com/en-us/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) omezení více než jednou.  
  
 Seznam omezení pro parametr typu můžete určit, že typ argument předaný parametr tohoto typu musí být typu hodnoty (s [struktury (Visual Basic)](http://msdn.microsoft.com/en-us/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) omezení) nebo musí být typu odkazu (s `Class` omezení) . Nelze zadat obě omezení na stejný parametr typu a některé z nich nelze zadat více než jednou.  
  
 ID chyby: BC32101  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny redundantní `Class` klíčová slova. Měli byste mít pouze jeden v seznamu omezení.
