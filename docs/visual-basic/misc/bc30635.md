# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;Jen pro čtení&#39; a &#39;WriteOnly&#39; nelze kombinovat
`ReadOnly` a `WriteOnly` modifikátory se vzájemně vylučují.  
  
 **ID chyby:** BC30635  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `ReadOnly` nebo `WriteOnly`, ale ne obojí.
