# <a name="39finally39-must-end-with-a-matching-39end-try39"></a>&#39;Nakonec&#39; musí končit odpovídající &#39;End Try&#39;
A `Finally` příkazu se zobrazí ve vašem kódu bez odpovídající `End Try` příkaz. `Finally` příkazy musí být uzavřena s `End Try` příkaz.  
  
 **ID chyby:** BC30442  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Finally` příkaz.  
  
2.  Přidat `End Try` příkaz dokončete bloku.
