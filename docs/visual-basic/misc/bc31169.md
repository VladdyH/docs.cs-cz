# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Znak & č. 39; znak & č. 39; (0 x&lt;hexadecimální&gt;) není povolená na začátku název XML
Nepodporovaný znak je zahrnuta na začátku názvu literál XML nebo ve vlastnosti osy XML.  
  
 **ID chyby:** BC31169  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nepodporovaný znak, od začátku název literál XML nebo vlastnost osy XML.
