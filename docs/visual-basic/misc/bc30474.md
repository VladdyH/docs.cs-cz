# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-it-is-a-39declare39-statement"></a>& č. 39; &lt;method1&gt;& č. 39; nelze přepsat & č. 39;&lt; method2&gt;& č. 39; protože je & č. 39; Declare & č. 39; – příkaz
Pokoušíte se o přepsání delegáta na název základní třídy, která byla deklarována s `Declare` příkaz.  
  
 **ID chyby:** BC30474  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změnit přepsaného člen, takže není `Declare` příkaz.  
  
2.  Nepokoušejte se přepsat tuto metodu.
