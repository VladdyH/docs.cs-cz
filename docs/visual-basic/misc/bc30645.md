# <a name="attribute-39ltattributenamegt39-cannot-be-applied-to-a-method-with-optional-parameters"></a>Atribut & č. 39; &lt;attributename&gt;& č. 39; nelze použít pro metodu s volitelné parametry
Atribut použít jenom s metody, které používají povinnými argumenty nebo žádné argumenty.  
  
 **ID chyby:** BC30645  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Definujte metoda bez volitelné parametry.  
  
2.  Použijte atribut, který lze použít s metody, které mají volitelné parametry.  
  
3.  Definujte vlastní atribut, který lze použít v tomto kontextu.
