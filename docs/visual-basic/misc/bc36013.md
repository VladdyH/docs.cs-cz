# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;V případě chyby&#39; příkazy nejsou platné v rámci &#39;pomocí&#39; – příkazy
`On Error` Příkazu se zobrazí v rámci `Using` příkaz ale není platný v tomto kontextu.  
  
 **ID chyby:** BC36013  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použít strukturované zpracování, jako například chyb `Try…Catch` bloku, v místě `On Error` příkaz.
