# <a name="39ltmembernamegt39-is-not-declared"></a>& č. 39; &lt;membername&gt;& č. 39; není deklarovaný
'`<membername>`' není deklarovaný. `Debug`objekt funkce je k dispozici v `System.Diagnostics.Debug` v `System` sestavení.  
  
 Název zadaného člena nelze nalézt.  
  
 **ID chyby:** BC30816  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda člen.  
  
2.  Použijte `Imports` příkaz nebo plnému určení členů definovaných v jiných oborech názvů. Například `WriteLine` v deklaraci funkce `System.Diagnostics.Debug` oboru názvů.
