# <a name="39option-compare39-must-be-followed-by-39text39-or-39binary39"></a>&#39;Možnost porovnání&#39; musí následovat &#39;Text&#39; nebo &#39;binární&#39;
`Option Compare` Příkaz obsahuje nesprávné nastavení nebo žádná nastavení. Pouze nastavení povoleno v `Option Compare` jsou `Text` a `Binary`.  
  
 **ID chyby:** BC30207  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, pokud je zadáno chybně specifikátor nastavení.  
  
2.  Buď přidejte `Text` nebo `Binary` k `Option Compare` příkaz, například `Option Compare Text`.
