# <a name="only-one-xml-comment-block-is-allowed-per-language-element"></a>Jeden element jazyka je povolen pouze jeden blok komentáře XML
Do elementu jazyk použilo více než jeden blok komentáře XML.  
  
 **ID chyby:** BC42301  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné blok komentáře XML.
