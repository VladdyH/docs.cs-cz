# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;Sdílené&#39; nelze kombinovat s &#39; &lt;specifikátor&gt; &#39; na deklarace vlastnosti
Pokusili jste se zkombinovat `Shared` modifikátor s specifikátor, který není platný v kombinaci, na deklarace vlastnosti.  
  
 **ID chyby:** BC30502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
