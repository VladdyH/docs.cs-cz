# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-from-39lttype1gt39-to-39lttype2gt39"></a>Argument odpovídající parametru &#39; &lt;parametername&gt; &#39; zúží z &#39; &lt;type1&gt; &#39; k &#39; &lt;type2&gt;&#39;
Provedli jste volání přetížené metody, ale kompilátor nebyla nalezena metoda, kterou lze volat bez zužující převod. Zužující převod změní hodnotu na datový typ, který nemusí být možné přesněji obsahovat některé z možných hodnot.  
  
 **ID chyby:** BC30520  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte `Option Strict Off`.
