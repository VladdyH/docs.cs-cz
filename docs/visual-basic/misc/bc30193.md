# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>Specifikátory a atributy nejsou platná na tento příkaz
Deklaraci, která nemůže být upraven, jako například `Inherits`, obsahuje jeden nebo více atributů, nebo jeden nebo více klíčová slova, jako `Public` nebo `ReadOnly`. Pouze upravitelnými deklarace může obsahovat tyto klíčová slova nebo atributy.  
  
 **ID chyby:** BC30193  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte atribut nebo – klíčové slovo z deklaraci.
