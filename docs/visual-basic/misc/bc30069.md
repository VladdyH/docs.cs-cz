# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>Pro proměnnou řízení smyčky &#39; &lt;NázevProměnné&gt; &#39; již používán v příkazu vnější smyčky For
Vnořené `For` smyčky všechny musí používat proměnné jedinečný ovládacího prvku.  
  
 **ID chyby:** BC30069  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte jednu z proměnné názvy ovládacích prvků.
