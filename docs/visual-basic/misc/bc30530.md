# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr nemůže mít stejný název jako jeho definující funkce
Použili jste se stejným názvem v parametru a jeho definující funkce.  
  
 **ID chyby:** BC30530  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte názvy parametrů v rámci funkce a změňte konfliktní parametr název.
