# <a name="members-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Členové v modulu nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Použili jste specifikátor, který není platný pro člen v rámci `Module` příkaz. Moduly nikdy může být vytvořena, nepodporují dědičnosti a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30436  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
