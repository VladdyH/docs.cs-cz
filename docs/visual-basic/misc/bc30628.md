# <a name="structures-cannot-have-39inherits39-statements"></a>Struktury nemůže mít &#39;Inherits&#39; – příkazy
Na rozdíl od třídy struktury nepodporují dědičnosti.  
  
 **ID chyby:** BC30628  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz ze struktury.  
  
-   Změnit návrh aplikace místo toho použít třídy.
