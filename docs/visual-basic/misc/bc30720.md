# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; na chyby & č. 39; příkazy nejsou platné v příkazové podokno
`On Error`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30720  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `On Error` příkazy z ladění kódu.
