# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Všechny projekty v řešení Visual Basic musí mít jako cíl stejnou platformu, ale projekt, který se pokoušíte přidat cílí na platformě než ten, který je určeného řešení
Všechny projekty v [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] řešení musí mít jako cíl stejnou platformu, ale projektu, které přidáváte cílí na platformě než ten, který je zadán vaše řešení.  
  
 **ID chyby:** BC30765  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte jenom projektů cílených určeného řešení pro stejnou platformu.
