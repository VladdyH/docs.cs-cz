# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39;WithEvents&#39; proměnná nevyvolá všechny události
Objektová proměnná deklarovat s `WithEvents` – klíčové slovo není odkaz na instanci třídy, který může vyvolat události.  
  
 **ID chyby:** BC30591  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že proměnná objektu odkazuje na instanci třídy, který může vyvolat události.
