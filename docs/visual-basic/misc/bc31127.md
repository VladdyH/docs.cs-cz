# <a name="39addhandler39-is-already-declared"></a>& č. 39; AddHandler & č. 39; je již deklarována
Více než jeden `AddHandler` deklarace se zobrazí v deklaraci vlastní události. `AddHandler` Deklarace deklaruje postup použít k přidání obslužné rutiny události.  
  
 **ID chyby:** BC31127  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `AddHandler` příkaz.
