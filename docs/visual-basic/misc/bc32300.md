# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>Objekt již existuje z důvodu chyby nebo odstranění
Na kód element přístup došlo k chybě uvnitř [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilátoru.  
  
 **ID chyby:** BC32300  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kompilace programu zjistíte, pokud se chyba objeví znovu.  
  
2.  Pokud se chyba objeví znovu, přeinstalujte [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilátoru.  
  
3.  Pokud chyba přetrvává i po přeinstalování, shromážděte informace o okolnostech a upozornit služby Microsoft Product Support Services.
