# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39;Sub New&#39; nelze implementace členů rozhraní
`Sub New` je konstruktor a nelze ho implementovat členy.  
  
 **ID chyby:** BC31042  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Implements` příkazy z `Sub New` postupy.
