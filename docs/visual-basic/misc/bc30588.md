# <a name="omitted-argument-cannot-match-a-paramarray-parameter"></a>Není zadán argument se nemůže shodovat parametr ParamArray
Jste zapomněli zadat argument vyžadovanou pro shodu `ParamArray` parametr. `ParamArray`, která umožňuje předat libovolný počet argumentů proceduře, lze použít pouze u poslední argument seznam argumentů.  
  
 **ID chyby:** BC30588  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte vynechání argument.
