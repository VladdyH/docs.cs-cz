# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Částečné metody &#39; &lt;methodname&gt; &#39; nelze použít &#39;implementuje&#39; – klíčové slovo
Definice částečné metody nelze implementovat, rozhraním nebo člena rozhraní.  
  
 **ID chyby:** BC31439  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Implements` příkaz z definice částečné metoda.
