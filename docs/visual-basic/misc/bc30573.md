# <a name="xml-namespace-prefix-39prefix39-is-already-declared"></a>Předpona oboru názvů XML & č. 39; předponu & č. 39; je již deklarována
Předponu oboru názvů XML určené `Imports` příkaz odpovídá dříve definovaném předpony oboru názvů XML.  
  
 **ID chyby:** BC30573  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použijte jedinečný název předpony oboru názvů XML.
