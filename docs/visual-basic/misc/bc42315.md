# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39declare-sub39-language-element"></a>Značka komentáře XML & č. 39; vrátí & č. 39; není povolená na & č. 39; deklarovat sub & č. 39; jazyk – element
Značka komentáře XML, vrátí' není povolená u prvku, deklarovat sub' jazyk. Komentáře XML budou ignorovány.  
  
 Komentáře jazyka XML pro `Declare Sub` nemůže obsahovat deklarace `<`vrátí`>` značky.  
  
 **ID chyby:** BC42315  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat nepodporovanou `<`vrátí`>` značky.
