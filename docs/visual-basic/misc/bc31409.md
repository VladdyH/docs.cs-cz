# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;Metoda&gt; &#39; &lt;methodname1&gt; &#39; musí být deklarován &#39;přetížení&#39; protože jiné &#39; &lt;methodname2&gt; &#39; je deklarovaná &#39; Přetížení&#39;
Další vlastnosti nebo metody se stejným názvem je deklarovaný s `Overloads` modifikátor.  
  
 **ID chyby:** BC31409  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Overloads` modifikátor vlastnost nebo metoda.  
  
2.  Odeberte `Overloads` modifikátor ze všech vlastností nebo metod se stejným názvem.
