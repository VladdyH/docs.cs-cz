# <a name="39exit-operator39-is-not-valid-use-39return39-to-exit-an-operator"></a>&#39;Ukončete operátor&#39; není platný. Použití &#39;vrátit&#39; ukončíte operátor
`Exit Operator` Příkazu se zobrazí v `Operator` postupu.  
  
 Je nutné použít [příkaz Return](../../visual-basic/language-reference/statements/return-statement.md) mají být vráceny od `Operator` postupu. [Ukončovací příkaz](../../visual-basic/language-reference/statements/exit-statement.md) nepřijímá `Operator` – klíčové slovo a `End Operator` příkaz nevrátí kód volání ovládacího prvku.  
  
 **ID chyby:** BC33008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte `Exit Operator` příkaz s `Return` příkaz.
