# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>& č. 39; RaiseEvent – koncové & č. 39; musí být první příkaz na řádek
`End RaiseEvent` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End RaiseEvent`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31120  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
