# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>Parametr s názvem již byl deklarován &#39; &lt;název argumentu argumentname&gt;&#39;
Deklarace procedury definuje dva argumenty se stejným názvem. Všechny názvy argumentu v deklaraci postup musí být jedinečný.  
  
 **ID chyby:** BC30237  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název jednoho z argumentů.
