# <a name="argument-39ltargumentnamegt39-cannot-be-a-multidimensional-array"></a>Argument & č. 39; &lt;název argumentu argumentname&gt;& č. 39; nemůže být multidimenzionální pole
Pořadí argument pole je větší než 1, ale může být pouze jednorozměrné pole.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte pořadí argument na 1.
