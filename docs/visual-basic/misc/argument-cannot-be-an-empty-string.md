# <a name="argument-cannot-be-an-empty-string"></a>Argument nesmí být prázdný řetězec.
Argument byla definována jako prázdný řetězec ("").  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte platnou hodnotu pro argument.
