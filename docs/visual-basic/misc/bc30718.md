# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Vyberte & č. 39; příkazy nejsou platné v příkazové podokno
`Select`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30718  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Select` příkazy z ladění kódu.
