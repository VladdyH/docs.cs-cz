# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>ParamArray parametr musí být jednorozměrné pole
A `ParamArray` argument musí být deklarován jako typ jednorozměrné pole. Určuje více než jednou dimenzí pro deklaraci `ParamArray` argument.  
  
 **ID chyby:** BC30051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny čárkami uvnitř závorek následujících `ParamArray` argument název.
