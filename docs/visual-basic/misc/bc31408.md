# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39;&lt;specifier1&gt; &#39; a &#39; &lt;specifier2&gt; &#39; nelze kombinovat
Dva nekompatibilní specifikátory, jako například `Shadows` a `Overrides` používají společně.  
  
 **ID chyby:** BC31408  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte jednu nebo jiných specifikátor.
