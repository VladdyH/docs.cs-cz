# <a name="option-win32manifest-ignored"></a>/ Win32manifest – možnost Ignorovat
/ Win32manifest – možnost Ignorovat. Lze zadat, pouze pokud jsou cílem sestavení.  
  
 Visual Basic – kompilátor byl předán `/win32manifest` při – možnost kompilátoru `/target` je možnost nastavena na `module`.  
  
 **ID chyby:** BC2034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `/win32manifest` – možnost kompilátoru nebo nastaví `/target` možnost k `exe`, `winexe`, nebo `library`.
