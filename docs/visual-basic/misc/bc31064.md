# <a name="39set39-parameter-must-have-the-same-type-as-the-containing-property"></a>&#39;Nastavit&#39; parametr musí mít stejný typ jako vlastnost obsahující
Parametr pro `Set` procedura property má jiný typ z vlastnosti, které patří do.  
  
 **ID chyby:** BC31064  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Umožňuje změnit datový typ parametru k `Set` tak, aby odpovídala datového typu pro vlastnost. Příklad:  
  
    ```  
    Class Class1  
       ' Declare a local variable to hold the property value.  
       Private Fval As Integer  
  
       Property F() As Integer  
          Get  
             Return Fval  
          End Get  
          Set(ByVal Value As Integer)  
             Fval = Value  
          End Set  
       End Property  
    End Class  
    ```
