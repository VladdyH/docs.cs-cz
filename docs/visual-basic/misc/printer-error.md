# <a name="printer-error"></a>Chyba tiskárny
Došlo k chybě na tiskárně, ale žádné jiné informace byl vrácen do počítače, který odeslal soubor.  
  
### <a name="to-correct-the-error"></a>Opravte chybu  
  
-   Fyzicky zkontrolujte tiskárny. Ujistěte se, že všechna připojení mezi počítačem a tiskárny jsou pevné. Většina tiskáren slouží k zobrazení informace o chybě, například "Offline" nebo "došel papír."
