# <a name="39end-event39-must-be-preceded-by-a-matching-39custom-event39"></a>& č. 39; Koncová událost & č. 39; musí předcházet odpovídající & č. 39; Vlastní události & č. 39;
`End Event` Příkaz probíhá, aniž by odpovídající `Custom Event` příkaz. `End Event`musí předcházet odpovídající `Custom Event` příkaz.  
  
 **ID chyby:** BC31123  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zajistěte, aby předchozím `Custom Event` příkaz je platná a správně zadané.
