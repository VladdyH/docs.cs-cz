# <a name="xml-name-expected"></a>Byl očekáván název XML
Soubor XML literál nebo vlastnost osy XML byla zadána bez názvu.  
  
 **ID chyby:** BC31146  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte název literál XML nebo vlastnost osy XML.
