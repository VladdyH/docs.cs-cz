# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Vlastní události&#39; musí končit odpovídající &#39;koncové události&#39;
A `Custom Event` musí končit deklarace `End Event` příkaz.  
  
 **ID chyby:** BC31114  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda `Custom Event` deklarace končí `End Event` příkaz.
