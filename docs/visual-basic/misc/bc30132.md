# <a name="label-39ltlabelnamegt39-is-not-defined"></a>Popisek &#39; &lt;název_jmenovky&gt; &#39; není definován
Popisek čáry nebo číslo řádku je uvedené, ale není definován v rámci oboru odkazu. Popis musí být v rámci procedury, která obsahuje odkaz na.  
  
 **ID chyby:** BC30132  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit strukturu kód tak, aby popisek čáry a odkaz na v stejným způsobem.
