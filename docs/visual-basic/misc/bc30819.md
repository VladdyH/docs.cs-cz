# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;mathfunction1&gt; &#39; není deklarovaný
'\<mathfunction1 >' není deklarovaný. Funkce se přesunul na třídě System.Math a se nyní označuje jako "\<mathfunction2 >".  
  
 Několik funkcí, které byly vnitřní do jazyka Visual Basic v předchozích verzích byla přesunuta do <xref:System.Math?displayProperty=nameWithType> oboru názvů. Díky jejich funkce více obecně dostupné pro všechny programovací jazyky.  
  
 **ID chyby:** BC30819  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použít metody deklarované v <xref:System.Math?displayProperty=nameWithType>.
