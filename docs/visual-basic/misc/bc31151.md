# <a name="element-is-missing-an-end-tag"></a>Element chybí koncová značka
Literál XML elementu chybí koncová značka XML nebo identifikátor prázdný značky '/' (například `<tagName />`).  
  
 **ID chyby:** BC31151  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat koncovou značku pro značku XML nebo přidání značka prázdný identifikátor ' /' pro soubor XML značky.
