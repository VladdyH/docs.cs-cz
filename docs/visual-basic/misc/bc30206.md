# <a name="39option39-must-be-followed-by-39compare39-39explicit39-39infer39-or-39strict39"></a>&#39;Možnost&#39; musí následovat &#39;porovnat&#39;, &#39;explicitní&#39;, &#39;Infer&#39;, nebo &#39;Strict&#39;
Pouze možnosti, které lze nastavit pomocí `Option` prohlášení o `Compare`, `Explicit`, `Infer`, a `Strict`.  
  
 **ID chyby:** BC30206  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, jestli option – klíčové slovo (`Compare`, `Explicit`, `Infer`, nebo `Strict`) je zadáno chybně.  
  
-   Zkontrolujte, zda `Option` příkaz není úplná.
