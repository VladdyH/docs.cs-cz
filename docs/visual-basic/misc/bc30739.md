# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent –&#39; příkazy nejsou platné v příkazové podokno
`RaiseEvent` nelze použít v **Immediate** okno.  
  
 **ID chyby:** BC30739  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Test `RaiseEvent` příkazy z kódu v **Editor kódu**.
