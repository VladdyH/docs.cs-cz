# <a name="implemented-type-must-be-an-interface"></a>Musí být implementovaná typu rozhraní
`Implements` Příkaz neurčuje rozhraní. Třída může implementovat pouze rozhraní.  
  
 **ID chyby:** BC30232  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že rozhraní název je napsán správně.  
  
2.  Pokud příkaz určuje třídu, zvažte použití `Inherits` příkaz.
