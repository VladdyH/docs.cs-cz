# <a name="xml-attribute-39attribute139-must-appear-before-xml-attribute-39attribute239"></a>Atribut XML & č. 39; atribut1 & č. 39; musí být před atribut XML & č. 39; atribut2 & č. 39;
Atributy v dokumentu XML literálu jsou určené v neplatný pořadí. Literál dokumentu XML platný atribut pořadí podle specifikace XML 1.0. Například `encoding` okamžitě postupujte atribut literál dokumentu XML `version` atribut.  
  
 **ID chyby:** BC31157  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Aktualizujte atribut pořadí pro literál dokumentu XML pomocí pokynů uvedených v specifikace XML 1.0.
