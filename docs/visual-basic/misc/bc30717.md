# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Zkuste & č. 39; příkazy nejsou platné v příkazové podokno
`Try`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30717  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Try` příkazy z ladění kódu.
