# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic – kompilátor se nepodařilo obnovit z k následující chybě: &lt;chyba&gt;
Visual Basic – kompilátor se nepodařilo obnovit z k následující chybě: <`error`>.  
  
 Některé funkce IDE byly zakázány.  
  
 V kompilátoru jazyka Visual Basic došlo k vnitřní chybě.  
  
 **ID chyby:** BC31021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Uložit práci a restartujte Visual Studio.  
  
2.  Pokud se chyba objeví znovu, znovu nainstalujte Visual Studio.  
  
3.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
