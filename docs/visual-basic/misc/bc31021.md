# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic – kompilátor se nepodařilo obnovit z k následující chybě: &lt;chyba&gt;
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilátoru se nepodařilo obnovit z k následující chybě: <`error`>.  
  
 Některé funkce IDE byly zakázány.  
  
 Došlo k vnitřní chybě v [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] kompilátoru.  
  
 **ID chyby:** BC31021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Uložit práci a restartujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Pokud se chyba objeví znovu, přeinstalujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
