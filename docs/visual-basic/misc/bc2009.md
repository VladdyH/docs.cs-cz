# <a name="option-ltoptionnamegt-can-be-followed-only-by-3939-or-39-39"></a>možnost &lt;název_možnosti&gt; může následovat pouze &#39;+&#39; nebo &#39;-&#39;
Text, který následuje možnost není znaménko plus (+) nebo znaménkem minus (-).  
  
 **ID chyby:** BC2009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte text, který následuje možnost znaménkem plus (+) možnost zapnout.  
  
     – nebo –  
  
-   Nahraďte text, který následuje možnost se znaménkem minus (-) Chcete-li možnost vypnout.
