# <a name="no-input-sources-specified"></a>Vstupní zdroje nebyly zadány
Při kompilování, nebyly zadány žádné vstupní zdroje.  
  
 **ID chyby:** BC2008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že jste do souboru zdrojového kódu.
