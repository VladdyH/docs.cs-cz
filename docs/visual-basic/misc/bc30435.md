# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Členové struktury nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Použili jste specifikátor, který není platný pro člen v rámci `Structure` příkaz.  
  
 **ID chyby:** BC30435  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
