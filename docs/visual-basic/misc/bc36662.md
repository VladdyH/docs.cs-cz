# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-the-lambda-expression-and-delegate-39ltdelegatenamegt39"></a><span data-ttu-id="317be-102">Možnost striktní na neumožňuje zužující v typu implicitní převody mezi výrazu lambda a delegáta &#39; &lt;vlastnost delegatename&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="317be-102">Option Strict On does not allow narrowing in implicit type conversions between the lambda expression and delegate &#39;&lt;delegatename&gt;&#39;</span></span>
<span data-ttu-id="317be-103">S `Option Strict` , nemůže mít zužující převod mezi datový typ parametru v delegáta a s odpovídajícím parametrem výrazu lambda přiřazený k proměnné tohoto typu delegáta.</span><span class="sxs-lookup"><span data-stu-id="317be-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a lambda expression assigned to a variable of that delegate type.</span></span> <span data-ttu-id="317be-104">Například následující kód, delegovat `Del` má jeden parametr typu `Integer`.</span><span class="sxs-lookup"><span data-stu-id="317be-104">For example, in the following code, delegate `Del` has one parameter of type `Integer`.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
```  
  
 <span data-ttu-id="317be-105">Proto s odpovídajícím parametrem jakýkoli výraz lambda přiřazený k proměnné typu `Del` může být `Integer` nebo kterýkoli typ dat, pro kterou je rozšiřující převod z `Integer`.</span><span class="sxs-lookup"><span data-stu-id="317be-105">Therefore, the corresponding parameter of any lambda expression assigned to a variable of type `Del` can be an `Integer` or any data type for which there is a widening conversion from `Integer`.</span></span>  
  
```vb  
' Valid.  
Dim example1 As Del = Function(n As Integer) "Valid"  
Dim example2 As Del = Function(n As Long) "Valid"  
  
' Not valid.  
Dim example3 As Del = Function(n As Short) "Not Valid"  
```  
  
 <span data-ttu-id="317be-106">**ID chyby:** BC36662</span><span class="sxs-lookup"><span data-stu-id="317be-106">**Error ID:** BC36662</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="317be-107">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="317be-107">To correct this error</span></span>  
  
-   <span data-ttu-id="317be-108">Změňte datový typ parametru v delegát nebo výrazu lambda, takže požadovaná rozšiřující relace existuje.</span><span class="sxs-lookup"><span data-stu-id="317be-108">Change the data type of the parameter in the delegate or the lambda expression so that the required widening relationship exists.</span></span>  
  
-   <span data-ttu-id="317be-109">Nezadávejte parametr datové typy ve výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="317be-109">Do not specify parameter data types in the lambda expression.</span></span> <span data-ttu-id="317be-110">Typy bude odvodit z odpovídajících parametrů v delegát.</span><span class="sxs-lookup"><span data-stu-id="317be-110">Types will be inferred from the corresponding parameters in the delegate.</span></span>
