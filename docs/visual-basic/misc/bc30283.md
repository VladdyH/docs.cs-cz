# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>&#39;Sub New&#39; nelze deklarovat &#39;přepsání&#39;
Konstruktor označuje, že přepíše zděděné konstruktor. Konstruktory nelze přepsat.  
  
 **ID chyby:** BC30283  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Overrides` – klíčové slovo z `Sub` deklarace.
