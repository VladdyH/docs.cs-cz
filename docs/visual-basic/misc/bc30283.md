# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>& č. 39; Sub – nové & č. 39; nelze deklarovat & č. 39; přepsání & č. 39;
Konstruktor označuje, že přepíše zděděné konstruktor. Konstruktory nelze přepsat.  
  
 **ID chyby:** BC30283  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Overrides` – klíčové slovo z `Sub` deklarace.
