# <a name="resource-name-39ltresourcenamegt39-cannot-be-used-more-than-once"></a>Název prostředku &#39; &lt;resourcename&gt; &#39; nelze použít více než jednou.
Název zadaný prostředek propojené dynamické knihovny nebo spustitelný soubor byl použit více než jednou.  
  
 **ID chyby:** BC31502  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jedinečné jméno prostředky.
