# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argument &#39;přístup&#39; není platný. Platné hodnoty pro výstup režimu jsou &#39;OpenAccess.Write&#39; a &#39;OpenAccess.Default&#39;
K přístupu hodnota není platná v režimu výstup.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte hodnotu na buď `OpenAccess.Write` nebo `OpenAccess.Default`.
