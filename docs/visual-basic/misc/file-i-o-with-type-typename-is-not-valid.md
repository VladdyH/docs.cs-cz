# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>Soubor vstupně-výstupních operací s typem &#39; &lt;typename&gt; &#39; není platný
Pokoušíte se nepodporovaný typ zapisovat do souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte strukturu k určení, kde dojde k nepodporovaný typ a pak ji změňte platného typu.
