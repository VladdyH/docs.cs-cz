# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;TypeName&gt; &#39; musí být deklarován &#39;MustInherit&#39; protože obsahuje metody deklarované &#39;MustOverride&#39;
Typy, které obsahují `MustOverride` členy musí být deklarován `MustInherit`.  
  
 **ID chyby:** BC31411  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `MustInherit` modifikátor typu.
