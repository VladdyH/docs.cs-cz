# <a name="expected-3939-for-xml-end-tag"></a>Očekávaný & č. 39; / & č. 39; pro koncová značka XML
Koncová značka XML pro literál XML chybí požadované '/' znak.  
  
 **ID chyby:** BC31167  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Znak '/' přidejte na začátek koncová značka XML pro literál XML.
