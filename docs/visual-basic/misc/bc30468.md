# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Typ deklarace znaky nejsou platné v tomto kontextu
Pokusili jste se použít znak, který není platný v tomto kontextu, jako je například $.  
  
 **ID chyby:** BC30468  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte znak.
