# <a name="39end-enum39-must-be-preceded-by-a-matching-39enum39"></a>&#39;Ukončení výčtu&#39; musí předcházet odpovídající &#39;výčtu&#39;
`End Enum` Příkaz probíhá, aniž by odpovídající `Enum` příkaz. `End Enum` musí předcházet odpovídající `Enum` příkaz.  
  
 **ID chyby:** BC30184  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zajistěte, aby předchozím `Enum` příkaz není překlepu nebo jinak neplatných.  
  
2.  Ujistěte se, že `Enum` členové jsou správně naformátován.
