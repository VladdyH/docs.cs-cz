# <a name="you-must-specify-a-file-name"></a>Musíte zadat název souboru
Jste se pokusili operace, jako je například přesunutí nebo zkopírování souboru, který vyžaduje zadávání názvu souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadáte název souboru.  
  
-   Zajistěte, aby cesta nekončí oddělovací znak adresáře (\\).
