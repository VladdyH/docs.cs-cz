# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Throw&#39; příkazy nejsou platné v příkazové podokno
`Throw` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30725  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Throw` příkazy z ladění kódu.
