# <a name="methods-cannot-be-declared-39static39"></a>Metody nelze deklarovat &#39;statické&#39;
`Static` Modifikátor nelze použít na celou postupy.  
  
 **ID chyby:** BC30810  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Static` modifikátor deklarovat jednotlivé položky v rámci procedury jako statické.
