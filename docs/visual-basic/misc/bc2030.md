# <a name="the-moduleassemblyname-option-may-only-be-specified-when-building-a-target-of-type-39module39"></a>/ Moduleassemblyname možnost lze zadat pouze při vytváření cílový typ & č. 39; modulu & č. 39;
Visual Basic – kompilátor byl předán `/moduleassemblyname` při – možnost kompilátoru `/target` možnost nastavena na hodnotu s výjimkou `module`.  
  
 **ID chyby:** BC2030  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `/moduleassemblyname` – možnost kompilátoru nebo nastaví `/target` možnost k `module`.
