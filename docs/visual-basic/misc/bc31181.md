# <a name="attribute-value-is-not-valid-expecting-39value39"></a>Hodnota atributu není platná. byla očekávána & č. 39; hodnota & č. 39;
Byla zadána neplatná hodnota atributu v literál dokumentu XML. V specifikace XML 1.0 je zadaný neplatný atribut hodnoty pro literál dokumentu XML.  
  
 **ID chyby:** BC31181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Aktualizujte hodnotu atributu na platnou hodnotu, podle specifikace XML 1.0.
