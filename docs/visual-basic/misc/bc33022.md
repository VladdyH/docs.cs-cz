# <a name="either-the-parameter-type-or-the-return-type-of-this-conversion-operator-must-be-the-containing-type"></a>Typ parametru nebo návratový typ této operátor převodu musí obsahovat typ
Definice operátora převodu určuje parametr a návratový typ s typy jiné než u třídu nebo strukturu, ve kterém je definovaný operátor.  
  
 Pokud definice operátora převodu ve třídě nebo struktuře, je nutné převést do nebo z typu třídy nebo struktura.  
  
 **ID chyby:** BC33022  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte typ parametru nebo návratový typ na typ třídu nebo strukturu, ve kterém je definovaný operátor.
