# <a name="evaluation-of-expression-or-statement-stopped"></a>Vyhodnocení výrazu nebo příkazu zastavena
**Zastavit vyhodnocení** nebyla vybrána položka nabídky, způsobuje zkušební verze na Zastavit.  
  
 **ID chyby:** BC30721  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Povolit spuštění pro pokračování bez výběru **zastavit vyhodnocení**.
