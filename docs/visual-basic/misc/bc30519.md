# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-without-a-narrowing-conversion-lterrorgt"></a>Rozlišení přetěžování se nezdařilo, protože žádné dostupné &#39; &lt;metoda&gt; &#39; lze volat bez zužující převod: &lt;chyba&gt;
Provedli jste volání přetížené metody, ale kompilátor nebyla nalezena metoda, kterou lze volat bez zužující převod. Zužující převod změní hodnotu na datový typ, který nemusí být možné přesněji obsahovat některé z možných hodnot.  
  
 **ID chyby:** BC30519  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte `Option Strict Off`.
