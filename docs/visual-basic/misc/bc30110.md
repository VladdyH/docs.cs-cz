# <a name="39lttypenamegt39-is-a-structure-type-and-cannot-be-used-as-an-expression"></a>&#39;&lt;TypeName&gt; &#39; je typ struktury a nelze použít jako výraz
Název typu nastane, kdy je potřeba výrazu. Výraz musí obsahovat kombinaci proměnné, konstanty, literály, vlastnosti, a `Function` volání procedur.  
  
 **ID chyby:** BC30110  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte název typu a vytvořit pomocí elementů. platný výraz.
