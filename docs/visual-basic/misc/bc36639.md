<span data-ttu-id="7ecb4-103">Výraz lambda deklarované v rámci `Sub` nebo funkce nemůžete použít žádnou `ByRef` parametry této `Sub` nebo funkce.</span><span class="sxs-lookup"><span data-stu-id="7ecb4-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="7ecb4-104">Následující kód například způsobí této chybě, protože `ByRef` parametr `n` je použít ve výrazu lambda.</span><span class="sxs-lookup"><span data-stu-id="7ecb4-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="7ecb4-105">**ID chyby:** BC36639</span><span class="sxs-lookup"><span data-stu-id="7ecb4-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7ecb4-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="7ecb4-106">To correct this error</span></span>  
  
-   <span data-ttu-id="7ecb4-107">Přiřazení `ByRef` parametr pro místní proměnné a použití místní proměnné ve výrazu lambda, jak je uvedené v následující kód:</span><span class="sxs-lookup"><span data-stu-id="7ecb4-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>

    ```  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
