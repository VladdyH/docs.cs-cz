# <a name="try-must-have-at-least-one-39catch39-or-a-39finally39"></a>Zkuste musí mít alespoň jeden &#39;Catch&#39; nebo &#39;Finally&#39;
A `Try` bloku musí obsahovat buď `Finally` bloku nebo alespoň jedna `Catch` před zablokovat `End Try` příkaz.  
  
 **ID chyby:** BC30030  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Catch` nebo `Finally` blokovat v rámci `Try` bloku.
