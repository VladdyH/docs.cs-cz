# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>Konstruktor, musí být deklarován Sub, ne jako funkce
Pokusili jste se deklarovat `Function New`. Musí být konstruktory deklarována jako `Sub New`.  
  
 **ID chyby:** BC30493  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Sub` místo `Function`.
