# <a name="format-not-valid-in-resource-file"></a>Neplatný formát souboru prostředku
Soubor prostředků ve vašem projektu nedodrží ve standardním formátu pro soubor prostředků systému Windows nebo došlo k neshodě mezi souboru prostředků a [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] projektu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Znovu vytvořte zdroj zdrojů a soubory prostředků a dodržovat syntaxe zdokumentována Windows Software Development Kit.  
  
2.  Odeberte soubor prostředků z projektu.
