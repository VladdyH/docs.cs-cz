# <a name="39ltmembernamegt39-cannot-be-declared-39shadows39-outside-of-a-class-structure-or-interface"></a>&#39;&lt;MemberName&gt; &#39; nelze deklarovat &#39;stínů&#39; mimo třídy, struktury nebo rozhraní
`Shadows` – Klíčové slovo se zobrazí v deklaraci na obor názvů, modul nebo na úrovni souborů. Stínový provoz má smysl pouze v rámci programovací element, který může dědit vlastnosti z base element.  
  
 **ID chyby:** BC32200  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Shadows` – klíčové slovo nebo přesunutí deklaraci uvnitř třídy, struktury nebo rozhraní.
