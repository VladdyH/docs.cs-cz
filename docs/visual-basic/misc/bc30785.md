# <a name="parameter-specifier-is-duplicated"></a>Parametr specifikátor je duplicitní.
Specifikátor parametru je použita více než jednou na parametr.  
  
 **ID chyby:** BC30785  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní specifikátory.
