# <a name="operator-39ltoperatorgt39-must-have-one-parameter"></a>Operátor &#39; &lt;operátor&gt; &#39; musí mít jeden parametr
Unární operátor je definována bez parametrů nebo s více než jeden parametr.  
  
 Unární operátor musí mít přesně jeden parametr.  
  
 **ID chyby:** BC33014  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Upravte definici zadat přesně jeden parametr.  
  
-   Pokud budete potřebovat dva parametry, je nutné zadat binární operátor.  
  
-   Pokud potřebujete žádné parametry nebo větší než dvě, je nutné použít [příkaz funkce](../../visual-basic/language-reference/statements/function-statement.md) k definování `Function` postup místo operátor.
