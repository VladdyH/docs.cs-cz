# <a name="xml-comment-parameter-must-have-a-39name39-attribute"></a>Parametr komentáře XML musí mít &#39;název&#39; atribut
Parametr komentáře XML musí mít atribut 'name'. Komentáře XML budou ignorovány.  
  
 Značky XML chybí povinná `name` atribut.  
  
 **ID chyby:** BC42308  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte požadované `name` atributů pro značku.
