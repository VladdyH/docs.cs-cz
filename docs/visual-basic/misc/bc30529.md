# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Všechny parametry musí být explicitně typu Pokud jsou některé
Používáte smíšené různé typy deklarace typu ve funkci.  
  
 **ID chyby:** BC30529  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda že jsou všechny parametry explicitně typu.
