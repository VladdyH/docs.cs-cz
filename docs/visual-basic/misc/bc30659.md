# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>Vlastnost nebo pole &#39; &lt;název&gt; &#39; nemá platný typ atributu
Může být pomocí atributu konstruktor, který nepřijímá platný typ atributu, i když atribut může mít jiné odpovídající konstruktory.  
  
 **ID chyby:** BC30659  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, jestli tento atribut.
