# <a name="local-variable-cannot-have-the-same-name-as-the-function-containing-it"></a>Místní proměnná nemůže mít stejný název jako funkce, které ji obsahují
A `Dim` příkaz určuje proměnné se stejným názvem jako `Function` postup, který jej obsahuje.  
  
 **ID chyby:** BC30290  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte deklarace proměnné, nebo změňte název proměnné.
