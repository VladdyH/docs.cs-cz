# <a name="insufficient-security-permissions-to-set-the-system-date"></a>Nedostatečná oprávnění zabezpečení k nastavení systémového data
Oprávnění zabezpečení nejsou dostatečná k nastavení systémové datum.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte oprávnění zabezpečení tak, aby nastavení systémové datum.
