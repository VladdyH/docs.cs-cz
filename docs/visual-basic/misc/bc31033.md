# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Rozhraní &#39; &lt;interfacename&gt; &#39; může být implementována pouze jednou tento typ
Zadané rozhraní je už implementované tento typ a nelze znovu implementovat.  
  
 **ID chyby:** BC31033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Implementace tohoto rozhraní pomocí tohoto typu pouze jednou.
