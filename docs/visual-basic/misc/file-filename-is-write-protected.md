# <a name="file-39ltfilenamegt39-is-write-protected"></a>Soubor &#39; &lt;filename&gt; &#39; je chráněna zápisu
Soubor, který se pokoušíte zapisovat je chráněna proti zápisu a proto nemůže být změněn.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změna oprávnění u souboru pro čtení a zápis nebo zapisovat na jiný soubor.
