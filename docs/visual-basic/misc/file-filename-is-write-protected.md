# <a name="file-39ltfilenamegt39-is-write-protected"></a>& Souborů č. 39; &lt;filename&gt;& č. 39; je chráněna zápisu

## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změna oprávnění u souboru pro čtení a zápis nebo zapisovat na jiný soubor.
