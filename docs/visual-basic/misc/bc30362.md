# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt; &#39; nejde přepsat &#39; &lt;Název_vlastnosti2&gt; &#39; protože se liší &#39;jen pro čtení&#39; nebo &#39;WriteOnly&#39;
Pokusili jste se přepsat vlastnost s druhou vlastností, která se liší od prvního pouze systémem `ReadOnly` nebo `WriteOnly`.  
  
 **ID chyby:** BC30362  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že vlastnosti jsou rozlišené pomocí více než `ReadOnly` a `WriteOnly`.
