# <a name="comma-or-3939-expected-type-parameter-list"></a>Čárkou nebo & č. 39;) & č. 39; očekávaný (seznam parametrů typu)
Seznam typů parametrů obsahuje neočekávaný element. Použít čárky (`,`) pro jednotlivé parametry typu v tento seznam a používání kulaté závorky (`)`) na konec seznamu.  
  
 **ID chyby:** BC32099  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání čárkou a pokračovat v seznamu parametrů typu nebo přidání ukončovací znak závorky ukončit seznamu.
