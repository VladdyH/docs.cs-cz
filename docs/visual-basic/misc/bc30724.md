# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Chyba&#39; příkazy nejsou platné v příkazové podokno
`Error` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30724  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Error` příkazy z ladění kódu.
