# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39;WithEvents&#39; proměnných musí mít &#39;jako&#39; – klauzule
Není poskytnut `As` klauzuli with klíčové slovo `WithEvents`. Deklarujte proměnnou jako určité třídy, který může vyvolat události.  
  
 **ID chyby:** BC30412  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `As` klauzuli, která určuje třídu, která může být spojeno události.
