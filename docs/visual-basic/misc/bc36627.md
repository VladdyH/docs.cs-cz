# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Vyhodnocení výrazů lambda není platný v ladicím programu
Byl zadán výraz lambda v **sledovat** nebo **Immediate** okno během relace ladění. Lambda – výrazy nejsou k dispozici při ladění systému windows.  
  
 **ID chyby:** BC36627  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Otestujte výrazy lambda v editoru kódu.
