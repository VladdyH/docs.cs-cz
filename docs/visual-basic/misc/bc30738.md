# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub Main&#39; je deklarován více než jednou v &#39; &lt;obor názvů&gt;&#39;: &lt;zpráv&gt;
`Sub Main` lze deklarovat pouze jednou uvnitř oboru názvů.  
  
 **ID chyby:** BC30738  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, je jediným `Sub Main` postupu ve vašem projektu.
