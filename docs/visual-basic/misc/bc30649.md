# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;TypeName&gt; &#39; je nepodporovaný typ.
Zadaný typ nemůže být reprezentovaná Visual Studio.  
  
 **ID chyby:** BC30649  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pomocí podporovaného typu.  
  
-   Vytvořte nový typ.
