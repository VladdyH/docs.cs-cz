# <a name="39ltkeywordgt39-is-valid-only-within-a-class"></a>&#39;&lt;– klíčové slovo&gt; &#39; je platná pouze v rámci třídy
Klíčové slovo související s třídami, jako například `Me` nebo `MyClass`, je použít mimo definici třídy.  
  
 **ID chyby:** BC32002  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud kód pomocí klíčového slova zahrnuje instancí třídy, přesuňte jej do implementace třídy.  
  
-   Pokud kód pomocí klíčového slova se nevztahuje na třídy, odeberte neplatné – klíčové slovo.
