# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-with-these-argumentslterrorgt"></a>Rozlišení přetěžování se nezdařilo, protože žádné dostupné &#39; &lt;metoda&gt; &#39; nelze volat s těmito argumenty:&lt;chyba&gt;
Provedli jste volání přetížené metody, ale kompilátor nelze najít metodu s odpovídající seznam argumentů.  
  
 **ID chyby:** BC30518  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte seznam argumentů, které jste zadali.
