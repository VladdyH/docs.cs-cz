# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Hodnotu typu &#39; &lt;type1&gt; &#39; nelze převést na &#39; &lt;type2&gt; &#39; protože typech pole mají odlišné počty dimenzí
Pokusili jste se převést jedno pole na jiný s různým počtem dimenzí.  
  
 **ID chyby:** BC30414  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jeden nebo oba pole změnit tak, aby odpovídalo počet dimenzí.
