# <a name="alias-39ltnamespacenamegt39-is-already-declared"></a>Alias &#39; &lt;parametr namespacename&gt; &#39; je již deklarována
Pokusili jste se použít `Imports` importovat obor názvů, který již byl deklarován.  
  
 **ID chyby:** BC30572  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte obor názvů, který se pokoušíte importovat.
