# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Metoda do struktury nelze deklarovat &#39;chráněné&#39; nebo &#39;Protected Friend&#39;
Metoda ve struktuře byla deklarována jako `Protected` nebo `Protected Friend`.  
  
 **ID chyby:** BC31067  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat metody uvnitř struktury jako `Public` nebo `Private`.
