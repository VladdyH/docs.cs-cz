# <a name="39ltspecifiergt39-is-not-valid-on-an-interface-event-declaration"></a>& č. 39; &lt;specifikátor&gt;& č. 39; není platný v deklaraci události rozhraní
`Event` Příkaz uvnitř rozhraní obsahuje klíčové slovo, které není povolena, jako například `Implements`. Rozhraní můžete definovat pouze členy, není jejich implementaci.  
  
 **ID chyby:** BC30275  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte klíčové slovo z příkazu deklarace.  
  
2.  Přesuňte implementace členů rozhraní na třídu, která implementuje rozhraní.
