# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>Pouze první osm znaků název vlastního protokolu jsou důležité
Při kontrole názvy protokolu událostí pro jedinečnosti, jsou považovány za pouze prvních 8 znaků. Z protokolů událostí, které sdílejí jejich prvních 8 znaků může vést ke konfliktu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pojmenujte protokol událostí ve kterém jsou jedinečné prvních 8 znaků.
