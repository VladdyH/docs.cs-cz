# <a name="39end39-statement-cannot-be-used-in-class-library-projects"></a>&#39;End&#39; příkaz nelze použít v projektů knihovny tříd
Použít k vytvoření knihovny DLL projektů knihovny tříd zakázat použití `End` – klíčové slovo zastavit provádění kódu v postupu.  
  
 **ID chyby:** BC30615  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použít řídicí struktury jako `While` a `For` k řízení toku spuštění programu.
