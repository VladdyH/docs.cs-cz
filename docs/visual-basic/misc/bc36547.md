<span data-ttu-id="808af-103">Název vlastnosti lze vytvořit pouze jednou v deklaraci anonymního typu.</span><span class="sxs-lookup"><span data-stu-id="808af-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="808af-104">Například následující deklarace nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="808af-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.

'' Not valid, because the property name inferred for both properties is

' Dim anonType2 = New With {Key product.Name, Key car1.Name}
```  
