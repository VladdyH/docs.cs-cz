# <a name="39catch39-block-never-reached-ltexceptiongt-handled-above-in-the-same-39try39-statement"></a>&#39;Catch –&#39; bloku nikdy dosaženo; &lt;výjimka&gt; zpracovává výše ve stejném &#39;zkuste&#39; – příkaz
A `Catch` bloku v kódu nelze získat přístup, protože je zpracován v předchozích `Try` bloku.  
  
 Ve výchozím nastavení je tato zpráva upozornění. Další informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)  
  
 **ID chyby:** BC42031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte příkaz redundantní.
