# <a name="named-argument-cannot-match-a-paramarray-parameter"></a>Se nemůže shodovat s názvem argument parametr ParamArray
Jste zadali. pojmenovaný argument (zadaný argument je deklarovaný název následovaným dvojtečkou a symbolem rovná, za nímž následuje hodnota argumentu s); pole parametrů ale nemůžete předat podle názvu. Při volání postupu zadáte nekonečný počet textový soubor s oddělovači argumenty pro parametr pole a kompilátor nelze přidružit více než jeden argument s jedním názvem.  
  
 **ID chyby:** BC30587  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Argument předání umístěním, nikoli podle názvu.
