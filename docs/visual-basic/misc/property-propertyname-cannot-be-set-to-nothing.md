# <a name="property-ltpropertynamegt-cannot-be-set-to-nothing"></a>Vlastnost &lt;propertyname&gt; nelze nastavit na hodnotu Nothing
Vlastnost vyžaduje hodnotu než `Nothing`. To může být způsobeno nesprávně výpočtu hodnoty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte platnou hodnotu.
