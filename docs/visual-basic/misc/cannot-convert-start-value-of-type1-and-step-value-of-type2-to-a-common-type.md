# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>Nelze převést počáteční hodnota typu č. 39; &lt;type1&gt;& č. 39; a krok hodnota typu č. 39;&lt; Type2&gt;& č. 39; do stejného typu
Zadaný `For...Next` hodnoty řízení mají různé typy a proto nejsou platné vzhledem k tomu, že je nelze převést do stejného typu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte typ jednoho nebo více hodnot, tak, aby všechny hodnoty lze převést na stejného typu.
