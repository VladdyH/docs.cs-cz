# <a name="this-inheritance-causes-circular-dependencies-between-lttype1gt-39lttypename1gt39-and-its-nested-lttype2gt-39lttypename2gt39"></a>Tento vztah dědičnosti způsobí, že cyklické závislosti mezi &lt;type1&gt; & č. 39;&lt; NázevTypu1&gt;& č. 39; a její vnořené &lt;type2&gt; & č. 39;&lt; NázevTypu2&gt;& č. 39;
Strukturu dědičnosti výsledkem cyklická závislost mezi vnořené třídy, tedy dvě třídy, které dědí od sebe navzájem.  
  
 Následující kód lze vygenerovat tato chybová zpráva.  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 V předchozí kód třídy `c1` dědí z třídy `c4`, ale `c4` je vnořit `c3`, který dědí z `c2`, vnořených uvnitř `c1`.  
  
 **ID chyby:** BC30907  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte strukturu dědičnosti tak, aby bylo žádné cyklické závislosti.
