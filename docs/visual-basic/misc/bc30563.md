# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;název&gt; &#39; je nejednoznačný aplikační objekty &#39; &lt;seznamu&gt;&#39;
Jste zadali název, který je nejednoznačný a proto je v konfliktu s jiným názvem. Visual Basic – kompilátor nemá žádné pravidel řešení konfliktů; rozlišení názvů musí být sami. V seznamu součástí Podrobnosti chybové zprávy objekty aplikací, kde název je nejednoznačný.  
  
 **ID chyby:** BC30563  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Plnému určení názvu.
