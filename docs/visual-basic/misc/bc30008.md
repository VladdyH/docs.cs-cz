# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Odkaz na je vyžadován na modul &#39; &lt;modulename&gt; &#39; obsahující základní třídu &#39; &lt;elementem classname&gt;&#39;
Odkaz na je vyžadován na modul,\<modulename >' obsahující základní třídu\<classname >'. Přidáte do projektu.  
  
 Je třída definovaná v modulu, který není přímo odkazuje ve vašem projektu. Visual Basic – kompilátor vyžaduje odkaz, aby se zabránilo nejednoznačnosti v případě, že je třída definovaná v více než jeden modul.  
  
 **ID chyby:** BC30008  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrňte název neregistrované modulu odkazy projektu.
