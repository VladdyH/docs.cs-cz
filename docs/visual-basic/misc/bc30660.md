# <a name="attributes-cannot-be-applied-to-local-variables"></a>Atributy nelze použít pro lokální proměnné
Atributy nelze použít na místní proměnné.  
  
 **ID chyby:** BC30660  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Tuto proměnnou na úrovni modulu nebo další vyšší úroveň oboru, například třídu deklarujte.
