# <a name="3939-is-not-allowed"></a>&#39;:&#39; není povoleno
':' není povolen. XML kvalifikovaný názvy nelze použít v tomto kontextu.  
  
 Literál XML byla zadána s kvalifikovaný název do neplatné umístění, například cíl v příkazu zpracování literálu.  
  
 **ID chyby:** BC31186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte XML literálu s úplný název, nebo ho přesunout do platné umístění.  
  
     Informace o tom, které jsou platné v názvech XML znaků najdete v tématu [jazyka XML (Extensible Markup) 1.0](https://www.w3.org/TR/xml).
