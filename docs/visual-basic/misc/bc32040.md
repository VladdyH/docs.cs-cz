# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39; &lt;– Klíčové slovo&gt; &#39; – klíčové slovo se používá k přetížení zděděné členy; nepoužívejte &#39; &lt;– klíčové slovo&gt; &#39; – klíčové slovo přetížení &#39;Sub New&#39;
Konstruktor je deklarovaný s `Overloads` – klíčové slovo.  
  
 Visual Basic nepodporuje dědění nebo přetížení konstruktory.  
  
 **ID chyby:** BC32040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Overloads` – klíčové slovo z všechny deklarace konstruktor.
