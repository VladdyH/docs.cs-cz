# <a name="3939-expected-declaration"></a>&#39;=&#39; očekává (deklarace)
A `Mid` příkaz nebo `#Const` kompilace direktiva neobsahuje přiřazení hodnoty.  
  
 **ID chyby:** BC30249  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat symbolem rovná (`=`) a hodnotu na řetězec nebo konstanta kompilace přiřazena.
