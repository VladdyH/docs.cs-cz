# <a name="cannot-rename-with-different-drive"></a>Nelze přejmenovat s jinou jednotku
`Rename` Příkaz nelze zadat písmeno jednotky v cestě k souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte písmeno jednotky z cesty k souboru.
