# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>Třída &#39; &lt;classname&gt; &#39; nebylo možné vytvořit: &lt;chyba&gt;
Třída inicializace se nezdařila. Možnou příčinou je poškozený [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] instalace.  
  
 **ID chyby:** BC32400  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte uvozovkách chybovou zprávu a proveďte příslušnou akci.  
  
2.  Znovu nainstalujte [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] a překompilování program.  
  
3.  Pokud se chyba objeví znovu, přeinstalujte kompilátoru jazyka Visual Basic.  
  
4.  Pokud chyba přetrvává i po přeinstalování, shromážděte informace o okolnostech a upozornit služby Microsoft Product Support Services.
