# <a name="39sub39-or-39function39-expected"></a>&#39;Sub –&#39; nebo &#39;funkce&#39; očekávání
A `Declare` příkaz neurčuje buď `Sub` nebo `Function`. `Declare` identifikuje externí procedura a určit, jaký druh procedury.  
  
 **ID chyby:** BC30215  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat buď `Sub` nebo `Function` – klíčové slovo chcete `Declare` příkaz.
