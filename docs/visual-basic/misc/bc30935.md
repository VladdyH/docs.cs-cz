<span data-ttu-id="c8336-103">Procedura nebo vlastnost pokusí přepsat na zděděné postup nebo vlastnost, ale kompilátor vyhledá více než jedna verze základní postup nebo vlastnosti se stejným názvem a podpis.</span><span class="sxs-lookup"><span data-stu-id="c8336-103">A procedure or property attempts to override an inherited procedure or property, but the compiler finds more than one version of the base procedure or property with the same name and signature.</span></span>  
  
 <span data-ttu-id="c8336-104">Této chybě může dojít v případě pomocí sestavené obecné typy, jak znázorňují následující kostru deklarace.</span><span class="sxs-lookup"><span data-stu-id="c8336-104">This error can occur in a situation with constructed generic types, as the following skeleton declarations illustrate.</span></span>
