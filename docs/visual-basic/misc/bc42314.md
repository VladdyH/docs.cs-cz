# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-lttypegt"></a>Komentáře XML nelze použít více než jednou pro částečné &lt;typu&gt;
Komentáře XML nelze použít více než jednou pro částečné \<typ >. XML – komentáře pro tento \<typ > budou ignorovány.  
  
 Blok komentáře XML se může zobrazit nad pouze jedna část částečné typu.  
  
 Pokud blok komentáře XML se zobrazí nad více než jednu část částečné typu, toto upozornění se vytvoří pro každý blok komentáře a nejvyšší úrovně komentáře XML budou ignorovány.  
  
 **ID chyby:** BC42314  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné komentář bloky.
