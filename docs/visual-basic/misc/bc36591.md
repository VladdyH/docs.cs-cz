# <a name="too-many-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Příliš mnoho argumentů typu rozšíření metoda &#39; &lt;methodName&gt; &#39; definované v &#39; &lt;typeName&gt;&#39;
Obecný rozšiřující metoda metoda byla volána s další argumenty typu, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC36591  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Argumenty typu odeberte ze seznamu argument typu, takže je jedna pro každý parametr typu definované obecná metoda, která je vyvolán.
