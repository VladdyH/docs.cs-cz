# <a name="type-arguments-cannot-be-applied-to-the-expression-39ltexpressiongt39"></a>Argumenty typu nelze použít pro výraz & č. 39; &lt;výraz&gt;& č. 39;
Alias importu je definovaná pomocí [z](../../visual-basic/language-reference/statements/of-clause.md) klauzule, předává zadejte argumenty pro import alias.  
  
 Argumenty typu se používají pro obecné typy a pouze třídy, struktury, rozhraní, postupy a delegáti může být obecný. Obory názvů ani import aliasů může být obecný.  
  
 **ID chyby:** BC32058  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Of` klauzule a jeho argumenty typu z aliasu importu.
