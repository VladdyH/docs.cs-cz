# <a name="39withevents39-variable-does-not-raise-any-instance-events-that-are-accessible-to-39ltcontainernamegt39"></a>&#39;WithEvents&#39; proměnná nevyvolá všechny instance události, které jsou k dispozici &#39; &lt;containername&gt;&#39;
Objekt, který je odkazováno na nevyvolá přístupné události.  
  
 **ID chyby:** BC30600  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Modifikátory přístupu pro událost upravit, nebo zvolte událost, která je přístupná v aktuálním kontextu.
