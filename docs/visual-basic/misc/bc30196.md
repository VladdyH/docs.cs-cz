# <a name="comma-expected"></a>Byla očekávána čárka
Seznam argumentů chybí čárka (,) mezi dvěma z argumentů.  
  
 **ID chyby:** BC30196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda že všechny argumenty jsou odděleny čárkami.
