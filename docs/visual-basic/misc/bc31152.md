# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Předpona oboru názvů XML & č. 39; předponu & č. 39; je vyhrazená pro použití XML a obor názvů, který nelze změnit identifikátor URI
A vyhrazena Předpona oboru názvů XML, jako například `xmlns`, jsou využívány literál XML.  
  
 **ID chyby:** BC31152  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte na hodnotu, která není vyhrazena Předpona oboru názvů XML.
