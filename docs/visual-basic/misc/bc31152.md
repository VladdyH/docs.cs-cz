# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Předpona oboru názvů XML &#39;předponu&#39; je vyhrazená pro použití XML a nelze změnit identifikátor URI oboru názvů
A vyhrazena Předpona oboru názvů XML, jako například `xmlns`, jsou využívány literál XML.  
  
 **ID chyby:** BC31152  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte na hodnotu, která není vyhrazena Předpona oboru názvů XML.
