# <a name="implicit-conversion-from-39lttypename1gt39-to-39lttypename2gt39"></a>Implicitní převod z &#39; &lt;NázevTypu1&gt; &#39; k &#39; &lt;NázevTypu2&gt;&#39;
Výraz nebo příkaz přiřazení přebírá hodnotu jeden datový typ a převede jej k jinému typu. Vzhledem k tomu, že je použit žádný převod – klíčové slovo, převod je *implicitní*.  
  
 Ve výchozím nastavení je tato zpráva upozornění. Informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je to možné používejte hodnoty stejného typu dat, takže není nutné provádět jakékoli převody jazyka Visual Basic.  
  
-   Použití `CType` nebo jeden z jiných klíčová slova převodu tak, aby převod *explicitní*.
