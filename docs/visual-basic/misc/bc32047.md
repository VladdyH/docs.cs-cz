# <a name="type-parameter-39lttypeparameternamegt39-can-only-have-one-constraint-that-is-a-class"></a>Zadejte parametr &#39; &lt;typeparametername&gt; &#39; může mít pouze jedno omezení, která je třída
Omezení seznam obsahuje více než jednu třídu.  
  
 Seznam omezení pro parametr typu můžete zadat libovolný počet rozhraní, ale maximálně jednu třídu. Argument typu zadaná pro parametr typu musí dědit z této třídy, a Visual Basic nepodporuje vícenásobná dědičnost.  
  
 **ID chyby:** BC32047  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zvolit jednu třídu a zahrnout jenom třídy do seznamu omezení.  
  
-   Je možné definovat další typ parametry pro uložení třídu nebo třídy, které nelze zahrnout do tohoto seznamu omezení.
