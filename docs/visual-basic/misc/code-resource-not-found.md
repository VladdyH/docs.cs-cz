# <a name="code-resource-not-found"></a>Kód prostředek nebyl nalezen
Došlo k volání procedury v prostředku kód, ale kód prostředek nebyl nalezen.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda je prostředek k dispozici a správně odkazované.
