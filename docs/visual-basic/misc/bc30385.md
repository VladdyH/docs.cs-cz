# <a name="39ltmodifiergt39-is-not-valid-on-a-delegate-declaration"></a>&#39;&lt;Modifikátor&gt; &#39; není platný v deklaraci delegáta
Jste se pokusili použít modifikátor, jako například `Shared`, v deklaraci delegáta. Delegáti jsou objekty používá k volání metody jiné objekty, definováním konstruktor, který je předán specifikace metodu objektu. Není platný k určení modifikační v deklaraci delegáta.  
  
 **ID chyby:** BC30385  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte modifikátor.
