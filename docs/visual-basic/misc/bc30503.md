# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Vlastnosti v modulu nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Pokusili jste se použít specifikátorem, například `MustOverride`, `NotOverridable`, nebo `Overridable`, u vlastnosti v rámci modulu.  
  
 **ID chyby:** BC30503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
