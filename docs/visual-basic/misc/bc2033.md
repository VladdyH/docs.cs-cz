# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Chyba při vkládání Win32 manifest: / win32manifest – možnost v konfliktu s/nowin32manifest
Visual Basic – kompilátor byl předán i `/win32manifest` – možnost kompilátoru a `/nowin32manifest` – možnost kompilátoru. Pouze jeden z těchto možností může být předána do kompilátoru jazyka Visual Basic.  
  
 **ID chyby:** BC2033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Buď odeberte `/win32manifest` – možnost kompilátoru nebo `/nowin32manifest` – možnost kompilátoru.
