# <a name="39implements39-not-valid-in-modules"></a>& č. 39; implementuje & č. 39; není platný v modulech
`Implements` Příkaz nebo klauzule se provádí v modulu. Moduly nelze implementovat rozhraní.  
  
 **ID chyby:** BC30231  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Implements` příkaz nebo klauzuli nebo znovu zadejte modul jako třída.
