# <a name="type-arguments-unexpected"></a><span data-ttu-id="943bc-102">Argumenty typu neočekávané</span><span class="sxs-lookup"><span data-stu-id="943bc-102">Type arguments unexpected</span></span>
<span data-ttu-id="943bc-103">`Implements` Klauzule poskytuje argumenty typu pro člena rozhraní implementuje.</span><span class="sxs-lookup"><span data-stu-id="943bc-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="943bc-104">`Implements` Klauzule měli identifikovat pouze v uživatelském rozhraní a člen je implementace.</span><span class="sxs-lookup"><span data-stu-id="943bc-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="943bc-105">To znamená, že pokud jsou deklarace obecný postup, `Of` klauzule a argumenty typu objevit v hlavní část deklaraci, stejně jako Pokud nebyly implementace proceduru rozhraní.</span><span class="sxs-lookup"><span data-stu-id="943bc-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="943bc-106">Tato chyba může generovat následující kód.</span><span class="sxs-lookup"><span data-stu-id="943bc-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="943bc-107">Předchozí deklarace `Implements` klauzule by měl vypadat jako definice rozhraní, s výjimkou možné přidávat modifikátory přístupu nebo postupu.</span><span class="sxs-lookup"><span data-stu-id="943bc-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="943bc-108">Následující kód zabraňuje chyba.</span><span class="sxs-lookup"><span data-stu-id="943bc-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="943bc-109">**ID chyby:** BC32088</span><span class="sxs-lookup"><span data-stu-id="943bc-109">**Error ID:** BC32088</span></span>
