# <a name="cannot-determine-array-type-because-it-is-nothing"></a>Nelze určit typ pole, protože je nic
Typ pole nelze určit, protože byla nastavena na hodnotu `Nothing`.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte hodnotu než `Nothing` pro pole.
