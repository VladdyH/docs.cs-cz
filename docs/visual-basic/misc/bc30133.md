# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; GoTo & č. 39; příkazy nejsou platné v příkazové podokno
`GoTo` Příkaz provede vytvoření větve a není povolen v kontextu ladění.  
  
 **ID chyby:** BC30133  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nevydávají `GoTo` příkaz v **Immediate** okno.
