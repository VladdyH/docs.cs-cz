# <a name="expression-or-statement-is-not-valid-in-debug-windows"></a>Výraz nebo příkaz není platný v systému windows ladění
`Try...Catch...Finally`nelze použít v kontextu ladění.  
  
 **ID chyby:** BC30706  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat `Try...Catch...Finally` příkazy z ladění kódu.
