# <a name="too-many-type-arguments"></a>Příliš mnoho argumentů typu
Obecná metoda metoda byla volána s další argumenty typu, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC36579  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Argumenty typu odeberte ze seznamu argument typu, takže je jedna pro každý parametr typu Obecné metody, kterou vyvoláváte.
