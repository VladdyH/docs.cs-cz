# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Popisek &#39; &lt;název_jmenovky&gt; &#39; již je definována v aktuální metody nebo víceřádkového výrazu lambda
Název popisku je duplikována v rámci procesu. Všechny názvy v rámci procedury musí být jedinečný.  
  
 **ID chyby:** BC30094  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přejmenujte duplicitní popisek.
