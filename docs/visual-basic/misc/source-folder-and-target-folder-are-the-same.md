# <a name="source-folder-and-target-folder-are-the-same"></a>Zdrojové složky a cílové složky jsou shodné
Byl proveden pokus o kopírování nebo přesunutí souboru, ale adresář zadaný zdrojový a cílový adresář jsou stejné. To může být způsobeno nesprávně analyzovat cestu k souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda že jste zadali správné zdrojové a cílové složky.
