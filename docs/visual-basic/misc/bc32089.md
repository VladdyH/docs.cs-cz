# <a name="39ltparameternamegt39-is-already-declared-as-a-type-parameter-of-this-method"></a>&#39;&lt;Název parametru&gt; &#39; je již deklarována jako parametr typ této metody
Obecný postup definuje normální parametr nebo místní proměnné se stejným názvem jako parametr typu.  
  
 Každý parametr procedury, včetně každý parametr typu obecné procedury, musí mít název liší od všech dalších parametrů. Protože parametry procedur slouží jako místní proměnné, všechny místní proměnné deklarované v rámci postupu musí také mít název liší od všech parametrů a parametry typu.  
  
 **ID chyby:** BC32089  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název normální parametr nebo místní proměnné.
