# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Značka komentáře XML & č. 39; &lt;značka&gt;& č. 39; se zobrazí s stejné atributy více než jednou ve stejné blok komentáře XML
Značka komentáře XML \<značka > se zobrazí s stejné atributy více než jednou ve stejné blok komentáře XML. Komentáře XML budou ignorovány.  
  
 Komentáře XML obsahuje více než jednu značku s stejné atributy.  
  
 **ID chyby:** BC42305  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní značky.
