# <a name="argument-39conversion39-is-not-valid"></a>Argument & č. 39; Převod & č. 39; není platný
Hodnotu `VbStrConv` výčet není platný, pravděpodobně protože ji je menší než nula. Hodnoty výčtu musí být větší než nebo rovna hodnotě nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že hodnota je větší než nebo rovna hodnotě nula.
