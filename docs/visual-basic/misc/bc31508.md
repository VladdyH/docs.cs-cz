# <a name="cannot-find-net-framework-directory-lterrorgt"></a>Nelze najít adresář rozhraní .NET Framework: &lt;chyba&gt;
Adresář kde [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] ukládají sestavení nelze najít.  
  
 **ID chyby:** BC31508  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kompilace programu zjistíte, pokud se chyba objeví znovu.  
  
2.  Pokud potíže potrvají, uložte si práci a restartujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Pokud se chyba objeví znovu, přeinstalujte [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
