# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>Nelze vytvořit rozhraní .NET Runtime: &lt;chyba&gt;
Došlo k chybě vnitřní kompilátoru při vytváření rozhraní nebo volání metody na rozhraní se nezdařilo.  
  
 **ID chyby:** BC31024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Uložit práci a restartujte Visual Studio.  
  
2.  Pokud se chyba objeví znovu, znovu nainstalujte Visual Studio.  
  
3.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
