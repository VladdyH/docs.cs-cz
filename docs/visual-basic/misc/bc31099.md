# <a name="property-accessors-cannot-be-declared-39ltkeywordgt39"></a>Přístupové objekty vlastnosti nelze deklarovat &#39; &lt;– klíčové slovo&gt;&#39;
A `Get` nebo `Set` deklarace postup obsahuje klíčové slovo, které na procedury vlastnosti není platné.  
  
 [Získat příkaz](../../visual-basic/language-reference/statements/get-statement.md) a [příkaz Set](../../visual-basic/language-reference/statements/set-statement.md) povolit pouze modifikátor přístupu (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **ID chyby:** BC31099  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Get` nebo `Set` příkaz.
