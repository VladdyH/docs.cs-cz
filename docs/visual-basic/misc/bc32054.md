# <a name="39ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>& č. 39; &lt;membername&gt; má stejný název jako parametr typu
– Obecná třída nebo struktura má stejný název jako parametr typu třídy nebo struktura.  
  
 Každý proměnná, konstanta, – výčet, postupu, vlastnost, události a delegáta musí mít název liší od každý parametr typu.  
  
 **ID chyby:** BC32054  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte název člena třídu nebo strukturu tak, aby se liší od každý název typu parametru.
