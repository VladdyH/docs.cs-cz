# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39;ByVal&#39; a &#39;ByRef&#39; nelze kombinovat
`ByVal` a `ByRef` klíčová slova se vzájemně vylučují.  
  
 **ID chyby:** BC30641  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `ByVal` nebo `ByRef`, ale ne obojí.
