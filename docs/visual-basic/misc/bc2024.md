# <a name="the-default-assembly-reference-39ltreferencegt39-has-already-been-automatically-added-ignored"></a>Odkaz sestavení výchozí &#39; &lt;odkaz&gt; &#39; již byla automaticky přidal; ignorovat
Byl proveden pokus o přidat odkaz na výchozí sestavení, ale sestavení již byl přidán, takže pokus se ignoruje.  
  
 **ID chyby:** BC2024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné kódu.
