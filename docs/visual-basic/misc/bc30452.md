# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>Operátor & č. 39; &lt;operatorname&gt;& č. 39; není definován pro typy & č. 39;&lt; Type1&gt;& č. 39; a & č. 39;&lt; Type2&gt;& č. 39;
Pokusili jste se použít k provedení určité operace, která není platná pro dvě vrací hodnotu kódu elementy související se situací binární operátor.  
  
 **ID chyby:** BC30452  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte dva elementy a ujistěte se, že jsou kompatibilní.
