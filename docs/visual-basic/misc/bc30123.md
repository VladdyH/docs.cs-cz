# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; & Konce č. 39; příkazy nejsou platné v příkazové podokno
`Stop` a `End` příkazy pozastavení provádění a nejsou povoleny v kontextu ladění.  
  
 **ID chyby:** BC30123  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nevydávají `End` nebo `Stop` příkaz v **Immediate** okno.
