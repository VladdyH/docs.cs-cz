# <a name="range-variable-ltvariablegt-is-already-declared"></a>Proměnná rozsahu &lt;proměnná&gt; je již deklarována
Název proměnné rozsahu zadaný v `Select` klauzuli, nebo `Into` součástí `Aggregate`, `Group By`, nebo `Group Join` klauzuli duplikuje název proměnné rozsahu, který je již zadán v klauzuli dotazu.  
  
 **ID chyby:** BC36600  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zajistěte, aby všechny proměnné rozsahu v klauzuli dotazu jedinečné názvy.
