# <a name="loop-control-variable-of-type-39lttypenamegt39-does-not-implement-the-39systemicomparable39-interface"></a>Smyčky řídicí proměnná typu &#39; &lt;typename&gt; &#39; neimplementuje &#39;System.IComparable&#39; rozhraní
Je proměnná typu, který neimplementuje `System.IComparable` rozhraní. `System.IComparable` musí být implementované typ hodnoty nebo třídu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda proměnná typu hodnoty.
