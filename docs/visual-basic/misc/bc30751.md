# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>Nelze získat informace o typu pro &#39; &lt;typename&gt;&#39;
Bylo odkazováno typu, která nebyla načtena podle modul common language runtime.  
  
 **ID chyby:** BC30751  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit strukturu kódu, takže typ je definovaná v aktuálním oboru.
