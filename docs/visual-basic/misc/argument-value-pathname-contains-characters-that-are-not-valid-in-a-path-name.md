# <a name="argument-value-39ltpathnamegt39-contains-characters-that-are-not-valid-in-a-path-name"></a>Hodnota argumentu & č. 39; &lt;pathname&gt;& č. 39; obsahuje znaky, které nejsou platné v názvu cesty
Cesta obsahuje jeden nebo více znaků, které nejsou platné.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte znak z název cesty.
