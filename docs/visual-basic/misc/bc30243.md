# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;Modifikátor&gt; &#39; není platný v deklaraci události
`Event` Příkaz obsahuje neplatný klíčové slovo, jako například `ReadOnly`.  
  
 **ID chyby:** BC30243  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte neplatné klíčové slovo z `Event` příkaz.
