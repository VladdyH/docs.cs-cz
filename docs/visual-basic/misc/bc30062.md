# <a name="39me39-cannot-be-the-target-of-an-assignment"></a>& č. 39; Mi & č. 39; nemůže být cílem přiřazení
`Me` – Klíčové slovo vždy odkazuje na konkrétní instanci třídy, kde právě probíhá kód. Jeho hodnotu nelze změnit.  
  
 **ID chyby:** BC30062  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte příkaz, který přiřazuje hodnotu na `Me`.
