# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>Očekáván začátek &#39; &lt; &#39; pro značku XML
Značka XML pro literál XML chybí požadované začátku ' <' znak.  
  
 **ID chyby:** BC31165  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat ' <' znak na začátek značky XML pro literál XML.
