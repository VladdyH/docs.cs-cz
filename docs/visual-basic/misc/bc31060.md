# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;PropertyName&gt; &#39; implicitně definuje &#39; &lt;membername&gt;&#39;, který je v konfliktu s člen se stejným názvem v &lt;typ&gt; &#39; &lt; TypeName&gt;&#39;
Název člena typ je v konfliktu s názvem člena vytvořena implicitně pro vlastnost. Vlastnosti implicitně vytvořit několik implicitní proměnné, které může dojít ke konfliktu se definované členy.  
  
 **ID chyby:** BC31060  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte explicitně deklarované člena odebrat ke konfliktu názvů.
