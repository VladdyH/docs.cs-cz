# <a name="interface-39ltinterfacenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Rozhraní &#39; &lt;interfacename&gt; &#39; nelze indexovat, protože nemá žádné výchozí vlastnost
Výrazy index musí způsobit hodnotu s typem je pole, jehož typ má sadu vlastností přetížené výchozí hodnotu nebo sadu přetížené vlastností. Rozhraní může mít pouze jeden výchozí metody nebo vlastnosti. To znamená, že buď může dědit vlastnosti rozhraní obsahující výchozí metody nebo vlastnosti, nebo jeho definice blok může obsahovat jeden metody nebo vlastnosti, která je označena jako výchozí.  
  
 **ID chyby:** BC30547  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte výchozí vlastnost v rozhraní.
