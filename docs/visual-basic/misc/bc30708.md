# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>Příkazy smyčky nejsou platné v příkazové podokno
`Loop`nelze použít v **Immediate** okno.  
  
 **ID chyby:** BC30708  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Loop` příkazy z ladění kódu.
