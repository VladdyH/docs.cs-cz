# <a name="you-must-specify-a-path-that-is-under-the-current-folder-or-one-of-its-sub-folders"></a>Musíte zadat cestu, která je v aktuální složce nebo jeden z jejích podsložkách
Zadaná cesta není v aktuálním adresáři. To může být způsobeno nesprávně analýza cest k souborům, při zadávání relativní cesta.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že zadáváte správnou cestu.
