# <a name="cannot-save-file-to-temp"></a>Soubor nelze uložit do dočasného
Součást nemůže najít adresář s názvem TEMP, nebo disku nebo oddílu adresáře TEMP nemá dostatek místa k uložení informace.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Vytvořte adresář s názvem "TEMP" a nastavte dočasné proměnné prostředí rovná jeho cesty.  
  
2.  Zkontrolujte místo na disku podle mazání zbytečné soubory, nebo vytvoření dočasného adresáře na druhý oddíl a nastavte dočasné proměnné prostředí rovná jeho cesty.
