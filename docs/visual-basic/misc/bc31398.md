# <a name="type-characters-are-not-allowed-on-imports-aliases"></a>Znaky typu nejsou povoleny na aliasy importy
Alias v importu `Imports` příkaz obsahuje alespoň jeden znak typu identifikátoru.  
  
 Alias importu musí být platný název jazyka Visual Basic. Povolené znaky nesmí obsahovat žádný z identifikátor – znaky typu (`%`, `&`, `@`, `!`, `#`, a `$`). V tématu [deklarované názvy elementů](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC31398  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Z aliasu import odeberte – znak typu identifikátoru nebo znaky.
