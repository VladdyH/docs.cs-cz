# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Pokračovat & č. 39; příkazy nejsou platné v příkazové podokno
`Continue`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30780  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Continue` příkaz z ladění kódu.
