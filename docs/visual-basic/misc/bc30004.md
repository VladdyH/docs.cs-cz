# <a name="character-constant-must-contain-exactly-one-character"></a>Konstanta znaků musí obsahovat přesně jeden znak.
Konstanta znaků není platný, protože buď obsahuje více než jeden znak nebo prázdný znak (všech jsou nula).  
  
 **ID chyby:** BC30004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte konstanta znaků jeden znak nebo změňte znak hodnoty null platný znak.
