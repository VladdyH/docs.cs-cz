# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39;#Region&#39; příkaz musí končit odpovídající &#39;#End Region&#39;
Použití `#Region` – direktiva blok kódu, který můžete rozbalit nebo sbalit když používáte funkci osnovy z editoru kódu sady Visual Studio. Začátek a konec `#Region` příkazy musí být ve stejné blok kódu.  
  
 **ID chyby:** BC30681  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Vložit `#End Region` na příslušné místo v kódu po `#Region` příkaz.
