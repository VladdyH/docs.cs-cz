# <a name="39lteventnamegt39-is-not-an-event-of-39ltcontainernamegt39"></a>& č. 39; &lt;eventname&gt;& č. 39; není událost z & č. 39;&lt; ContainerName&gt;& č. 39;
Událost zadaný není deklarovaný pro objekt.  
  
 **ID chyby:** BC30676  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název události.  
  
2.  Zkontrolujte, zda přistupujete správný objekt. Použijte plně kvalifikované odkazy na objekt, nebo použijte `Imports` příkaz pro import odpovídající obor názvů, v případě potřeby.
