# <a name="39gt39-expected"></a>&#39;&gt;&#39;očekávání
Syntaxe pro zadaný kód je nesprávná. Tato chyba často znamená, že chybí ukončovací lomená závorka literál XML elementu nebo atributu.  
  
 **ID chyby:** BC30636  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte syntaxi pro kód byl použit.
