# <a name="type-parameter-cannot-have-the-same-name-as-its-defining-function"></a>Parametr typu nemůže mít stejný název jako jeho definující funkce
Se stejným názvem jako obecný typ je deklarovaný typ parametr obecného typu.  
  
 V seznamu Typ parametr obecného typu musí mít každý parametr typu název liší od všechny tyto názvy:  
  
-   Každý druhý parametr typu v seznamu parametrů stejný typ  
  
-   Každý parametr typu v seznamu Typ parametru obsahující obecného typu a  
  
-   Název obecného typu sám sebe.  
  
 **ID chyby:** BC32090  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte parametr typu je odlišná od každých názvu uvedených v seznamu na této stránce nápovědy.
