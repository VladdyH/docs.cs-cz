# <a name="cannot-create-an-instance-of-module-39ltmodulenamegt39"></a>Nelze vytvořit instanci modulu &#39; &lt;název modulu&gt;&#39;
Modul existuje jenom jako jediné sdílené instance, a další instance nelze vytvořit.  
  
 **ID chyby:** BC30166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte modul třídu nebo nahraďte ji v `New` klauzuli with název třídy.
