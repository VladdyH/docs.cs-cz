# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;Dědí&#39; příkaz musí předcházet všechny deklarace ve třídě
`Inherits` Příkazy musí být na začátku definice tříd.  
  
 **ID chyby:** BC30683  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `Inherits` příkaz do horní části třídy.
