# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Metoda & č. 39; &lt;methodname&gt;& č. 39; nemůže zpracovat událost & č. 39;&lt; EventName&gt;& č. 39; protože nemají stejným podpisem
Parametry nebo typy parametrů pro obslužnou rutinu zadanou událost neodpovídají definicím definované zadané události.  
  
 **ID chyby:** BC31029  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Znovu definujte události nebo obslužná rutina události tak, aby parametr uvádí shodu.
