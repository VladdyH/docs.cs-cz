# <a name="interface-39ltinterfacenamegt39-is-not-implemented-by-this-class"></a>Rozhraní & č. 39; &lt;interfacename&gt;& č. 39; touto třídou není implementována
Členem této třídě nebo struktuře se pokouší implementovat členem rozhraní, které třídu nebo strukturu neimplementuje.  
  
 **ID chyby:** BC31035  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Implements` příkaz na začátku třídu nebo strukturu, které implementuje rozhraní vhodné.  
  
-   Odeberte `Implements` – klíčové slovo z člena, který způsobí, že k této chybě.
