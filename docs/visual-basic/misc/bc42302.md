# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>Komentáře XML musí být první příkaz na řádek
Komentáře XML musí být první příkaz na řádek. Komentáře XML budou ignorovány.  
  
 Komentáře jazyka XML je ponecháno za jiný jazyk element, který není povolen.  
  
 **ID chyby:** BC42302  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte komentáře XML na nový řádek.
