# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>Plně kvalifikované názvy pro typy v oborech názvů nesmí být delší než &lt;číslo&gt; znaků
Plně kvalifikovaný název typu nemůže být delší než zadaný počet znaků.  
  
 **ID chyby:** BC30031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkraťte název typu.
