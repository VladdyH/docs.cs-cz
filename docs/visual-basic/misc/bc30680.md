# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;#End region&#39; musí předcházet párový příkaz &#39;#Region&#39;
S `#Region` můžete určit blok kódu, který rozbalit nebo sbalit při použití funkce sbalování z editoru kódu sady Visual Studio. Začátku a konce `#Region` příkazy musí být ve stejném bloku kódu.  
  
 **ID chyby:** BC30680  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vložit `#Region` v příslušném umístění před odpovídajícími `#End Region` příkazu.
