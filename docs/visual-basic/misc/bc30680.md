# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;#End region&#39; musí předcházet odpovídající &#39;#Region&#39;
S `#Region` zadáte blok kódu rozbalit nebo sbalit, když používáte funkci osnovy z editoru kódu sady Visual Studio. Začátek a konec `#Region` příkazy musí být ve stejné blok kódu.  
  
 **ID chyby:** BC30680  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vložit `#Region` na příslušné místo před odpovídající `#End``Region` příkaz.
