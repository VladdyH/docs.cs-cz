# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Znak pořadí &#39;–&#39; není povolen v komentáře jazyka XML
Literál komentáře XML obsahuje posloupnost znaků "–", což není podporováno.  
  
 **ID chyby:** BC31171  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat sekvence znaků '--' z literál komentáře XML.
