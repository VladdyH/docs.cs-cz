# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>V tomto prohlášení není možné zadat parametry typu
Programovací element je deklarovaný s seznam parametrů typu, ale programovací element není smí být obecného typu.  
  
 Programovací elementy, které nejsou smí být obecný obsahovat vlastnosti, operátory, události a konstruktory. Deklarování některý z těchto prvků s typem parametru seznamu výsledkem této chyby.  
  
 **ID chyby:** BC32065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Of` – klíčové slovo a parametry typu z deklarace.
