# <a name="39ltmodifiergt39-is-not-valid-on-a-method-declaration"></a>& č. 39; &lt;modifikátor&gt;& č. 39; není platný v deklaraci – metoda
A `Function` nebo `Sub` příkaz obsahuje neplatné klíčové slovo.  
  
 **ID chyby:** BC30242  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Function` nebo `Sub` příkaz.
