# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39;Na GoTo&#39; a &#39;na GoSub&#39; příkazy již nejsou podporovány.
Již není platná pro použití hodnota proměnné nebo výraz s `GoTo` a `GoSub` příkazů do ovládacího prvku programu toku.  
  
 **ID chyby:** BC30817  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změny struktury aplikace k používání `If...Then...Else` nebo `Case` příkazy.
