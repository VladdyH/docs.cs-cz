# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39;Na GoTo&#39; a &#39;na GoSub&#39; příkazy již nejsou podporovány.
Už nejsou platné. použijte hodnotu proměnné nebo výrazu `GoTo` a `GoSub` příkazů do ovládacího prvku programu toku.  
  
 **ID chyby:** BC30817  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změny struktury aplikace pomocí `If...Then...Else` nebo `Case` příkazy.
