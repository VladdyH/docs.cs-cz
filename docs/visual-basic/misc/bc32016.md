# <a name="39ltprocedurenamegt39-has-no-parameters-and-its-return-type-cannot-be-indexed"></a>&#39;&lt;procedurename&gt; &#39; nemá žádné parametry a návratový typ není možné indexovat.
Volání `Function` nebo `Sub` postup poskytuje jeden nebo více argumentů, ale postup nevyžaduje žádné argumenty a její návratový typ (Pokud je `Function`) není typu pole.  
  
 **ID chyby:** BC32016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrání volání procedury argument nebo argumenty.
