# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39; není deklarována
"\<rsetstmt >' není deklarována. Příkazy RSet již nejsou podporovány. Použijte místo nich příkazy Microsoft.VisualBasic.RSet.  
  
 Několik funkcí, které byly vnitřní do jazyka Visual Basic v předchozích verzích se přesunuly do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> oboru názvů. Díky jejich funkce obecně k dispozici pro všechny programovací jazyky.  
  
 **ID chyby:** BC30821  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `RSet` fungovat v `Microsoft.VisualBasic` obor názvů místo.
