# <a name="39ltrsetstmtgt39-is-not-declared"></a>& č. 39; &lt;rsetstmt&gt;& č. 39; není deklarovaný
'\<rsetstmt >' není deklarovaný. Příkazy RSet již nejsou podporovány. Místo toho použijte Microsoft.VisualBasic.RSet.  
  
 Několik funkcí, které byly vnitřní k [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] v předchozích verzích byl přesunut do <xref:Microsoft.VisualBasic?displayProperty=nameWithType> oboru názvů. Díky jejich funkce více obecně dostupné pro všechny programovací jazyky.  
  
 **ID chyby:** BC30821  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `RSet` fungovat v `Microsoft.VisualBasic` obor názvů místo.
