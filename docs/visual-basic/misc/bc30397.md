# <a name="39ltmodifiergt39-is-not-valid-on-an-interface-declaration"></a>&#39;&lt;Modifikátor&gt; &#39; není platný v deklaraci rozhraní
Použili jste modifikátor, který není platný na `Interface` deklarace. Jediné platné modifikátory pro `Sub`, `Function`, nebo `Property` příkazy deklarované v `Interface` deklarace jsou `Overloads` a `Default` klíčová slova. Další modifikátory, jako například `Public`, `Private`, `Friend`, `Protected`, `Shared`, `Static`, `Overrides`, `MustOverride` a `Overridable`, nejsou platné.  
  
 **ID chyby:** BC30397  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
