# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Odkaz vyžadoval sestavení &#39; &lt;assemblyname&gt; &#39; obsahující implementované rozhraní &#39; &lt;interfacename&gt;&#39;
Odkaz vyžadoval sestavení '\<assemblyname >' obsahující implementované rozhraní '\<interfacename >'. Přidáte do projektu.  
  
 Rozhraní je definováno v dynamické knihovně (DLL) nebo sestavení, který není přímo odkazuje ve vašem projektu. Visual Basic – kompilátor vyžaduje odkaz, aby se zabránilo nejednoznačnosti v případě, že rozhraní je definováno více než jeden knihovny DLL nebo sestavení.  
  
 **ID chyby:** BC30009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrňte název neregistrované knihovny DLL nebo sestavení odkazy projektu.
