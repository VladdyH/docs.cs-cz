# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Odkaz vyžadoval sestavení &#39; &lt;assemblyname&gt; &#39; obsahující typ &#39; &lt;členské jméno&gt;&#39;
Odkaz vyžadoval sestavení '\<assemblyname >' obsahující typ '\<členské jméno > ". Přidáte do projektu.  
  
 Projekt obsahuje typ, který závisí na jiném typu ze sestavení, které nejsou součástí odkazy na projekt.  
  
 **ID chyby:** BC30652  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte odkaz na projekt u závislých typů v projektu.
