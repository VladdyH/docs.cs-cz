# <a name="const-declaration-cannot-have-an-array-initializer"></a>Deklarace Const nemůže používat inicializátor pole
Jste se pokusili získat pomocí inicializátoru pole v `Const` deklarace.  
  
 **ID chyby:** BC30445  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte inicializátoru na platný výraz odebráním okolního složené závorky.
