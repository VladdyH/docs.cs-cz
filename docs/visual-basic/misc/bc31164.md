# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>Očekávaný odpovídající zavřením dvojitých uvozovek pro hodnotu atributu XML
Hodnota atributu, která je oddělená dvojitých uvozovek chybí ukončovací znak uvozovky.  
  
 **ID chyby:** BC31164  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte koncový znak dvojitých uvozovek pro vymezení hodnotu atributu.
