# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>Žádné dostupné přetížený &#39; &lt;methodname&gt; &#39; nelze volat s těmito argumenty bez zužující převod
Volání přetížené metody, ale žádná metoda se najde shoda s seznam zadané argumenty bez zužující převod.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte `Option Strict Off`.  
  
2.  Změňte argumenty tak, aby odpovídala údajům přetížené metody.
