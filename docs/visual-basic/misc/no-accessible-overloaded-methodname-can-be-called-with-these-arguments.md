# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion"></a>Žádné dostupné přetížené & č. 39; &lt;methodname&gt;& č. 39; lze volat s těmito argumenty bez zužující převod
Volání přetížené metody, ale žádná metoda se najde shoda s seznam zadané argumenty bez zužující převod.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte `Option Strict Off`.  
  
2.  Změňte argumenty tak, aby odpovídala údajům přetížené metody.
