# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>& č. 39; &lt;název&gt;& č. 39; je deklarovaný v projektu & č. 39;&lt; projectname1&gt;& č. 39; který není odkazuje projektu & č. 39;&lt; projectname2&gt;& č. 39;
Programovací element je definována v projektu, ke které nemá v aktuálním projektu odkaz.  
  
 **ID chyby:** BC32004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   V aktuálním projektu přidejte odkaz na projekt, který obsahuje požadovaný element.
