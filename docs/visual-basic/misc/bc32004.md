# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;název&gt; &#39; je deklarován v projektu &#39; &lt;projectname1&gt;&#39;, která neodkazuje projektu &#39; &lt;projectname2&gt;&#39;
Programovací element je definována v projektu, ke které nemá v aktuálním projektu odkaz.  
  
 **ID chyby:** BC32004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   V aktuálním projektu přidejte odkaz na projekt, který obsahuje požadovaný element.
