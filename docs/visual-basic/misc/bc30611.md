# <a name="array-dimensions-cannot-have-a-negative-size"></a>Rozměry pole nemůže být záporná čísla
Jeden nebo více mezí pole zadané je záporná. Záporné dolní index můžete zadat, jenom když použijete horní mez -1 deklarovat prázdné pole. Takové pole má nulovým počtem elementů, ale není `Nothing`.  
  
 **ID chyby:** BC30611  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte záporné pole svázané kladné číslo.
