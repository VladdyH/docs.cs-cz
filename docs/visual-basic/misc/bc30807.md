# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>& č. 39; Umožňují & č. 39; a & č. 39; & Nastavit č. 39; příkazy přiřazení již nejsou podporovány.
`Let`a `Set` nelze použít v přiřazení.  
  
 **ID chyby:** BC30807  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vynechat `Let` a `Set` z přiřazení příkazů; například:  
  
     `X = 2`
