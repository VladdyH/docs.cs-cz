# <a name="conversion-from-39lttype1gt39-to-39lttype2gt39-cannot-occur-in-a-constant-expression-used-as-an-argument-to-an-attribute"></a><span data-ttu-id="74062-102">Převod z & č. 39; &lt;type1&gt;& č. 39; na & č. 39;&lt; Type2&gt;& č. 39; nelze provést v použít jako argument do atribut konstantní výraz</span><span class="sxs-lookup"><span data-stu-id="74062-102">Conversion from &#39;&lt;type1&gt;&#39; to &#39;&lt;type2&gt;&#39; cannot occur in a constant expression used as an argument to an attribute</span></span>
<span data-ttu-id="74062-103">Výraz použitý pro argument atributu vyhodnocen jako datový typ, který se liší od odpovídajícího atributu parametru, a [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] neumožňuje požadovaný typ převodu pro argumenty atributu.</span><span class="sxs-lookup"><span data-stu-id="74062-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="74062-104">Atribut poskytuje metadata pro element, který se použije k a kompilátor musí být schopen vytvořit všechna metadata v době kompilace.</span><span class="sxs-lookup"><span data-stu-id="74062-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="74062-105">Z tohoto důvodu každý atribut musí používat hodnoty, které jsou v době kompilace konstantní, a proto každý argument atributu se musí vyhodnotit na konstantní hodnotu kompilaci.</span><span class="sxs-lookup"><span data-stu-id="74062-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>  
  
 <span data-ttu-id="74062-106">Převody určitých typů nelze vytvořit hodnoty, které jsou v době kompilace konstantní.</span><span class="sxs-lookup"><span data-stu-id="74062-106">Certain type conversions cannot produce values that are constant at compile time.</span></span> <span data-ttu-id="74062-107">Například převod `String` k `Double` nebo `Date` závisí na nastavení národního prostředí v době běhu.</span><span class="sxs-lookup"><span data-stu-id="74062-107">For example, converting a `String` to a `Double` or a `Date` depends on the locale setting at run time.</span></span> <span data-ttu-id="74062-108">Jiné převody, jako je například odvozeného typu na pole pole `Object`, k dispozici různé problémy, které nepovoluje kompilátoru a povolení jejich na argumenty atributu.</span><span class="sxs-lookup"><span data-stu-id="74062-108">Other conversions, such as an array of a derived type to an array of `Object`, present a variety of problems that do not permit the compiler to allow them on attribute arguments.</span></span>
