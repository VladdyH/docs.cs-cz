# <a name="an-embedded-expression-may-not-be-used-here"></a>Zde není možné použít výraz embedded
Výraz embedded byl součástí na nepodporované umístění v literál XML. Všimněte si, že vložené výrazy nelze použít k určení hodnoty obor názvů XML.  
  
 **ID chyby:** BC31172  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte embedded výraz z nepodporované umístění.
