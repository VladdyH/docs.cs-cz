# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>Nelze použít atributy zabezpečení k & č. 39; &lt;název&gt;& č. 39;: &lt;chyba&gt;
Při použití atributů zabezpečení došlo k chybě. Příčinou může být důsledkem nesprávné použití atributů zabezpečení nebo mohlo by to znamenat problém s překladačem.  
  
 **ID chyby:** BC31520  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že se syntaxe a argumenty pro atribut.  
  
2.  Kompilace programu zjistíte, pokud bude chyba opakovat.  
  
3.  Pokud potíže potrvají, uložte si práci a restartujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
4.  Pokud se chyba znova objeví, přeinstalujte [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
5.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
