# <a name="construct-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39lttypenamegt39"></a>Konstrukce vytváří nepřímý odkaz na sestavení &#39; &lt;assemblyname&gt;&#39;, který obsahuje &#39; &lt;typename&gt;&#39;
Konstrukce vytváří nepřímý odkaz na sestavení '\<assemblyname >', která obsahuje \<typename >. Přidání odkazu na soubor \<filename > do projektu.  
  
 Výraz používá typu, například třída, struktura, rozhraní, výčet nebo delegáta, ale vaše sestavení nemá odkaz na sestavení, která definuje typ projektu.  
  
 **ID chyby:** BC31528  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ve vlastnostech projektu přidejte odkaz na soubor, který obsahuje sestavení, která definuje typ, který používáte.
