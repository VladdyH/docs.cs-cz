# <a name="baselogname-cannot-be-nothing-or-an-empty-string"></a>BaseLogName nemůže být Nothing nebo prázdný řetězec.
Hodnota <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> vlastnost nesmí být `Nothing` nebo prázdný řetězec.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> Vlastnost určuje základní název pro soubory protokolu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> vlastnost na řetězec, který obsahuje alespoň jeden znak.
