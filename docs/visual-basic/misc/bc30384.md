# <a name="39try39-must-end-with-a-matching-39end-try39"></a>&#39;Zkuste&#39; musí končit odpovídající &#39;End Try&#39;
`Try` slouží k zahájení `Try` blokovat; proto se může vyskytovat pouze na začátku bloku, s odpovídající `End Try` příkaz ukončení bloku. Máte buď redundantní `Try`, nebo můžete neukončil vaše `Try` blokovat s `Finally`.  
  
 **ID chyby:** BC30384  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Vyhledejte a odebrat nadbytečné `Try`, nebo konce bloku s odpovídající `End Try`.
