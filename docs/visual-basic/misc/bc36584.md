# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parametr &#39; &lt;parametername&gt; &#39; v metodě rozšíření &#39; &lt;methodname&gt; &#39; definované v &#39; &lt;typename&gt; &#39; již má odpovídající argument
Argument je předána podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda jeden ze zadaných názvů argument je zadáno chybně.  
  
-   Odeberte duplicitní argument ze seznamu argumentů.
