# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parametr & č. 39; &lt;parametername&gt;& č. 39; v rozšíření metoda & č. 39;&lt; methodName&gt;& č. 39; definovaným v & č. 39;&lt; TypeName&gt;& č. 39; už má odpovídající argument
Argument je předána podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda jeden ze zadaných názvů argument je zadáno chybně.  
  
-   Odeberte duplicitní argument ze seznamu argumentů.
