# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>Moduly nelze deklarovat &#39; &lt;– specifikátor&gt;&#39;
Modul byla deklarována s neplatným modifikátorem přístupu.  
  
 **ID chyby:** BC31052  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Modifikátor přístupu nahraďte modifikátor platný přístup, jako například `Public` nebo `Friend`.
