# <a name="lambda-expression-cannot-be-converted-to-39lttypenamegt39-because-39lttypenamegt39-is-not-a-delegate-type"></a><span data-ttu-id="6a7dd-102">Výraz lambda, nelze převést na &#39; &lt;typename&gt; &#39; protože &#39; &lt;typename&gt; &#39; není typem delegáta</span><span class="sxs-lookup"><span data-stu-id="6a7dd-102">Lambda expression cannot be converted to &#39;&lt;typename&gt;&#39; because &#39;&lt;typename&gt;&#39; is not a delegate type</span></span>
<span data-ttu-id="6a7dd-103">Lambda – výrazy lze, kde jsou platné delegáti.</span><span class="sxs-lookup"><span data-stu-id="6a7dd-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="6a7dd-104">Mohou být převedeny na kompatibilní delegáta typy, ale ne na jiný typ.</span><span class="sxs-lookup"><span data-stu-id="6a7dd-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="6a7dd-105">Například můžete definovat typem delegáta a přiřadit výraz lambda nebo odeslat jako argument výraz lambda <xref:System.Func%601> parametr.</span><span class="sxs-lookup"><span data-stu-id="6a7dd-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="6a7dd-106">Tyto příklady jsou uvedeny v následující kód.</span><span class="sxs-lookup"><span data-stu-id="6a7dd-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```
