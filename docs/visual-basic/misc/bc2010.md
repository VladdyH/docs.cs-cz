# <a name="compilation-failed-ltmessagegt"></a>Kompilace se nezdařila: &lt;zpráv&gt;

 **ID chyby:** BC2010  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte chybové zprávy a určete přesnou příčinu selhání.
