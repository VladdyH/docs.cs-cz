# <a name="39end-function39-expected"></a>&#39;Ukončení funkce&#39; očekávání
`End` Příkaz musí odpovídat bloku ukončí. A `Function` definice procedury nebyl správně zavřen.  
  
 **ID chyby:** BC30027  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Function` příkaz k ukončení režimu.
