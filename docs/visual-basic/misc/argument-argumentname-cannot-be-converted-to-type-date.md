# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>Argument &#39; &lt;název argumentu argumentname&gt; &#39; nelze převést na typ &#39;datum&#39;
Převod se pokusil změnit proměnné na typ `Date`, ale hodnota nebyla rozpoznána jako platné datum.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Najít převodu, který má smysl.
