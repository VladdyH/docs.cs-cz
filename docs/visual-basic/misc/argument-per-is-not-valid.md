# <a name="argument-39per39-is-not-valid"></a>Argument &#39;za&#39; není platný
Argument zadaný pro `Per`, která musí být `Double` určující platební období v rozsahu 1 až `NPer`, není platný.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že argument spadá v platném rozsahu.
