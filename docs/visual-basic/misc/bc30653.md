# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Odkaz na je vyžadován na modul &#39; &lt;modulename&gt; &#39; obsahující typ &#39; &lt;členské jméno&gt;&#39;
Odkaz na je vyžadován na modul,\<modulename >' obsahující typ '\<membername >'. Přidáte do projektu.  
  
 Projekt obsahuje typ, který závisí na jiném typu ze sestavení, které nejsou součástí odkazy na projekt.  
  
 **ID chyby:** BC30653  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte odkaz na projekt u závislých typů v projektu.
