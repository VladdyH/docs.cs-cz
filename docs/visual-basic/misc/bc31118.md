# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>& č. 39; Ukončení AddHandler & č. 39; musí být první příkaz na řádek
`End AddHandler` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End AddHandler`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC31118  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
