# <a name="too-few-type-arguments"></a>Příliš málo argumentů typu
Obecná metoda metoda byla volána s méně argumenty typu, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC36578  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Tak, aby je jedna pro každý parametr typu Obecné metody, která je vyvolán, přidejte do seznamu argument typu argumenty typu.
