# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Ukončení&#39; příkazy nejsou platné v příkazové podokno
`Exit` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30713  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Exit` příkazy z ladění kódu.
