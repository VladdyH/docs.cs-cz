# <a name="statement-cannot-appear-outside-of-a-method-bodymultiline-lambda"></a>Příkaz nelze vložit mimo textu nebo víceřádkového výrazu lambda – metoda

 **ID chyby:** BC30689  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Příkaz přesunete do procedury nebo podprogramu.
