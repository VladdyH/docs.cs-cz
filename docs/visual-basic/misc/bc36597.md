# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>Přejít na štítek &#39; &lt;název_jmenovky&gt; &#39; nelze použít v rámci výrazy lambda
Tělo výrazu lambda musí být výraz, který vypočítá a vrátí jednu hodnotu. Jsou povoleny žádné příkazy, včetně `GoTo` příkazy.  
  
 **ID chyby:** BC36597
