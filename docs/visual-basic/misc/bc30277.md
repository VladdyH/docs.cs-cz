# <a name="type-character-39ltcharacternamegt39-does-not-match-declared-data-type-39lttypegt39"></a>Zadejte znak &#39; &lt;název_postavy&gt; &#39; neodpovídá deklarované datový typ &#39; &lt;typu&gt;&#39;
Proměnné je deklarován jednoho datového typu, ale označuje typ znakem představující typ nekompatibilní data, například s `K$ = 10` při `K` je deklarovaná `Integer`.  
  
 **ID chyby:** BC30277  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit typ dat deklarovaný proměnné, nebo změnit nebo odebrat znak typu v dokumentaci.
