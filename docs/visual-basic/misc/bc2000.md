# <a name="initialization-error"></a>Chyba inicializace
Došlo k chybě při inicializaci programu. To může být způsobeno tím, když je stále probíhá zrušení instalace.  
  
 **ID chyby:** BC2000  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokuste se znovu spustit instalaci.
