# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39;Ukončení struktura&#39; musí předcházet odpovídající &#39;struktura&#39;
`End Structure` Příkaz lze použít pouze na konci bloku struktura, který začíná `Structure` příkaz.  
  
 **ID chyby:** BC30621  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že aktuálnímu bloku struktura začíná `Structure` příkaz.
