# <a name="39ltstatementnamegt39-statement-requires-an-array"></a>&#39;&lt;statementname&gt; &#39; příkaz vyžaduje pole
`Erase` a `ReDim` příkazy vyžadují pole datové typy. Zadaný typ není pole.  
  
 **ID chyby:** BC30049  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte datový typ pro typ pole.
