# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Vlastnost &#39; &lt;propertyname&gt; &#39; musí být deklarován &#39;výchozí&#39; protože přepisuje výchozí vlastnost na základní &#39; &lt;název&gt;&#39;
A `Property` přepsání příkaz ve vašem kódu `Default` vlastnost na základní třídy, struktury, nebo rozhraní a proto musí být deklarován `Default`.  
  
 **ID chyby:** BC30360  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat vlastnost `Default`.
