# <a name="set-not-supported-at-run-time"></a>Nastavit není podporována v době běhu
Pokusili jste se nastavit nebo změnit vlastnost, jehož hodnota lze nastavit pouze v době návrhu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte odkaz na vlastnost z vašeho kódu.  
  
2.  Změňte odkaz na hodnotu vlastnosti vrátit pouze v době běhu.
