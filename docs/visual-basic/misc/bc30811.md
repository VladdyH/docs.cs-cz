# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>& č. 39; ReDim & č. 39; příkazy už lze deklarovat proměnné pole
`ReDim`můžete použít pouze ke změně velikosti pole existujících.  
  
 **ID chyby:** BC30811  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte velikost pole, pokud jsou deklarovány; například:  
  
    ```  
    Dim X(20) As Integer  
    ```
