Název Typ člena v konfliktu s názvem členem implicitně deklarované v všechny výčty.  
  
 Visual Basic – kompilátor vytvoří implicitní členy odpovídající určité programovací elementy, které je deklarovat. Výčty implicitně deklarovat člen `value__member`.
