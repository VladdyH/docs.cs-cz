# <a name="collection-index-must-be-in-the-range-1-to-the-size-of-the-collection"></a>Index kolekce musí být v rozsahu od 1 do velikosti kolekce
Zadaný index hodnota spadá mimo platný rozsah je 1 velikosti kolekce.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadaná hodnota změňte na platný.
