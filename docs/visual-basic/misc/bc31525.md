# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Typ & č. 39; &lt;typename&gt;& č. 39; je naimportovaná ze různé verze sestavení & č. 39;&lt; AssemblyName&gt;& č. 39;
Typ '\<typename >' je importovat z různých verzí sestavení '\<assemblyname >'. Různé verze stejného sestavení nelze použít ve stejném projektu.  
  
 Typ importovaného z různé verze stejného sestavení, vytváření nejednoznačnosti.  
  
 **ID chyby:** BC31525  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte odkazy na více verzí stejného sestavení.
