# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Literály XML a vlastnosti osy XML nejsou k dispozici v této relaci ladění, protože se používá v tomto sestavení
Bylo odkazováno na literál XML nebo vlastnost osy XML v **sledovat** nebo **Immediate** okno během ladicí relace, ve které XML v jazyce Visual Basic funkce nejsou k dispozici. To platí pro sestavení, které se nepoužívá soubor XML v jazyce Visual Basic funkce nebo je sestavení pro vydání.  
  
 **ID chyby:** BC31196  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahájit novou relaci ladění použitím ladění sestavení sestavení.
