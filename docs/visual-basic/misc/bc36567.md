# <a name="parameter-39ltparameternamegt39-already-has-a-matching-argument"></a>Parametr & č. 39; &lt;parametername&gt;& č. 39; už má odpovídající argument
Argument je předaná název více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda jeden ze zadaných názvů argument je zadáno chybně.  
  
-   Odeberte duplicitní argument ze seznamu argumentů.
