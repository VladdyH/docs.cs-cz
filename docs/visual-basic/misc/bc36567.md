# <a name="parameter-39ltparameternamegt39-already-has-a-matching-argument"></a>Parametr &#39; &lt;parametername&gt; &#39; již má odpovídající argument
Argument je předaná název více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC36567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda jeden ze zadaných názvů argument je zadáno chybně.  
  
-   Odeberte duplicitní argument ze seznamu argumentů.
