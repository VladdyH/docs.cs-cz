# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>Komentář XML nemůže být uvnitř metody nebo vlastnosti
Komentář XML se nemůže objevit v rámci metody nebo vlastnosti. Komentář XML bude ignorován. Bloky komentáře XML se nemůže objevit v rámci metody nebo vlastnosti.  
  
 **ID chyby:** BC42303  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte blok komentáře nebo přesunout do umístění vhodnější.
