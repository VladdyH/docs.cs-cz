# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>& č. 39; #If & č. 39; blok musí končit odpovídající & č. 39; #End If & č. 39;
`#If`představuje direktivu Podmíněná kompilace. `#If` Blok není ukončila příkazem `#End If` – direktiva.  
  
 **ID chyby:** BC30012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `#End If` na konci bloku Podmíněná kompilace.
