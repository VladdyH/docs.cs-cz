# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39; bloku musí končit odpovídající &#39;#End If&#39;
`#If` představuje direktivu Podmíněná kompilace. `#If` Blok není ukončila příkazem `#End If` – direktiva.  
  
 **ID chyby:** BC30012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `#End If` na konci bloku Podmíněná kompilace.
