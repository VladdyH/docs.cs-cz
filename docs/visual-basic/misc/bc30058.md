# <a name="statements-and-labels-are-not-valid-between-39select-case39-and-first-39case39"></a>Příkazy a popisky nejsou platné mezi &#39;Select Case&#39; první a &#39;případu&#39;
Příkaz, který není komentář se zobrazí mezi otevření `Select` nebo `Select Case` příkaz a jeho první `Case` příkaz.  
  
 **ID chyby:** BC30058  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud příkaz použité komentář, předcházet s oddělovačem komentář (`'` nebo `REM`). Jinak přesuňte nebo odstraňte příkaz.
