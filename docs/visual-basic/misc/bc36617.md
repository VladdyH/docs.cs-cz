# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Název agregační funkce nelze použít s – znak typu
– Znak typu byl součástí na konci názvu agregační funkci.  
  
 **ID chyby:** BC36617  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte znak typu od názvu agregační funkci.
