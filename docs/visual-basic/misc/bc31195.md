# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>Operátor &#39;operátor&#39; není definován pro typy &#39;type1&#39; a &#39;type2&#39;
Operátor 'operátor' není definována pro typy 'type1' a 'type2'. Vlastnost 'Hodnota' můžete získat hodnotu řetězce první prvek '\<parentElement > ".  
  
 Používá se literál XML v binárním nebo `Boolean` porovnání.  
  
 **ID chyby:** BC31195  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Value` vlastnost literál pro binárního souboru XML nebo `Boolean` porovnání.
