# <a name="xml-literals-and-xml-axis-properties-are-not-available"></a>Literály XML a vlastnosti osy XML nejsou k dispozici
Literály XML a vlastnosti osy XML nejsou k dispozici. Přidejte odkazy na System.Xml System.Xml.Linq a System.Core.  
  
 Kód kompilován zahrnuje literálů XML nebo vlastnosti osy XML, ale neobsahuje odkaz na sestavení, které jsou potřebné ke kompilaci literálů XML nebo vlastnosti osy XML.  
  
 **ID chyby:** BC31190  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte odkazy na sestavení System.Xml.dll knihovně System.Xml.Linq.dll a System.Core.dll.
