# <a name="nullable-parameters-must-specify-a-type"></a>Parametry umožňující hodnotou null je nutné zadat typ
Parametr obsahuje modifikátor typu s povolenou hodnotou Null (?), ale neurčuje typu.  
  
 **ID chyby:** BC36632  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte typ při deklarování parametr hodnotu Null.
