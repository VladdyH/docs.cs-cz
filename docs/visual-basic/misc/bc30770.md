# <a name="event-39lteventnamegt39-event-specified-by-the-39defaultevent39-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Událost &#39; &lt;eventname&gt; &#39; událostí určeného &#39;DefaultEvent&#39; atribut není veřejně přístupná událost pro tuto třídu.
<xref:System.ComponentModel.DefaultEventAttribute> Atribut musíte zadat název veřejně přístupná události ve třídě, na který se použije atribut.  
  
 **ID chyby:** BC30770  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, že třída definuje veřejně přístupná událost.  
  
2.  Ujistěte se, jestli název událostí ve třídě, odpovídá názvu určeného <xref:System.ComponentModel.DefaultEventAttribute> atribut.
