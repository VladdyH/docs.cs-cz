# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>Značka komentáře XML &#39; &lt;značka&gt; &#39; není povolená u &#39; &lt;element&gt; &#39; prvek jazyka
Značka komentáře XML \<značka > není povolená u '\<element >' element. Komentáře XML budou ignorovány.  
  
 Značky pro komentáře XML nepodporuje jazyk elementu, které je použito.  
  
 **ID chyby:** BC42306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nepodporované značka komentáře XML.
