# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>Značka komentáře XML &#39; &lt;značky&gt; &#39; není povolená u &#39; &lt;element&gt; &#39; prvek jazyka
Značka komentáře XML \<značka > není povolená u "\<element >' elementu. Komentář XML bude ignorován.  
  
 Značka komentáře XML není podporován jazyk elementu, na který se použije.  
  
 **ID chyby:** BC42306  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nepodporované značka komentáře XML.
