# <a name="enum-39ltenumnamegt39-must-contain-at-least-one-member"></a>Výčet & č. 39; &lt;enumname&gt;& č. 39; musí obsahovat alespoň jeden člen
Nastane, prázdný výčet. Musí být deklarován nejméně jeden člen mezi `Enum` příkaz a `End Enum` příkaz.  
  
 **ID chyby:** BC30280  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidejte deklaraci člen výčtu.
