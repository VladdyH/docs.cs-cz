# <a name="39lib39-expected"></a>&#39;Lib&#39; očekávání
A `Declare` příkaz určuje název knihovny bez použití `Lib` – klíčové slovo.  
  
 **ID chyby:** BC30218  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Lib` – klíčové slovo před název knihovny.
