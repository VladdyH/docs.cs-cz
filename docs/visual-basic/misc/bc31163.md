# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>Očekávaný odpovídající zavření v jednoduchých uvozovkách pro hodnotu atributu XML
Hodnota atributu, která je oddělená apostrofech chybí ukončovací znak uvozovky.  
  
 **ID chyby:** BC31163  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte pro vymezení hodnotu atributu ukončovací znak jednoduché uvozovky (').
