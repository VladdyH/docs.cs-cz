# <a name="39d39-can-no-longer-be-used-to-indicate-an-exponent-use-39e39-instead"></a>&#39;D&#39; už nedá používat k označení exponentu &#39;E&#39; místo
Měl ' znak nelze použít k označení exponenciální zápis.  
  
 **ID chyby:** BC30827  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `^` operátor nebo `E+` znaků k označení exponentem je k dispozici. Příklad:  
  
    ```  
    Const Mole = 6.02E+23 ' Same as 6.02D23  
    Const Mole2 = 6.02 * 10 ^ 23 ' Same as 6.02D23  
    ```
