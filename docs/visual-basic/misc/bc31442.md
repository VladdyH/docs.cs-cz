# <a name="parameter-name-39ltparametername1gt39-does-not-match-the-name-of-the-corresponding-parameter-39ltparametername2gt39-defined-on-the-partial-method-declaration-39ltmethodnamegt39"></a>Název parametru &#39; &lt;parametername1&gt; &#39; neodpovídá názvu odpovídajícího parametru &#39; &lt;parametername2&gt;&#39;definovaná v deklaraci &#39; &lt;methodname&gt;&#39;
Pokud jsou zadány parametry pro deklarace a implementace částečné metody, názvy odpovídající parametry musí být stejné. Tuto chybu způsobí například následující kód.  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged(ByVal newVal As Integer)  
    End Sub  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method. This error is  
    ' reported for parameter val.  
    ' Private Sub valueChanged(ByVal val As Integer)  
    '     MsgBox(Value was changed to " & Me.Quantity)  
    ' End Sub  
  
End Class  
```  
  
 **ID chyby:** BC31442  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Změňte parametr název nebo názvy v deklaraci nebo implementace tak, aby odpovídající parametry mají stejný název.
