# <a name="property-without-an-39as39-clause-type-of-object-assumed"></a>Vlastnost bez &#39;jako&#39; klauzule; Typ objektu předpokládá, že
Deklarace vlastnosti neurčuje `As` klauzule.  
  
 `As` Klauzuli určuje datový typ, který se má přidružit programovací element. V [Property – příkaz](../../visual-basic/language-reference/statements/property-statement.md), určuje datový typ hodnoty, vlastnosti `Get` postup vrátí kód volání. Pokud nepoužijete `As` klauzuli v `Property` příkaz výchozí typ dat vlastnosti `Object`.  
  
 Ve výchozím nastavení je tato zpráva upozornění. Další informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42022  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Patří `As` klauzuli v `Property` lze zadat typ dat vlastnosti.
