# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-lambda-expression"></a>Instance s omezeným přístupem typu &#39; &lt;typename&gt; &#39; nelze použít ve výrazu lambda
Instance zakázané typy, jako například <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>, a <xref:System.TypedReference>, nelze použít ve výrazu lambda nebo výrazu dotazu.  
  
 **ID chyby:** BC36640
