# <a name="line-ltnumbergt-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>Řádek &lt;číslo&gt; nelze přečíst, protože přesahuje maximální délka řádku.
Řádek nelze přečíst, protože překračuje maximální délka řádku. Obvykle to znamená, že je znak uvozovek v souboru, ale není pravé uvozovky.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vložte uzavírací uvozovky v souboru.
