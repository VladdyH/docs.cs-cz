# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39;&lt;Modifikátor&gt; &#39; není platná místní deklarace konstanty
A `Const` příkaz uvnitř procedury obsahuje neplatný klíčové slovo, jako například `Overloads`.  
  
 **ID chyby:** BC30246  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte neplatné klíčové slovo z `Const` příkaz.
