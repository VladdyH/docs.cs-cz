# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Znaky typu nelze použít v rozsahu deklarace proměnných
– Znak typu byl součástí na konci názvu deklarace proměnné rozsahu v dotazu LINQ.  
  
 **ID chyby:** BC36601  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrání znak typu deklarace proměnné rozsahu.
