# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39;Ukončení RaiseEvent –&#39; musí předcházet odpovídající &#39;RaiseEvent –&#39; deklarace
`End RaiseEvent` Příkazu došlo k chybě bez odpovídající `RaiseEvent` příkaz. `End RaiseEvent` musí předcházet odpovídající `RaiseEvent` příkaz.  
  
 **ID chyby:** BC31126  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že předchozí `RaiseEvent` příkaz je platná a správně zadané.
