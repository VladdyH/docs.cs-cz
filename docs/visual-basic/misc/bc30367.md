# <a name="class-39ltclassnamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Třída &#39; &lt;classname&gt; &#39; nelze indexovat, protože nemá žádné výchozí vlastnost
Se nepodařilo zadat výchozí vlastnost pro třídu.  
  
 **ID chyby:** BC30367  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Určete vlastnost jako vlastnost výchozí od jeho deklaraci s `Default` – klíčové slovo.
