# <a name="no-default-member-found-for-type-39lttypenamegt39"></a>Výchozí člen nebyl nalezen pro typ & č. 39; &lt;typename&gt;& č. 39;
Pozdní vazba index `Get` nebo `Set` došlo k volání.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že objekt obsahuje vlastnost s výchozí `Get` nebo `Set`, v závislosti na tom, který se pokoušíte provést.
