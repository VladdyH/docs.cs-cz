# <a name="39catch39-cannot-catch-type-39lttypenamegt39-because-it-is-not-39systemexception39-or-a-class-that-inherits-from-39systemexception39"></a>&#39;Catch –&#39; nemůže zachytit typ &#39; &lt;typename&gt; &#39; protože není &#39;System.Exception&#39; nebo třídu, která dědí z &#39;System.Exception&#39;
`Catch` můžete pouze zachytávat výjimky a pokusili jste se catch něco, co není odvozen od výjimku.  
  
 **ID chyby:** BC30392  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Catch` prohlášení, nebo změnit cíl `Catch` skutečné výjimce.
