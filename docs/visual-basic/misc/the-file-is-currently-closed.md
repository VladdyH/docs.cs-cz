# <a name="the-file-is-currently-closed"></a>Soubor je momentálně uzavřeno
Byl proveden pokus o provedení určité operace, které lze provést pouze na otevření souboru, ale soubor je momentálně uzavřeno.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Otevřete soubor a opakujte operaci znovu.
