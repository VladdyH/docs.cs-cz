# <a name="39ltmethod1gt39-and-39ltmethod2gt39-cannot-overload-each-other-because-they-differ-only-by-parameters-declared-39byref39-or-39byval39"></a>&#39;&lt;Method1&gt; &#39; a &#39; &lt;method2&gt; &#39; nelze přetížení navzájem, protože se liší pouze parametry deklarovaný &#39;ByRef&#39; nebo &#39;ByVal&#39;
Pokusili jste se přetížení metody s jinou metodu, která se liší od prvního pouze systémem parametr deklarován jako `ByRef` nebo `ByVal`.  
  
 **ID chyby:** BC30345  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že tyto metody jsou rozlišené pomocí více než název `ByRef` nebo `ByVal` parametr.
