# <a name="39end-set39-must-be-the-first-statement-on-a-line"></a>&#39;Sady ukončit&#39; musí být první příkaz na řádek
`End Set` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End Set` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
