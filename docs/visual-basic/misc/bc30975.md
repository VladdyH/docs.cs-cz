# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Pro&#39; smyčky řídicí proměnná již používán v příkazu vnější &#39;pro&#39; smyčky
Vnořené `For` smyčky musí všechny používat proměnné jedinečný ovládacího prvku.  
  
 **ID chyby:** BC30975  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte jednu z proměnné názvy ovládacích prvků.
