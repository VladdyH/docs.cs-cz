# <a name="too-few-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Příliš málo argumentů typu rozšíření metoda & č. 39; &lt;methodname&gt;& č. 39; definovaným v & č. 39;&lt; TypeName&gt;& č. 39;
Obecný rozšiřující metoda metoda byla volána s méně argumenty typu, než je typ parametry v definici metoda.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC36590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Tak, aby je jedna pro každý typ parametru metody rozšíření, která je vyvolán, přidejte do seznamu argument typu argumenty typu.
