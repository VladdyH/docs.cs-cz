# <a name="39ltnamegt39-is-already-declared-as-39ltdeclarationgt39-in-this-ltdeclarationspacegt"></a>& č. 39; &lt;název&gt;& č. 39; je již deklarována jako & č. 39;&lt; deklarace&gt;& č. 39; v tomto &lt;declarationspace&gt;
Dva programovací elementy jsou deklarovány se stejným názvem ve stejném prostoru deklarace (třída, modulu, rozhraní nebo struktura).  
  
 **ID chyby:** BC30260  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jedna z deklarací přesunout do jiné deklarace místa, nebo změňte název jednoho z deklarované elementy.
