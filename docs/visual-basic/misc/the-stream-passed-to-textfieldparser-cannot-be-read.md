# <a name="the-stream-passed-to-textfieldparser-cannot-be-read"></a>Datový proud předaný TextFieldParser nelze číst.
[Objektu TextFieldParser](../../visual-basic/language-reference/objects/textfieldparser-object.md) nelze číst datový proud, který byl předán do ní. To může být způsobeno při čtení souboru, který není do textového souboru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, ujistěte se, že soubor je textový soubor.
