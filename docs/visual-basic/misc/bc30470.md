# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39; nelze použít mimo třídu
Použili jste `MyClass` – klíčové slovo mimo struktura třídy.  
  
 **ID chyby:** BC30470  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `MyClass` deklaraci do struktury vaší třídy.
