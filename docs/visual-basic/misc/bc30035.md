# <a name="syntax-error-visual-basic-error"></a>Chyba syntaxe (Chyba jazyka Visual Basic)
Řádek kódu zdroj generuje nespecifikované syntaktickou chybu.  
  
 **ID chyby:** BC30035  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Vyhledejte v dokumentaci pro každou hesel používaných v řádku kódu zdroje. Porovnejte ukázkový kód v dokumentaci k proti řádku zdroje generování této chyby.  
  
2.  Pokud nelze určit příčinu chyby, shromažďovat informace o okolnostech a upozornění služby Microsoft Product Support Services.
