# <a name="39get39-statements-are-no-longer-supported-visual-basic"></a>& č. 39; Get & č. 39; příkazy již nejsou podporovány (Visual Basic)
`Get`příkazy již nejsou podporovány. Funkce souborových vstupně-výstupních operací je dostupné v `Microsoft.VisualBasic` obor názvů, ale cílové verze rozhraní .NET Compact Framework ji nepodporuje.  
  
 **ID chyby:** BC30767  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Provedení operace se soubory s funkcí definovaných v <xref:System.IO> oboru názvů.
