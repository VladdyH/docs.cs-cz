# <a name="argument-39ltargumentnamegt39-is-nothing-or-empty"></a>Argument & č. 39; &lt;název argumentu argumentname&gt;& č. 39; není nic nebo musí být prázdná
Název argument není platný, protože je inicializováno `Nothing` nebo je prázdný. Argument musí mít specifické datový typ a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte název platný argument.
