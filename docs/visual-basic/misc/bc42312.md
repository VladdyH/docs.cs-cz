# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>Blok komentáře XML nemůže být přidružena libovolný element jazyk, který podporuje aplikace dokumentační komentáře XML
Blok komentáře XML nemůže být přidružena libovolný element, platný jazyk. Komentáře XML budou ignorovány. K této chybě dojde, pokud má značky XML v bloku komentář žádný odpovídající element jazyka.  
  
 **ID chyby:** BC42312  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Najít a odebrat nadbytečné značky XML nebo zadejte element jazyk, které popisuje.
