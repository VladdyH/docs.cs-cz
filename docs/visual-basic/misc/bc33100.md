# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Modifikátor s možnou hodnotou Null nelze zadat pro proměnnou a její typ
Modifikátor typ s možnou hodnotou Null (?) byla zahrnuta pro deklarované proměnné a příslušným deklarovaným typem.  
  
 **ID chyby:** BC33100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrání proměnné nebo příslušným deklarovaným typem modifikátor typ s možnou hodnotou Null (?).
