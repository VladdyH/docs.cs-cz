# <a name="39ltfunctionnamegt39-is-not-declared-visual-basic-error"></a>& č. 39; &lt;%{FunctionName/&gt;& č. 39; není deklarovaný (Chyba jazyka Visual Basic)
'\<%{FunctionName/ >' není deklarovaný. Funkce souborových vstupně-výstupních operací je k dispozici v Microsoft.VisualBasic – obor názvů.  
  
 **ID chyby:** BC30815  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pomocí funkce vstupně-výstupní definované v `My.Computer.FileSystem`, `System.IO` obor názvů, Visual Basic – spuštění funkce, nebo `Microsoft.VisualBasic.FileSystem` objektu.
