# <a name="file-name-already-declared-with-a-different-guid-and-checksum-value"></a>Název souboru již deklarována s jinou hodnotou GUID a kontrolního součtu
Byl proveden pokus o deklarovat soubor, ale soubor s jinou hodnotou GUID a kontrolních součtů již existuje.  
  
 **ID chyby:** BC42034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte název souboru a cestu a ujistěte se, že každý je správná.  
  
-   Odstraňte redundantní soubor, pokud není nutné.
