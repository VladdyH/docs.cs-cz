# <a name="39as39-comma-or-3939-expected"></a>&#39;Stejně jako&#39;, čárkou nebo &#39;)&#39; očekávání
Chybí některý typ parametru, které poskytl `As`, čárkami (`,`), nebo pravé závorky ( `)` ).  
  
 **ID chyby:** BC32100  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte chybějící `As`, čárkou nebo kulaté závorky.
