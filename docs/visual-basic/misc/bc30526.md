# <a name="property-39ltpropertynamegt39-is-39readonly39"></a>Vlastnost &#39; &lt;propertyname&gt; &#39; je &#39;jen pro čtení&#39;
Pokusili jste se o přiřazení hodnoty k vlastnosti, která je deklarovaná `ReadOnly`.  
  
 **ID chyby:** BC30526  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `ReadOnly` specifikátor z deklarace vlastnosti.
