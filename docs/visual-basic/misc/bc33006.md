# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>& č. 39; Ukončení operátor & č. 39; musí být první příkaz na řádek
`End Operator` Příkaz následuje dvojtečka (`:`) příkaz oddělovače. `End Operator`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC33006  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
