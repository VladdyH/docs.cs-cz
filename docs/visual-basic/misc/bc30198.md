# <a name="3939-expected"></a>& č. 39;) & č. 39; očekávání
Seznam, jako je například seznam argumentů v definici postup nebo volání procedury, není ukončila příkazem kulaté závorky (`)`) znaků.  
  
 **ID chyby:** BC30198  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat uzavírací kulatá závorka znak na konec seznamu.
