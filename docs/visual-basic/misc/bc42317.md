# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parametr typu komentáře XML &lt;parametr&gt; neodpovídá typu parametru na odpovídajícím &lt;– klíčové slovo&gt; – příkaz
Parametr komentáře XML \<parametr > neodpovídá typu parametru na odpovídajícím \<– klíčové slovo > příkaz. Komentáře XML budou ignorovány.  
  
 Značky pro komentáře XML byl zadán pro některé, ale ne všechny parametry typu v metodě.  
  
 **ID chyby:** BC42317  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte značky pro komentáře XML pro všechny parametry typu v metodě.
