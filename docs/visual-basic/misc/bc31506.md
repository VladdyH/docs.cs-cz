# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>& č. 39; &lt;typename&gt;& č. 39; nelze použít jako atribut, protože je deklarovaná & č. 39; MustInherit & č. 39;
Vlastní třídy atributu nelze deklarovat jako `MustInherit`.  
  
 **ID chyby:** BC31506  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `MustInherit` modifikátor z vlastních atributů.
