# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Případ & č. 39; příkazy nejsou platné v příkazové podokno
`Case`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30719  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Case` příkazy z **Immediate** okno.
