# <a name="39finally39-can-only-appear-once-in-a-39try39-statement"></a>&#39;Nakonec&#39; může být pouze jednou v &#39;zkuste&#39; – příkaz
`Finally` slouží k dokončení `Try...Catch...Finally` blokovat; proto se může vyskytovat pouze jednou na konci bloku.  
  
 **ID chyby:** BC30381  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Vyhledejte a odstraňte redundantní `Finally`.
