# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Zastavit&#39; příkazy nejsou platné v příkazové podokno
`Stop` a `End` příkazy pozastavení provádění a nejsou povoleny v kontextu ladění.  
  
 **ID chyby:** BC30122  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nevydávají `Stop` nebo `End` příkaz v **Immediate** okno.
