# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>& č. 39; &lt;membername1&gt;& č. 39; a & č. 39;&lt; membername2&gt;& č. 39; nelze přetížení navzájem, protože se liší pouze typy volitelné parametry
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30696  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte alespoň jeden požadovaný parametr jako na jiný datový typ.
