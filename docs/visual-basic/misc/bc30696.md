# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39;&lt;membername1&gt; &#39; a &#39; &lt;membername2&gt; &#39; nelze přetížení navzájem, protože se liší pouze typy volitelné parametry
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30696  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte alespoň jeden požadovaný parametr jako na jiný datový typ.
