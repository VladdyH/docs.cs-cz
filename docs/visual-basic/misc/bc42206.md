# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Byl překročen maximální počet upozornění
Maximální počet upozornění kompilace je 100. Kompilátor ukládá tento limit ke snížení velikosti chyby a upozornění sestavy.  
  
 Ve výchozím nastavení je tato zpráva upozornění. Další informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42206  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Další upozornění vygenerovaných tento kompilace zkontrolovat a vyřešit co nejvíce. Pokud zdrojový kód obsahuje více než 100 upozornění, může se zobrazit další upozornění v další kompilace.
