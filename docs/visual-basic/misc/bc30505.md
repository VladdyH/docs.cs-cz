# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-39shared39"></a>Metody nebo událostí, které implementují rozhraní, které nelze deklarovat členy & č. 39; Sdílené & č. 39;
Pokusili jste se deklarovat jako `Shared` metoda nebo událost, která implementuje rozhraní člen. Metody a události se implementuje v třídě nelze určené `Shared` nebo `Private`, kromě Neděditelný třídu.  
  
 **ID chyby:** BC30505  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Shared` – klíčové slovo.
