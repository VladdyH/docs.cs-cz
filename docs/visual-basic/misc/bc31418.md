# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>& č. 39; Obslužné rutiny & č. 39; musíte zadat v modulech a & č. 39; WithEvents & č. 39; Proměnná kvalifikovaný pomocí jednoho identifikátoru
Chcete-li určit obslužné rutiny události, `Handles` příkazy musíte zadat objektová proměnná, která byla deklarována s `WithEvents` – klíčové slovo.  
  
 **ID chyby:** BC31418  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `WithEvents` modifikátor deklarovat proměnné, které budou použity s `Handles` příkaz.
