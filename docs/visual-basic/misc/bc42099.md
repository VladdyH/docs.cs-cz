# <a name="unused-local-constant-39ltconstantgt39"></a>Nepoužívané místní konstanta: &#39; &lt;konstant&gt;&#39;
Místní konstanta bylo deklarováno, který není použit kdekoli v kódu.  
  
 **ID chyby:** BC42099  
  
### <a name="to-remove-this-warning"></a>Chcete-li odebrat toto upozornění  
  
1.  Tato chyba upozornění a můžete ji ignorovat. Pokud chcete odebrat toto upozornění, odeberte nepoužívané deklarace konstanty z vašeho kódu.
