# <a name="39inherits39-not-valid-in-modules"></a>& č. 39; dědí & č. 39; není platný v modulech
`Inherits` Příkaz se provádí v modulu. Moduly nelze dědí z třídy.  
  
 **ID chyby:** BC30230  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz nebo znovu zadejte modul jako třída.
