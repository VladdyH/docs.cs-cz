# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-they-have-different-access-levels"></a>&#39;&lt;declaration1&gt; &#39; nejde přepsat &#39; &lt;declaration2&gt; &#39; protože mají různé úrovně přístupu
Procedura nebo vlastnost deklarace pokusí přepsat zděděné element se stejným názvem, ale Určuje jinou usnadnění než zděděné element. Zděděná elementu usnadnění, jako například `Public` nebo `Private`, musí být zachováno v přepsání.  
  
 **ID chyby:** BC30266  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte usnadnění přepsání elementu odpovídat zděděné elementu.
