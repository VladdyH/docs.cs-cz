# <a name="constructors-cannot-implement-interface-methods"></a>Konstruktory nelze implementovat, metody rozhraní
Pokusili jste se použít `New` – klíčové slovo implementovat metodu rozhraní, která není platná.  
  
 **ID chyby:** BC30550  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Implementace rozhraní s `Implements` – klíčové slovo.
