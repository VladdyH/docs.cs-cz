# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>& č. 39; &lt;specifikátor&gt;& č. 39; není platný v deklaraci struktury
Použili jste specifikátorem, jako například `Dim` nebo `Override`, který není platný na `Structure` deklarace.  
  
 **ID chyby:** BC30395  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
