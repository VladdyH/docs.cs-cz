# <a name="specifiers-are-not-valid-on-39addhandler39-39removehandler39-and-39raiseevent39-methods"></a>Specifikátory nejsou platná na &#39;AddHandler&#39;, &#39;RemoveHandler&#39; a &#39;RaiseEvent –&#39; metody
`AddHandler`, `RemoveHandler`, A `RaiseEvent` metoda deklarace se nedá změnit klíčová slova, jako `Public` nebo `ReadOnly`. Tato klíčová slova může obsahovat pouze upravitelnými deklarace.  
  
 **ID chyby:** BC31135  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte klíčové slovo z deklarace metody.
