# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>Značka komentáře XML &#39;vrátí&#39; není povolená u &#39;WriteOnly&#39; vlastnost
Značka komentáře XML, vrátí' není povolená u vlastnosti 'WriteOnly'. Komentáře XML budou ignorovány.  
  
 Komentáře jazyka XML pro deklarace vlastnosti jen pro zápis, nemůže obsahovat \<vrátí > značky.  
  
 **ID chyby:** BC42313  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat nepodporovanou \<vrátí > značky.
