# <a name="overload-resolution-failed-because-no-39ltmethodgt39-is-accessible"></a>Rozlišení přetěžování se nezdařilo, protože žádné &#39; &lt;metoda&gt; &#39; přístupný
Provedli jste volání přetížené metody, ale kompilátor nelze najít metodu přístupné pomocí seznamu parametr, který odpovídá z hlediska počet parametrů.  
  
 **ID chyby:** BC30517  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte seznam argumentů, které jste zadali.  
  
2.  Zkontrolujte metodu, kterou se pokoušíte volání pro ověření, zda je přístupný.
