# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>Proměnné v modulech nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Použili jste specifikátorem, jako například `MustInherit`, na proměnnou v `Module` příkaz. Moduly nikdy může být vytvořena, nepodporují dědičnosti a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30593  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
