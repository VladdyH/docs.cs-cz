# <a name="xml-dtds-are-not-supported"></a>Specifikace DTD XML nejsou podporovány.
Definice typu dokumentu (DTD) byl zadán jako literál XML.  
  
 **ID chyby:** BC31175  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte DTD. Definice schématu XML (XSD) použijte k určení schéma pro dokument XML.
