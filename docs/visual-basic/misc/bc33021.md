# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-39lttypenamegt39"></a>Alespoň jeden typ parametru tohoto binárního operátoru musí obsahovat typ &#39; &lt;typename&gt;&#39;
Definice binární operátor určuje oba parametry s typy jiné než u třídu nebo strukturu, ve kterém je definovaný operátor.  
  
 Když definujete operátor ve třídě nebo struktuře, alespoň jeden z parametrů musí být typu třídy nebo struktura.  
  
 **ID chyby:** BC33021  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte typ jeden nebo oba parametry typu třídu nebo strukturu, ve kterém je definovaný operátor.
