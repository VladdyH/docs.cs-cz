# <a name="databits-must-be-greater-than-0"></a>DataBits musí být větší než 0.
`DataBits` Zadaný pro argument `My.Computer.Ports.OpenSerialPort` metoda musí být větší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte hodnotu `DataBits` argument kladné číslo.
