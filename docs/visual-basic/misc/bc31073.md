# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39;&lt;name1&gt; &#39; je v konfliktu s &lt;typ&gt; se stejným názvem definované v &#39; &lt;name2&gt;&#39;
Byl zjištěn název, který je v konfliktu s typem se stejným názvem.  
  
 **ID chyby:** BC31073  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte jedinečný název.  
  
2.  Použijte hranaté závorky kolem konfliktní název člena.
