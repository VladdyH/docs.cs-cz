# <a name="property-missing-39end-property39"></a>Chybí vlastnost & č. 39; Vlastnost end & č. 39;
`End` Příkaz musí odpovídat bloku ukončí. A `Property` definice nebyl správně zavřen.  
  
 **ID chyby:** BC30025  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Property` příkaz na konec definici vlastnosti.
