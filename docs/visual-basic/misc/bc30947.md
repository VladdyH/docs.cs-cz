# <a name="suspend-request-is-pending"></a>Pozastavení čeká na vyřízení žádosti
V ladicím programu sady Visual Studio výraz určuje volání procedury, ale je požadavek na pozastavení vlákno.  
  
 Ladicí program nebude pokoušet o volání procedury na vlákno, které není aktivní.  
  
 **ID chyby:** BC30947  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je to možné určete zdroj žádost o pozastavení vlákno, takže ho můžete zabránit dalšímu.  
  
-   Ukončit a znovu spusťte ladění.
