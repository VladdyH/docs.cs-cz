# <a name="cannot-implement-39ltinterfacename1gtltmembernamegt39-because-its-implementation-could-conflict-with-the-implementation-for-39ltinterfacename2gtltmembernamegt39-for-some-type-arguments"></a><span data-ttu-id="65f87-102">Nelze implementace & č. 39; &lt;interfacename1&gt;.&lt; MemberName&gt;& č. 39; protože jeho implementace může dojít ke konfliktu s implementaci pro & č. 39;&lt; interfacename2&gt;.&lt; MemberName&gt;& č. 39; u některých argumentů</span><span class="sxs-lookup"><span data-stu-id="65f87-102">Cannot implement &#39;&lt;interfacename1&gt;.&lt;membername&gt;&#39; because its implementation could conflict with the implementation for &#39;&lt;interfacename2&gt;.&lt;membername&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="65f87-103">Třída implementuje více než jeden generické rozhraní, z nichž jeden dědí z jiné, a dvě implementace člena rozhraní může dojít ke konfliktu určité hodnoty typu argumentů.</span><span class="sxs-lookup"><span data-stu-id="65f87-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="65f87-104">Tato chyba může generovat následující příkazy.</span><span class="sxs-lookup"><span data-stu-id="65f87-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="65f87-105">Protože `iFace2` dědí z `iFace1` pomocí vlastní typ parametru (`u`), `testClass` by implementovat dvě verze `iFace1.testSub` s identickými podpisy, pokud bylo předáno stejný argument typu `y` a `z`.</span><span class="sxs-lookup"><span data-stu-id="65f87-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="65f87-106">Tato akce by vytvořila nejednoznačnosti, kterou verzi má přístup.</span><span class="sxs-lookup"><span data-stu-id="65f87-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="65f87-107">**ID chyby:** BC32125</span><span class="sxs-lookup"><span data-stu-id="65f87-107">**Error ID:** BC32125</span></span>
