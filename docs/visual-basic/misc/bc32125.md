# <a name="cannot-implement-39ltinterfacename1gtltmembernamegt39-because-its-implementation-could-conflict-with-the-implementation-for-39ltinterfacename2gtltmembernamegt39-for-some-type-arguments"></a><span data-ttu-id="b057f-102">Nelze implementovat, &#39; &lt;interfacename1&gt;.&lt; MemberName&gt; &#39; vzhledem k tomu, že jeho implementace může dojít ke konfliktu s implementací pro &#39; &lt;interfacename2&gt;.&lt; MemberName&gt; &#39; pro některé argumenty typů</span><span class="sxs-lookup"><span data-stu-id="b057f-102">Cannot implement &#39;&lt;interfacename1&gt;.&lt;membername&gt;&#39; because its implementation could conflict with the implementation for &#39;&lt;interfacename2&gt;.&lt;membername&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="b057f-103">Třída implementuje více než jeden generické rozhraní, z nichž jeden dědí z jiné, a dvě implementace člena rozhraní může dojít ke konfliktu určité hodnoty typu argumentů.</span><span class="sxs-lookup"><span data-stu-id="b057f-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="b057f-104">Tato chyba může generovat následující příkazy.</span><span class="sxs-lookup"><span data-stu-id="b057f-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="b057f-105">Protože `iFace2` dědí z `iFace1` pomocí vlastní typ parametru (`u`), `testClass` by implementovat dvě verze `iFace1.testSub` s identickými podpisy, pokud bylo předáno stejný argument typu `y` a `z`.</span><span class="sxs-lookup"><span data-stu-id="b057f-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="b057f-106">Tato akce by vytvořila nejednoznačnosti, kterou verzi má přístup.</span><span class="sxs-lookup"><span data-stu-id="b057f-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="b057f-107">**ID chyby:** BC32125</span><span class="sxs-lookup"><span data-stu-id="b057f-107">**Error ID:** BC32125</span></span>

-   <span data-ttu-id="b057f-109">Změnit strukturu dědičnosti rozhraní tak, aby `iFace1` nebylo implementováno s dva argumenty jiného typu.</span><span class="sxs-lookup"><span data-stu-id="b057f-109">Change the inheritance structure of the interfaces so that `iFace1` could not be implemented with two different type arguments.</span></span>  
  
     <span data-ttu-id="b057f-110">-nebo-</span><span class="sxs-lookup"><span data-stu-id="b057f-110">-or-</span></span>
