# <a name="non-intrinsic-type-names-are-not-allowed-in-conditional-compilation-expressions"></a>Názvy bez vnitřního typu nejsou povoleny ve výrazech Podmíněná kompilace
Podmíněná kompilace výraz odkazuje na jiné než vnitřní typ, ale jsou povoleny pouze vnitřní typy.  
  
 Další informace o vnitřního typu v jazyce Visual Basic najdete v tématu [souhrn datových typů](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **ID chyby:** BC31426  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte – vnitřní typ vnitřního typu.
