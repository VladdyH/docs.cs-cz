# <a name="39default39-cannot-be-combined-with-39ltspecifiergt39"></a>& č. 39; výchozí & č. 39; nelze kombinovat s & č. 39; &lt;specifikátor&gt;& č. 39;
Pokusili jste se zkombinovat `Default` – klíčové slovo s specifikátorem, jako například `Shared` nebo `Private`, která je v tomto kontextu není platný.  
  
 **ID chyby:** BC30490  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Default`.  
  
     -nebo-  
  
-   Odeberte konfliktní specifikátor.
