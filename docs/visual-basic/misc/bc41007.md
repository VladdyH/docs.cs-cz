# <a name="attribute-39conditional39-is-only-valid-on-39sub39-declarations"></a>Atribut &#39;podmíněného&#39; je platné jenom pro &#39;Sub&#39; deklarace
<xref:System.Diagnostics.ConditionalAttribute> Byl použit atribut k deklaraci metody jiné než `Sub` postupu.  
  
 **ID chyby:** BC41007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte <xref:System.Diagnostics.ConditionalAttribute> z neplatná deklarace.
