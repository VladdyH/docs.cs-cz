# <a name="argument-basepath-must-be-a-path-to-a-folder"></a>Argument BasePath musí být cesta ke složce
Argument `BasePath` musí obsahovat cesta ke složce. Můžete být nesprávně analýze řetězec a poskytnutí hodnotu, která není rozpoznán jako platná cesta.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte hodnotu se poskytuje pro `BasePath` a ujistěte se, je platnou cestu ke složce.
