# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Chyba při vkládání Win32 manifest: &lt;manifest&gt;
Pro zadaný soubor `/win32manifest` – možnost kompilátoru nelze vložit. Může to být způsobeno tím, nedostatečná oprávnění k souboru manifestu.  
  
 **ID chyby:** BC31191  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že identita spuštěním příkazu kompilátoru jazyka Visual Basic má přístup k souboru manifestu Win32.
