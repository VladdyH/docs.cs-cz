# <a name="event-39lteventnamegt39-cannot-be-found"></a>Událost &#39; &lt;eventname&gt; &#39; nebyl nalezen
Pokusili jste se zpracovat událost, která kompilátor nelze najít.  
  
 **ID chyby:** BC30590  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda že je správně zadán název události.
