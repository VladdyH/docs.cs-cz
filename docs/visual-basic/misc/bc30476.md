# <a name="39withevents39-variables-cannot-be-typed-as-arrays"></a>& č. 39; WithEvents & č. 39; proměnné nemůže být typu pole
Pokusili jste se deklarovat pole pomocí `WithEvents`. Můžete deklarovat tolik jednotlivé proměnné, jako je jako `WithEvents`, ale pole nelze deklarovat tímto způsobem.  
  
 **ID chyby:** BC30476  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte proměnné jednotlivě.
