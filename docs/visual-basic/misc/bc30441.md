# <a name="39catch39-must-end-with-a-matching-39end-try39"></a>& č. 39; Catch & č. 39; musí končit odpovídající & č. 39; Zkuste end & č. 39;
A `Catch` příkazu se zobrazí ve vašem kódu bez odpovídající `End Try` příkaz. `Catch`příkazy musí být uzavřena s `End Try` příkaz.  
  
 **ID chyby:** BC30441  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Catch` příkaz.  
  
2.  Přidat `End Try` příkaz dokončete bloku.
