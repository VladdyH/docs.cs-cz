# <a name="bad-record-number"></a>Počet chybných záznamů
Počet záznamů v `a FileGet`, `FilePut`, `FileGetObject`, nebo `FilePutObject` příkaz je menší než nebo rovna nule.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte výpočty použité při vytváření číslo záznamu. Zkontrolujte pravopis proměnné obsahující číslo záznamů nebo při výpočtu záznamu čísla. Chybný název proměnné je implicitně deklarovaný a inicializovat na hodnotu nula, pokud jste použili `Option Explicit On` v modulu.
