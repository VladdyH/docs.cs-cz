# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Možnost explicitní&#39; může následovat pouze &#39;na&#39; nebo &#39;vypnuto&#39;
`Option Explicit` příkazy musíte zadat buď `On` nebo `Off`. Všechny ostatní hodnoty jsou neplatné.  
  
 **ID chyby:** BC30640  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `Option Explicit On` nebo `Option Explicit Off`.
