# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39; nelze použít s metodou &#39; &lt;methodname&gt; &#39; vzhledem k tomu, že je deklarovaná &#39;MustOverride&#39;
Pokusili jste se použít `MyBase` pomocí metody, která byla deklarována `MustOverride`.  
  
 **ID chyby:** BC30399  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `MustOverride` deklarace.
