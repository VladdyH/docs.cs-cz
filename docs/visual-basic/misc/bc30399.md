# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>& č. 39; MyBase & č. 39; nelze použít s metoda & č. 39; &lt;methodname&gt;& č. 39; protože je deklarovaná & č. 39; MustOverride & č. 39;
Pokusili jste se použít `MyBase` pomocí metody, která byla deklarována `MustOverride`.  
  
 **ID chyby:** BC30399  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `MustOverride` deklarace.
