# <a name="operators-must-be-declared-39shared39"></a>Operátory musí být deklarován &#39;sdílené&#39;
[Operator – příkaz](../../visual-basic/language-reference/statements/operator-statement.md) nezahrnuje [sdílené](../../visual-basic/language-reference/modifiers/shared.md) – klíčové slovo.  
  
 `Operator` Postup vyžaduje, i [veřejné](../../visual-basic/language-reference/modifiers/public.md) a `Shared` klíčová slova a operátora převodu také buď vyžaduje [Widening](../../visual-basic/language-reference/modifiers/widening.md) nebo [Narrowing](../../visual-basic/language-reference/modifiers/narrowing.md)– klíčové slovo.  
  
 **ID chyby:** BC33012  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Shared` – klíčové slovo chcete `Operator` příkaz.
