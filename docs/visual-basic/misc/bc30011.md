# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Vnitřní chyba kompilátoru: Generátor kódu přijal poškozený vstup
Visual Basic – kompilátor nemůže interpretovat úspěšně jeden z jeho interně napsané souborů.  
  
 **ID chyby:** BC30011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kompilace programu zjistíte, pokud se chyba objeví znovu.  
  
2.  Pokud se chyba objeví znovu, přeinstalujte kompilátoru jazyka Visual Basic.  
  
3.  Pokud chyba přetrvává i po přeinstalování, shromážděte informace o okolnostech a upozornit služby Microsoft Product Support Services.
