# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>Událost &#39; &lt;eventname&gt; &#39; nelze zpracovat, protože není přístupná z &#39; &lt;název&gt;&#39;
Pokusili jste se zpracovat událost, která není přístupná. Například pokud `Handles` je sdílené proměnné, metoda zpracování události je nutné taky sdílet.  
  
 **ID chyby:** BC30585  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že událost je dostupné.
