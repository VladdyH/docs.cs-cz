# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argument &#39;přístup&#39; není platný. Platné hodnoty pro vstup režimu jsou &#39;OpenAccess.Read&#39; a &#39;OpenAccess.Default&#39;
K přístupu hodnota není platná v `Input` režimu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte hodnotu na buď `OpenAccess.Read` nebo `OpenAccess.Default`.
