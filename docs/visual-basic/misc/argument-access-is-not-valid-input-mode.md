# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argument & č. 39; Přístup k & č. 39; není platný. Platné hodnoty pro vstup režimu jsou &#39;OpenAccess.Read & č. 39; a &#39;OpenAccess.Default & č. 39;
K přístupu hodnota není platná v `Input` režimu.

-   Změňte hodnotu na buď `OpenAccess.Read` nebo `OpenAccess.Default`.
