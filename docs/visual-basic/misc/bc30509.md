# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;typ&gt; nemůže Zdědit z &lt;specifier2&gt; &lt;typ&gt; protože rozšiřuje přístup základní &lt;typu&gt;
Pokusili jste se mají veřejnou třídu způsobem zděděn ze základní třídy označena `Private` nebo `Friend`.  
  
 **ID chyby:** BC30509  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Základní třídu deklarovat `Public` nebo dědičných třídu deklarovat `Private` nebo `Friend`.
