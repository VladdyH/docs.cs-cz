# <a name="only-one-of-39notoverridable39-39mustoverride39-or-39overridable39-can-be-specified"></a>Pouze jeden z &#39;NotOverridable&#39;, &#39;MustOverride&#39;, nebo &#39;Overridable&#39; lze zadat
Konfliktní přepsání specifikace dojít v deklarace elementu. Přepsání klíčová slova se vzájemně vylučují.  
  
 **ID chyby:** BC30177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vyberte jeden přepsání specifikátor a odeberte ostatní klíčová slova z deklarace.
