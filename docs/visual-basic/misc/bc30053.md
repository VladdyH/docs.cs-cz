# <a name="arrays-cannot-be-declared-with-39new39"></a><span data-ttu-id="75213-102">Pole nelze deklarovat s &#39;nový&#39;</span><span class="sxs-lookup"><span data-stu-id="75213-102">Arrays cannot be declared with &#39;New&#39;</span></span>
<span data-ttu-id="75213-103">`New` – Klíčové slovo může vyskytovat pouze v části inicializace deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="75213-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="75213-104">To znamená `New` musí být na pravé straně znaménkem rovnosti (`=`), může vytvořit nový typ pole přiřazení proměnné pole.</span><span class="sxs-lookup"><span data-stu-id="75213-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="75213-105">Zástupce pro inicializaci třídy není k dispozici pro pole.</span><span class="sxs-lookup"><span data-stu-id="75213-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="75213-106">Následující kód dva řádky jsou obě platný a jsou ekvivalentní, protože se inicializovat objekt ze třídy.</span><span class="sxs-lookup"><span data-stu-id="75213-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="75213-107">Inicializace pole však nelze použít stejný zástupce jako třída inicializace.</span><span class="sxs-lookup"><span data-stu-id="75213-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="75213-108">Všimněte si, že `New` klauzule pro pole musí obsahovat obě závorkách `()`a složené závorky `{}`.</span><span class="sxs-lookup"><span data-stu-id="75213-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="75213-109">Závorkách určíte, že je nový typ pole a zadejte hodnoty inicializace složené závorky.</span><span class="sxs-lookup"><span data-stu-id="75213-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="75213-110">Kompilátor vyžaduje složené závorky i v případě, že jsou prázdná, to znamená, i když nejsou inicializace pole hodnot.</span><span class="sxs-lookup"><span data-stu-id="75213-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="75213-111">**ID chyby:** BC30053</span><span class="sxs-lookup"><span data-stu-id="75213-111">**Error ID:** BC30053</span></span>
