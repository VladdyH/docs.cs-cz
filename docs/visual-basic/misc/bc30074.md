# <a name="constant-cannot-be-the-target-of-an-assignment"></a>Konstanta nemůže být cílem přiřazení
Konstanta nastane v kontextu, který přiřazuje hodnotu. Hodnoty, které jsou jim přiřazeny během provádění mohou obsahovat pouze zapisovatelné proměnné, vlastnosti a prvků pole.  
  
 **ID chyby:** BC30074  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte konstanta zapisovatelné samostatná proměnná, vlastnost nebo pole elementu.
