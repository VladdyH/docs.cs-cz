# <a name="interface-can-inherit-only-from-another-interface"></a>Rozhraní může dědit vlastnosti pouze z jiné rozhraní
Pokusili jste se dědí něco, co není rozhraní.  
  
 **ID chyby:** BC30354  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že základní rozhraní nebo rozhraní, ze které se pokoušíte dědění jsou ve skutečnosti rozhraní.
