# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Příkazy deklarace proměnné nejsou platná ve příkazové podokno
Deklarace proměnných nelze použít v **Immediate** okno.  
  
 **ID chyby:** BC30709  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Proměnné v nedeklarujte **Immediate** okno.
