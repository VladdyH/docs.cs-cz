# <a name="option-39ltoptiongt39-cannot-be-followed-by-3939-or-39-39-ignored"></a>možnost &#39; &lt;možnost&gt; &#39; nemůže následovat &#39;+&#39; nebo &#39;-&#39;; ignorováno
Možnost kompilátoru byl zadán s hodnotou `+` nebo `-`; tyto hodnoty nejsou podporovány.  
  
 Další informace o možnostech kompilátoru a hodnoty, které podporují najdete v tématu [jazyka Visual Basic možnosti kompilátoru uvedené abecedně](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **ID chyby:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Opravte tuto chybu a použijte možnost  
  
-   Zadejte možnost kompilátoru s žádnou hodnotu.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Opravte tuto chybu a nechcete použít možnost  
  
-   Remove – možnost kompilátoru.
