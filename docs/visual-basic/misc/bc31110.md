# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Pole použít jako argumenty atributu zapotřebí explicitně zadat hodnoty pro všechny elementy
Element s hodnotou neurčené je pole použít jako argument atributu.  
  
 **ID chyby:** BC31110  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte všechny hodnoty elementu.
