# <a name="39resume39-or-39goto39-expected"></a>&#39;Obnovit&#39; nebo &#39;GoTo&#39; očekávání
`On Error` Příkaz neobsahuje buď `Resume` klauzule nebo `GoTo` klauzule. Musí používat jednu klauzuli nebo dalších.  
  
 **ID chyby:** BC32019  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat buď `Resume` klauzule nebo `GoTo` klauzule `On Error` příkaz.
