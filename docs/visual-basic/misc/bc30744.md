# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>Odkazovaný prvek pole má hodnotu &#39;nic&#39;
Element pole se na ně odkazovat nebyl nastaven, nebo byla dříve nastavena na `Nothing`.  
  
 **ID chyby:** BC30744  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, jestli jsou se správně nastavené hodnoty v poli.
