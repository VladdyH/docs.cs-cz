# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>Není zadán argument pro parametr &#39; &lt;parametername&gt;&#39;
Argument pro požadovaný parametr nebyla zadána.  
  
 **ID chyby:** BC36569  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte argument pro zadaný parametr.
