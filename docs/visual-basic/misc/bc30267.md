# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39notoverridable39"></a>& č. 39; &lt;declaration1&gt;& č. 39; nelze přepsat & č. 39;&lt; declaration2&gt;& č. 39; protože je deklarovaná & č. 39; NotOverridable & č. 39;
Procedura nebo vlastnost deklarace pokusí přepsat zděděné element se stejným názvem, ale zděděné elementu je zadán jako `NotOverridable`.  
  
 **ID chyby:** BC30267  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `NotOverridable` – klíčové slovo z deklarace zděděné elementu nebo odeberte deklaraci přepsání.
