# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39notoverridable39"></a>&#39;&lt;declaration1&gt; &#39; nejde přepsat &#39; &lt;declaration2&gt; &#39; vzhledem k tomu, že je deklarovaná &#39;NotOverridable&#39;
Procedura nebo vlastnost deklarace pokusí přepsat zděděné element se stejným názvem, ale zděděné elementu je zadán jako `NotOverridable`.  
  
 **ID chyby:** BC30267  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `NotOverridable` – klíčové slovo z deklarace zděděné elementu nebo odeberte deklaraci přepsání.
