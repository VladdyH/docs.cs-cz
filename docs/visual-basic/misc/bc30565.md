# <a name="array-initializer-has-too-few-dimensions"></a>Inicializátor pole má příliš málo dimenze
Není zadáno dost rozměry v inicializátoru pro pole.  
  
 **ID chyby:** BC30565  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte vaše inicializátor pole k určení, kolik dimenzí, které jsou nezbytné.  
  
-   Zadejte chybí argument nebo argumenty.
