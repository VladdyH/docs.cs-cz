# <a name="inheriting-from-39systemltclassnamegt39-is-not-valid"></a>Dědění z &#39;systému. &lt;classname&gt; &#39; není platný
Třída nemůže Zdědit z `System.Array`, `System.Delegate`, `System.Enum`, nebo `System.ValueType`.  
  
 **ID chyby:** BC30015  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Inherits` příkaz nebo změňte ho na platné základní třídy dědí.
