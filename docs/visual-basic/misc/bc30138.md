# <a name="unable-to-create-temp-file-in-path-39ltfilenamegt39-lterror-messagegt"></a>Nelze vytvořit dočasný soubor v cestě & č. 39; &lt;filename&gt;& č. 39;: &lt;chybová zpráva&gt;
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilátoru volá Linker sestavení (Al.exe, také známé jako Alink) ke generování sestavení s manifestu. Linkeru ohlásilo chybu soubor vytvoření nebo zápis k prostředku v paměti k němu. To může být k potížím s konfigurací.  
  
 **ID chyby:** BC30138  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Prověřením uvozovkách chybové zprávy a podívejte se téma [Al.exe (Linker sestavení)](../../framework/tools/al-exe-assembly-linker.md) pro další vysvětlení a doporučení.  
  
2.  Pokud potíže potrvají, shromažďovat informace o okolnostech a upozornění služby Microsoft Product Support Services.
