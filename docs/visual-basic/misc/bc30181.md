# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Specifikátory platný pouze na začátku prohlášení
Klíčová slova usnadnění přístupu a další označení musí být před deklarace – klíčové slovo, například `Dim` nebo `Function`.  
  
 **ID chyby:** BC30181  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte všechny specifikátory mezi seznamu atributů (pokud existuje) a klíčové slovo deklarace.
