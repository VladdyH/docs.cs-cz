# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39;Vlastní&#39; modifikátor lze použít pouze bezprostředně před &#39;událostí&#39; deklarace
`Custom` – Klíčové slovo musí předcházet pouze `Event` – klíčové slovo v deklaraci vlastní události.  
  
 **ID chyby:** BC31140  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `Custom` – klíčové slovo na atributy stála před `Event` – klíčové slovo.  
  
     —nebo—  
  
-   Odeberte `Custom` – klíčové slovo.
