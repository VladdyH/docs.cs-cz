# <a name="3939-expected"></a>&#39;(&#39; očekávání
Seznam argumentů ve volání procedury nezačíná počáteční závorky (`(`) znaků.  
  
 **ID chyby:** BC30199  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat počáteční závorky znak na začátek seznamu.
