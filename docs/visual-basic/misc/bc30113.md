# <a name="39ltnamespacegt39-is-not-a-valid-name-and-cannot-be-used-as-the-root-namespace-name"></a>& č. 39; &lt;obor názvů&gt;& č. 39; není platný název a nelze použít jako název kořenového oboru názvů
Opravňující název oboru názvů není platný název elementu. Název musí obsahovat pouze znaky abecedy, číslice desítkové soustavy a podtržítka a musí začínat na znak abecedy nebo podtržítkem (`_`).  
  
 **ID chyby:** BC30113  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je obor názvů součástí sadu vnořené obory názvů, zkontrolujte, pokud některé vnější názvy obsahují neplatné znaky.
