# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39;Rozhraní&#39; musí končit odpovídající &#39;End rozhraní&#39;
`Interface` Příkaz probíhá, aniž by odpovídající `End Interface` příkaz. Je nutné použít `End Interface` příkaz k ukončení definice rozhraní.  
  
 **ID chyby:** BC30253  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda `Interface` členové jsou správně naformátován.  
  
2.  Přidat `End Interface` příkaz položky celkem Konec definice rozhraní.
