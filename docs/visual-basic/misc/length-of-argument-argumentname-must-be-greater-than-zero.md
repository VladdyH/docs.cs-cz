# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Délka argumentu & č. 39; &lt;název argumentu argumentname&gt;& č. 39; musí být větší než nula.
Argument má hodnotu, která je rovna nebo menší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte argument, který je větší než nula.
