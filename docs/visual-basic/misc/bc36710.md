# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-does-not-have-a-signature-compatible-with-delegate-39ltdelegatenamegt39"></a>Metody rozšíření &#39; &lt;methodname&gt; &#39; definované v &#39; &lt;modulename&gt; &#39; nemá kompatibilní s delegáta podpis &#39; &lt;vlastnost delegatename&gt;&#39;
Došlo k neshodě mezi signatur metodě rozšíření a delegáta, který se pokoušíte použít. `Delegate` Příkaz definuje parametry a návratové typy delegát třídy. Libovolná procedura s kompatibilní parametry, parametry a návratový typ může použít k vytvoření instance tohoto typu delegáta.
