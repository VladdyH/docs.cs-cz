# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Třída v rozhraní nelze deklarovat &#39; &lt;– specifikátor&gt;&#39;
Neplatný modifikátor byl zadán s třídou definována v rozhraní.  
  
 **ID chyby:** BC31070  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat modifikátory jako `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
