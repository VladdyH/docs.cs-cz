## <a name="example"></a><span data-ttu-id="bf0a9-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="bf0a9-110">Example</span></span>  
 <span data-ttu-id="bf0a9-111">Následující třídy obsahuje tři vlastnosti definice: jeden pro `TotalItems` a dva jsou pro `Item`, což je přetížena.</span><span class="sxs-lookup"><span data-stu-id="bf0a9-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="bf0a9-112">`TotalItems` Vlastnost vyžaduje bez argumentů a může být inicializována v seznamu inicializace objektu, jak je znázorněno v následující prohlášení.</span><span class="sxs-lookup"><span data-stu-id="bf0a9-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="bf0a9-113">`Item` Vlastnost je přetížena a každé přetížení vyžaduje argument.</span><span class="sxs-lookup"><span data-stu-id="bf0a9-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="bf0a9-114">Proto `Item` nemůže vyskytovat v seznamu inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="bf0a9-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="bf0a9-115">Chcete-li se vyhnout této chybě, inicializovat `Item` vlastnost mimo inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="bf0a9-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>

```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
