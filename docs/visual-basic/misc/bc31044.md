# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>Události deklarovat s & č. 39; Jako & č. 39; klauzule musí mít typ, delegát
Byl zadán typ, který není delegáta.  
  
 **ID chyby:** BC31044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `As` klauzule.  
  
2.  Zadejte delegáta s `As` klauzule.
