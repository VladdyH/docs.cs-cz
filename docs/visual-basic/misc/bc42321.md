# <a name="badly-formed-xml-file-39ltcommentfilegt39-cannot-be-included"></a>Chybně formátovaný XML. Soubor &#39; &lt;commentfile&gt; &#39; nelze zahrnout.
`<include>` Značky v jednom z komentáře XML odkazuje na soubor, který neexistuje, nebo soubor, který obsahuje chybně formátu XML.  
  
 **ID chyby:** BC42321  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda soubor existuje.  
  
2.  Ověřte, zda je platný kód XML v souboru.
