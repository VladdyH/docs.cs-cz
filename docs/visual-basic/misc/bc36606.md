# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-39object39-class"></a>Název proměnné rozsahu nesmí shodovat s názvem členem &#39;objekt&#39; – třída
Proměnná rozsahu v dotazu LINQ odpovídá názvu členem <xref:System.Object> třídy. Tím se vytvoří konflikt s objekty vytvořené pomocí jazyka Visual Basic pro dotaz LINQ.  
  
 **ID chyby:** BC36606  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zvolte nový název pro proměnnou rozsahu, která neodpovídá názvu libovolný člen <xref:System.Object> třídy.
