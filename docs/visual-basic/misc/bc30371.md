# <a name="module-39ltmodulenamegt39-cannot-be-used-as-a-type"></a>Modul &#39; &lt;modulename&gt; &#39; nelze použít jako typ
Pokusili jste se použít jako typ modulu. Modul nelze použít s výjimkou jako `Get` typu.  
  
 **ID chyby:** BC30371  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Modul nahraďte něco, co lze použít jako platného typu.
