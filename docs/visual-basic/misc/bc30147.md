# <a name="39return39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Vrátí & č. 39; příkazy nejsou platné v příkazové podokno
`Return` Příkaz provede vytvoření větve a není povolen v kontextu ladění.  
  
 **ID chyby:** BC30147  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nevydávají `Return` příkaz v **Immediate** okno.
