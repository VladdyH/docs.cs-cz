# <a name="39if39-operator-requires-either-two-or-three-operands"></a>&#39;Pokud&#39; operátor vyžaduje dva nebo tři operandy
`If` Operátor přijímá pouze dvě nebo tři argumenty.  
  
 **ID chyby:** BC33104  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte `If` výraz, který se podívat, jestli číslo operátorů lze snížit nebo rozšířit na dvě nebo tři argumenty.  
  
-   Zvažte použití jiného podmíněné konstrukce, jako například `If...Then...Else` příkaz nebo vnořený `If...Then...Else` příkaz k uspořádání argumenty.
