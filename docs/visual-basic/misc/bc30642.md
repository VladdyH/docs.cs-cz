# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39;Volitelné&#39; a &#39;ParamArray&#39; nelze kombinovat
`Optional` a `ParamArray` modifikátory se vzájemně vylučují.  
  
 **ID chyby:** BC30642  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `ParamArray` nebo `Optional`, ale ne obojí.
