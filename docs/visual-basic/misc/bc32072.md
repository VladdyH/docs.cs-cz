# <a name="cannot-implement-interface-39ltinterfacename1gt39-because-its-implementation-could-conflict-with-the-implementation-of-another-implemented-interface-39ltinterfacename2gt39-for-some-type-arguments"></a><span data-ttu-id="4a7fa-102">Nelze implementovat rozhraní &#39; &lt;interfacename1&gt; &#39; vzhledem k tomu, že jeho implementace může dojít ke konfliktu s prováděním jiné implementovaných rozhraní &#39; &lt;interfacename2&gt; &#39; pro některé argumenty typů</span><span class="sxs-lookup"><span data-stu-id="4a7fa-102">Cannot implement interface &#39;&lt;interfacename1&gt;&#39; because its implementation could conflict with the implementation of another implemented interface &#39;&lt;interfacename2&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="4a7fa-103">Zahrnuje deklaraci třídy `Implements` příkaz, který určuje dvě nebo více rozhraní, ale alespoň jednomu z rozhraní je obecná a dva implementací může dojít ke konfliktu určité hodnoty typu argumentů.</span><span class="sxs-lookup"><span data-stu-id="4a7fa-103">A class declaration includes an `Implements` statement that specifies two or more interfaces, but at least one of the interfaces is generic and two of the implementations could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="4a7fa-104">Tato chyba může generovat následující příkazy.</span><span class="sxs-lookup"><span data-stu-id="4a7fa-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1  
    Sub testSub(ByVal arg As String)  
End Interface  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface  
Public Class testClass  
    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="4a7fa-105">Protože `iFace2` je vytvořený pomocí `String`, `testClass` musí implementovat dvě verze `testSub` s identickými podpisy.</span><span class="sxs-lookup"><span data-stu-id="4a7fa-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="4a7fa-106">Díky tomu byste mohli vytvořit nejednoznačnosti, kterou verzi má přístup.</span><span class="sxs-lookup"><span data-stu-id="4a7fa-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="4a7fa-107">**ID chyby:** BC32072</span><span class="sxs-lookup"><span data-stu-id="4a7fa-107">**Error ID:** BC32072</span></span>
