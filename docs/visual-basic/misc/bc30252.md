# <a name="39end-interface39-must-be-preceded-by-a-matching-39interface39"></a>&#39;Ukončení rozhraní&#39; musí předcházet odpovídající &#39;rozhraní&#39;
`End Interface` Příkaz probíhá, aniž by odpovídající `Interface` příkaz. `End Interface` musí předcházet odpovídající `Interface` příkaz.  
  
 **ID chyby:** BC30252  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, pokud předchozím `Interface` příkaz je chybný nebo jinak neplatných.  
  
2.  Ujistěte se, že `Interface` členové jsou správně naformátován.
