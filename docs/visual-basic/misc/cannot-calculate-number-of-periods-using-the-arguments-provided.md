# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>Nemůže vypočítat počet období pomocí zadaných argumentů
Volání `NPer` funkce neobsahuje všechny požadované argumenty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že `Rate`, `Prnt` a `PV` hodnoty jsou zahrnuty ve volání funkce.
