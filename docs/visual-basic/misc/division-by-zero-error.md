# <a name="division-by-zero-visual-basic-error"></a>Chyba dělení nulou (Visual Basic)
Hodnota výrazu použité jako dělitel je nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda proměnné ve výrazu. Proměnnou překlepu implicitně vytvořit číselný proměnnou inicializovat na nulu.  
  
2.  Zkontrolujte předchozí operace na proměnné ve výrazu, především těch, které jsou předány do procesu jako argumenty z další postupy.
