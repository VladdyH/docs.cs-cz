# <a name="argument-ltargumentnamegt-must-be-greater-than-or-equal-to-zero"></a>Argument &lt;název argumentu argumentname&gt; musí být větší než nebo rovna hodnotě nula.
Hodnota argumentu menší než nula. Argument hodnoty musí být větší než nebo rovna hodnotě nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zajistěte, aby byl argument na hodnotu větší než nebo rovna hodnotě nula.  
  
2.  Pokud je argument zadaný výpočtem, zkontrolujte hodnoty zpracovávaných výpočtu.
