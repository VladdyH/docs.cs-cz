# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Výraz není pole nebo metoda a nemůže mít seznam argumentů
Seznam argumentů jste zadali pro výraz, který není pole nebo metoda. Seznamy argumentů může mít pouze pole nebo metody.  
  
 **ID chyby:** BC30471  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte seznam argumentů.
