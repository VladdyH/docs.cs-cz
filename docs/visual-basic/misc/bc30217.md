# <a name="string-constant-expected"></a>Byla očekávána řetězcová konstanta
Znak než dvojité uvozovky (") dojde, pokud kontext nevyžaduje řetězcová konstanta. Možnou příčinou je chybějící z řetězce v uvozovkách.  
  
 **ID chyby:** BC30217  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že je řetězec správně uzavřena v uvozovkách.  
  
2.  Zadejte řetězcová konstanta.
