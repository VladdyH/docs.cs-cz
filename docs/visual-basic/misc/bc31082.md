# <a name="39ltvariablenamegt39-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-39catch39-variable"></a>& č. 39; &lt;NázevProměnné&gt;& č. 39; není místní proměnné nebo parametru a proto jej nelze použít jako & č. 39; Catch & č. 39; Proměnná
Proměnné v `Try...Catch...Finally` příkazy musí být deklarován místně, nebo být parametry pro aktuální procedury.  
  
 **ID chyby:** BC31082  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarovat místní proměnné nebo parametru pro `Try...Catch...Finally` příkazy.
