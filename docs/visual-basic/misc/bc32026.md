# <a name="specifiers-and-attributes-are-not-valid-on-39namespace39-statements"></a>Specifikátory a atributy nejsou platné v & č. 39; Namespace & č. 39; příkazy
Obory názvů se používají pro hierarchická organizace program bloky a programovací elementy. Jejich nepřebírají žádné modifikátory nebo atributy.  
  
 **ID chyby:** BC32026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny klíčová slova a atribut bloků z `Namespace` příkaz.
