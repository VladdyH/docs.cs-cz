# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="2ed34-102">&#39;Pokud&#39; operandy nemůže mít název argumenty</span><span class="sxs-lookup"><span data-stu-id="2ed34-102">&#39;If&#39; operands cannot be named arguments</span></span>
<span data-ttu-id="2ed34-103">Použití s názvem argumentů operandy `If` operátor není platný.</span><span class="sxs-lookup"><span data-stu-id="2ed34-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="2ed34-104">Následující příklad způsobí, že tuto chybu:</span><span class="sxs-lookup"><span data-stu-id="2ed34-104">The following example causes this error:</span></span>  
  
```

Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="2ed34-105">Tím se liší od `IIf` funkce, které umožňují pojmenované argumenty, jak je znázorněno v následujícím kódu:</span><span class="sxs-lookup"><span data-stu-id="2ed34-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```

```
