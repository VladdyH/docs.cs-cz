# <a name="specifier-is-duplicated"></a>Specifikátor je duplicitní.
Klíčové slovo vyskytuje více než jednou ve stejné deklaraci.  
  
 **ID chyby:** BC30178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní vzhledy klíčové slovo z deklarace.
