# <a name="static-local-variable-39ltvariablenamegt39-is-already-declared"></a>Statické místní proměnné &#39; &lt;NázevProměnné&gt; &#39; je již deklarována
Jiné statické místní proměnné se stejným názvem již byl deklarován.  
  
 **ID chyby:** BC31401  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte redundantní statické místní deklarace.  
  
2.  Zadejte jedinečný název každé statické místní proměnné.
