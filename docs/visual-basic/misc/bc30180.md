# <a name="keyword-does-not-name-a-type"></a>Klíčové slovo neurčuje typu
Klíčové slovo se zobrazí, kdy je potřeba název typu. Kontext žádá základní datový typ nebo název deklarovaný element představující typu.  
  
 **ID chyby:** BC30180  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Klíčové slovo nahraďte název platného typu.
