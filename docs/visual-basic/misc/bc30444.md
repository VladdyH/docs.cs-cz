# <a name="39set39-is-already-declared"></a>&#39;Nastavit&#39; je již deklarována
A `Set` příkaz vyskytuje více než jednou v bloku kódu. `Set` Příkaz deklaruje `Set` procedura property sloužící k přidělování hodnotu vlastnosti.  
  
 **ID chyby:** BC30444  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `Set` příkaz.
