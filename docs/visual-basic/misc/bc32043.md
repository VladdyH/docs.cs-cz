# <a name="too-many-type-arguments-to-39ltgenericmethodnamegt39"></a>Příliš mnoho argumentů typu & č. 39; &lt;genericMethodName&gt;& č. 39;
Obecná metoda metoda byla volána s další argumenty typu, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC32043  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Argumenty typu odeberte ze seznamu argument typu, takže je jedna pro každý parametr typu na obecná metoda, kterou vyvoláváte.
