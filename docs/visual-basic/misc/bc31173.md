# <a name="missing-required-white-space"></a>Chybí požadované prázdné znaky
Literál XML byla zadána bez požadovaný oddělovací mezery. Je například prostoru požadovaný oddělovací mezery mezi dalších atributů v literál XML elementu.  
  
 **ID chyby:** BC31173  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte oddělovač požadované místo.
