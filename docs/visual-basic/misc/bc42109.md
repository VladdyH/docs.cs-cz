# <a name="variable-39ltvariablenamegt39-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>Proměnné &#39; &lt;NázevProměnné&gt; &#39; slouží předtím, než byl přiřazen hodnotu (Chyba jazyka Visual Basic)
Proměnná se používá, než byl přiřazen hodnotu, což je <xref:System.NullReferenceException>.  
  
 **ID chyby:** BC42109  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že proměnná nemá přiřazenou hodnotu.
