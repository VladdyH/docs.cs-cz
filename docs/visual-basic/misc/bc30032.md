# <a name="events-cannot-have-a-return-type"></a>Události nemůže mít typ vrácené hodnoty.
Události mohou přijímat argumenty, ale nemůže vrátit přímo hodnoty.  
  
 **ID chyby:** BC30032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte z návratový typ `Event` příkaz.
