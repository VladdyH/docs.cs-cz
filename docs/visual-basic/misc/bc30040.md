# <a name="first-statement-of-a-method-body-cannot-be-on-the-same-line-as-the-method-declaration"></a>První příkaz těla metody nemůže být na stejném řádku jako deklaraci – metoda
A `Function`, `Sub`, `Get`, `Set`, nebo `Property` příkaz musí být samostatně na řádku zdrojového kódu.  
  
 **ID chyby:** BC30040  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte všechny popisek čáry předchází deklaraci postupu.  
  
2.  Přesuňte všechny příkaz předchází deklaraci postup předchozí řádek zdrojového kódu.  
  
3.  Přesuňte všechny příkaz následující postup deklaraci do řádku následné zdrojového kódu.
