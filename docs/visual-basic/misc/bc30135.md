# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39;SyncLock –&#39; příkazy nejsou platné v příkazové podokno
`SyncLock` Příkaz synchronizuje vláken a není povolen v kontextu ladění.  
  
 **ID chyby:** BC30135  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nevydávají `SyncLock` příkaz v **Immediate** okno.
