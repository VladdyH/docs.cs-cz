# <a name="attribute-value-is-not-valid-expecting-39value139-or-39value239"></a>Hodnota atributu není platná. byla očekávána & č. 39; value1 & č. 39; nebo & č. 39; value2 & č. 39;
Byla zadána neplatná hodnota atributu v literál dokumentu XML. Atribut platné hodnoty pro literál dokumentu XML se podle specifikace XML 1.0.  
  
 **ID chyby:** BC31182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte hodnotu atributu na hodnotu, která je platná podle specifikace XML 1.0.
