# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Catch –&#39; příkazy nejsou platné v příkazové podokno
`Catch` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30715  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Catch` příkazy z ladění kódu.
