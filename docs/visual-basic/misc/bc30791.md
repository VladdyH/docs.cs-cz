# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39;Deklarovat&#39; příkazy do struktury nelze deklarovat &#39; &lt;– specifikátor&gt;&#39;
A `Declare` deklarace má specifikátor, který není platný v rámci `Structure` deklarace.  
  
 **ID chyby:** BC30791  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
