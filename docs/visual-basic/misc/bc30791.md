# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>& č. 39; deklarovat & č. 39; příkazy do struktury nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
A `Declare` deklarace má specifikátor, který není platný v rámci `Structure` deklarace.  
  
 **ID chyby:** BC30791  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
