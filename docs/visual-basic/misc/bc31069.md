# <a name="enum-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Výčet v rozhraní nelze deklarovat &#39; &lt;– specifikátor&gt;&#39;
Je zadaná neplatná modifikátor se výčet definována v rozhraní.  
  
 **ID chyby:** BC31069  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat modifikátory jako `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
