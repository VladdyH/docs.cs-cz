# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>& č. 39; Sub – nové & č. 39; nelze deklarovat & č. 39; &lt;modifikátor&gt;& č. 39;
Pokusili jste se deklarovat `Sub New` pomocí modifikátoru, jako například `Overrides`, který není platný.  
  
 **ID chyby:** BC30364  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte modifikátor.
