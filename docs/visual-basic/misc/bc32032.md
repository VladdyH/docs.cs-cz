# <a name="39end-function39-must-be-the-first-statement-on-a-line"></a>& č. 39; Funkce end & č. 39; musí být první příkaz na řádek
`End Function` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End Function`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32032  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
