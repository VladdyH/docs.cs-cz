# <a name="white-space-may-not-appear-here"></a>Prázdné znaky se nemůže nacházet v tomto poli
Mezeru nebo mezery, je obsažen v názvu literál XML elementu.  
  
 **ID chyby:** BC31177  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrání název literál XML elementu místa nebo mezery.
