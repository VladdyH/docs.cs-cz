# <a name="device-unavailable"></a>Zařízení není k dispozici
Buď došlo ke ztrátě připojení k síti, nebo zařízení, které se pokoušíte získat přístup, je offline nebo neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kontrola výkonu k zařízení a kabely připojení počítače k zařízení. Pokud se pokoušíte získat přístup k tiskárně přes síť, ujistěte se, že existuje logický připojení mezi vaším počítačem a tiskárny, třeba připojení LPT1 přidružením ID sítě tiskárny.  
  
2.  Připojení k síti a akci opakujte.
