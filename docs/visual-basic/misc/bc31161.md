# <a name="expected-closing-39--gt39-for-xml-comment"></a>Očekávaný ukončovací & č. 39;--&gt;& č. 39; pro komentáře XML
Literál komentáře jazyka XML chybí požadované uzavírací "-->" znaky.  
  
 **ID chyby:** BC31161  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte požadované uzavírací "-->" znaky konec literál komentáře XML.
