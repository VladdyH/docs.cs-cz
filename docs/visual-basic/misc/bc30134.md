# <a name="labels-are-not-valid-in-the-immediate-window"></a>Popisky nejsou platné v příkazové podokno
Označení řádku povolte vytvoření větve a nejsou povoleny v kontextu ladění.  
  
 **ID chyby:** BC30134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nezadávejte název řádku v **Immediate** okno.
