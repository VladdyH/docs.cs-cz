# <a name="warning-treated-as-error-lterrorgt"></a>Upozornění vyhodnocena jako chyba: &lt;chyba&gt;
Při spuštění sestavení kompilovat s došlo upozornění **považovat upozornění jako chyby** zapnuta možnost.  
  
 **ID chyby:** BC31072  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Opravte upozornění.  
  
2.  Kompilace sestavení s **považovat upozornění jako chyby** možnost vypnuta.
