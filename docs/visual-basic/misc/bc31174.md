# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>Zpracování instrukcí název & č. 39; název & č. 39; XML není platný

 **ID chyby:** BC31174  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Opravte všechny chyby názvu instrukcí pro zpracování XML, nebo ji odeberte.
