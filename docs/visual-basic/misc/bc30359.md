# <a name="39default39-can-be-applied-to-only-one-property-name-in-a-lttypegt"></a>& č. 39; výchozí & č. 39; lze použít pouze jednu vlastnost název v &lt;typu&gt;
Pokusili jste se použít `Default` více než jeden název vlastnosti v rámci třídy, rozhraní nebo struktura.  
  
 **ID chyby:** BC30359  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte všechny kromě jednoho `Default` deklarace.
