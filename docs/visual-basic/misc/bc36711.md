# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>Argument odpovídající parametru &#39; &lt;parametername&gt; &#39; zmenší tak, aby &#39; &lt;typename&gt;&#39;
Byl proveden volání přetížené metody, ale kompilátor nebyla nalezena metoda, kterou lze volat bez zužující převod. A *zužující převod* změní hodnotu na datový typ, který nemusí obsahovat některé z možných hodnot.  
  
 **ID chyby:** BC36711
