# <a name="properties-cannot-be-declared-39ltmodifiernamegt39"></a>Vlastnosti nelze deklarovat &#39; &lt;modifiername&gt;&#39;
Dojde k chybě v syntaxi používány `Property` deklarace. Tato chyba obvykle označuje, že zadané klíčové slovo, jako například `WithEvents`, nelze použít v `Property` deklarace.  
  
 **ID chyby:** BC30639  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte syntaxi `Property` deklarace.
