# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>&#39;Z&#39; při zadání argumentů typu pro obecný typ nebo metoda vyžaduje
Příkaz se pokusí vytvořit typ od obecného typu nebo proveďte volání obecné metody, bez použití [z](../../visual-basic/language-reference/statements/of-clause.md) klauzule.  
  
 Syntaxe jazyka Visual Basic – obecné typy volání pro typ parametry a argumenty typu nutno zavést `Of` – klíčové slovo. Kromě toho seznam parametrů typu nebo typ seznam argumentů musí být uzavřena v uvozovkách.  
  
 **ID chyby:** BC32093  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrnout `Of` – klíčové slovo na začátku argument typu seznamu a uzavřete celý seznam v uvozovkách.
