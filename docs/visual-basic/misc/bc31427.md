# <a name="syntax-error-in-conditional-compilation-expression"></a>Chyba syntaxe ve výrazu Podmíněná kompilace
Podmíněná kompilace výraz generuje nespecifikované syntaktickou chybu.  
  
 **ID chyby:** BC31427  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte dokumentaci Podmíněná kompilace a klíčová slova použít ve výrazu. Porovnejte ukázkový kód v dokumentaci k proti řádku zdroje generování této chyby.  
  
2.  Pokud nelze určit příčinu chyby, shromažďovat informace o okolnostech a upozornění služby Microsoft Product Support Services.
