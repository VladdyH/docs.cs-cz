# <a name="could-not-find-library-39ltlibrarynamegt39"></a>Nebylo možné najít knihovnu &#39; &lt;NázevKnihovny&gt;&#39;
Zadaná knihovna nebyla nalezena.  
  
 Tomuto problému může dojít při kompilaci z příkazového řádku a použití odpověď (.rsp) souboru, který odkazuje na mnoho sestavení  
  
 **ID chyby:** BC2017  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte název knihovny a ujistěte se, zda že jsou správná.
