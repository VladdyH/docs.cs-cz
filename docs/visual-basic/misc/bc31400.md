# <a name="local-variables-within-methods-of-structures-cannot-be-declared-39static39"></a>Lokální proměnné v rámci metod struktury nelze deklarovat & č. 39; Statické & č. 39;
`Static` Modifikátor nelze použít s místní proměnné v struktury.  
  
 **ID chyby:** BC31400  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Static` modifikátor z místní proměnné.  
  
2.  Deklarujte proměnnou jako statické proměnná s širší oboru.
