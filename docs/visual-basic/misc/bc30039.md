# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>Smyčky řídicí proměnná nemůže být vlastnost nebo pole indexované pozdní vazba
Proměnná použít k iteraci v rámci `For` smyčky musí být číselným datovým typem.  
  
 **ID chyby:** BC30039  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte deklaraci řídicí proměnná smyčky k určení `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, nebo `Decimal`.
