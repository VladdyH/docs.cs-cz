# <a name="expected-closing-39gt39-for-xml-tag"></a>Očekávaný ukončovací & č. 39; &gt;& č. 39; pro – značka XML
Značka XML pro literál XML chybí požadované ukončovací ' >' znak.  
  
 **ID chyby:** BC31166  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidání ukončovací ' >' znak na konci značky XML pro literál XML.
