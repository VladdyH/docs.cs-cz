# <a name="39ltspecifiergt39-is-not-valid-on-a-constant-declaration"></a>& č. 39; &lt;specifikátor&gt;& č. 39; není platná deklarace konstanty
A `Const` příkaz obsahuje neplatné klíčové slovo. A `Const` výraz může obsahovat pouze `Friend`, `Private`, `Protected`, `Public`, nebo `Shadows` klíčová slova.  
  
 **ID chyby:** BC30233  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, jestli je správně zadané klíčové slovo.  
  
2.  Odeberte neplatné klíčové slovo z `Const` příkaz.
