# <a name="39raiseevent39-is-already-declared"></a>& č. 39; RaiseEvent – & č. 39; je již deklarována
Více než jeden `RaiseEvent` deklarace zobrazovaly v deklaraci vlastní události. A `RaiseEvent` deklarace deklaruje procedury používají k vyvolání události.  
  
 **ID chyby:** BC31129  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `RaiseEvent` příkaz.
