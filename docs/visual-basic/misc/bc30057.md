# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>Příliš mnoho argumentů pro & č. 39; &lt;procedurename&gt;& č. 39;

 **ID chyby:** BC30057  
  
## <a name="to-correct-this-error"></a>Oprava této chyby
