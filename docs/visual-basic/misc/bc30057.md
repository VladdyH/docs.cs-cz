# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>Příliš mnoho argumentů pro &#39; &lt;procedurename&gt;&#39;
Počet argumentů zadaný ve volání procedury překračuje počet argumentů definované pro proceduru.  
  
 **ID chyby:** BC30057  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Projděte si definice procedury a odebrat další argumenty z volání.
