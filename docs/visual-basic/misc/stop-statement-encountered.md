# <a name="stop-statement-encountered"></a>Byl zjištěn příkaz Stop
A `Stop` příkaz, který pozastaví spuštění, došlo.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Stop` příkaz.
