# <a name="39ltmodifiergt39-is-not-valid-on-a-declare"></a>&#39;&lt;Modifikátor&gt; &#39; není platný v příkazu Declare
A `Declare` příkaz obsahuje neplatný klíčové slovo, jako například `ReadOnly`.  
  
 **ID chyby:** BC30244  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Declare` příkaz.
