# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent –&#39; deklarace musí končit odpovídající &#39;RaiseEvent – End&#39;
A `RaiseEvent` musí končit deklarace `End RaiseEvent` příkaz.  
  
 **ID chyby:** BC31117  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda `RaiseEvent` deklarace končí `End RaiseEvent` příkaz.
