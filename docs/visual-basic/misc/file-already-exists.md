# <a name="file-already-exists"></a>Soubor již existuje.
K této chybě dojde v důsledku pojmenování nový soubor s názvem existujícího souboru nebo pomocí **uložit jako** příkaz pro uložení aktuálně načtených projektu s názvem, který již existuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte název a buď ho změnit na takový, který není v konfliktu, nebo odstraňte existující soubor nebo projektu, pokud již není potřeba.
