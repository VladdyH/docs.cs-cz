# <a name="array-initializer-has-too-many-dimensions"></a>Inicializátor pole má příliš mnoho dimenzí
Jste zadali příliš mnoho rozměry v inicializátoru pro pole.  
  
 **ID chyby:** BC30566  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte vaše inicializátor pole k určení, kolik dimenzí, které jsou nezbytné.  
  
-   Odeberte nadbytečné argument nebo argumenty.
