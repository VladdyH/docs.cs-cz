# <a name="labels-are-not-valid-outside-methodsmultiline-lambdas"></a>Štítky jsou neplatné mimo metody nebo víceřádkových výrazů lambda
Popisek prohlášení můžete přidat pouze uvnitř `Sub`, `Function`, vlastnost `Get`, nebo vlastnost `Set` postupu. Jenom příkazu spustitelný soubor může mít štítek a všechny spustitelné příkazy musí být uvnitř procedury.  
  
 **ID chyby:** BC30016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte popisek z příkazu nebo přesunout příkaz uvnitř procedury.
