# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Struktury v rozhraní nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Neplatný modifikátor byla zadána se strukturou definována v rozhraní.  
  
 **ID chyby:** BC31071  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat modifikátory jako `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, nebo `Overridable`.
