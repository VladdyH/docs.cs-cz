# <a name="line-ltnumbergt-cannot-be-parsed-using-the-current-delimiters"></a>Řádek &lt;číslo&gt; nelze analyzovat pomocí aktuální oddělovače
Zadaný řádek se nezdařila, protože používá oddělovače než tyto zadané.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Upravit `Delimiters` tak, aby řádek správně analyzovat nebo vložení kódu zpracování výjimek ke zpracování na řádku.
