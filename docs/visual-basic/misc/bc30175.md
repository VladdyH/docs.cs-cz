# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-declared-in-39ltnamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;Type1&gt; &#39; &lt;typename&gt; &#39; a &lt;type2&gt; &#39; &lt;typename&gt;&#39;, deklarované v &#39; &lt;název&gt; &#39;, konfliktu v &lt;obor názvů&gt; &#39; &lt;parametr namespacename&gt;&#39;
Dva deklarované elementy, jako jsou třídy a modul, mají stejný název v stejného oboru názvů.  
  
 **ID chyby:** BC30175  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte deklaraci jednoho prvku na jiný obor názvů.  
  
-   Změňte název jednoho prvku.
