# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Importy alias &#39; &lt;název&gt; &#39; je v konfliktu s &#39; &lt;name2&gt; &#39; deklarované v kořenového oboru názvů
Název alias použitý v `Imports` příkaz je v konfliktu s jiným názvem deklarované v kořenového oboru názvů.  
  
 **ID chyby:** BC31403  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Vyberte jedinečný název pro alias používané `Imports` příkaz.
