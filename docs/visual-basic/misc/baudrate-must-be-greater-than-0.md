# <a name="baudrate-must-be-greater-than-0"></a>Přenosová rychlost musí být větší než 0.
`BaudRate` Zadaný pro argument `My.Computer.Ports.OpenSerialPort` metoda musí být větší než nula.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změňte hodnotu `BaudRate` argument kladné číslo.
