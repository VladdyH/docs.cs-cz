# <a name="definition-of-method-ltmethodgt-is-not-accessible-in-this-context"></a>Definice metody &lt;metoda&gt; není dostupný v tomto kontextu
Byl součástí agregační funkci, která neexistuje `Into` operátor.  
  
 **ID chyby:** BC36594  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte správnost názvu agregační funkci a provést potřebné změny.  
  
-   Přidejte metodu jako vlastní agregační funkci (metoda rozšíření). Další informace o vytváření vlastní agregační funkce najdete v tématu [Aggregate – klauzule](../../visual-basic/language-reference/queries/aggregate-clause.md).
