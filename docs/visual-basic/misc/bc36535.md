## <a name="to-correct-this-error"></a><span data-ttu-id="74a22-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="74a22-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="74a22-107">Vytvořit kopii instanci členu nebo hodnoty vrácené odkaz na `Me` a použít kopii ve výrazu dotazu, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="74a22-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb

    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv
```  
