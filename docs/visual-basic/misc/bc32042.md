# <a name="too-few-type-arguments-to-39ltgenericmethodnamegt39"></a>Příliš málo argumentů typu &#39; &lt;genericMethodName&gt;&#39;
Obecná metoda metoda byla volána s méně argumenty typu, než je počet parametrů typu.  
  
 Při vyvolání obecné metody, je nutné zadat jeden argument typu pro každý parametr typu definované dané metody.  
  
 **ID chyby:** BC32042  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Argumenty typu přidáte do seznamu argument typu, aby je jedna pro každý parametr typu na obecná metoda, kterou vyvoláváte.
