# <a name="paramarray-parameters-must-be-declared-39byval39"></a>ParamArray parametry musí být deklarován &#39;ByVal&#39;
`ParamArray` Parametry nelze použít `ByRef` modifikátor.  
  
 **ID chyby:** BC30667  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat `ParamArray` parametry s využitím `ByVal` modifikátor.
