# <a name="exponent-is-not-valid"></a>Exponent není platný
Pokusili jste se použít exponent, který není platný s literál s plovoucí desetinnou čárkou. Tato chyba obvykle nastává kvůli s plovoucí desetinnou čárkou literál typ je v rozporu s velikost požadované pro uchovávání literál s plovoucí desetinnou čárkou.  
  
 **ID chyby:** BC30495  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte typ s plovoucí desetinnou čárkou literál a změňte ji na platný typ.
