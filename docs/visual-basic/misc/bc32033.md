# <a name="39end-get39-must-be-the-first-statement-on-a-line"></a>&#39;Ukončení Get&#39; musí být první příkaz na řádek
`End Get` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End Get` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32033  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
