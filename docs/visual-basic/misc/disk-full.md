# <a name="disk-full-visual-basic"></a>Disk plný (Visual Basic)
Není dostatek místa na disku pro dokončení buď `Print`, `Write`, nebo `FileClose` operaci, nebo k vytvoření požadované soubory.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunutí některých souborů na jiný disk nebo odstranit některé soubory vytvořte více místa na disku.
