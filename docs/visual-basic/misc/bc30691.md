# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;TypeName&gt; &#39; je typu v &#39; &lt;containername&gt; &#39; a nedá se použít jako výraz
Jeden ze zadaných názvů používá je již deklarována jako název typu a nelze použít ve výrazu.  
  
 **ID chyby:** BC30691  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jednoznačně název člena.  
  
-   Použijte hranaté závorky kolem název člena.
