# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Inicializátory anonymního typu nejsou platné ve windows ladění
Inicializátoru anonymní typ byl zadán v **sledovat** nebo **Immediate** okno během relace ladění. Inicializátory anonymního typu nejsou k dispozici při ladění systému windows.  
  
 **ID chyby:** BC36577  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Otestujte anonymního typu inicializátory v editoru kódu.
