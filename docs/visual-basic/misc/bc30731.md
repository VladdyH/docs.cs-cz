# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39;AddressOf&#39; výrazy nejsou platné ve windows ladění
`AddressOf` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30731  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat `AddressOf` příkazy z ladění kódu.
