# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Parametr komentáře XML & č. 39; &lt;parametr&gt;& č. 39; neodpovídá parametr na odpovídajícím &lt;– klíčové slovo&gt; – příkaz
Parametr komentáře XML \<parametr > neodpovídá parametr na odpovídajícím \<– klíčové slovo > příkaz. Komentáře XML budou ignorovány.  
  
 Značky pro komentáře XML byl zadán pro některé, ale ne všechny parametry v metodu.  
  
 **ID chyby:** BC42307  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte komentář značka XML pro všechny parametry v metodě.
