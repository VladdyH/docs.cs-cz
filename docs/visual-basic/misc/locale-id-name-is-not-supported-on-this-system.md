# <a name="locale-id-39ltnamegt39-is-not-supported-on-this-system"></a>ID národního prostředí &#39; &lt;název&gt; &#39; není v tomto systému podporován.
Pokoušíte se použít `VbStrConv`, ale váš systém nepodporuje zadaný národní prostředí.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Upgradujte počítač na podporu národní prostředí a spuštění aplikace na počítači, který podporuje zadaný národní prostředí.
