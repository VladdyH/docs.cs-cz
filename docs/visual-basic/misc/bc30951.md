# <a name="wrong-number-of-type-arguments-passed-to-method-39ltgenericprocedurenamegt39"></a>Nesprávný počet argumentů typu předaný metoda & č. 39; &lt;genericprocedurename&gt;& č. 39;
Obecný postup se nazývá s číslem argumenty typu, který neodpovídá počtu parametrů typu, s nimiž je definována.  
  
 **ID chyby:** BC30951  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte typ argumentu pro každý parametr typu definované pro obecný postup.  
  
     -nebo-  
  
-   Volání obecné procedury bez argumentů. typ vůbec a umožní kompilátoru pokus o odvození argumenty typu.
