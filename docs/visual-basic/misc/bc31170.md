# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>Znak & č. 39; znak & č. 39; (0 x&lt;hexadecimální&gt;) není povolen v název XML
Nepodporovaný znak je obsažen v názvu literál XML nebo ve vlastnosti osy XML.  
  
 **ID chyby:** BC31170  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nepodporovaný znak z název literál XML nebo vlastnost osy XML.
