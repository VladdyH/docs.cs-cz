# <a name="local-variable-39ltvariablenamegt39-is-already-declared-in-the-current-block"></a>Místní proměnné & č. 39; &lt;NázevProměnné&gt;& č. 39; je již deklarována v aktuální bloku
Dvě proměnné se stejným názvem, které jsou deklarované v na stejný blok (který může být například procedury nebo řídicí blok `If...Then`).  
  
 **ID chyby:** BC30288  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Jedna z deklarací přesunout na jiný blok, nebo změňte název jednoho z proměnné.
