# <a name="picture-is-not-valid"></a>Obrázek není platný
Pokusili jste se přiřadit formátu grafického než rastrového obrázku, ikona nebo WMF k `Picture` vlastnost formuláře nebo ovládacího prvku.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zajistěte, aby se pokoušíte načíst do souboru `Picture` vlastnost je soubor platný grafiky Visual Basic nepodporuje.
