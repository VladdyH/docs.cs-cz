# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="1ad0b-102">Zadejte parametr s &#39;struktura&#39; omezení nelze použít jako omezení</span><span class="sxs-lookup"><span data-stu-id="1ad0b-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="1ad0b-103">Parametr typu `Structure` omezení slouží jako omezení pro jiný typ parametru.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="1ad0b-104">`Structure` Omezení vyžaduje, aby argument typu předaný parametr jeho typu typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="1ad0b-105">Typ hodnoty však nelze implementován nebo zděděná, takže není smysluplný můžete použít jako omezení, která by vyžadovala další parametr typu k implementaci, nebo nastavte dědičnost z něj.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="1ad0b-106">Má smysl pouze výklad této situaci je, že oba argumenty typu musí být přesně stejného typu.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="1ad0b-107">Pokud je to tento případ, obecného typu je potřeba jenom jeden parametr typu.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="1ad0b-108">Tato chyba může generovat následující příkaz.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="1ad0b-109">Předaný typ `t2` nelze implementovat, nebo nastavte dědičnost typu předaný `t1`, protože typ předaný `t1` musí být typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="1ad0b-110">**ID chyby:** BC32114</span><span class="sxs-lookup"><span data-stu-id="1ad0b-110">**Error ID:** BC32114</span></span>

-   <span data-ttu-id="1ad0b-112">Odeberte parametr typu omezená na `Structure` ze seznamu omezení na druhý parametr typu.</span><span class="sxs-lookup"><span data-stu-id="1ad0b-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>
