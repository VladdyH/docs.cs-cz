# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="744bd-102">Parametr typu s &#39;struktura&#39; omezení nejde používat jako omezení</span><span class="sxs-lookup"><span data-stu-id="744bd-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="744bd-103">Parametr typu `Structure` omezení je používat jako omezení pro parametr jiného typu.</span><span class="sxs-lookup"><span data-stu-id="744bd-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="744bd-104">`Structure` Omezení vyžaduje, aby argument typu předaný do parametru jeho typ typem hodnoty.</span><span class="sxs-lookup"><span data-stu-id="744bd-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="744bd-105">Typ hodnoty však nelze implementovat nebo zděděno, takže není smysl použít jako omezení, která by vyžadovala další parametr typu na jeho implementaci nebo z něj dědí.</span><span class="sxs-lookup"><span data-stu-id="744bd-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="744bd-106">Má smysl pouze výklad této situaci je, že oba argumenty typu musí být přesně stejného typu.</span><span class="sxs-lookup"><span data-stu-id="744bd-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="744bd-107">Pokud je to tento případ, obecný typ musí pouze jeden parametr typu.</span><span class="sxs-lookup"><span data-stu-id="744bd-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="744bd-108">Tato chyba může generovat následující příkaz.</span><span class="sxs-lookup"><span data-stu-id="744bd-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="744bd-109">Typ předaný `t2` nelze neimplementuje ani nedědí typ předaný `t1`, protože typ předaný `t1` musí být typem hodnoty.</span><span class="sxs-lookup"><span data-stu-id="744bd-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="744bd-110">**ID chyby:** BC32114</span><span class="sxs-lookup"><span data-stu-id="744bd-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="744bd-111">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="744bd-111">To correct this error</span></span>  
  
-   <span data-ttu-id="744bd-112">Odeberte parametr typu s omezením `Structure` ze seznamu omezení na jiné parametr typu.</span><span class="sxs-lookup"><span data-stu-id="744bd-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="744bd-113">Pokud oba parametry typu vyžadují stejného typu hodnoty, definice obecného typu s pouze jedním parametrem typu.</span><span class="sxs-lookup"><span data-stu-id="744bd-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
