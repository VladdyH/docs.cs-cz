# <a name="the-connectiontimeout-must-be-greater-than-0"></a>ConnectionTimeout musí být větší než 0.
Při odesílání a stahování souborů s [My.Computer.Network – objekt](../../visual-basic/language-reference/objects/my-computer-network-object.md), je nutné zadat `connectionTimeout` větší než `0`.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte `connectionTimeout` větší než `0`.
