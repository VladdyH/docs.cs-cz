# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39;Možnost&#39; příkazy musí předcházet všechny deklarace nebo &#39;importy&#39; – příkazy
`Option` příkazy by měl být první příkazy bez komentáře ve zdrojovém kódu.  
  
 **ID chyby:** BC30627  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `Option` příkazů do horní části zdrojový kód bezprostředně před `Imports` a `Namespace` příkazy.
