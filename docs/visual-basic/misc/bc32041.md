# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>– Znak typu nelze použít v deklaraci parametrů typu
Parametr deklaraci typu obsahuje alespoň jeden znak typu identifikátoru.  
  
 Parametr typu obecného typu musí být platný název jazyka Visual Basic. Povolené znaky nesmí obsahovat žádný z identifikátor – znaky typu (`%`, `&`, `@`, `!`, `#`, a `$`). V tématu [deklarované názvy elementů](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **ID chyby:** BC32041  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte – znak typu identifikátoru nebo znaky z deklarace typu parametru.
