# <a name="bracketed-identifier-is-missing-closing-3939"></a>V závorkách identifikátor chybí uzavírací &#39;]&#39;
Hranaté závorce v názvu objektu uvozovacími znaky musí dojít k odpovídajícím páry.  
  
 **ID chyby:** BC30034  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pravá závorka umístíte na konci názvu uvozovacími znaky.
