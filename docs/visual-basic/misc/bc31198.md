<span data-ttu-id="462e4-103">Deklarace literálů XML se používá ve výrazu v umístění, které se nejednoznačný kompilátoru jazyka Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="462e4-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="462e4-104">To znamená, nelze určit Visual Basic – kompilátor zda je menší – než znak (<) je určený jako srovnávací operátor nebo spuštění literál XML.</span><span class="sxs-lookup"><span data-stu-id="462e4-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="462e4-105">Následující kód představuje příklad:</span><span class="sxs-lookup"><span data-stu-id="462e4-105">The following code provides an example:</span></span>  

```vb

                  Select element  
```
