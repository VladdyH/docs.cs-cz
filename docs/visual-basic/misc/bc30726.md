# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>& č. 39; S & č. 39; kontexty a příkazy nejsou platné ve windows ladění

 **ID chyby:** BC30726  
  
## <a name="to-correct-this-error"></a>Oprava této chyby
