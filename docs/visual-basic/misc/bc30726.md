# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;S&#39; kontexty a příkazy nejsou platné ve windows ladění
`With` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30726  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `With` příkazy z ladění kódu.
