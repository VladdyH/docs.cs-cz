# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Sdílené & č. 39; Sub – nové & č. 39; nemůže mít žádné parametry

 **ID chyby:** BC30479  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte parametr nebo parametry.
