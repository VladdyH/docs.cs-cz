# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Sdílené &#39;Sub New&#39; nemůže mít žádné parametry
Parametr nebo parametry jste zahrnuli do sdílenou `Sub New` příkaz, který není platný.  
  
 **ID chyby:** BC30479  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte parametr nebo parametry.
