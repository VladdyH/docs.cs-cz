# <a name="statement-cannot-appear-within-a-method-body"></a>Příkaz nelze vložit do těla metody
Procedura obsahuje neplatný příkaz například jiný postup deklarace. Deklarace procedury nelze vnořit.  
  
 **ID chyby:** BC30289  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte deklaraci postup nebo přesunout mimo nadřazených postupu.
