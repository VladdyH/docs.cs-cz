# <a name="39ltkeywordgt39-is-not-valid-within-a-structure"></a>& č. 39; &lt;– klíčové slovo&gt;& č. 39; není platný v rámci struktury
Struktury jsou typy hodnot, ne odkazové typy. Struktury není instance vytvořené z třídy, proto `Me`, `MyClass`, a `MyBase` klíčová slova jsou smysl ve struktuře.  
  
 **ID chyby:** BC30044  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Změnit strukturu na třídu nebo odeberte klíčové slovo v postupu.
