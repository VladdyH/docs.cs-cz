# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>Nerozpoznaný možnost &lt;název_možnosti&gt;; ignorováno
Byla zadána možnost, která není platná z příkazového řádku.  
  
 **ID chyby:** BC2007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte zadané možnosti najít ten, který není platný.
