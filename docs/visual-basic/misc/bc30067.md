# <a name="39exit-function39-is-not-valid-in-a-sub-or-property"></a>& č. 39; Ukončit funkce & č. 39; není platný v Sub nebo vlastnost
`Exit Function`Zobrazí se v `Sub` postup nebo `Property` postupu. `Exit` Příkaz musí odpovídat bloku ve kterém dochází.  
  
 **ID chyby:** BC30067  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte `Exit Function` s `Exit Sub` nebo `Exit Property` příkaz podle potřeby.
