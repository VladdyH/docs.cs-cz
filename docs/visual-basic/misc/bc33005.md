# <a name="39end-operator39-expected"></a>&#39;Ukončení operátor&#39; očekávání
`End` Příkaz musí odpovídat bloku ukončí. `Operator` Definice procedury nebyl správně zavřen.  
  
 **ID chyby:** BC33005  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `End Operator` příkaz k ukončení režimu.
