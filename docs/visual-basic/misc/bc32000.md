# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Místní proměnné &#39; &lt;NázevProměnné&gt; &#39; nemůže odkazovat na předtím, než je deklarovaná
Proměnnou uvnitř procedury se používá před jeho deklaraci.  
  
 **ID chyby:** BC32000  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesuňte deklaraci předcházet prvního odkazu na proměnnou.
