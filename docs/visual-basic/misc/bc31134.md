# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39;AddHandler&#39; a &#39;RemoveHandler&#39; parametry metody nelze deklarovat &#39;ByRef&#39;
Parametry `AddHandler` a `RemoveHandler` metody nelze deklarovat s `ByRef` modifikátor.  
  
 **ID chyby:** BC31134  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `ByRef` – klíčové slovo z parametru.
