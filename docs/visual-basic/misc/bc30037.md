# <a name="character-is-not-valid"></a>Znak není platný
Aktuální kontext nepovoluje tento znak. Například tildou (`~`) nebo svislá čára (`|`) nemůže být součástí názvu deklarovaný element.  
  
 **ID chyby:** BC30037  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda byla do programovací element omylem vložena další znak.  
  
2.  Zkontrolujte, že programovací element je napsaný nebo vytvářeny pomocí pouze znaky, které jsou platné pro tento typ elementu.
