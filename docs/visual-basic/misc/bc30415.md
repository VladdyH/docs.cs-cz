# <a name="39redim39-cannot-change-the-number-of-dimensions-of-an-array"></a>& č. 39; ReDim & č. 39; nelze změnit počet rozměrů pole
Pokusili jste se použít `ReDim` Chcete-li změnit pořadí pole (počet dimenzí). `ReDim` Příkaz lze použít ke změně velikosti jeden nebo více rozměry pole, které již byla dříve deklarována, ale nedá se změnit pořadí pole.  
  
 **ID chyby:** BC30415  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda chcete pořadí místo velikosti pole a pokud je to možné, použijte `Dim` deklarovat nové pole se požadované pořadí.
