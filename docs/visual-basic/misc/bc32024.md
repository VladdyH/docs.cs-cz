# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-39optional39"></a>Nelze zadat výchozí hodnoty pro parametry, které nejsou deklarované &#39;volitelné&#39;
Při volání procedury, a jejich nepřebírají výchozí hodnoty v deklaraci je nutné zadat povinnými argumenty.  
  
 **ID chyby:** BC32024  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarovat argumentem být `Optional` nebo odeberte výchozí hodnotu.
