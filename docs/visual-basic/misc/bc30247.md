# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>& č. 39; &lt;modifikátor&gt;& č. 39; není platný v deklaraci lokální proměnné
A `Dim` příkaz uvnitř procedury obsahuje neplatný klíčové slovo, jako například `Overloads`.  
  
 **ID chyby:** BC30247  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné klíčové slovo z `Dim` příkaz.
