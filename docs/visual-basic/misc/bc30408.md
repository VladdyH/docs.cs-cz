# <a name="method-39ltmethodnamegt39-does-not-have-the-same-signature-as-delegate-39ltdelegatenamegt39"></a>Metoda & č. 39; &lt;methodname&gt;& č. 39; nemá stejným podpisem jako delegáta & č. 39;&lt; Vlastnost delegateName&gt;& č. 39;
Došlo k neshodě mezi podpisy metody a delegáta, který se pokoušíte použít. `Delegate` Příkaz definuje parametry a návratové typy delegát třídy. Libovolná procedura s odpovídající parametry, typy a návratový typ může použít k vytvoření instance tohoto typu delegáta. Každá třída delegáta definuje konstruktor, který je předán specifikace metodu objektu.  
  
 **ID chyby:** BC30408
