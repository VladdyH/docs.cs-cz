# <a name="registry-key-39ltkeynamegt39-could-not-be-found"></a>Klíč registru &#39; &lt;keyname&gt; &#39; nebyl nalezen
Nelze nalézt klíč registru.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda název klíče.  
  
2.  Ujistěte se, zda máte dostatečná oprávnění k přístupu ke klíči.
