# <a name="3939-unexpected"></a>& č. 39; (& č. 39; neočekávané
'(' Unexpected. Pole obecných typů bez instancí nejsou povoleny.  
  
 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]pole nelze kompilovat, pokud je konkrétní datového typu. Datový typ parametr obecného typu nelze použít jako datový typ pole.  
  
 **ID chyby:** BC32095  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud budete muset použít pole, je potřeba deklarovat mohla být konkrétní datového typu. Datový typ parametru nelze použít.  
  
-   Pokud potřebujete skupinu elementů typu dat, který má být zadaná pro parametr typu dat, musíte použít kolekci místo pole.
