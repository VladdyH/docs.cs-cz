# <a name="argument-39path39-is-nothing-or-empty"></a>Argument &#39;cesta&#39; nic nebo musí být prázdná
Cesta není platný, protože je inicializováno `Nothing` nebo je prázdný. Cesta musí mít specifické datový typ a platný název.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte platný název cesty.
