# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;Ukončení pomocí&#39; musí předcházet odpovídající &#39;Using&#39;
`End Using` Příkazu se zobrazí s neexistuje odpovídající `Using` deklaraci předchozích ho.  
  
 **ID chyby:** BC36007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `End Using` příkaz, pokud je redundantní.  
  
-   Zadejte chybějící [příkazu Using](../../visual-basic/language-reference/statements/using-statement.md) Pokud některý chybí.  
  
-   Přesunout `End Using` příkaz na příslušné místo v kódu.
