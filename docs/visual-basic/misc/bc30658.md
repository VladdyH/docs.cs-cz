# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Vlastnost &#39; &lt;propertyname&gt; &#39; bez parametrů nelze nalézt
Atribut používaný má vlastnost, která vyžaduje jeden nebo více parametrů.  
  
 **ID chyby:** BC30658  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte parametry pro vlastnost.
