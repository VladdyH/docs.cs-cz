# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Příliš mnoho argumentů pro rozšíření metoda &#39; &lt;methodname&gt; &#39; definované v &#39; &lt;typename&gt;&#39;
Počet argumentů zadat ve volání metody rozšíření překračuje počet argumentů pro metodu definován.  
  
 **ID chyby:** BC36582  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Projděte si definici metody rozšíření a odebrat další argumenty z volání.
