# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Příliš mnoho argumentů pro rozšíření metoda & č. 39; &lt;methodname&gt;& č. 39; definovaným v & č. 39;&lt; TypeName&gt;& č. 39;
Počet argumentů zadat ve volání metody rozšíření překračuje počet argumentů pro metodu definován.  
  
 **ID chyby:** BC36582  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Projděte si definici metody rozšíření a odebrat další argumenty z volání.
