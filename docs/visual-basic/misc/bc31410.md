# <a name="overloading-methods-declared-in-multiple-base-interfaces-is-not-valid"></a>Přetížení metody deklarované v několika základní rozhraní není platný
Více zděděné rozhraní implicitně přetížení stejnou metodu.  
  
 **ID chyby:** BC31410  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Shadows` modifikátor místo `Overloads` modifikátor.
