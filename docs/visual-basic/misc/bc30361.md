# <a name="39ltproperty1gt39-and-39ltproperty2gt39-cannot-overload-each-other-because-only-one-is-declared-39default39"></a>& č. 39; &lt;vlastnost1&gt;& č. 39; a & č. 39;&lt; vlastnost2&gt;& č. 39; nelze přetížení navzájem, protože je deklarovaná jenom jeden & č. 39; výchozí & č. 39;
Pokud vlastnost určuje `Default`, musíte zadat také všechny vlastnosti přetížený na tento název `Default`.  
  
 **ID chyby:** BC30361  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zajistěte, aby všechny přetížené vlastnosti jsou deklarovány `Default`.
