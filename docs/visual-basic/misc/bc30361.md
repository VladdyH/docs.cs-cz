# <a name="39ltproperty1gt39-and-39ltproperty2gt39-cannot-overload-each-other-because-only-one-is-declared-39default39"></a>&#39;&lt;vlastnost1&gt; &#39; a &#39; &lt;vlastnost2&gt; &#39; nelze přetížení navzájem, protože je deklarovaná jenom jeden &#39;výchozí&#39;
Pokud vlastnost určuje `Default`, musíte zadat také všechny vlastnosti přetížený na tento název `Default`.  
  
 **ID chyby:** BC30361  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zajistěte, aby všechny přetížené vlastnosti jsou deklarovány `Default`.
