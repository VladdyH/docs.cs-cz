# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Pomocí & č. 39; příkazy nejsou platné v příkazové podokno
`Using`příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30186  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Using` příkaz z **Immediate** okno.
