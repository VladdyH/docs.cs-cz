# <a name="could-not-obtain-memory-information-due-to-internal-error"></a>Nelze získat informace o paměti z důvodu vnitřní chyby
Volání do jedné z vlastností informace o paměti `My.Computer.Info` objektu se nezdařilo.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Try...Catch` bloku kolem volání vlastnost informace o paměti `My.Computer.Info` objektu.
