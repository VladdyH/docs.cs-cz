# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>& č. 39; &lt;typename&gt;& č. 39; nelze použít jako atribut, protože se nejedná o třídu
Došlo pokusu o použití typu, který není třída jako atribut.  
  
 **ID chyby:** BC31503  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Definovat vlastní atributy třídy, které jsou odvozeny od `System.Attribute`.
