# <a name="extension-method-39ltmethodnamegt39-defined-in-39ltmodulenamegt39-is-not-generic-or-has-no-free-type-parameters-and-so-cannot-have-type-arguments"></a>Metody rozšíření &#39; &lt;methodname&gt; &#39; definované v &#39; &lt;modulename&gt; &#39; není obecná (nebo nemá žádné parametry volného typu) a proto nemůže mít argumenty typů
Argument typu byla zadána při volání metody rozšíření, která buď nemá žádné obecné parametry, nebo nemá žádné obecné parametry, pro které není typu již zadán. Tuto chybu způsobí například následující kód.  
  
```vb  
' The extension method is not generic.  
<Extension()> _  
Sub Example(ByVal str As String)  
    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```  
  
 **ID chyby:** BC36907  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidáte parametr typu k definici – metoda rozšíření.  
  
-   Argument typu navíc odebrání volání procedury.
