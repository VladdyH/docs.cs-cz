# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Odkaz na je vyžadován na modul &#39; &lt;modulename&gt; &#39; obsahující definici pro události &#39; &lt;eventname&gt;&#39;
Odkaz na je vyžadován na modul ' <`modulename`>' obsahující definici pro události se <`eventname`> ". Přidáte do projektu.  
  
 Událost je definována v modulu, který není přímo odkazuje ve vašem projektu. Visual Basic – kompilátor vyžaduje odkaz, aby se zabránilo nejednoznačnosti v případě, že událost je definována v více než jeden modul.  
  
 **ID chyby:** BC30006  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrňte název neregistrované modulu odkazy projektu.
