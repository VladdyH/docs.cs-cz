# <a name="error-setting-assembly-manifest-option-lterror-messagegt"></a>Chyba při nastavování možnosti manifestu sestavení: &lt;chybová zpráva&gt;
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Kompilátoru volá Linker sestavení (Al.exe, také známé jako Alink) ke generování sestavení s manifestu. Linkeru ohlásil chybu hledání soubor klíče nebo klíče jméno, které k podepsání sestavení.  
  
 **ID chyby:** BC30139  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Prověřením uvozovkách chybové zprávy a informace uvedené na chyby [Al.exe (Linker sestavení)](../../framework/tools/al-exe-assembly-linker.md) pro další vysvětlení a doporučení.  
  
2.  Pokud potíže potrvají, shromažďovat informace o okolnostech a upozornění služby Microsoft Product Support Services.
