# <a name="39ltspecifiergt39-is-not-valid-on-an-interface-property-declaration"></a>&#39;&lt;specifikátor&gt; &#39; není platný v deklaraci vlastnosti rozhraní
A `Property` příkaz uvnitř rozhraní obsahuje neplatný klíčové slovo, jako například `Implements`. Rozhraní můžete definovat pouze členy, není jejich implementaci.  
  
 **ID chyby:** BC30273  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné – klíčové slovo z příkazu deklarace.  
  
-   Přesuňte implementace členů rozhraní na třídu, která implementuje rozhraní.
