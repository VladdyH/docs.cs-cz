# <a name="drive-39ltdrivenamegt39-not-found"></a>Jednotka & č. 39; &lt;drivename&gt;& č. 39; nebyla nalezena
Nebylo možné najít jednotku. Buď došlo ke ztrátě připojení k síti, nebo jednotku, kterou se pokoušíte získat přístup, je offline nebo neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte energie k jednotce.  
  
2.  Zkontrolujte, jestli jsou správně připojené kabely připojení počítače k síti.  
  
3.  Pro přístup k jednotce přes síť, zkontrolujte, zda logické připojení mezi vaším počítačem a jednotku.  
  
4.  Připojení k síti a akci opakujte.
