# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Metody v modulu nelze implementace členů rozhraní
Členové rozhraní musí být implementované třídy nebo struktury.  
  
 **ID chyby:** BC31083  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Implementace členů rozhraní z třídy nebo struktury.
