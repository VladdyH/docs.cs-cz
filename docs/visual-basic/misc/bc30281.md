# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Struktura &#39; &lt;%{structurename/&gt; &#39; musí obsahovat alespoň jednu instanci členské proměnné nebo deklarace událostí
Dojde k funkčně prázdnou strukturou. Musí být deklarován nejméně jeden člen proměnné mezi `Structure` příkaz a `End Structure` příkaz.  
  
 **ID chyby:** BC30281  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Dim` příkaz nebo `Event` příkaz na strukturu.
