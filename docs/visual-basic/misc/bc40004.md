# <a name="lttype1gt-39ltmembernamegt39-conflicts-with-lttype2gt-39ltmembernamegt39-on-the-base-class-lttype3gt-39ltclassnamegt39-and-should-be-declared-39shadows39"></a>&lt;Type1&gt; &#39; &lt;membername&gt; &#39; je v konfliktu s &lt;type2&gt; &#39; &lt;membername&gt; &#39; na základní třídě &lt; Type3&gt; &#39; &lt;classname&gt; &#39; a musí být deklarován &#39;stínů&#39;
Programovací element je deklarovaná se stejným názvem jako element definovaný v základní třídě. V takovém případě by měl stínové elementu v této třídě prvku základní třídy.  
  
 Tato zpráva je upozornění. `Shadows` ve výchozím nastavení se předpokládá. Další informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC40004  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Shadows` – klíčové slovo na deklaraci nebo změňte název elementu se deklarovat.
