# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>Inicializátor pole chybí &lt;číslo&gt; elementy
Není zadáno dostatečný počet elementů v inicializátoru pro pole.  
  
 **ID chyby:** BC30567  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte chybí argument nebo argumenty.  
  
-   Použití `ReDim` ke změně velikosti pole.
