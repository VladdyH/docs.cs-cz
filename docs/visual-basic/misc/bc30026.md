# <a name="39end-sub39-expected"></a>& č. 39; End Sub & č. 39; očekávání
`End` Příkaz musí odpovídat bloku ukončí. A `Sub` definice procedury nebyl správně zavřen.  
  
 **ID chyby:** BC30026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Sub` příkaz k ukončení režimu.
