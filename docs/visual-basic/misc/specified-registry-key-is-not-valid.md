# <a name="specified-registry-key-is-not-valid-because-it-contains-two-or-more-consecutive-backslashes"></a>Zadaný klíč registru není platný, protože obsahuje dvě nebo více po sobě jdoucích zpětná lomítka
Klíč registru zadaný s cestou obsahuje dvě nebo více po sobě jdoucích zpětná lomítka. To může být způsobeno kombinování několik řetězce, které vytvoří cestu a nechtěně včetně příliš mnoho zpětná lomítka.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte klíč registru specifikované k určení, kde a proč vkládání navíc zpětná lomítka.
