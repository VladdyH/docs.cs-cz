# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>Atribut konstanta & č. 39; &lt;constantname&gt;& č. 39; nemůže být cílem přiřazení
Proběhl pokus o přiřazení hodnoty na konstantu deklarované v atributu.  
  
 **ID chyby:** BC31510

-   Pokud používáte vlastní atributy, které jste vytvořili, znovu definujte člen v atributu jako pole.
