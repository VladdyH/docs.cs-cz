# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>Atribut konstanta &#39; &lt;constantname&gt; &#39; nemůže být cílem přiřazení
Proběhl pokus o přiřazení hodnoty na konstantu deklarované v atributu.  
  
 **ID chyby:** BC31510  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte chybné přiřazení.  
  
-   Pokud používáte vlastní atributy, které jste vytvořili, znovu definujte člen v atributu jako pole.
