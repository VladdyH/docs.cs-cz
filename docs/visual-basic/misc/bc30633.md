`Set` procedury vlastností musí končit `End Set` příkaz.  
  
 **ID chyby:** BC30633  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že `Set` procedura property končí `End Set` vytvořit.
