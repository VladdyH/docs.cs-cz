# <a name="39is39-expected"></a>& č. 39; je & č. 39; očekávání
A `TypeOf` operátor probíhá, aniž by `Is` klauzule.  
  
 **ID chyby:** BC30224  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Is` následující klauzule `TypeOf` operátor, například `TypeOf MyControl Is ComboBox`.
