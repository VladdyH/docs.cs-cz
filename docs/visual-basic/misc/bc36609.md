# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Výrazy dotazů nejsou platné ve windows ladění
Dotaz LINQ byl zadán v **sledovat** nebo **Immediate** okno během relace ladění. Dotazy nejsou k dispozici v systému windows ladění.  
  
 **ID chyby:** BC36609
