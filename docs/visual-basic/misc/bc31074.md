# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Delegovat konstruktor pro typ & č. 39; &lt;typename&gt;& č. 39; nezbytné k provedení delegáti, nebyl nalezen
Sestavení obsahující typ může být poškozen nebo není platná.  
  
 **ID chyby:** BC31074  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda je sestavení obsahující tohoto typu platný.  
  
2.  Kompilace programu zjistíte, pokud se chyba objeví znovu.  
  
3.  Pokud potíže potrvají, uložte si práci a restartujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
