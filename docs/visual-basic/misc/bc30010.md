# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Odkaz na je vyžadován na modul &#39; &lt;modulename&gt; &#39; obsahující implementované rozhraní &#39; &lt;interfacename&gt;&#39;
Odkaz na je vyžadován na modul,\<modulename >' obsahující implementované rozhraní '\<interfacename >'. Přidáte do projektu.  
  
 Rozhraní je definována v modulu, který není přímo odkazuje ve vašem projektu. Visual Basic – kompilátor vyžaduje odkaz, aby se zabránilo nejednoznačnosti v případě, že rozhraní je definováno ve více než jeden modul.  
  
 **ID chyby:** BC30010  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zahrňte název neregistrované modulu odkazy projektu.
