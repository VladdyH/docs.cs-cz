# <a name="39gosub39-statements-are-no-longer-supported"></a>&#39;GoSub&#39; příkazy již nejsou podporovány.
`GoSub` nelze použít k volání procedury.  
  
 **ID chyby:** BC30814  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Volání procedury přímo bez použití `GoSub`; například:  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
