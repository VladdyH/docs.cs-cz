Člen '\<membername >' nebyl nalezen v třídě\<classname >'. Tento stav je obvykle výsledkem neodpovídající 'Microsoft.VisualBasic.dll'.  
  
 Nelze najít definovaný člen.  
  
 **ID chyby:** BC31097  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Kompilace programu zjistíte, pokud se chyba objeví znovu.  
  
2.  Pokud se chyba objeví znovu, uložte si práci a restartujte [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Pokud chyba přetrvává, přeinstalujte [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Pokud chyba přetrvává i po přeinstalování, upozorněte služby Microsoft Product Support Services.
