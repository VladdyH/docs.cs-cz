# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Objekt odkazovaný &#39; &lt;objectname&gt; &#39; má hodnotu &#39;nic&#39;
Odkazovaný objekt nebyl nastaven, nebo byla nastavena na hodnotu `Nothing`.  
  
 **ID chyby:** BC30705  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, že název objektu je napsán správně a nastavený objekt.
