# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Objekt odkazovaný & č. 39; &lt;objectname&gt;& č. 39; má hodnotu & č. 39; Nic & č. 39;
Odkazovaný objekt nebyl nastaven, nebo byla nastavena na hodnotu `Nothing`.  
  
 **ID chyby:** BC30705  
  
## <a name="to-correct-this-error"></a>Oprava této chyby
