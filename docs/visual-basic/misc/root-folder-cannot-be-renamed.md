# <a name="root-folder-cannot-be-renamed"></a>Kořenovou složku nelze přejmenovat.
Byl proveden pokus o přejmenování kořenový adresář. To může být způsobeno nesprávně Analyzovaná souboru cestu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte cestu k souboru a ujistěte se, zda je správný.
