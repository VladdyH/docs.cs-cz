# <a name="events-in-interfaces-cannot-be-declared-39ltimplementsgt39"></a>Události v rozhraní nelze deklarovat & č. 39; &lt;implementuje&gt;& č. 39;
Události deklarované v rozhraních nelze implementovat, události jiných rozhraní.  
  
 **ID chyby:** BC30688  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Implements` příkaz.  
  
2.  Implementujte události v rámci třídu nebo strukturu.
