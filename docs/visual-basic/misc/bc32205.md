# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>Délka názvu proměnné WithEvents nesmí překročit 1019 znaků
Název nebo identifikátor každé proměnné deklarovány pomocí `WithEvents` je omezený na 1019 znaků.  
  
 **ID chyby:** BC32205  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkraťte identifikátor.
