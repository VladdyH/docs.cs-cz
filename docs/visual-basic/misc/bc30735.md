# <a name="type-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Typ modulu nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
V modulu byla deklarována typu s neplatnou modifikátory.  
  
 **ID chyby:** BC30735  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nepoužívejte `Protected` nebo `Protected Friend` modifikátory na typech deklarované v rámci modulu.
