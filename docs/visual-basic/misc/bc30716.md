# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Nakonec&#39; příkazy nejsou platné v příkazové podokno
`Finally` příkazy lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30716  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Finally` příkazy z ladění kódu.
