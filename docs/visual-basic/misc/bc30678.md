# <a name="39end39-statement-not-valid"></a>&#39;End&#39; příkaz není platný
`End` – Klíčové slovo je nesprávně použitá nebo nevhodná v aktuálním kontextu.  
  
 **ID chyby:** BC30678  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Použití `End` samostatně způsobí postup ztratit oboru.  
  
2.  Použití `End` s žádná jiná klíčová slova ukončit blok kódu, jako je například struktura nebo vlastnost.
