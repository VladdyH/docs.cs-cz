# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Převody z & č. 39; &lt;NázevTypu1&gt;& č. 39; na & č. 39;&lt; NázevTypu2&gt;& č. 39; musí být explicitní
Tento převod nelze provést implicitně.  
  
 **ID chyby:** BC30664  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `CType`, nebo jeden z konkrétní převod funguje, jako například `CInt` nebo `CDec`.
