# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace nebo typ &#39; &lt;název&gt; &#39; již byla naimportována
Byl proveden pokus o import obor názvů více než jednou.  
  
 **ID chyby:** BC31051  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat redundantní `Imports` příkazy.
