# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39;NotOverridable&#39; nelze zadat pro metody, které nepřepisují jinou metodu
Vlastnosti a metody jsou `NotOverridable` ve výchozím nastavení. `NotOverridable` Modifikátor lze použít pouze na metody, které potlačí jinou vlastnost nebo metoda.  
  
 **ID chyby:** BC31088  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `NotOverridable` modifikátor z vlastnosti a metody, které nepřepisují jiného člena.
