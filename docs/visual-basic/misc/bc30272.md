# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;Název argumentu ArgumentName&gt; &#39; není parametr &#39; &lt;methoddeclaration&gt;&#39;
Volání procedur poskytuje další argument, který není definován procedurou.  
  
 **ID chyby:** BC30272  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné argument ze seznamu argumentů.
