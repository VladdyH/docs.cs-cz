# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Pole &#39; &lt;název pole&gt; &#39; je nepodporovaného typu
Pole je deklarován jako typ, který nemůže být reprezentovaná Visual Studio.  
  
 **ID chyby:** BC30656  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Definujte pole jako podporovaného typu.  
  
2.  Vytvořte nový typ, který popisuje data v poli.
