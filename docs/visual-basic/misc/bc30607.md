# <a name="39notinheritable39-classes-cannot-have-members-declared-39ltspecifiernamegt39"></a>&#39;NotInheritable&#39; třídy nemůže mít členy deklarovaný &#39; &lt;specifiername&gt;&#39;
Modifikátory přepsání nelze použít s `NotInheritable` třídy, protože jejich členové nelze přepsat.  
  
 **ID chyby:** BC30607  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat přepsání modifikátory jako `Overridable`, `NotOverridable`, nebo `MustOverride`, z definice třídy.
