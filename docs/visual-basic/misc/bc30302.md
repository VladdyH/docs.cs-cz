# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Zadejte znak &#39; &lt;znak&gt; &#39; nelze použít v deklaraci explicitního typu
Použili jste – znak typu, například `$`, když explicitním zadáním typu.  
  
 **ID chyby:** BC30302  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat znak typu nebo odebrat `As <Type>` klauzule z deklarace.
