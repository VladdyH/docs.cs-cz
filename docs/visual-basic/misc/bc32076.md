# <a name="39lttypenamegt39-is-a-generic-type-and-requires-type-arguments"></a>& č. 39; &lt;typename&gt;& č. 39; je obecný typ a vyžaduje argumenty typů
Proměnnou, parametr procedury nebo funkce návratový je deklarovaná tak, aby měl typ obecné třídu nebo strukturu, ale deklaraci neposkytuje žádné argumenty typu.  
  
 Svou povahou každý – obecná třída, struktura je definován s alespoň jeden typ parametru. Pokud používáte obecného typu deklarovat sestavené třídu nebo strukturu, je nutné zadat typ argumentu pro každý parametr typu definované obecného typu.  
  
 **ID chyby:** BC32076  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte seznam typů k deklaraci, uzavřený v závorkách a počínaje `Of` – klíčové slovo.
