# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>& č. 39; &lt;filename&gt;& č. 39; nelze odkazovat, protože se nejedná o sestavení
Odkaz byl proveden do souboru, který není sestavení .NET.  
  
 **ID chyby:** BC31519  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte název a umístění sestavení.  
  
2.  Ověřte, že sestavení není poškozený.
