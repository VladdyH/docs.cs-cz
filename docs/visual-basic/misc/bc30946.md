# <a name="stop-request-is-pending"></a>Požadavek na zastavení čeká na vyřízení
V ladicím programu sady Visual Studio určuje volání procedury, výraz, ale neexistuje požadavek na zastavení vlákna.  
  
 Ladicí program nebude pokoušet o volání procedury na vlákno, které není aktivní.  
  
 **ID chyby:** BC30946  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je to možné zjištění příčiny žádost o ukončení vlákna, takže můžete zabránit jeho dalšímu.  
  
-   Ukončit a znovu spusťte ladění.
