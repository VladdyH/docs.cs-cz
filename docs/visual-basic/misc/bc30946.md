# <a name="stop-request-is-pending"></a>Požadavek na zastavení čeká na vyřízení
V ladicím programu sady Visual Studio výraz určuje volání procedury, ale je požadavek na zastavení vlákno.  
  
 Ladicí program nebude pokoušet o volání procedury na vlákno, které není aktivní.  
  
 **ID chyby:** BC30946  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Pokud je to možné určete zdroj požadavek na zastavení vlákno, takže ho můžete zabránit dalšímu.  
  
-   Ukončit a znovu spusťte ladění.
