# <a name="property-39ltpropertynamegt39-cannot-be-declared-39default39-because-it-overrides-a-property-on-the-base-lttypegt-39lttypenamegt39-that-is-not-default"></a>Vlastnost & č. 39; &lt;propertyname&gt;& č. 39; nelze deklarovat & č. 39; výchozí & č. 39; protože přepisuje vlastnost základní &lt;typ&gt; & č. 39;&lt; TypeName&gt;& č. 39; které není výchozí
Pokusili jste se deklarace vlastnosti `Default`, což není platná deklarace protože přepisuje vlastnost, která nebyla deklarována `Default` na základní třídy.  
  
 **ID chyby:** BC30504  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `Default` deklarace.
