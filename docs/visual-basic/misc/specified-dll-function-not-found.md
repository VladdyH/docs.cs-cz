# <a name="specified-dll-function-not-found"></a>Zadaná funkce DLL nebyla nalezena
Dynamická knihovna (DLL) do knihovny odkazu uživatel nebyl nalezen, ale funkce knihovny DLL určené nebyl nalezen v rámci knihovnu DLL.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte platný pořadí v deklaraci funkce.  
  
2.  Ujistěte se, že název knihovny DLL a alias jsou správné.
