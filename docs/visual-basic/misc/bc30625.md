# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Modul&#39; příkaz musí končit odpovídající &#39;End modulu&#39;
Modul bloky musí končit `End``Module` příkaz.  
  
 **ID chyby:** BC30625  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Module` příkaz na konec `Module` bloku.
