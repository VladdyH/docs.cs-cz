# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Modul&#39; příkaz musí být ukončen párovým klíčovým &#39;End Module&#39;
Bloky modul musí končit `End Module` příkazu.  
  
 **ID chyby:** BC30625  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Module` příkaz na konec objektu `Module` bloku.
