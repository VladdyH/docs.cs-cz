# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;Catch –&#39; bloku nikdy dosaženo, protože &#39; &lt;name1&gt; &#39; dědí z &#39; &lt;name2&gt;&#39;
A `Catch` bloku v rámci kód není přístupný.  
  
 Ve výchozím nastavení je tato zpráva upozornění. Další informace o zobrazení nebo skrytí upozornění práce upozornění jako chyby najdete v tématu [Konfigurace upozornění v jazyce Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **ID chyby:** BC42029  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte nadbytečné `Catch` bloku.
