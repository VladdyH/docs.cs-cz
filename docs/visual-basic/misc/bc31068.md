# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Delegát v rozhraní nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Je zadaná neplatná modifikátor se delegáta definována v rozhraní.  
  
 **ID chyby:** BC31068  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrat modifikátory jako `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, a `Overridable`.
