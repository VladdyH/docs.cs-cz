# <a name="registry-key-39ltkeynamegt39-could-not-be-created"></a>Klíč registru &#39; &lt;keyname&gt; &#39; nebylo možné vytvořit
Zadaný klíč registru se nezdařilo, možná z důvodu nedostatečnými oprávněními zabezpečení.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, zda máte dostatečná oprávnění k přístupu ke klíči.
