# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>&#39;Ukončení modulu&#39; musí předcházet odpovídající &#39;modulu&#39;
Modul bloky začínají `Module` příkaz a konec s `End Module` vytvořit.  
  
 **ID chyby:** BC30622  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že modul bloku začíná `Module` příkaz.
