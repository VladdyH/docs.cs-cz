# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>& č. 39; Ukončení modulu & č. 39; musí předcházet odpovídající & č. 39; Modul & č. 39;
Modul bloky začínají `Module` příkaz a konec s `End Module` vytvořit.  
  
 **ID chyby:** BC30622  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že modul bloku začíná `Module` příkaz.
