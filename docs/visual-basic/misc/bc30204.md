# <a name="integer-constant-expected"></a>Byla očekávána konstanta celé číslo
`#ExternalSource` – Direktiva dojde, ve kterém druhý argument není typu integer literálu. V tomto kontextu platí pouze celé literálu. Člen s názvem konstanta nebo výčtu není platná.  
  
 **ID chyby:** BC30204  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Použijte pojmenované člen výčtu nebo konstanta místo literál.  
  
2.  Zadejte celé číslo literálu jako druhý argument `#ExternalSource` – direktiva.
