# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39;Možnost &lt;specifikátor&gt; &#39; příkaz může být pouze jednou za souboru
`Option Compare`, `Option Explicit`, A `Option Strict` příkazy dá použít jenom jeden čas ve stejném souboru zdroje.  
  
 **ID chyby:** BC30225  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte duplicitní `Option` příkaz.
