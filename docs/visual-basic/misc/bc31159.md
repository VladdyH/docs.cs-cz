# <a name="expected-closing-39gt39-for-embedded-expression"></a>Očekávaný ukončovací & č. 39; %&gt;& č. 39; pro embedded výraz
Vložené výraz v literál XML chybí požadované ukončovací `%>` znaků.  
  
 **ID chyby:** BC31159  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte požadované ukončovací `%>` znaků na konec embedded výraz.
