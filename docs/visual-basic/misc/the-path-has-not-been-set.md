# <a name="the-path-has-not-been-set"></a>Cesta nebyla nastavena.
Operace se soubory, jako je například přesunutí nebo kopírování souboru, došlo k pokusu o bez nutnosti zadávat nezbytné cestu. Může to být důsledkem nesprávně analýze cesty.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda není zadaná cesta k souboru a je správný.
