# <a name="expression-expected"></a>Byl očekáván výraz
Operátor jako `+` nebo `=` vyžaduje výraz pro pravý operand, ale zdá žádný rozpoznatelném výraz.  
  
 **ID chyby:** BC30201  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte platný výraz pro pravý operand operátoru.
