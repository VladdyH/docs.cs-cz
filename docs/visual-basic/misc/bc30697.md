# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>& č. 39; &lt;membername1&gt;& č. 39; nelze přepsat & č. 39;&lt; membername2&gt;& č. 39; protože se liší podle typu volitelné parametry
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30697  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte alespoň jeden požadovaný parametr jako na jiný datový typ.
