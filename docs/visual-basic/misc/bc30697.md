# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39;&lt;membername1&gt; &#39; nejde přepsat &#39; &lt;membername2&gt; &#39; protože se liší podle typu volitelné parametry
Přetěžované členy musí mít různé datové typy pro jeden nebo více požadovaných parametrů.  
  
 **ID chyby:** BC30697  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte alespoň jeden požadovaný parametr jako na jiný datový typ.
