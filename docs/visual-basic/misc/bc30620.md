# <a name="39option-strict39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Strict&#39; může následovat pouze &#39;na&#39; nebo &#39;vypnuto&#39;
`Option Strict` příkazy musíte zadat buď `On` nebo `Off`. Všechny ostatní hodnoty jsou neplatné.  
  
 **ID chyby:** BC30620  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `On` nebo `Off` pro `Option Strict` příkaz.
