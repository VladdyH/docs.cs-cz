# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Argumenty metody musí být uzavřen v závorkách
Pravidla pro volání procedur jsou jednodušší v novějších verzích jazyka Visual Basic. Všechny argumenty musí být uzavřen v závorkách.  
  
 **ID chyby:** BC30800  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Uvést seznam argumentů v závorkách; Příklad:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
