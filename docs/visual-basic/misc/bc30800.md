# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Metoda argumenty musí být uzavřena v závorkách
Pravidla pro volání procedur jsou jednodušší v novějších verzích [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]. Všechny argumenty musí být uzavřena v závorkách.  
  
 **ID chyby:** BC30800  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Uveďte seznam argumentů v závorkách; například:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
