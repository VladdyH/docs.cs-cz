# <a name="file-specified-by-filename-does-not-use-the-encoding-specified-by-fileencoding"></a>Soubor určený touto FileName nepoužívá kódování určeného FileEncoding
Byl proveden pokus o čtení ze souboru, ale zadaný kódování nepoužívá soubor.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, ujistěte se, že zadáváte správný soubor kódování.
