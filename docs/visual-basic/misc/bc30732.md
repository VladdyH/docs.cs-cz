# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Webové metody nelze vyhodnotit v systému windows ladění
Metody webové služby XML nelze testovat v rámci okna ladění.  
  
 **ID chyby:** BC30732  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Test metody webové služby XML z editoru kódu.
