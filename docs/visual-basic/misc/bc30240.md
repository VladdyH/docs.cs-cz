# <a name="39exit39-must-be-followed-by-39sub39-39function39-39property39-39do39-39for39-39while39-39select39-or-39try39"></a>&#39;Ukončení&#39; musí následovat &#39;Sub&#39;, &#39;funkce&#39;, &#39;vlastnost&#39;, &#39;provést&#39;, &#39;pro&#39;, &#39;při&#39;, &#39;vyberte&#39;, nebo &#39;akci&#39;
`Exit` Příkaz obsahuje neplatné klíčové slovo. `Exit` musíte zadat bloku, ze kterého řízení je potřeba přenést příkaz následující bloku; například `End While`.  
  
 **ID chyby:** BC30240  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte následující platné – klíčové slovo `Exit`, nebo odebrat `Exit` příkaz.
