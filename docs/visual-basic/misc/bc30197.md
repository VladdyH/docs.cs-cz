# <a name="39as39-expected"></a>& č. 39; Jako & č. 39; očekávání
Dojde k deklaraci, která neposkytuje `As` klauzule pro jeden nebo více elementů se deklarovat.  
  
 **ID chyby:** BC30197  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Poskytne `As` klauzuli podle potřeby nebo zapnout `Option Strict Off`.
