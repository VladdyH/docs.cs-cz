# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>& č. 39; & Nastavit č. 39; parametr nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Byl zadán neplatný modifikátor s parametrem v `Set` procedura property.  
  
 **ID chyby:** BC31065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte neplatné modifikátory, jako například `ByRef`, `ParamArray`, a `Optional`.
