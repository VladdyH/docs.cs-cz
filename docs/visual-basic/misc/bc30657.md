# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;název_procedury&gt; &#39; návratový typ, který není podporován nebo typy parametrů, které nejsou podporovány
Postup je deklarována s parametry nebo návratový typ, který nemůže být reprezentována sady Visual Studio.  
  
 **ID chyby:** BC30657  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definujte návratové typy nebo parametry jako podporovaného typu.  
  
-   Vytvořte nový typ, který popisuje data používá tento postup.
