# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;procedurename&gt; &#39; má návratový typ, který není podporován nebo typy parametrů, které nejsou podporovány
Postup je deklarovaný s parametry nebo návratový typ, který nemůže být reprezentovaná Visual Studio.  
  
 **ID chyby:** BC30657  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Návratové typy nebo parametry definujte jako podporovaného typu.  
  
-   Vytvořte nový typ, který popisuje data používá tento postup.
