# <a name="39ltname1gt39-for-the-imports-39ltname2gt39-does-not-refer-to-a-namespace-class-structure-enum-or-module"></a>&#39;&lt;name1&gt; &#39; pro importy &#39; &lt;name2&gt; &#39; neodkazuje na Namespace, třída, struktura, výčet nebo modulu
Pokusili jste se použít `Imports` příkazu na něco, který není `Namespace`, `Class`, `Structure`, `Enum`, nebo `Module`. `Imports` Příkaz importuje názvy oborů názvů z odkazovaného projekty a sestavení, nebo ji importuje názvy oborů názvů, které jsou definované v rámci stejného projektu jako modul, ve kterém se zobrazí příkaz.  
  
 **ID chyby:** BC30467  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte entity, že se pokoušíte importovat, a ujistěte se, že je platná pro použití s `Imports` příkaz.
