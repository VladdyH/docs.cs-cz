# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;Výčet&#39; musí končit odpovídající &#39;End výčtu&#39;
`Enum` Příkaz probíhá, aniž by odpovídající `End Enum` příkaz. `End Enum` Třeba zadat příkaz k ukončení výčtu.  
  
 **ID chyby:** BC30185  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda `Enum` členové jsou správně naformátován.  
  
2.  Přidat `End Enum` příkaz na konec výčtu.
