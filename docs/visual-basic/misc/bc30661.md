# <a name="field-or-property-39ltmembernamegt39-is-not-found"></a>Pole nebo vlastnost &#39; &lt;membername&gt; &#39; nebyl nalezen
Pro atribut je odkazováno na nedefinované pole nebo vlastnost.  
  
 **ID chyby:** BC30661  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte, zda pole nebo vlastnost.  
  
2.  Ověřte, zda projektu můžete přístup ke třídě správný atribut a přidat jej do odkazů projektu v případě potřeby.
