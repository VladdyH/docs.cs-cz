# <a name="39widening39-and-39narrowing39-cannot-be-combined"></a>&#39;Rozšiřující&#39; a &#39;Narrowing&#39; nelze kombinovat
[Operator – příkaz](../../visual-basic/language-reference/statements/operator-statement.md) určuje obě [Widening](../../visual-basic/language-reference/modifiers/widening.md) a [Narrowing](../../visual-basic/language-reference/modifiers/narrowing.md).  
  
 Když definujete operátora převodu, je potřeba deklarovat jako `Widening` nebo `Narrowing`. Toto jsou vzájemně se vylučuje charakteristiky, takže nelze zadat současně.  
  
 **ID chyby:** BC33001  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Buď odeberte `Widening` nebo `Narrowing` – klíčové slovo z `Operator` příkaz. Musíte zadat jeden z nich.
