# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>Nelze zahrnout XML fragment &#39; &lt;objektu elementpath&gt; &#39; souboru &#39; &lt;commentfile&gt;&#39;
`<include>` Značku komentáře jazyka XML obsahuje výraz XPath pro element, který neexistuje, nebo soubor obsahuje chybně formátovaný XML.  
  
 **ID chyby:** BC42320  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda je platný kód XML v souboru.  
  
2.  Ověřte, zda XML fragment v souboru.
