# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>Vlastnost &#39; &lt;propertyname&gt; &#39; je nepodporovaného typu
Vlastnost v odkazovaném sestavení obsahuje typ, který nelze.  
  
 **ID chyby:** BC30643  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte typ, který lze použít s vaší aplikací.
