# <a name="properties-with-no-required-parameters-cannot-be-declared-39default39"></a>Vlastnosti s žádné požadované parametry nelze deklarovat & č. 39; výchozí & č. 39;
Vlastnost zadaná bez požadované parametry, ale s `Default` modifikátor.  
  
 **ID chyby:** BC31048  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Deklarujte požadované parametry pro výchozí vlastnosti.  
  
-   Odeberte `Default` modifikátor z definice vlastnost.
