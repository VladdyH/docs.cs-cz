# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Při vyhodnocování výrazu došlo k dělení nulou
Při vyhodnocování výrazu, jste se pokusili získat dělení čísla nulou.  
  
 **ID chyby:** BC30542  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte hodnoty konstanty a proměnnou k určení, jak v vaší výraz jmenovatel dojde nula.
