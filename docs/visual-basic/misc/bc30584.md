# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;typ&gt; &#39; nelze zděděná více než jednou.
Mají duplicitní `Inherits` příkaz.  
  
 **ID chyby:** BC30584  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `Inherits` příkaz.
