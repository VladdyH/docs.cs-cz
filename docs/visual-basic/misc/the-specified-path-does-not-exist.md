# <a name="the-specified-path-does-not-exist"></a>Zadaná cesta neexistuje.
Zadaná cesta neexistuje, pravděpodobně kvůli chybě kombinovaných řetězců.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že je cesta správná.
