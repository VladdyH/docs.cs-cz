# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>& č. 39; Obnovit & č. 39; příkazy nejsou platné v příkazové podokno

 **ID chyby:** BC30714  
  
## <a name="to-correct-this-error"></a>Oprava této chyby
