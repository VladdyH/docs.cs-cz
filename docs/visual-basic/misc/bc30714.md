# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Obnovit&#39; příkazy nejsou platné v příkazové podokno
Příkazy obnovení lze použít pouze ve zdrojovém kódu.  
  
 **ID chyby:** BC30714  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odebrat `Resume` příkazy z ladění kódu.
