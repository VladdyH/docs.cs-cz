# <a name="constants-must-have-a-value"></a>Konstanty musí mít hodnotu
Konstanta je deklarovaná bez jeho přiřazení hodnotu.  
  
 **ID chyby:** BC30438  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přiřadíte hodnotu konstanty.
