# <a name="type-expected"></a>Očekávaný typ
Zdrojový kód znaků dojít k, které nejsou rozpoznat jako název typu. Kontext žádá základní datový typ nebo název deklarovaný element představující typu.  
  
 **ID chyby:** BC30182  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, že typ byla definována.  
  
-   Zkontrolujte, jestli typ název je napsán správně.
