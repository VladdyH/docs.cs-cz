# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39;Nové&#39; nelze použít pro třídu &#39; &lt;classname&gt; &#39; protože obsahuje &#39;MustOverride&#39; člena, který nebyl přepsán
Pokusili jste se použít `New` na třídu, která obsahuje `MustOverride` člena, který nebyl přepsán.  
  
 **ID chyby:** BC30376  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `New` příkaz.
