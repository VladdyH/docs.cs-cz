# <a name="statement-cannot-appear-within-a-property-body"></a>Příkaz nemůže být použit v těle vlastnosti
Příkaz nelze vložit v těle vlastnosti. Konec předpokládá, že vlastnost.  
  
 Uvnitř vlastnost byl zjištěn neočekávaný příkaz.  
  
 **ID chyby:** BC30634  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ověřte, zda vaše vlastnost používá správnou syntaxi a ukončí se `End``Property` vytvořit.
