# <a name="enums-must-be-declared-as-an-integral-type"></a>Výčty musí být deklarován jako typ integrální
Můžete použít v výčty pouze platné typy jsou `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, a `ULong`. Můžete použít jiné datové typy.  
  
 **ID chyby:** BC30650  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte typ dat `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, nebo `ULong`.
