# <a name="operator-39ltoperatorgt39-must-have-either-one-or-two-parameters"></a>Operátor &#39; &lt;operátor&gt; &#39; musí mít jeden nebo dva parametry
Operátor, který může být buď unární nebo binární je definována bez parametrů nebo s více než dva parametry.  
  
 Operátor unární nebo binární musí mít jeden nebo dva parametry.  
  
 **ID chyby:** BC33016  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Upravte definici určit jeden nebo dva parametry.  
  
-   Pokud potřebujete žádné parametry nebo větší než dvě, je nutné použít [příkaz funkce](../../visual-basic/language-reference/statements/function-statement.md) k definování `Function` postup místo operátor.
