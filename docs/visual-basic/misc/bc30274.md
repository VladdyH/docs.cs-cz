# <a name="parameter-39ltparameternamegt39-of-39ltmethodnamegt39-already-has-a-matching-argument"></a>Parametr & č. 39; &lt;parametername&gt;& č. 39; & č. 39;&lt; methodName&gt;& č. 39; už má odpovídající argument
Argument je předána podle názvu více než jednou ve stejném volání procedury.  
  
 **ID chyby:** BC30274  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, pokud jeden ze zadaných názvů argument je zadáno chybně.  
  
-   Odeberte duplicitní argument ze seznamu argumentů.
