# <a name="39get39-is-already-declared"></a>& č. 39; Get & č. 39; je již deklarována
A `Get` příkaz vyskytuje více než jednou v bloku kódu. `Get` Příkaz deklaruje `Get` procedury vlastnosti použít k získání hodnoty vlastnosti.  
  
 **ID chyby:** BC30443  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `Get` příkaz.
