# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39;Případ&#39; nelze provést &#39;Case Else&#39; ve stejné &#39;vyberte&#39; – příkaz
A `Case Else` příkaz uvádí příkazy, které mají být provedeny, pokud není nalezena žádná shoda pro počáteční `Case`. A `Case` příkaz nebyl nalezen po `Case Else` ve stejné `Select` bloku.  
  
 **ID chyby:** BC30321  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `Case Else` do příslušného umístění po `Case` příkaz.
