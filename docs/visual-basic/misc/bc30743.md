# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Typ &#39; &lt;typename&gt; &#39; není definován nebo není v relaci ladění načíst modul, který ji obsahuje
Typ bylo odkazováno, není k dispozici nebo není definován.  
  
 **ID chyby:** BC30743  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, že název typu je napsán správně.  
  
2.  Definování typu v rámci aktuální třídy nebo modulu.
