# <a name="39ltelementnamegt39-cannot-be-declared-39partial39-because-partial-methods-must-be-subs"></a>&#39;&lt;Vlastnost elementname&gt; &#39; nelze deklarovat &#39;částečné&#39; protože částečné metody musí být předplatných
Pouze `Sub` postupy lze deklarovat jako částečné metody. Například následující kód způsobí této chybě, protože `partialMethod` je funkce.  
  
```  
' Partial Private Function partialMethod(ByVal n As Integer) As Integer  
' End Function  
```  
  
 **ID chyby:** BC31437  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Převést jsou deklarace jako částečné metodu `Sub`.  
  
-   V takovém případě nepoužívejte částečné metoda.
