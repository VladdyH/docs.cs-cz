# <a name="declare-statements-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Deklarovat příkazy v modulu nelze deklarovat &#39; &lt;– specifikátor&gt;&#39;
A `Declare` deklarace má specifikátor, který není platný v rámci `Module` deklarace. Moduly nikdy může být vytvořena, nepodporují dědičnosti a nemohou implementovat rozhraní.  
  
 **ID chyby:** BC30786  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
