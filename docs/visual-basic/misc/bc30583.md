# <a name="39ltmethodnamegt39-cannot-be-implemented-more-than-once"></a>& č. 39; &lt;methodname&gt;& č. 39; nemůže být implementována více než jednou.
Pokusili jste se implementovat metodu, která již byla implementována.  
  
 **ID chyby:** BC30583  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte název metody a ujistěte se, že máte napsaný správně.  
  
2.  Pokud je metoda redundantní, odeberte navíc deklaraci.
