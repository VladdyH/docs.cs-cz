# <a name="39ltmethodnamegt39-cannot-be-implemented-more-than-once"></a>&#39;&lt;methodName&gt; &#39; nelze implementovat více než jednou.
Pokusili jste se implementovat metodu, která již byla implementována.  
  
 **ID chyby:** BC30583  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte název metody a ujistěte se, že máte napsaný správně.  
  
2.  Pokud je metoda redundantní, odeberte navíc deklaraci.
