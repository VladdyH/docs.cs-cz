# <a name="keyword-is-not-valid-as-an-identifier"></a>Není platný jako identifikátor – klíčové slovo
Klíčové slovo nastane, kdy je potřeba název elementu. Kontext volání pro daný element jako proměnné nebo procedury.  
  
 **ID chyby:** BC30183  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Klíčové slovo nahraďte platný název elementu.
