# <a name="xml-comment-type-parameter-must-have-a-39name39-attribute"></a>Parametr typu komentáře XML musí mít & č. 39; název & č. 39; atribut
Parametr typu komentáře XML musí mít atribut 'name'. Komentáře XML budou ignorovány.  
  
 Vyžaduje značka komentáře XML `name` atribut, který chybí nebo není úplná.  
  
 **ID chyby:** BC42318  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat nebo dokončení požadované `name` atribut ke značce.
