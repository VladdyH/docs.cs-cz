# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39;Ukončení operátor&#39; musí předcházet odpovídající &#39;– operátor&#39;
`End Operator` Příkazu se zobrazí s neexistuje odpovídající `Operator` deklaraci předchozích ho.  
  
 **ID chyby:** BC33007  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte `End Operator` příkaz, pokud je redundantní.  
  
-   Zadejte chybějící `Operator` postup, pokud některý chybí.  
  
-   Přesunout `End Operator` příkaz na příslušné místo v kódu.
