# <a name="could-not-complete-operation-since-target-directory-is-under-source-directory"></a>Operaci nelze provést, protože cílový adresář je pod zdrojový adresář
Cyklická operace se nezdařila. Cyklická operace cyklus a proto nelze dokončit. Například objekt A pokusit dědění z objektu B, který zase dědí z objektu A.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Když je zděděný, ujistěte se, že neexistují žádné cyklické odkazy.
