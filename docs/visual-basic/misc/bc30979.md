# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Infer&#39; může následovat pouze &#39;na&#39; nebo &#39;vypnuto&#39;
`Option Infer` příkazy musíte zadat buď `On` nebo `Off`. Jiné hodnoty nejsou platné.  
  
 **ID chyby:** BC30979  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte buď `Option Infer On` nebo `Option Infer Off`.
