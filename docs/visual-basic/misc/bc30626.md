# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39; příkaz musí končit odpovídající &#39;End Namespace&#39;
`Namespace` bloky musí končit `End Namespace` příkaz.  
  
 **ID chyby:** BC30626  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Namespace` příkaz na konec `Namespace` bloku.
