# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>& č. 39; Namespace & č. 39; příkaz musí končit odpovídající & č. 39; Ukončení Namespace & č. 39;
`Namespace`bloky musí končit `End Namespace` příkaz.  
  
 **ID chyby:** BC30626  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Namespace` příkaz na konec `Namespace` bloku.
