# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Chyba při vkládání Win32 manifest: soubor manifestu &lt;soubor&gt; nebyl nalezen
Pro zadaný soubor `/win32manifest` – možnost kompilátoru nebyl nalezen.  
  
 **ID chyby:** BC31192  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zadejte platný název souboru a cesty pro `/win32manifest` – možnost kompilátoru.
