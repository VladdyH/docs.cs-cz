# <a name="39removehandler39-is-already-declared"></a>&#39;RemoveHandler&#39; je již deklarována
Více než jeden `RemoveHandler` deklarace zobrazovaly v deklaraci vlastní události. `RemoveHandler` Deklarace deklaruje postup slouží k odebírání obslužné rutiny události.  
  
 **ID chyby:** BC31128  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte redundantní `RemoveHandler` příkaz.
