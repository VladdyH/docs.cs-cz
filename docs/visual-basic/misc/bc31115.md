# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39;AddHandler&#39; deklarace musí končit odpovídající &#39;End AddHandler&#39;
`AddHandler` Musí končit deklarace `End AddHandler` příkaz.  
  
 **ID chyby:** BC31115  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda `AddHandler` deklarace končí `End AddHandler` příkaz.
