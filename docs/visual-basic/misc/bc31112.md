# <a name="statement-cannot-appear-within-an-event-body"></a>Příkaz nemůže být použit v těle události
Příkaz nelze vložit v těle události. Konec předpokládá, že událost.  
  
 Příkaz, který není platný v těle události se zobrazí uvnitř jeden.  
  
 **ID chyby:** BC31112  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `End Event` před příkaz.
