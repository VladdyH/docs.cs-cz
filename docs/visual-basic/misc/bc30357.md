# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>& č. 39; dědí & č. 39; příkazy musí předcházet všechny deklarace v rozhraní
`Inherits` Příkaz musí být první neprázdné, bez komentář řádek v definici třídy.  
  
 **ID chyby:** BC30357  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout vaše `Inherits` prohlášení, které předchází všechny deklarace.
