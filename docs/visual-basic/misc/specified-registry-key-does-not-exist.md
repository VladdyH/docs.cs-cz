# <a name="specified-registry-key-does-not-exist"></a>Zadaný klíč registru neexistuje.
Klíč registru, který byl zadán neexistuje.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte klíč registru, který se zadávání a ujistěte se, že jste správně zadali cestu.  
  
-   Vytvořte klíč registru dřív, než se do něj zapisovat.
