# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray parametr musí být pole.
A `ParamArray` argument musí být deklarován jako typ jednorozměrné pole. Deklaraci nezahrnuje závorkách po `ParamArray` argument název.  
  
 **ID chyby:** BC30050  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat pár závorek po `ParamArray` argument název.
