# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>& č. 39; ReDim & č. 39; příkazy vyžadují, aby v závorkách seznamu nové hranice Každá dimenze pole
Zadejte novou velikost pole v rámci `ReDim` příkaz.  
  
 **ID chyby:** BC30670  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte velikost každé pořadí pole; například:  
  
    ```  
    ReDim arr(5, 6)  
    ```
