# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Členové rozhraní musí být metody, vlastnosti, události nebo definice typů
Pouze vnořené typy, metody, vlastnosti a události může být členy rozhraní.  
  
 **ID chyby:** BC30602  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že položka je metoda, vlastnost, události nebo vnořené typy, jako je například výčet.
