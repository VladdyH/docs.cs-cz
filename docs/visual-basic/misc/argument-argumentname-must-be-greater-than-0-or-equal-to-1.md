# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>Argument &#39; &lt;název argumentu argumentname&gt; &#39; musí být větší než 0 nebo rovno -1
Hodnota pro argument je menší než 0, ale není rovno -1.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ujistěte se, že hodnota argumentu spadá v platném rozsahu.  
  
2.  Pokud je argument zadaný výpočtem, zkontrolujte procesy výpočtu hodnoty.
