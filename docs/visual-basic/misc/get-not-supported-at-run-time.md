# <a name="get-not-supported-at-run-time"></a>Metoda Get není podporována v době běhu
Pokusili jste se načíst vlastnost za běhu, který je k dispozici pouze v době návrhu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Zkontrolujte vlastnosti a určit, za jakých podmínek může být nastaven.  
  
2.  Odeberte odkaz na vlastnost.
