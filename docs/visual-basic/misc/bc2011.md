# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>Nelze otevřít soubor odpovědí & č. 39; &lt;filename&gt;& č. 39;
Soubor, který se pokoušíte získat přístup k není k dispozici.  
  
 **ID chyby:** BC2011  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že je stále aktivní připojení k síti.  
  
-   Ověřte oprávnění u souboru.
