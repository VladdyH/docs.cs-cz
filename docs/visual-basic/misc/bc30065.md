# <a name="39exit-sub39-is-not-valid-in-a-function-or-property"></a>& č. 39; Ukončit dílčí & č. 39; není platný v funkci nebo vlastnost
`Exit Sub`Zobrazí se v `Function` postup nebo `Property` postupu. `Exit` Příkaz musí odpovídat bloku ve kterém dochází.  
  
 **ID chyby:** BC30065  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nahraďte `Exit Sub` s `Exit Function` nebo `Exit Property` příkaz podle potřeby.
