<span data-ttu-id="53b9b-103">– Znak typu nelze použít na název vlastnosti, když deklarujte instanci anonymního typu.</span><span class="sxs-lookup"><span data-stu-id="53b9b-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="53b9b-104">Datový typ vlastnosti je odvozeno z hodnotu přiřazenou k němu.</span><span class="sxs-lookup"><span data-stu-id="53b9b-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="53b9b-105">Například následující deklarace nejsou platné.</span><span class="sxs-lookup"><span data-stu-id="53b9b-105">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.  
'Dim anon1 = New With {.ID$ = "abc"}  
'Dim anon2 = New With {.ID$ = 42}  
```
