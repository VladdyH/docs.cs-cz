# <a name="39compare39-expected"></a>&#39;Porovnání&#39; očekávání
`Option` Příkaz určuje `Text` nebo `Binary`, ale ne `Compare`.  
  
 **ID chyby:** BC30208  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Compare` – klíčové slovo chcete `Option` příkaz, například `Option Compare Binary`.
