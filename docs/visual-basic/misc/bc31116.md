# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39;RemoveHandler&#39; deklarace musí končit odpovídající &#39;End RemoveHandler&#39;
A `RemoveHandler` musí končit deklarace `End RemoveHandler` příkaz.  
  
 **ID chyby:** BC31116  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte, zda `RemoveHandler` deklarace končí `End RemoveHandler` příkaz.
