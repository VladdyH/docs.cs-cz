# <a name="39next39-expected"></a>&#39;Další&#39; očekávání
`On Error Resume` Konstrukce probíhá, aniž by končí `Next`.  
  
 **ID chyby:** BC30003  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidat `Next` – klíčové slovo na konec `On Error` příkaz.
