# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Vlastnost&gt; parametry nelze deklarovat & č. 39; ByRef & č. 39;
Parametry pro vlastnosti musí být deklarován pomocí `ByVal` modifikátor.  
  
 **ID chyby:** BC30651  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Definujte parametry vlastnost jako `ByVal`.
