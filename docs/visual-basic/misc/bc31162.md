# <a name="expected-closing-39gt39-for-xml-cdata-section"></a>Očekávaný ukončovací &#39;]]&gt; &#39; pro oddíl XML CDATA
Literál XML CDATA chybí požadované ukončovací ']] >' znaků.  
  
 **ID chyby:** BC31162  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte požadované ukončovací ']] > se znaky na konec literál XML CDATA.
