# <a name="39sub39-or-39function39-expected-after-39delegate39"></a>&#39;Sub –&#39; nebo &#39;funkce&#39; očekávané po &#39;delegáta&#39;
A `Delegate` příkaz neurčuje `Sub` nebo `Function` postupu. `Sub` Nebo `Function` okamžitě postupujte – klíčové slovo `Delegate` – klíčové slovo.  
  
 **ID chyby:** BC30278  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `Sub` nebo `Function` – klíčové slovo ihned po `Delegate`.  
  
2.  Zadejte název procedury, seznam argumentů a návratový typ podle potřeby.
