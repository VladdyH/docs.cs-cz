# <a name="39redim39-cannot-change-the-number-of-dimensions"></a>& č. 39; ReDim & č. 39; nelze změnit počet dimenzí
Operace se pokusí použít `ReDim` příkaz Chcete-li změnit pořadí pole (počet dimenzí). `ReDim`můžete změnit velikost jeden nebo více rozměry pole, které již byla dříve deklarována, ale nedá se změnit pořadí tohoto pole.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, kterou chcete změnit pořadí tohoto pole a není velikosti jeho dimenzí a pokud je to možné, používejte `Dim` deklarovat nové pole se požadované pořadí.
