# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39;&lt;methodName&gt; &#39; nejde přepsat metodu, která byla stíněné
Metoda došlo k pokusu o přepsání metody stín jiného člena.  
  
 **ID chyby:** BC31406  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Použití `Shadows` modifikátor přepsat stíněné členy.
