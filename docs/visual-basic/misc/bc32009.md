# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Deklarační příkazy metoda musí být první logického řádku
A `Function` nebo `Sub` příkaz následuje oddělovač příkaz dvojtečkou (:). `Function`nebo `Sub` musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32009  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
