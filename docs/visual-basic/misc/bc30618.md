# <a name="39namespace39-statements-can-occur-only-at-file-or-namespace-level"></a>&#39;Namespace&#39; příkazy lze používat pouze na úrovni souboru nebo oboru názvů
`Namespace` příkazy musí být po `Option` příkazy, `Imports` příkazy a globálními atributy, ale před všechny deklarace ve zdrojovém souboru.  
  
 **ID chyby:** BC30618  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přesunout `Namespace` příkaz na začátek oboru názvů deklarace nebo zdrojový soubor.
