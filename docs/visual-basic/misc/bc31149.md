# <a name="duplicate-xml-attribute-39attributename39"></a>Duplicitní atribut XML &#39;attributeName&#39;
Atribut parametru literál XML elementu odpovídá atribut, který dříve byla definována pro literál XML elementu. Názvy atributů v literálech XML element musí být jedinečný.  
  
 **ID chyby:** BC31149  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přejmenujte atribut název, který je jedinečný pro literál XML elementu.
