# <a name="39end-sub39-must-be-the-first-statement-on-a-line"></a>& č. 39; End Sub & č. 39; musí být první příkaz na řádek
`End Sub` Příkaz následuje oddělovač příkaz dvojtečkou (:). `End Sub`musí být jediným příkazem na stejném řádku zdroje.  
  
 **ID chyby:** BC32031  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Oddělte více příkazů na různé řádky.
