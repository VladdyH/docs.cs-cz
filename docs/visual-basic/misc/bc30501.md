# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;Sdílené&#39; nelze kombinovat s &#39; &lt;specifikátor&gt; &#39; na deklaraci – metoda
Pokusili jste se zkombinovat `Shared` modifikátor s specifikátorem, například `Overridable`, `NotOverridable`, nebo `MustOverride`, který není platný v kombinaci, v deklaraci metody.  
  
 **ID chyby:** BC30501  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
