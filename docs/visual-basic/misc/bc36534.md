# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>Výraz nelze převést na strom výrazu se nezdařilo
Většina výrazy mohou být převedeny na stromů výrazů, ale některé nelze. Tato chyba znamená, že byl proveden pokus o převodu výrazu, kterou nelze převést.  
  
 **ID chyby:** BC36534
