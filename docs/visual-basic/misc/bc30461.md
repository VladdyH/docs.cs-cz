# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>Třídy nelze deklarovat & č. 39; &lt;specifikátor&gt;& č. 39;
Pokusili jste se deklarovat `Class` nebo `Module` s Neplatný specifikátor.  
  
 **ID chyby:** BC30461  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Odeberte specifikátor.
