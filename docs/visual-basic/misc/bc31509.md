# <a name="unable-to-open-resource-file-39ltfilenamegt39-lterrorgt"></a>Nelze otevřít zdrojový soubor &#39; &lt;filename&gt;&#39;: &lt;chyba&gt;
Došlo k chybě při otevírání souboru zadaný prostředek.  
  
 **ID chyby:** BC31509  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Ověřte, zda zadaný název a umístění jsou správné.  
  
2.  Ověřte, že soubor není poškozený.
