# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Počet upozornění &#39; &lt;číslo&gt; &#39; pro možnost &#39; &lt;název_možnosti&gt; &#39; nejde konfigurovat nebo platná
Upozornění ID není platný. ID musí být celočíselné hodnoty vyšší než 0.  
  
 **ID chyby:** BC2026  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zkontrolujte zadané a ujistěte se, že je platné ID upozornění.
