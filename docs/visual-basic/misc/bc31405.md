# <a name="optional-parameters-cannot-have-structure-types"></a>Volitelné parametry nemohou být strukturovaného typu
Volitelné parametry nemůže být strukturovaného typu.  
  
 **ID chyby:** BC31405  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Deklarujte parametry typu Struktura podle potřeby.  
  
2.  Statická třída člena se místo strukturu deklarujte.
