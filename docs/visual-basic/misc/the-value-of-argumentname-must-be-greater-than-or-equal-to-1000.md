# <a name="the-value-of-ltargumentnamegt-must-be-greater-than-or-equal-to-1000"></a>Hodnota &lt;název argumentu argumentname&gt; musí být větší než nebo rovno 1 000
Hodnota <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> vlastnost musí být větší než nebo rovno 1 000.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Vlastnost určuje maximální počet bajtů v souboru protokolu před vytvořením nového souboru protokolu.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Nastavte <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> vlastnost na číslo větší než nebo rovno 1 000.
