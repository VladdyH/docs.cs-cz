# <a name="security-attribute-39ltattributenamegt39-cannot-be-applied-to-a-module"></a>Atribut zabezpečení &#39; &lt;attributename&gt; &#39; nelze použít k modulu
Modul common language runtime nepřijímá atribut zabezpečení na modul.

**ID chyby:** BC30131

## <a name="to-correct-this-error"></a>Oprava této chyby

- Odeberte atribut z deklaraci.
