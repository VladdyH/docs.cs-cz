# <a name="expected-closing-3939-for-xml-entity"></a>Očekávané koncové &#39;; &#39; entity XML
Odkaz na entity XML (například `>`) chybí uzavírací středníkem (;).  
  
 **ID chyby:** BC31178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte středník na konci odkaz na entity XML.
