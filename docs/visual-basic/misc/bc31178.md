# <a name="expected-closing-3939-for-xml-entity"></a>Očekávaný ukončovací & č. 39; & č. 39; pro XML entity
Odkaz na entity XML (například `>`) chybí uzavírací středníkem (;).  
  
 **ID chyby:** BC31178  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Zadejte středník na konci odkaz na entity XML.
