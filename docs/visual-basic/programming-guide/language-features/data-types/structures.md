# <a name="structures-visual-basic"></a>Struktury (Visual Basic)
A *struktura* je generalizaci uživatelem definovaný typ (UDT) podporované v předchozích verzích nástroje jazyka Visual Basic. Kromě polí můžou zpřístupnit struktury vlastnosti, metod a události. Struktury můžete implementovat jednu nebo více rozhraní, a můžou deklarovat individuální úrovně přístupu pro každé pole.  
  
 Můžete kombinovat datovými položkami různých typů vytvořit strukturu. Struktury přidružuje jednu nebo více *elementy* mezi sebou a se strukturou sám sebe. Když jste definice struktury, stane se *složeného datového typu*, a můžou deklarovat proměnné daného typu.  
  
 Struktury jsou užitečné, pokud chcete jednu proměnnou pro uložení několik souvisejících informací. Například můžete chtít zachovat název zaměstnance, telefonní číslo a mzda společně. Tyto informace můžete použít několik proměnné, nebo může definovat strukturu a použít jej pro proměnnou jediného zaměstnance. Výhodou strukturu vyvstává zřejmá, pokud máte mnoho pracovníků a proto velký počet instancí proměnné.
