# <a name="creating-xml-in-visual-basic"></a>Vytvoření XML v jazyce Visual Basic
Visual Basic umožňuje používat *XML – literály* přímo v kódu. Představuje literálu syntaxe XML [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] objekty a je podobná syntaxe XML 1.0. Díky tomu je snazší vytvářet elementů XML, dokumenty a fragmenty prostřednictvím kódu programu, protože váš kód má stejné struktuře jako poslední XML.
