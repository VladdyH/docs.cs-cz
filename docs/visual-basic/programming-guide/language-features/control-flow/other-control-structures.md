# <a name="other-control-structures-visual-basic"></a>Ostatní řídicí struktury (Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]poskytuje řídicí struktury, které vám pomůžou odstranění prostředku nebo snižte počet, kolikrát je nutné zopakovat odkaz na objekt.  
  
## <a name="usingend-using-construction"></a>Pomocí... Ukončení pomocí konstrukce  
 `Using...End Using` Konstrukce vytvoří blok příkaz, ve kterém provedete využití prostředků, jako je připojení SQL. Volitelně můžete získat prostředek s `Using` příkaz. Po ukončení `Using` bloku [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] automaticky uvolní prostředku, aby byla dostupná pro ostatní kódu pro použití. Prostředek musí být místní a na jedno použití. Další informace najdete v tématu [pomocí příkazu](../../../../visual-basic/language-reference/statements/using-statement.md).  
  
## <a name="withend-with-construction"></a>S... Končit konstrukce  
 `With...End With` Konstrukce umožňuje zadat odkaz na objekt jednou a poté spusťte řadu příkazů, které přístup k její členy. To můžete zjednodušit kód a zlepšit výkon, protože [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] není nutné znovu vytvořit referenční informace pro každý příkaz, který přistupuje k ho. Další informace najdete v tématu [s... End With – příkaz](../../../../visual-basic/language-reference/statements/with-end-with-statement.md).
