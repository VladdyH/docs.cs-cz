# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="0e4f9-102">Rozlišení přetěžování (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="0e4f9-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="0e4f9-103">Když Visual Basic – kompilátor dojde volání procedury, která je definována v několika verzích přetížené, kompilátor musíte rozhodnout, které přetížení volání.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="0e4f9-104">Dělá to pomocí následujících kroků:</span><span class="sxs-lookup"><span data-stu-id="0e4f9-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="0e4f9-105">**Přístupnost**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-105">**Accessibility.**</span></span> <span data-ttu-id="0e4f9-106">Eliminuje žádné přetížení s úroveň přístupu, která brání jeho volání volající kód.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="0e4f9-107">**Počet parametrů.**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-107">**Number of Parameters.**</span></span> <span data-ttu-id="0e4f9-108">Eliminuje žádné přetížení, které definuje jiný počet parametrů, než se zadávají ve volání.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="0e4f9-109">**Datové typy parametrů.**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-109">**Parameter Data Types.**</span></span> <span data-ttu-id="0e4f9-110">Kompilátor poskytuje instance metody předvoleb nad rozšiřující metody.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="0e4f9-111">Pokud se najde libovolné metody instance, který vyžaduje pouze rozšiřující převody tak, aby odpovídaly volání procedury, všechny rozšiřující metody jsou vyřazen a kompilátor pokračuje pouze kandidáty metoda instance.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="0e4f9-112">Pokud se žádná instance metoda nenajde, pokračuje v instanci a rozšiřující metody.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="0e4f9-113">V tomto kroku eliminuje žádné přetížení, pro který datové typy argumentů volání nelze převést na typy parametrů, které jsou definované v přetížení.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="0e4f9-114">**Zužující převody.**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="0e4f9-115">Eliminuje žádné přetížení, které vyžaduje zužující převod z volání typy argumentů pro typy definované parametrů.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="0e4f9-116">To platí určení, zda kontrola typu přepnout ([Option Strict – příkaz](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) je `On` nebo `Off`.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="0e4f9-117">**Minimálně rozšíření.**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-117">**Least Widening.**</span></span> <span data-ttu-id="0e4f9-118">Kompilátor zvažuje zbývající přetížení v párech.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="0e4f9-119">Pro každý pár porovná datové typy definovaných parametrů.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="0e4f9-120">Pokud typy v jednom z přetížení všechny rozšířit na odpovídající typy v dalších, kompilátor eliminuje ty druhé.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="0e4f9-121">To znamená zachová přetížení, která vyžaduje nejnižší možné rozšíření.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="0e4f9-122">**Na jednom uchazeči.**</span><span class="sxs-lookup"><span data-stu-id="0e4f9-122">**Single Candidate.**</span></span> <span data-ttu-id="0e4f9-123">Pokračuje considering přetížení v párech dokud jenom jeden přetížení zůstane a jeho přeloží volání této přetížení.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="0e4f9-124">Pokud kompilátor nelze zmenšit přetížení do jediného kandidáta, vygeneruje se chyba.</span><span class="sxs-lookup"><span data-stu-id="0e4f9-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
