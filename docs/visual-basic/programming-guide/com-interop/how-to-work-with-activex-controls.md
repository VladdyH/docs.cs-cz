# <a name="how-to-work-with-activex-controls-visual-basic"></a>Postupy: Práce s ovládacími prvky ActiveX (Visual Basic)
ActiveX – ovládací prvky jsou komponenty modelu COM nebo objekty, které můžete vložit do webové stránky nebo jiná aplikace zabalené funkce, které má někdo jiný naprogramovaný tak znovu použít. ActiveX – ovládací prvky vytvořených pro Visual Basic 6.0 a starší verze můžete použít k přidání funkcí do **sada nástrojů** z [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>K přidávání ovládacích prvků ActiveX do sady nástrojů  
  
1.  Na **nástroje** nabídky, klikněte na tlačítko **výběr položek sady nástrojů**.  
  
     **Zvolte nástrojů** zobrazí se dialogové okno.  
  
2.  Klikněte **COM – součásti** kartě.  
  
3.  Zaškrtněte políčko vedle ovládací prvek ActiveX, které chcete použít a pak klikněte na **OK**.  
  
     Nový ovládací prvek se zobrazuje pomocí jiných nástrojů v **sada nástrojů**.  
  
    > [!NOTE]
    >  Můžete použít nástroj Aximp ručně vytvořit sestavení vzájemné spolupráce pro ovládací prvky ActiveX. Další informace najdete v tématu [Aximp.exe (Windows Forms ActiveX – Importér ovládacích prvků)](http://msdn.microsoft.com/library/482c0d83-7144-4497-b626-87d2351b78d0).
