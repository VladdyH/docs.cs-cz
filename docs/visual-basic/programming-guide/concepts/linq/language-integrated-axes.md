# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Language-Integrated osy v jazyce Visual Basic (technologie LINQ to XML)
Tato část popisuje funkce přímo integrovaná [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] jazyka usnadňují přístup XML. Mnoho příkladů v LINQ do dokumentace XML pomocí těchto integrované [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] osy.  
  
## <a name="in-this-section"></a>V tomto oddílu  
  
|Téma|Popis|  
|-----------|-----------------|  
|[Vlastnost osy podřízeného souboru XML](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Poskytuje přístup k podřízené objekty daného jednu z následujících: <xref:System.Xml.Linq.XElement> objektu, <xref:System.Xml.Linq.XDocument> objektu, kolekce <xref:System.Xml.Linq.XElement> objekty nebo kolekci <xref:System.Xml.Linq.XDocument> objekty. Je ekvivalentní této osy <xref:System.Xml.Linq.XContainer.Elements%2A> osy.|  
|[Vlastnost osy nástupce XML](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Poskytuje přístup k následníky následující: <xref:System.Xml.Linq.XElement> objektu, <xref:System.Xml.Linq.XDocument> objektu nebo kolekci <xref:System.Xml.Linq.XElement> objekty. Je ekvivalentní této osy <xref:System.Xml.Linq.XContainer.Descendants%2A> osy.|  
|[Vlastnost osy atributu XML](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Poskytuje přístup k atribut <xref:System.Xml.Linq.XElement> objektu. Je této osy zhruba odpovídá <xref:System.Xml.Linq.XElement.Attribute%2A> osy. Této osy se liší od <xref:System.Xml.Linq.XElement.Attribute%2A> osy v tom, že ji vrací hodnotu pro atribut, není <xref:System.Xml.Linq.XAttribute> objektu.|  
|[Vlastnost indexeru rozšíření](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Poskytuje přístup k jednotlivé elementy v kolekci.|
