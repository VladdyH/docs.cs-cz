# <a name="pure-functional-transformations-of-xml-visual-basic"></a>Čistý funkční transformace XML (Visual Basic)
Tato část obsahuje návod funkční transformace pro formát XML. To zahrnuje vysvětlení hlavní koncepty a jazyk vytvoří, musíte pochopit použít funkční transformace a příklady použití funkční transformace k manipulaci s dokument XML. I když v tomto kurzu poskytuje LINQ příklady kódu XML, všechny základní koncepty platí také pro jiné technologie LINQ.  
  
 [Kurz: manipulace s obsahu v dokumentu WordprocessingML (Visual Basic)](../../../../visual-basic/programming-guide/concepts/linq/tutorial-manipulating-content-in-a-wordprocessingml-document.md) kurzu poskytuje řadu příklady, každá je postavena na předchozí. Tyto příklady ukazují funkční způsob čistý transformational manipulace s kódem XML.  
  
 Tento kurz předpokládá praktické znalosti jazyka Visual Basic. Podrobné sémantiku jazykové konstrukty nejsou k dispozici v tomto kurzu, ale jsou uvedeny odkazy na dokumentaci jazyk podle potřeby.  
  
 Taky se předpokládá praktické znalosti základní počítač vědecké účely koncepty a XML, včetně obory názvů XML.
