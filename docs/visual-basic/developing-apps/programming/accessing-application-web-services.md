# <a name="accessing-application-web-services-visual-basic"></a>Přístup k aplikačním webovým službám (Visual Basic)
`My.WebServices` Objekt poskytuje instanci každé webové služby odkazuje v aktuálním projektu. Každá instance je vytvořena na vyžádání. Tyto webové služby můžete přistupovat prostřednictvím vlastnosti `My.WebServices` objektu. Název vlastnosti je stejný jako název webové služby, který má přístup k vlastnosti. Všechny třídy, která dědí z <xref:System.Web.Services.Protocols.SoapHttpClientProtocol> je webová služba.
