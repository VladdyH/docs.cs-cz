# <a name="accessing-application-settings-visual-basic"></a>Přístup k nastavení aplikace (Visual Basic)
Tento oddíl obsahuje témata s popisem `My.Settings` objektu a úlohy umožňuje provést.  
  
## <a name="mysettings"></a>My.Settings  
 Vlastnosti `My.Settings` objekt poskytnout přístup k nastavení aplikace. Chcete-li přidat nebo odebrat nastavení, použijte **nastavení** podokně **Návrhář projektu**.  
  
 Metody `My.Settings` objekt umožňují uložit aktuální nastavení uživatele nebo vrátit zpět uživatelské nastavení pro poslední uložené hodnoty.
