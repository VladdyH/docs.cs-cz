# <a name="help--visual-basic"></a>/help, /? (Visual Basic)
Zobrazí možnosti kompilátoru.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
/help  
' -or-  
/?  
```  
  
## <a name="remarks"></a>Poznámky  
 Pokud v kompilaci zahrnete tuto možnost, žádná výstupní soubor je vytvořen a žádné kompilace probíhá.  
  
> [!NOTE]
>  `/help` Možnost není k dispozici v rámci [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] vývojového prostředí; je k dispozici pouze při kompilaci z příkazového řádku.  
  
## <a name="example"></a>Příklad  
 Následující kód zobrazí nápovědu z příkazového řádku.  
  
```  
vbc /help  
```
