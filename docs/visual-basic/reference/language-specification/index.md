# <a name="visual-basic-language-specification"></a>Specifikace jazyka Visual Basic
Specifikace jazyka Visual Basic je autoritativní zdroj pro odpovědi na všechny dotazy o gramatice jazyka Visual Basic a syntaxi. Obsahuje podrobné informace o jazyce, včetně počtu bodů nejsou zahrnuta v referenční dokumentace jazyka Visual Basic.  
  
 Tato specifikace je k dispozici na [Microsoft Download Center](https://go.microsoft.com/fwlink/?LinkId=188623).  
  
Tento web obsahuje [specifikace jazyka Visual Basic 11](../../../../_vblang/spec/introduction.md). Je vytvořen ze souborů Markdown součástí [úložiště GitHub dotnet/vblang](https://github.com/dotnet/vblang/blob/master/spec/README.md).

Problémy na specifikaci by měl být vytvořen v [dotnet/vblang](https://github.com/dotnet/vblang/issues) úložiště. Nebo, pokud vás zajímá opravení chyb můžete najít, můžete odeslat [žádostí o přijetí změn](https://github.com/dotnet/vblang/pulls) do stejného úložiště.
