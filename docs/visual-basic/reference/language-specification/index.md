# <a name="visual-basic-language-specification"></a>Specifikace jazyka Visual Basic
Specifikace jazyka Visual Basic je autoritativní zdroj odpovědi na všechny otázky týkající se gramatika jazyka Visual Basic a syntaxe. Obsahuje podrobné informace o jazyce, včetně mnoha bodů, které nejsou zahrnuté v referenční dokumentaci k nástroji jazyka Visual Basic.  
  
 Specifikace je k dispozici na [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=188623).  
  
Tento web obsahuje [specifikace jazyka Visual Basic 11](../../../../_vblang/spec/introduction.md). Je sestaven z obsažené v souborech Markdownu [úložiště GitHub dotnet/vblang](https://github.com/dotnet/vblang/blob/master/spec/README.md).

Problémy na specifikaci by měl být vytvořen v [dotnet/vblang](https://github.com/dotnet/vblang/issues) úložiště. Nebo, pokud vás zajímá opravení chyb narazíte, můžete odeslat [žádost o přijetí změn](https://github.com/dotnet/vblang/pulls) do stejného úložiště.
