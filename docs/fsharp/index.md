# <a name="f-guide"></a>Průvodce F #

F # je open source napříč platformami programovací jazyk pro platformu .NET, která poskytuje prvotřídní podporu pro funkční programování, společně s podporou objektově orientované a imperativní programování.  Kompilátor Visual F # a nástrojů jsou implementaci společnosti Microsoft a nástrojů pro F # programovací jazyk, díky F # členem první třídy rozhraní .NET.

## <a name="if-youre-new-to-f"></a>Pokud jste ještě F # #

Pokud jste ještě F #, začínat [prohlídka z F #](tour.md) vám získat přehled o jazyce.

Doporučujeme projít si [funguje jako hodnoty první třídy](introduction-to-functional-programming/functions-as-first-class-values.md) <!--[Introduction to Functional Progamming](introduction-to-functional-programming/index.md)--> další funkční programování konceptů, které jsou nezbytné pro práci s F #.

[Kurzy](tutorials/getting-started/index.md) také mít podrobné návody pro různé úrovně dovedností a funkce jazyka.

## <a name="the-f-software-foundation"></a>Foundation softwaru F #

I když společnost Microsoft se na vývojáře primární jazyk F # a Visual F # Tooling, F # je také zálohován nezávislé foundation, F # softwaru Foundation (FSSF).

Zvláště Foundation softwaru F # je povýšit, ochranu a zálohy programovací jazyk, F # a pro podporu a usnadnění růst různých a mezinárodní komunita programátory v jazyce F #.

Další informace a zahrňte, podívejte se na [fsharp.org](http://fsharp.org).
