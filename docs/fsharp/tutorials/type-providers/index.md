<span data-ttu-id="cafa6-106">F # typ poskytovatele SQL můžete například vygenerovat typy reprezentující tabulky a sloupce v relační databázi.</span><span class="sxs-lookup"><span data-stu-id="cafa6-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="cafa6-107">Ve skutečnosti se jedná, co [SQLProvider](https://fsprojects.github.io/SQLProvider/) nemá typ poskytovatele.</span><span class="sxs-lookup"><span data-stu-id="cafa6-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="cafa6-108">Pokud že typy závisí na vstupní parametry pro zprostředkovatele typů.</span><span class="sxs-lookup"><span data-stu-id="cafa6-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="cafa6-109">Takové vstup může být ukázkového zdroje dat (například soubor schématu JSON), adresu URL přejdete přímo na externí službu nebo řetězec připojení ke zdroji dat.</span><span class="sxs-lookup"><span data-stu-id="cafa6-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="cafa6-110">Typ zprostředkovatele můžete také zajistit, že skupiny typy jsou pouze rozšířit na vyžádání; To znamená jsou rozšířit Pokud typy jsou ve skutečnosti odkazuje vašeho programu.</span><span class="sxs-lookup"><span data-stu-id="cafa6-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="cafa6-111">To umožňuje přímou integraci rozsáhlých informačních prostorů přístupných na vyžádání, jako jsou například online datové trhy, způsobem využívajícím silné typy.</span><span class="sxs-lookup"><span data-stu-id="cafa6-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="cafa6-112">Zprostředkovatelé typů tvoří a vymazaných</span><span class="sxs-lookup"><span data-stu-id="cafa6-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="cafa6-113">Zprostředkovatelé typů pocházejí ve dvou formách: tvoří a vymazaných.</span><span class="sxs-lookup"><span data-stu-id="cafa6-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="cafa6-114">Tvoří zprostředkovatelů typů vytvořit typy, které se dají zapsat jako typy .NET do sestavení, ve kterém se vytváří.</span><span class="sxs-lookup"><span data-stu-id="cafa6-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="cafa6-115">To umožňuje, aby z kódu v ostatních sestavení využívat.</span><span class="sxs-lookup"><span data-stu-id="cafa6-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="cafa6-116">To znamená, že reprezentaci typu zdroje dat musí obecně být ten, který je vhodný k reprezentaci s typy .NET.</span><span class="sxs-lookup"><span data-stu-id="cafa6-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="cafa6-117">Mazání zprostředkovatelů typů vytvořit typy, které mohou být využívány pouze v sestavení nebo projektu, který se generují z.</span><span class="sxs-lookup"><span data-stu-id="cafa6-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="cafa6-118">Typy jsou dočasné; To znamená nejsou zapsány do sestavení a nemůže být využívány službou kódu v ostatních sestavení.</span><span class="sxs-lookup"><span data-stu-id="cafa6-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="cafa6-119">Mohou obsahovat *odložené* členy, umožní vám použití zadané typy z prostoru potenciálně nekonečné informace.</span><span class="sxs-lookup"><span data-stu-id="cafa6-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="cafa6-120">Jsou užitečné pro používání malou část zdroje dat velké a vzájemně propojena.</span><span class="sxs-lookup"><span data-stu-id="cafa6-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
