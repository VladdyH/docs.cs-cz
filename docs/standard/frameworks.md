## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="17288-193">Určení cílových platforem</span><span class="sxs-lookup"><span data-stu-id="17288-193">How to specify target frameworks</span></span>

<span data-ttu-id="17288-194">Cílové architektury jsou uvedeny v souboru projektu.</span><span class="sxs-lookup"><span data-stu-id="17288-194">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="17288-195">Pokud je zadána jednu cílovou architekturu, použijte **TargetFramework** elementu.</span><span class="sxs-lookup"><span data-stu-id="17288-195">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="17288-196">Následující soubor projektu aplikace konzoly ukazuje, jak cílit na .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="17288-196">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

    <TargetFramework>netcoreapp2.0</TargetFramework>
  </PropertyGroup>

<span data-ttu-id="17288-199">Následující soubor Knihovního projektu cílí na rozhraní API z aplikaci .NET Standard (`netstandard1.4`) a rozhraní API rozhraní .NET Framework (`net40` a `net45`).</span><span class="sxs-lookup"><span data-stu-id="17288-199">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="17288-200">Použít množném čísle **TargetFrameworks** element s větším počtem cílových rozhraní.</span><span class="sxs-lookup"><span data-stu-id="17288-200">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="17288-201">Poznámka: Jak `Condition` atributy zahrnout balíčky specifický pro implementaci při kompilaci knihovny pro dva Tfm rozhraní .NET Framework:</span><span class="sxs-lookup"><span data-stu-id="17288-201">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```
