## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="e8c13-193">Určení cílových platforem</span><span class="sxs-lookup"><span data-stu-id="e8c13-193">How to specify target frameworks</span></span>

<span data-ttu-id="e8c13-194">Cílové architektury jsou uvedeny v souboru projektu.</span><span class="sxs-lookup"><span data-stu-id="e8c13-194">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="e8c13-195">Pokud je zadána jednu cílovou architekturu, použijte **TargetFramework** elementu.</span><span class="sxs-lookup"><span data-stu-id="e8c13-195">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="e8c13-196">Následující soubor projektu aplikace konzoly ukazuje, jak cílit na .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="e8c13-196">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.0</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="e8c13-197">Při zadávání více cílových rozhraní může podmíněně referenční sestavení pro každou cílovou architekturu.</span><span class="sxs-lookup"><span data-stu-id="e8c13-197">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="e8c13-198">Ve vašem kódu, můžete podmíněné kompilaci proti tato sestavení pomocí symboly preprocesoru s *if-then-else* logiku.</span><span class="sxs-lookup"><span data-stu-id="e8c13-198">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="e8c13-199">Následující soubor Knihovního projektu cílí na rozhraní API z aplikaci .NET Standard (`netstandard1.4`) a rozhraní API rozhraní .NET Framework (`net40` a `net45`).</span><span class="sxs-lookup"><span data-stu-id="e8c13-199">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="e8c13-200">Použít množném čísle **TargetFrameworks** element s větším počtem cílových rozhraní.</span><span class="sxs-lookup"><span data-stu-id="e8c13-200">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="e8c13-201">Poznámka: Jak `Condition` atributy zahrnout balíčky specifický pro implementaci při kompilaci knihovny pro dva Tfm rozhraní .NET Framework:</span><span class="sxs-lookup"><span data-stu-id="e8c13-201">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="e8c13-202">V rámci knihovny nebo aplikace napište kód podmíněné kompilace pro každou cílovou architekturu:</span><span class="sxs-lookup"><span data-stu-id="e8c13-202">Within your library or app, you write conditional code to compile for each target framework:</span></span>

    static void Main()
    {
#if NET40
        Console.WriteLine("Target framework: .NET Framework 4.0");
#elif NET45  
        Console.WriteLine("Target framework: .NET Framework 4.5");

#endif
    }
