## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="3ee34-190">Určení cílové rozhraní</span><span class="sxs-lookup"><span data-stu-id="3ee34-190">How to specify target frameworks</span></span>

<span data-ttu-id="3ee34-191">Cílové rozhraní jsou určené v souboru projektu.</span><span class="sxs-lookup"><span data-stu-id="3ee34-191">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="3ee34-192">Pokud je zadána jedné cílové rozhraní, použijte **TargetFramework** elementu.</span><span class="sxs-lookup"><span data-stu-id="3ee34-192">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="3ee34-193">Následující soubor projektu aplikace konzoly ukazuje, jak cílí na .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="3ee34-193">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.0</TargetFramework>
  </PropertyGroup>

```

<span data-ttu-id="3ee34-194">Pokud zadáte více cílové rozhraní, může podmíněně odkazujete na sestavení pro každé cílové rozhraní.</span><span class="sxs-lookup"><span data-stu-id="3ee34-194">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="3ee34-195">V kódu, můžete podmíněně zkompilovat proti tyto sestavení pomocí preprocesoru symboly s *if potom else* logiku.</span><span class="sxs-lookup"><span data-stu-id="3ee34-195">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="3ee34-196">Následující soubor projektu knihovny cílem rozhraní API z Standard .NET (`netstandard1.4`) a rozhraní API rozhraní .NET Framework (`net40` a `net45`).</span><span class="sxs-lookup"><span data-stu-id="3ee34-196">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="3ee34-197">Použijte množném čísle **TargetFrameworks** element s více cílové architektury.</span><span class="sxs-lookup"><span data-stu-id="3ee34-197">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="3ee34-198">Poznámka: Jak `Condition` atributy patří balíčky závisí na implementaci při kompilaci knihovny pro dvě TFMs Framework .NET:</span><span class="sxs-lookup"><span data-stu-id="3ee34-198">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->

    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="3ee34-199">V rámci knihovny nebo aplikací zápisu podmíněného kód mohl zkompilovat pro každé cílové rozhraní:</span><span class="sxs-lookup"><span data-stu-id="3ee34-199">Within your library or app, you write conditional code to compile for each target framework:</span></span>
