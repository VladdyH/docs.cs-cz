# <a name="introduction-to-the-docker-application-life-cycle"></a>Úvod do životního cyklu aplikace Docker
 
Životní cyklus kontejnerizované aplikací je cesty, který začíná na vývojáře. Vývojář se rozhodne implementovat kontejnery a Docker, protože eliminuje frictions v nasazení a IT oddělení, které nakonec umožní všem uživatelům se více agilní, zvýšit produktivitu začátku do konce a rychlejší.
