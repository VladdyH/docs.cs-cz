# <a name="run-manage-and-monitor-docker-production-environments"></a>Spouštět, spravovat a monitorovat Docker produkční prostředí

Vizi: Podnikové aplikace, které musí spustit s vysokou dostupnost a škálovatelnost vysoké; IT oddělení musí být schopné spravovat a monitorovat prostředí a aplikace samotné.

Tento poslední pilíře kontejnerizované životního cyklu aplikací Docker se jedná o tom, jak můžete spouštět, spravovat a monitorujte své aplikace v produkčním prostředí, škálovatelnou a vysoké dostupnosti (HA).

Jak spouštět kontejnerizované aplikace v produkčním prostředí (platformu a architektura technologie infrastruktury) je také velmi dobře související a úplně založená na zvolené architektury a vývoj platformy, které jsme se podívali na v kapitole 1 tohoto objektu elektronická kniha. Tato kapitola prozkoumá konkrétní produkty a technologie společnosti Microsoft a jiných dodavatelů, které můžete efektivně provozovat vysoce škálovatelné, HA distribuované aplikace a jak můžete spravovat a monitorovat je z hlediska oddělení IT.
