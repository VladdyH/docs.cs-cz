# <a name="copying-document-fragments"></a>Kopírování fragmenty dokumentu
Můžete vytvořit **XmlDocumentFragment** uzel a potom přidat uzly v něm. Když **XmlDocumentFragment** vložena s **InsertNode** metoda, **XmlDocumentFragment** uzlu nezkopíruje, ale jeho podřízené uzly jsou vloženy v souboru XML Modelu objektu dokumentu (DOM).
