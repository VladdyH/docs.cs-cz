 <span data-ttu-id="52b66-107">Po úspěšném ověření se použijí výchozí hodnoty schématu, textové hodnoty jsou převedeny na atomic hodnoty podle potřeby a informací o typu souvisí s položkami ověřené informace.</span><span class="sxs-lookup"><span data-stu-id="52b66-107">After successful validation, schema defaults are applied, text values are converted to atomic values as necessary, and type information is associated with validated information items.</span></span> <span data-ttu-id="52b66-108">V důsledku toho zadaných dat XML nahradí dříve netypové data XML.</span><span class="sxs-lookup"><span data-stu-id="52b66-108">As a result, typed XML data replaces previously untyped XML data.</span></span>
