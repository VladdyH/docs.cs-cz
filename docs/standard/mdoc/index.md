# <a name="building-documentation-with-mdoc"></a>Vytváření dokumentace s mdoc

mdoc je systém pro spravované referenční dokumentaci k nástroji sestavení dokumentaci. Mdoc odrážející přes vaše sestavení, vytvoří základní formát dokumentaci spravovaný odkaz, který doplňuje s vytvořené obsahem.
