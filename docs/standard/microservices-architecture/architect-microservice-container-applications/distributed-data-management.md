## <a name="challenge-4-how-to-design-communication-across-microservice-boundaries"></a><span data-ttu-id="d9774-176">Výzvy \#4: postup návrhu komunikace mezi hranic mikroslužby</span><span class="sxs-lookup"><span data-stu-id="d9774-176">Challenge \#4: How to design communication across microservice boundaries</span></span>
