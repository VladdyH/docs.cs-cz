# <a name="get-started"></a>Začínáme

Existuje několik způsobů, jak začít pracovat s rozhraním .NET.  Rozhraní .NET je masivní platformy, existuje více články v této dokumentaci, které ukazují, jak je můžete začít pracovat s rozhraním .NET, každý z hlediska různých.

## <a name="get-started-using-net-languages"></a>Začněte používat jazyky rozhraní .NET

* [C# Začínáme](../csharp/getting-started/index.md) články a [C# kurzy](../csharp/tutorials/index.md) umožňují mnoha způsoby, jak začít pracovat v jazyce C#-zaměřená na způsob.

* [F # Začínáme](../fsharp/tutorials/getting-started/index.md) kurzech nejdete tři primární způsoby použití F #: pomocí sady Visual Studio, Visual Studio Code nebo nástroje příkazového řádku.

* [Jazyka Visual Basic Začínáme](../visual-basic/getting-started/index.md) články poskytují příručky pro jazyka Visual Basic v sadě Visual Studio.

## <a name="get-started-using-net-core"></a>Začněte používat .NET Core

* [Začínáme s .NET Core](../core/get-started.md) poskytuje přehled články, které ukazují, jak začít pracovat s .NET Core na jiné operační systémy a pomocí jiných nástrojů.

* [.NET Core kurzy](../core/tutorials/index.md) podrobností mnoha různými způsoby, abyste mohli začít s .NET Core pomocí operačního systému a nástrojů výběru.
