# <a name="for-further-reading-parallel-programming"></a>Pro další čtení (paralelní programování)

Následující zdroje obsahují další informace o paralelní programování v rozhraní .NET:

- [Vzory pro paralelní programování: pochopení a použití Parallel Patterns pomocí rozhraní .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) dokument popisuje běžné paralelní vzory a osvědčené postupy pro vývoj těch, které využívá paralelní komponent vzory.

- [Vzory návrhu pro rozložením a spolupráce na Vícejádrový architektury](https://msdn.microsoft.com/library/ff963553.aspx) kniha popisuje vzory pro paralelní programování, které používají paralelní programování podporu byla zavedená v rozhraní .NET Framework 4.

- [Paralelní programování s rozhraním .NET](https://blogs.msdn.microsoft.com/pfxteam/) blog obsahuje mnoho podrobných článků o paralelní programování v rozhraní .NET.

- [Ukázky pro paralelní programování s rozhraním .NET Framework](https://code.msdn.microsoft.com/ParExtSamples) stránka obsahuje mnoho ukázky, která ukazují zprostředkující a pokročilé techniky paralelní programování.
