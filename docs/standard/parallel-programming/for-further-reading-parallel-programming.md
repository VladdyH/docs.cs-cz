# <a name="for-further-reading-parallel-programming"></a>Pro další čtení (paralelní programování)

Následující zdroje obsahují další informace o paralelní programování v rozhraní .NET:

- [Vzory pro paralelní programování: pochopení a použití Parallel Patterns pomocí rozhraní .NET Framework 4](https://www.microsoft.com/download/details.aspx?id=19222) dokument popisuje běžné paralelní vzory a osvědčené postupy pro vývoj těch, které využívá paralelní komponent vzory.
