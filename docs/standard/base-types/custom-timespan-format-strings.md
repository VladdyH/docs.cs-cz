> <span data-ttu-id="e5ba6-109">Vlastní <xref:System.TimeSpan> specifikátory formátu také nezahrnují symbolem, která umožňuje rozlišovat mezi kladnému i časové intervaly.</span><span class="sxs-lookup"><span data-stu-id="e5ba6-109">Custom <xref:System.TimeSpan> format specifiers also do not include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="e5ba6-110">Zahrnout symbolem, budete muset sestavit řetězec formátu pomocí podmíněnou logiku.</span><span class="sxs-lookup"><span data-stu-id="e5ba6-110">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="e5ba6-111">[Ostatní znaky](#Other) část obsahuje příklad.</span><span class="sxs-lookup"><span data-stu-id="e5ba6-111">The [Other Characters](#Other) section includes an example.</span></span>
