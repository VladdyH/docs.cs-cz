# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizovat existující aplikace .NET pro aplikace optimalizované pro Cloud

> Pro zpracování obrazu: Modernizace vašich stávajících aplikací rozhraní .NET Framework do optimalizovaných Cloudů aplikacím výrazně zvýšit vaší flexibilitu nasazení vám umožní rychlejší dodávání a snížit náklady na doručování vaší aplikace.

Abyste mohli využívat výhody cloudu a nové technologie, jako jsou kontejnery, by měl aspoň částečně modernizovat existující aplikace .NET. Nakonec modernizujete podnikové aplikace snížíte vaše celkové náklady na vlastnictví.

Částečně modernizace aplikace ale neznamená nutně plné migraci a rearchitecture. Zpočátku můžete modernizovat svoje stávající aplikace s důležitý, ale snadno provedete modernizaci. Díky čemuž můžete udržovat aktuální kód základní, napsané v existující verze rozhraní .NET Framework, se všechny závislosti Windows a služby IIS. Obrázek 4-1 poukazuje na to jak optimalizovaných pro cloudové aplikace jsou umístěny v modelech vyspělosti modernizaci aplikací Azure.
