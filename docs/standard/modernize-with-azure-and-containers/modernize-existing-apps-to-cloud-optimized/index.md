# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernizovat existující aplikace .NET do optimalizovaných Cloudů aplikací

> Vizi: Modernizovat existující aplikace rozhraní .NET Framework do optimalizovaných Cloudů aplikacím výrazně zlepšit flexibility vaše nasazení vám umožní dodávat rychlejší a nižší náklady doručení aplikace.

Abyste mohli využívat výhody cloudu a nové technologie jako kontejnery, by měl aspoň částečně modernizovat existující aplikace .NET. Nakonec modernizing podnikové aplikace bude nižší vaše náklady na vlastnictví.

Částečně modernizing aplikace nemusí znamenat úplné migrace a rearchitecture. Můžete zpočátku modernizovat existující aplikace s důležité ale snadné provést modernizace. Je možné zachovat aktuální kód základní, napsané v existující verze rozhraní .NET Framework, s Windows a IIS závislostí. Obrázek 4-1 označuje jak optimalizovaný pro cloudové aplikace jsou umístěny v aplikaci Azure modernizace vyspělosti modelech.
