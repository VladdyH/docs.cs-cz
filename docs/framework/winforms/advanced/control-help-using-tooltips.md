# <a name="control-help-using-tooltips"></a>Nápověda ovládacího prvku pomocí ToolTips
Můžete použít <xref:System.Windows.Forms.ToolTip> součást, kterou Struèné, specializuje nápovědu pro jednotlivé ovládacích prvků ve Windows Forms. <xref:System.Windows.Forms.ToolTip> Součást poskytuje vlastnosti, která určuje text, který zobrazí pro každý ovládací prvek na formuláři. Další informace o práci s <xref:System.Windows.Forms.ToolTip> součást tímto způsobem, najdete v části [postupy: nastavení ToolTips pro ovládací prvky na formuláři Windows v době návrhu](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md). Kromě toho můžete nakonfigurovat <xref:System.Windows.Forms.ToolTip> součásti tak, aby předtím, než se zobrazí, dochází ke zpoždění. Další informace najdete v tématu [postupy: Změna zpoždění součásti Windows Forms ToolTip](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
