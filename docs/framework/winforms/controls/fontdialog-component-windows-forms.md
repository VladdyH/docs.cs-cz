# <a name="fontdialog-component-windows-forms"></a>FontDialog – komponenta (Windows Forms)
Windows Forms <xref:System.Windows.Forms.FontDialog> součást je předem nakonfigurovaný dialogové okno. Stejná **písma** dialogové okno vystavené operačního systému Windows. Součást dědí z <xref:System.Windows.Forms.CommonDialog> třídy.
