# <a name="how-to-resize-controls-on-windows-forms"></a>Postupy: Změna velikosti ovládacích prvků ve formulářích Windows
Můžete změnit velikost jednotlivých ovládacích prvků, a můžete změnit velikost více ovládacích prvků stejného nebo jiného typu, například <xref:System.Windows.Forms.Button> a <xref:System.Windows.Forms.GroupBox> ovládací prvky.  
  
### <a name="to-resize-a-control"></a>Ke změně velikosti ovládacího prvku  
  
1.  Klikněte na tlačítko změnit a přetáhněte jeden z osmi úchyty ovládacího prvku.  
  
    > [!NOTE]
    >  Vyberte ovládací prvek a stisknutím klávesy se šipkami při podržíte stisknutou klávesu SHIFT a změnit velikost jednoho pixelu řízení najednou. Stisknutím klávesy se šipkami dolů nebo doprava a podržením klávesy SHIFT a CTRL ke změně velikosti ovládacího prvku velké přírůstcích.  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>Ke změně velikosti více ovládacích prvků ve formuláři  
  
1.  Podržte stisknutou klávesu SHIFT nebo CTRL a vyberte ovládací prvky, které chcete změnit. Velikost první ovládací prvek, který jste vybrali, se používá pro další ovládací prvky.  
  
2.  Na **formátu** nabídce zvolte **nastavit stejnou velikost**a vyberte jednu ze čtyř možností. První tři příkazy změnit rozměry ovládacích prvků tak, aby odpovídaly první vybraný ovládací prvek.
