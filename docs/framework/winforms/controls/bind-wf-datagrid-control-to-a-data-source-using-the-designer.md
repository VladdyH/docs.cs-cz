### <a name="to-data-bind-the-datagrid-control-to-a-single-table-in-the-designer"></a><span data-ttu-id="86eca-124">Pro data-bind DataGrid – ovládací prvek pro jednotlivé tabulky v Návrháři</span><span class="sxs-lookup"><span data-stu-id="86eca-124">To data-bind the DataGrid control to a single table in the designer</span></span>

2.  <span data-ttu-id="86eca-126">Pokud zdroj dat je datová sada, nastavte <xref:System.Windows.Forms.DataGrid.DataMember%2A> vlastnost na název tabulky pro vazbu.</span><span class="sxs-lookup"><span data-stu-id="86eca-126">If the data source is a dataset, set the <xref:System.Windows.Forms.DataGrid.DataMember%2A> property to the name of the table to bind to.</span></span>  
  
3.  <span data-ttu-id="86eca-127">Pokud je zdroj dat datové sady nebo zobrazení dat na základě datové sady tabulky, přidejte kód pro formulář k vyplnění datové sady.</span><span class="sxs-lookup"><span data-stu-id="86eca-127">If the data source is a dataset or a data view based on a dataset table, add code to the form to fill the dataset.</span></span>

    ```csharp  
    sqlDataAdapter1.Fill(DsCategories1);
    ```  

    ```cpp  
    sqlDataAdapter1->Fill(dsCategories1);
    ```  
