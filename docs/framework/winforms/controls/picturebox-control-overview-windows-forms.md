# <a name="picturebox-control-overview-windows-forms"></a>PictureBox – přehled ovládacího prvku (Windows Forms)
Windows Forms <xref:System.Windows.Forms.PictureBox> řízení se používá k zobrazení grafiky ve formátu rastrového obrázku, GIF, JPEG, metafile nebo ikonu.  
  
## <a name="key-properties-and-methods"></a>Klíčové vlastnosti a metody  
 Určuje obrázek, který se zobrazí <xref:System.Windows.Forms.PictureBox.Image%2A> vlastnosti, která lze nastavit v době běhu nebo v době návrhu. Můžete alternativně zadat bitovou kopii nastavením <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> vlastnost a pak můžete načíst image synchronně pomocí <xref:System.Windows.Forms.PictureBox.Load%2A> metoda nebo asynchronně pomocí <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> metoda. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> Vlastnost určuje, jak přizpůsobit bitové kopie a řízení mezi sebou. Další informace najdete v tématu [postupy: Změna velikosti či umístění obrázku za běhu](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
