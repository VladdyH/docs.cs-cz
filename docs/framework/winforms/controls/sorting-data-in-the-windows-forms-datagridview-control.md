# <a name="sorting-data-in-the-windows-forms-datagridview-control"></a>Řazení dat v ovládacím prvku Windows Forms DataGridView
Ve výchozím nastavení, mohou uživatelé seřadit data v `DataGridView` řízení kliknutím na záhlaví sloupce textové pole. Můžete upravit `SortMode` vlastnost konkrétní sloupců, aby uživatelé mohli seřadit jiné typy sloupců, když má smysl to udělat. Data můžete také řadit prostřednictvím kódu programu žádný sloupec nebo více sloupců.
