# <a name="how-to-copy-controls-between-windows-forms"></a>Postupy: Kopírování ovládacích prvků mezi formuláři Windows
Ovládací prvek mohou být zkopírovány do stejného formuláře, do jiného formuláře v rámci projektu nebo do schránky pro použití v jiných řešení.  
  
### <a name="to-copy-a-control"></a>Kopírování ovládacího prvku  
  
1.  Vyberte ovládací prvek a z **upravit** nabídce zvolte **kopie**.  
  
 Tento ovládací prvek lze nyní vložit do libovolného formuláře, který přijme daný typ ovládacího prvku. Kromě toho ovládacího prvku přidaná do schránky.
