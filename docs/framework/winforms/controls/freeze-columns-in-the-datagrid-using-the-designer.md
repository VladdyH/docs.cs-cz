### <a name="to-freeze-a-column-using-the-designer"></a><span data-ttu-id="2a385-115">Chcete-li ukotvit sloupec pomocí návrháře</span><span class="sxs-lookup"><span data-stu-id="2a385-115">To freeze a column using the designer</span></span>  
  
1.  <span data-ttu-id="2a385-116">Klikněte na inteligentní značky glyfy (![inteligentní značky glyfy](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) v pravém horním rohu <xref:System.Windows.Forms.DataGridView> řízení a potom vyberte **upravit sloupce**.</span><span class="sxs-lookup"><span data-stu-id="2a385-116">Click the smart tag glyph (![Smart Tag Glyph](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control, and then select **Edit Columns**.</span></span>  
  
2.  <span data-ttu-id="2a385-117">Vyberte sloupce z **vybrané sloupce** seznamu.</span><span class="sxs-lookup"><span data-stu-id="2a385-117">Select a column from the **Selected Columns** list.</span></span>  
  
3.  <span data-ttu-id="2a385-118">V **vlastnosti sloupce** mřížky, nastavte <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> vlastnost `true`.</span><span class="sxs-lookup"><span data-stu-id="2a385-118">In the **Column Properties** grid, set the <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> property to `true`.</span></span>

    >  <span data-ttu-id="2a385-119">Můžete také ukotvit sloupec při přidávání výběrem **zmrazené** pole **přidat sloupec** dialogové okno.</span><span class="sxs-lookup"><span data-stu-id="2a385-119">You can also freeze a column when adding it by selecting the **Frozen** box in the **Add Column** dialog box.</span></span>
