# <a name="linklabel-control-windows-forms"></a>LinkLabel – ovládací prvek (Windows Forms)
Windows Forms `LinkLabel` řízení umožňuje přidat webových odkazů do formulářové aplikace Windows. Můžete použít `LinkLabel` řízení pro všechno, který můžete použít <xref:System.Windows.Forms.Label> řízení pro; můžete také nastavit část textu jako odkaz na objekt nebo webovou stránku.
