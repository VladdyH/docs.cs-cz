# <a name="net-framework-client-profile"></a>Profil klienta rozhraní .NET Framework
Profil klienta rozhraní .NET je podmnožinou rozhraní .NET Framework, který byl poskytnut s [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] a starší verze a je optimalizovaný pro klientské aplikace. Rozhraní .NET Framework je platforma pro vývoj pro Windows, Windows Phone a Microsoft Azure a poskytuje prostředí pro spouštění spravovaných aplikací a knihovna tříd rozhraní .NET Framework. [!INCLUDE[net_v40_short](../../../includes/net-v40-short-md.md)] a starší verze k dispozici dvě možnosti nasazení: úplné rozhraní .NET Framework a profil klienta. Profil klienta podporuje rychlejší nasazení a menší instalační balíčky aplikace než plná verze rozhraní .NET Framework.  
  
 Od verze [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], profil klienta již není k dispozici a je k dispozici pouze úplná redistribuovatelného balíčku. Optimalizace poskytované [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], jako například menší velikost stahování a rychlejší nasazení odstranili potřebu samostatné nasazení balíčku. Jediná verze Redistributable zjednodušuje proces instalace a možnosti nasazení aplikace.  
  
 Ale pokud jsou cílení na rozhraní .NET Framework 4 nebo 3.5 a chcete se dozvědět víc o profil klienta a kdy ji použít, najdete v části [profil klienta rozhraní .NET Framework](http://msdn.microsoft.com/library/cc656912\(v=vs.100\).aspx) v dokumentaci k rozhraní .NET Framework 4.  
  
 Při instalaci [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], rozhraní .NET Framework 4 Client Profile se aktualizuje na plnou verzi rozhraní .NET Framework. Informace o instalaci [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], najdete v části [instalaci rozhraní .NET Framework pro vývojáře](../../../docs/framework/install/guide-for-developers.md).
