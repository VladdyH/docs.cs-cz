# <a name="dynamicresource-markup-extension"></a><span data-ttu-id="04bd3-102">DynamicResource – rozšíření značek</span><span class="sxs-lookup"><span data-stu-id="04bd3-102">DynamicResource Markup Extension</span></span>
<span data-ttu-id="04bd3-103">Poskytuje hodnotu pro všechny [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] vlastnost atribut rozlišením tuto hodnotu jako odkaz na prostředek definované.</span><span class="sxs-lookup"><span data-stu-id="04bd3-103">Provides a value for any [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] property attribute by deferring that value to be a reference to a defined resource.</span></span> <span data-ttu-id="04bd3-104">Chování vyhledávání pro tento prostředek je analogická spuštění vyhledávání.</span><span class="sxs-lookup"><span data-stu-id="04bd3-104">Lookup behavior for that resource is analogous to run-time lookup.</span></span>

