# <a name="xaml-in-wpf"></a>XAML ve WPF
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)]je programovací jazyk kódu pro deklarativní aplikace. [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)]implementuje aXAML procesoru implementace a poskytuje podporu pro jazyk XAML. [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] Typy jsou implementované tak, aby mohli poskytnout požadovaný typ zálohování pro znázornění XAML. Obecně platí, můžete vytvořit většina vaší [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] aplikace uživatelského rozhraní v XAML značek.
