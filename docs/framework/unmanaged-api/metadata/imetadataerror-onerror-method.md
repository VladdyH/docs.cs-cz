# <a name="imetadataerroronerror-method"></a>IMetaDataError::OnError – metoda
Poskytuje oznámení chyb, ke kterým došlo během metadat sloučení.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT OnError (  
    [in] HRESULT   hrError,   
    [in] mdToken   token  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `hrError`  
 [v] Hodnota chyby HRESULT vrácená volání metody.  
  
 `token`  
 [v] Token metadata objektu kód, který slučování když došlo k chybě.
