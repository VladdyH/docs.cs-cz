# <a name="imetadataimportgetuserstring-method"></a>IMetaDataImport::GetUserString – metoda
Získá reprezentována Zadaná metadata token řetězcového literálu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `stk`  
 [v] Token řetězec vrátit text přidružený k.  
  
 `szString`  
 [out] Kopie požadovaný řetězec.  
  
 `cchString`  
 [v] Maximální velikost v široké znaky požadované `szString`.  
  
 `pchString`  
 [out] Velikost v široké znaky vrácený `szString`.
