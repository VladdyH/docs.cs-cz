# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory – metoda
Uloží všechna metadata v aktuálním oboru do určené oblasti paměti.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pbData`  
 [out] Adresa, kam chcete zahájit zápis metadat.  
  
 `cbData`  
 [v] Velikost v bajtech, přidělené paměti.
