# <a name="metadata-global-static-functions"></a>Globální statické funkce metadat
Od verze [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)], v dokumentaci `_Cor*`, `CoInitialize*`, `CoUninitialize*`, a `CoEEShutDownCOM` funkce, které náleží k hostování, se nachází v [zastaralé funkce hostování CLR](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Nejsou žádné globální statické funkce, které se týkají metadat.
