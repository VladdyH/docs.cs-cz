# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA – metoda
Nastaví hodnotu globální proměnné pro relativní virtuální adresu pole odkazuje zadaný token.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `fd`  
 [v] Token pro cílové pole.  
  
 `ulRVA`  
 [v] Adresa kód nebo datové oblasti.
