# <a name="icordebugappdomain2-interface1"></a>Icordebugappdomain2 – Interface1
Poskytuje metody pro práci s pole, ukazatelů, ukazatelů na funkce a odkazové typy. Toto rozhraní je rozšíření rozhraní ICorDebugAppDomain.  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[GetArrayOrPointerType – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain2-getarrayorpointertype-method.md)|Získá zadaný typ, nebo ukazatel nebo odkaz na zadaný typ pole.|  
|[GetFunctionPointerType](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain2-getfunctionpointertype-method.md)|Získá ukazatel na funkci, která má daný podpis.|  
  
## <a name="remarks"></a>Poznámky  
  
> [!NOTE]
>  Toto rozhraní nepodporuje volané vzdáleně, mezi počítači nebo mezi procesy.
