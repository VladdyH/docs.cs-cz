|Metoda|Popis|

|[Next – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugprocessenum-next-method.md)|Získá zadaný počet `ICorDebugProcess` instancí z výčtu, počínaje na aktuální pozici.|
