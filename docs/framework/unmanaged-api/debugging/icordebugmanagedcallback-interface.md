 <span data-ttu-id="42624-161">Všechny zpětná volání jsou serializovat, názvem ve stejném vlákně, v a volána s procesu synchronizovaného stavu.</span><span class="sxs-lookup"><span data-stu-id="42624-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>
