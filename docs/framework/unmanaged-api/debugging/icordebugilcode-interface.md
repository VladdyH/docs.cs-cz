# <a name="icordebugilcode-interface"></a>Rozhraní ICorDebugILCode
[Podporované v rozhraní .NET Framework 4.5.2 a novějších verzích]  
  
 Představuje segment kód intermediate language (IL).  
  
## <a name="methods"></a>Metody  
  
|Metoda|Popis|  
|------------|-----------------|  
|[GetEHClauses – metoda](../../../../docs/framework/unmanaged-api/debugging/icordebugilcode-getehclauses-method.md)|Vrací ukazatel na seznam klauzule (EH), které jsou definovány pro tento IL zpracování výjimek.|
