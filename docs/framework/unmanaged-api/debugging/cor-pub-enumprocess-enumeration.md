# <a name="corpubenumprocess-enumeration"></a>COR_PUB_ENUMPROCESS – výčet
Určuje typ procesu výčet.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typedef enum {  
    COR_PUB_MANAGEDONLY    = 0x00000001  
} COR_PUB_ENUMPROCESS;  
```  
  
## <a name="members"></a>Členové  
  
|Název členu|Popis|  
|-----------------|-----------------|  
|`COR_PUB_MANAGEDONLY`|Spravované proces.|  
  
## <a name="remarks"></a>Poznámky  
 Aktuální verze nespravovaného rozhraní API pro ladění zobrazí jenom spravovaných procesy.
