# <a name="icordebuggetprocess-method"></a>ICorDebug::GetProcess – metoda
Získá ukazatel na instanci "ICorDebugProcess" pro určený proces.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetProcess (  
    [in] DWORD               dwProcessId,  
    [out] ICorDebugProcess   **ppProcess  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `dwProcessId`  
 [v] ID procesu.  
  
 `ppProcess`  
 [out] Ukazatel na adresu `ICorDebugProcess` instancí zadaného procesu.
