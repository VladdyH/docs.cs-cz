# <a name="cordebuggenerationtypes-enumeration"></a>CorDebugGenerationTypes – výčet
Spravovaná halda určuje generování oblasti paměti.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typedef enum CorDebugGenerationTypes {  
    CorDebug_Gen0 = 0,  
    CorDebug_Gen1 = 1,  
    CorDebug_Gen2 = 2,  
    CorDebug_LOH  = 3,  
} CorDebugRegionTypes;
```  
