# <a name="icordebugthread4getcurrentcustomdebuggernotification-method"></a>ICorDebugThread4::GetCurrentCustomDebuggerNotification – metoda
Získá aktuální [icordebugmanagedcallback3::customnotification –](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback3-customnotification-method.md) objekt na aktuální vlákno.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetCurrentCustomDebuggerNotification(  
    [out] ICorDebugValue **ppNotificationObject  
    );  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppNOtificationObject`  
 [out] Ukazatel na aktuální `ICorDebugManagedCallback3::CustomNotification` objekt na aktuální vlákno.  
  
## <a name="remarks"></a>Poznámky  
 Hodnota `ppNotificationObject` má hodnotu null, pokud metoda není volána v rámci `ICorDebugManagedCallback3::CustomNotification` zpětné volání, nebo pokud neexistuje žádný aktuální objekt oznámení.
