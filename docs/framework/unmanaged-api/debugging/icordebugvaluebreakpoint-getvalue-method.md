# <a name="icordebugvaluebreakpointgetvalue-method"></a>ICorDebugValueBreakpoint::GetValue – metoda
Získá ukazatele rozhraní do objektu "ICorDebugValue", který představuje hodnotu objektu, na kterém je nastaven breakpoint.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetValue (  
    [out] ICorDebugValue   **ppValue  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppValue`  
 [out] Ukazatel na adresu `ICorDebugValue` objektu.
