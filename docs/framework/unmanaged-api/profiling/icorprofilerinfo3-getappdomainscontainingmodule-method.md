# <a name="icorprofilerinfo3getappdomainscontainingmodule-method"></a>ICorProfilerInfo3::GetAppDomainsContainingModule – metoda
Získá identifikátory aplikační domény, ve kterých byl načten daného modulu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetAppDomainsContainingModule(  
            [in] ModuleID moduleId,  
            [in] ULONG32 cAppDomainIds,  
            [out] ULONG32 *pcAppDomainIds,  
            [out, size_is(cAppDomainIds), length_is(*pcAppDomainIds)]  
                    AppDomainID appDomainIds[]);  
```  
  
#### <a name="parameters"></a>Parametry  
 `moduleId`  
 [v] ID modulu načíst.  
  
 `cAppDomainIds`  
 [v] Velikost `appDomainIds` pole.  
  
 `pcAppDomainIds`  
 [out] Ukazatel na celkový počet vrácených elementy.  
  
 `appDomainIds`  
 [out] Pole hodnoty ID domény aplikace.  
  
## <a name="remarks"></a>Poznámky  
 Metoda používá volající přidělené vyrovnávací paměti.
