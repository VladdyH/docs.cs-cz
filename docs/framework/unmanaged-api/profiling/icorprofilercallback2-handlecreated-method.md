# <a name="icorprofilercallback2handlecreated-method"></a>ICorProfilerCallback2::HandleCreated – metoda
Upozorní profileru kód vytvořený popisovač kolekce paměti.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT HandleCreated(  
    [in] GCHandleID handleId,  
    [in] ObjectID initialObjectId);  
```  
  
#### <a name="parameters"></a>Parametry  
 `handleId`  
 [v] ID popisovač pro uvolnění paměti.  
  
 `initialObjectId`  
 [v] ID objektu, pro který byl vytvořen popisovač kolekce paměti.
