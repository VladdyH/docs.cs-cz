# <a name="profiling-structures"></a>Struktury pro profilaci
Tato část popisuje nespravované struktury, které používá profilaci API.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Struktura COR_PRF_ASSEMBLY_REFERENCE_INFO](../../../../docs/framework/unmanaged-api/profiling/cor-prf-assembly-reference-info-structure.md)  
 Poskytuje modul common language runtime s informacemi o odkaz na sestavení, které ho měli zvážit při procházení uzavření odkaz na sestavení.  
  
 [Cor_prf_code_info – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-code-info-structure.md)  
 Představuje jeden souvislý blok nativního kódu, které jsou uložené v paměti.  
  
 [Cor_prf_ex_clause_info – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-ex-clause-info-structure.md)  
 Obsahuje informace o instanci klauzule určité výjimky a jeho přidružené rámečku.  
  
 [Cor_prf_function – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-structure.md)  
 Poskytuje jedinečný reprezentace funkce kombinací jeho ID s ID jeho Rekompilované verze.  
  
 [Cor_prf_function_argument_info – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-argument-info-structure.md)  
 Představuje argumenty funkcí v pořadí zleva doprava.  
  
 [Cor_prf_function_argument_range – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-function-argument-range-structure.md)  
 Představuje blok argumenty funkce souvisle uložené v pořadí zleva doprava v paměti.  
  
 [Cor_prf_gc_generation_range – struktura](../../../../docs/framework/unmanaged-api/profiling/cor-prf-gc-generation-range-structure.md)  
 Popisuje rozsah (blok) paměti, která probíhá uvolňování paměti.
