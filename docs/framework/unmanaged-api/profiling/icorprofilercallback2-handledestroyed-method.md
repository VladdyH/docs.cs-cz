## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT HandleDestroyed(  
    [in] GCHandleID handleId);  
```  
  
#### <a name="parameters"></a>Parametry  
 `handleId`  
 [v] ID popisovač pro uvolnění paměti.
