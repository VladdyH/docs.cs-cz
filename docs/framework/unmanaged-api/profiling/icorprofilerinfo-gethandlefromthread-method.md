# <a name="icorprofilerinfogethandlefromthread-method"></a>ICorProfilerInfo::GetHandleFromThread – metoda
Mapuje ID vlákna popisovač podprocesu Win32.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetHandleFromThread(  
    [in]  ThreadID threadId,  
    [out] HANDLE  *phThread);  
```  
  
#### <a name="parameters"></a>Parametry  
 `threadId`  
 [v] Nejde mapovat ID vlákna.  
  
 `phThread`  
 [out] Ukazatel na popisovač podprocesu Win32.  
  
## <a name="remarks"></a>Poznámky  
 Profileru musí volat Win32 `DuplicateHandle` funkce na popisovač před jeho použitím.
