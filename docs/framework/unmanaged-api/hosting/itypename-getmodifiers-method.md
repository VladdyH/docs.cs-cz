# <a name="itypenamegetmodifiers-method"></a>ITypeName::GetModifiers – metoda

## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetModifiers (  
    [in] DWORD           count,  
    [out] DWORD*         rgModifiers,  
    [out, retval] DWORD* pCount

```
