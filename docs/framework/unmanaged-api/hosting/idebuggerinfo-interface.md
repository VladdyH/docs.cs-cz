|Metoda|Popis|  
|------------|-----------------|  
|[IsDebuggerAttached – metoda](../../../../docs/framework/unmanaged-api/hosting/idebuggerinfo-isdebuggerattached-method.md)|Získá hodnotu, která určuje, zda je pro tento proces připojen spravované ladicí program.|
