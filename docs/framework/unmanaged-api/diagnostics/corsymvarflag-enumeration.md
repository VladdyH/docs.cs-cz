# <a name="corsymvarflag-enumeration"></a>CorSymVarFlag – výčet
Určuje, zda je proměnná generované kompilátorem.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
typedef enum CorSymVarFlag   
{  
    VAR_IS_COMP_GEN = 1  
} CorSymVarFlag;  
```  
  
## <a name="members"></a>Členové  
  
|Člen|Popis|  
|------------|-----------------|  
|`VAR_IS_COMP_GEN`|Označuje, že daný proměnné je generované kompilátorem.|
