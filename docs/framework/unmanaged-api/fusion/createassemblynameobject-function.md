# <a name="createassemblynameobject-function"></a>CreateAssemblyNameObject – funkce
Získá ukazatele rozhraní k [iassemblyname –](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) instanci, která představuje jedinečné identity sestavení se zadaným názvem.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT CreateAssemblyNameObject (  
    [out] LPASSEMBLYNAME  *ppAssemblyNameObj,  
    [in]  LPCWSTR         szAssemblyName,  
    [in]  DWORD           dwFlags,  
    [in]  LPVOID          pvReserved  
 );  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppAssemblyNameObj`  
 [out] Vrácený `IAssemblyName`.  
  
 `szAssemblyName`  
 [v] Název sestavení, pro který chcete vytvořit nový `IAssemblyName` instance.  
  
 `dwFlags`  
 [v] Příznaky, které mají být předána do konstruktoru objektu.  
  
 `pvReserved`  
 [v] Vyhrazeno pro budoucí rozšíření. `pvReserved` musí být odkaz s hodnotou null.
