# <a name="fusion-enumerations"></a>Výčty fúzí
Tato část popisuje nespravovaná vyčíslení, které používá fusion rozhraní API.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [ASM_CACHE_FLAGS – výčet](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 Určuje zdroj sestavení reprezentována [iassemblycacheitem –](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md) v globální mezipaměti sestavení.  
  
 [ASM_CMP_FLAGS – výčet](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)  
 Určuje verzi, sestavení, jazykovou verzi, podpisu a tak dále, dvě sestavení, který se má porovnat pomocí [iassemblyname::isequal –](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-isequal-method.md) metoda.  
  
 [ASM_DISPLAY_FLAGS – výčet](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)  
 Určuje verzi, sestavení, jazykovou verzi, podpisu a tak dále, sestavení, jehož zobrazovaný název bude načten pomocí [iassemblyname::GetDisplayName –](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-getdisplayname-method.md) metoda.  
  
 [ASM_NAME – výčet](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)  
 Určuje verzi, sestavení, jazykovou verzi, podpisu a tak dále, sestavení, jehož vlastnosti se načíst nebo nastavit [iassemblyname –](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) metody.  
  
 [AssemblyComparisonResult – výčet](../../../../docs/framework/unmanaged-api/fusion/assemblycomparisonresult-enumeration.md)  
 Označuje ekvivalenční dvě sestavení identit, počítáno od [compareassemblyidentity –](../../../../docs/framework/unmanaged-api/fusion/compareassemblyidentity-function.md) funkce.  
  
 [CREATE_ASM_NAME_OBJ_FLAGS – výčet](../../../../docs/framework/unmanaged-api/fusion/create-asm-name-obj-flags-enumeration.md)  
 Určuje atributy `IAssemblyName` objektu, když je vytvořený pomocí [createassemblynameobject –](../../../../docs/framework/unmanaged-api/fusion/createassemblynameobject-function.md) funkce.
