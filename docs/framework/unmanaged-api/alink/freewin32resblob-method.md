# <a name="freewin32resblob-method"></a>FreeWin32ResBlob – metoda
Uvolní Win32 prostředků blob a přidružených prostředků.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT FreeWin32ResBlob(  
    const void** ppResBlob  
) PURE;  
```  
  
#### <a name="parameters"></a>Parametry  
 `ppResBlob`  
 Objekt blob prostředků k uvolnění. Tato metoda přiřadí objekt blob ukazatel na hodnotu NULL.  
  
## <a name="return-value"></a>Návratová hodnota  
 Vrátí S_OK, pokud metoda bude úspěšná.  
  
## <a name="requirements"></a>Požadavky  
 Vyžaduje alink.h
