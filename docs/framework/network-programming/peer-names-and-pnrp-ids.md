## <a name="pnrp-ids"></a><span data-ttu-id="a44bf-118">ID PNRP</span><span class="sxs-lookup"><span data-stu-id="a44bf-118">PNRP IDs</span></span>

 <span data-ttu-id="a44bf-120">ID PNRP se skládají z následujících akcí:</span><span class="sxs-lookup"><span data-stu-id="a44bf-120">PNRP IDs are composed of the following:</span></span>
