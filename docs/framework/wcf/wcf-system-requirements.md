# <a name="wcf-system-requirements"></a>WCF – požadavky na systém
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)]je komunikace infrastrukturu, která se používá k vytvoření distribuované aplikace. Následující seznam obsahuje požadavky, které umožňují [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] ke spuštění.  
  
## <a name="system-requirements"></a>Požadavky na systém  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)]ve výchozím nastavení je nainstalovaná na [!INCLUDE[wv](../../../includes/wv-md.md)].
