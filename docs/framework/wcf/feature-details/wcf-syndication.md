# <a name="wcf-syndication"></a>Syndikace WCF
Windows Communication Foundation (WCF) poskytuje podporu pro snadno pracovat s informační kanály syndikace v Atom, RSS nebo dalších vlastních formátů, které umožňuje číst a jejich vytvoření a také je vystavit na koncový bod služby. Témata v této části popisují tato programovací model pro syndikace podrobně.
