# <a name="support-for-json-and-other-data-transfer-formats"></a>Podpora formátu JSON a dalších formátů přenosu dat
JSON (JavaScript Object Notation) je formát kódování efektivní dat, který umožňuje rychlé výměnu malé množství dat mezi prohlížeče klienta a AJAX (asynchronní JavaScript a XML)-povolené webové služby.
