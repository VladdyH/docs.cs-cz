# <a name="message-security-with-issued-tokens"></a>Zabezpeční zpráv pomocí vydaných tokenů
Vystavený token klientovi umožňuje scénáře federované zabezpečení v [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)].  
  
 Federované zabezpečení umožňuje navázání vztahu důvěryhodnosti mezi doménami nezávislé. Každá doména, centralizuje ověřování a autorizace ve služby tokenů zabezpečení. Když je navázán vztah důvěryhodnosti mezi dvěma těchto služeb, členové každou doménu, mohou používat služby dalších.  
  
 Podrobné vysvětlení federované scénář, najdete v článku [federace a vystavené tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
