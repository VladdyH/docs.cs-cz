# <a name="message-security-with-issued-tokens"></a>Zabezpeční zpráv pomocí vydaných tokenů
Vydaný token klient umožňuje scénáře zabezpečení ve Windows Communication Foundation (WCF).  
  
 Zabezpečení umožňuje navázání vztahu důvěryhodnosti mezi doménami nezávislé. Každá doména centralizuje ověřování a autorizace pro služby tokenů zabezpečení. Po vytvoření vztahu důvěryhodnosti mezi dvěma takové služby, služby druhým můžou používat členy každé domény.  
  
 Podrobné vysvětlení o federovaných scénáři najdete v tématu [federace a vydané tokeny](../../../../docs/framework/wcf/feature-details/federation-and-issued-tokens.md).
