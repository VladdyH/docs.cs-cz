<span data-ttu-id="fc11a-103">*Popis stromu* je hierarchie typů (počínaje <xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType> třída) společně popisují všechny aspekty služby.</span><span class="sxs-lookup"><span data-stu-id="fc11a-103">The *description tree* is the hierarchy of types (starting with the <xref:System.ServiceModel.Description.ServiceDescription?displayProperty=nameWithType> class) that together describe every aspect of a service.</span></span> <span data-ttu-id="fc11a-104">Windows Communication Foundation (WCF) používá k vytvoření platný služby modulu runtime Publikovat Web Services Description Language (WSDL), jazyk definice schématu XML (XSD) a výrazy zásad (metadata) o službě, který můžou klienti používat pro popis stromu připojení k a používat službu a ke generování různé kódu a konfigurace souborů reprezentace strom hodnot popis.</span><span class="sxs-lookup"><span data-stu-id="fc11a-104">Windows Communication Foundation (WCF) uses a description tree to build a valid service runtime, to publish Web Services Description Language (WSDL), XML Schema definition language (XSD), and policy assertions (metadata) about the service that clients can use to connect to and use the service, and to generate various code and configuration file representations of the description tree values.</span></span>
