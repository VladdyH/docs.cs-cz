# <a name="correlation"></a>Korelace
Když aplikace služeb pracovního postupu komunikovat s ostatními službami, je důležité, že zprávy mezi nimi jsou odesílány do instance pracovního postupu odpovídající. Korelace poskytuje mechanismus pro to. Témata v této části poskytují přehled korelace a jak ji používat ve scénářích jiný pracovní postup služby.
