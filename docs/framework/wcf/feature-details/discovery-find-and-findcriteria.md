## <a name="findresponse"></a><span data-ttu-id="196b0-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="196b0-140">FindResponse</span></span>  
 <span data-ttu-id="196b0-141"><xref:System.ServiceModel.Discovery.FindResponse>má <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> vlastnost kolekce, která obsahuje všechny odpovědi poslal odpovídající služby v síti.</span><span class="sxs-lookup"><span data-stu-id="196b0-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="196b0-142">Pokud žádné služby zodpovězených, kolekce je prázdná.</span><span class="sxs-lookup"><span data-stu-id="196b0-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="196b0-143">Pokud jeden nebo více služeb zodpovězených, každou odpověď je uložen v <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> objektu, který obsahuje adresu, kontrakt a některé další informace o službě.</span><span class="sxs-lookup"><span data-stu-id="196b0-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="196b0-144">Následující příklad ukazuje, jak provádět operace hledání v kódu.</span><span class="sxs-lookup"><span data-stu-id="196b0-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```
