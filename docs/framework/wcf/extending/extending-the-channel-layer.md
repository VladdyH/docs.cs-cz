# <a name="extending-the-channel-layer"></a>Rozšíření vrstvy kanálu
Vrstvy kanálu je zodpovědná za výměny zpráv mezi klienty a služby. Kanál rozšíření můžete implementovat nové funkce protokolu, například zabezpečení nebo funkce přenosu, například implementace nový síťový přenos pro přenos protokolu SOAP zprávy.
