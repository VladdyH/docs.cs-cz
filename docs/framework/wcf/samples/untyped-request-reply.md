 <span data-ttu-id="81403-105">Tato ukázka je založena na [Začínáme](../../../../docs/framework/wcf/samples/getting-started-sample.md).</span><span class="sxs-lookup"><span data-stu-id="81403-105">This sample is based on the [Getting Started](../../../../docs/framework/wcf/samples/getting-started-sample.md).</span></span> <span data-ttu-id="81403-106">Kontrakt služby definuje jednu operaci, která přebírá typ zprávy jako argument a vrátí zprávu.</span><span class="sxs-lookup"><span data-stu-id="81403-106">The service contract defines one operation that takes in a message type as an argument and returns a message.</span></span> <span data-ttu-id="81403-107">Operaci shromažďuje všechna požadovaná data pro výpočetní součet z textu zprávy a poté odešle součet jako text v návratové zprávy.</span><span class="sxs-lookup"><span data-stu-id="81403-107">The operation collects all required data to compute the sum from the message body and then sends the sum as body in the return message.</span></span>

```  
Message ComputeSum(Message request);  
```  
  
 <span data-ttu-id="81403-108">Operaci na službu, načte poli celých čísel předaná vstupní zprávu a pak vypočítá součet.</span><span class="sxs-lookup"><span data-stu-id="81403-108">On the service, the operation retrieves the array of integers passed in the input message and then computes the sum.</span></span> <span data-ttu-id="81403-109">Odeslat zprávu odpovědi, ukázka vytvoří novou zprávu s verzí odpovídající zprávu a akci a přidá vypočítaný součet jako jeho text.</span><span class="sxs-lookup"><span data-stu-id="81403-109">To send a response message, the sample creates a new message with the appropriate message version and Action and adds the computed sum as its body.</span></span> <span data-ttu-id="81403-110">Následující vzorový kód ukazuje to.</span><span class="sxs-lookup"><span data-stu-id="81403-110">The following sample code demonstrates this.</span></span>  
  
```  
public Message ComputeSum(Message request)  
{  
    //The body of the message contains a list of numbers which will be   
    //read as a int[] using GetBody<T>  
    int result = 0;  
  
    int[] inputs = request.GetBody<int[]>();  
    foreach (int i in inputs)  
    {  
        result += i;  
    }  
  
    Message response = Message.CreateMessage(request.Version,   
                                      ReplyAction, result);  
    return response;  
}  
```  
  
 <span data-ttu-id="81403-111">Klient použije kód, který je generován [ServiceModel Metadata Utility Tool service.</span></span> <span data-ttu-id="81403-112">Pokud chcete odeslat zprávu požadavku, klient musí mít verzi zprávy, které závisí na základní kanálu.</span><span class="sxs-lookup"><span data-stu-id="81403-112">To send a request message, the client must have the message version, which depends on the underlying channel.</span></span> <span data-ttu-id="81403-113">Proto vytvoří novou `OutgoingMessageHeaders.MessageVersion` vlastnost.</span><span class="sxs-lookup"><span data-stu-id="81403-113">Thus, it creates a new <xref:System.ServiceModel.OperationContextScope> scoped to the <span data-ttu-id="81403-114">Klient předá vstupní pole jako text zprávy požadavku a potom se vyvolá `ComputeSum` na proxy serveru.</span><span class="sxs-lookup"><span data-stu-id="81403-114">The client passes an input array as the body to the request message and then invokes the `ComputeSum` on the proxy.</span></span> <span data-ttu-id="81403-115">Klient pak načte součet vstupních hodnot, je předaná přístup k `GetBody<T>` metodu na zpráva s odpovědí.</span><span class="sxs-lookup"><span data-stu-id="81403-115">The client then retrieves the sum of the inputs it passed by accessing the `GetBody<T>` method on the reply message.</span></span> <span data-ttu-id="81403-116">Následující vzorový kód ukazuje to.</span><span class="sxs-lookup"><span data-stu-id="81403-116">The following sample code demonstrates this.</span></span>  
  
```  
using (new OperationContextScope(client.InnerChannel))  
{  
    // Call the Sum service operation.  
    int[] values = { 1, 2, 3, 4, 5 };  
    Message request = Message.CreateMessage(  
        OperationContext.Current.OutgoingMessageHeaders.MessageVersion,   
        RequestAction, values);  
    Message reply = client.ComputeSum(request);  
    int response = reply.GetBody<int>();  
  
    Console.WriteLine("Sum of numbers passed (1,2,3,4,5) = {0}",   
                                                       response);  
}  
```
