# <a name="behaviors"></a>Chování
Tato část obsahuje příklady vysvětlující chování služby Windows Communication Foundation (WCF).

 [Souběžnost](../../../../docs/framework/wcf/samples/concurrency.md)  
 Ukazuje, jak pomocí <xref:System.ServiceModel.ServiceBehaviorAttribute> s <xref:System.ServiceModel.ConcurrencyMode> výčtu, který určuje, zda instance služby zpracovává zprávy postupně nebo souběžně.  
  
 [Výchozí chování služby](../../../../docs/framework/wcf/samples/default-service-behavior.md)  
 Ukazuje, jak lze nakonfigurovat nastavení chování služby.

 [Chování při publikování metadat](../../../../docs/framework/wcf/samples/metadata-publishing-behavior.md)

 [Chování při ladění služby](../../../../docs/framework/wcf/samples/service-debug-behavior.md)

 [Omezování](../../../../docs/framework/wcf/samples/throttling.md)

 [Zabezpečení chování](../../../../docs/framework/wcf/samples/behavior-security.md)  
 Představuje konfiguraci zabezpečení chování služby.
