# <a name="basic-resource-service"></a><span data-ttu-id="bd660-102">Služba základních prostředků</span><span class="sxs-lookup"><span data-stu-id="bd660-102">Basic Resource Service</span></span>
<span data-ttu-id="bd660-103">Tento příklad ukazuje, jak implementovat nějakou službu založenou na protokolu HTTP pomocí Windows Communication Foundation (WCF) REST programovacího modelu, který poskytuje kolekci zákazníků, který podporuje načtení, přidat, odstranit a nahrazovat operace.</span><span class="sxs-lookup"><span data-stu-id="bd660-103">This sample demonstrates how to implement a HTTP-based service using the Windows Communication Foundation (WCF) REST Programming model that exposes a collection of customers that supports the retrieve, add, delete and replace operations.</span></span> <span data-ttu-id="bd660-104">Tento příklad se skládá z 2 komponenty – v místním prostředí služby WCF HTTP (Service.cs) a Konzolová aplikace (program.cs), který vytvoří službu a provede volání do něj.</span><span class="sxs-lookup"><span data-stu-id="bd660-104">This sample consists of 2 components - a self-hosted WCF HTTP service (Service.cs) and a console application (program.cs) that creates the service and makes calls to it.</span></span>  
  
## <a name="sample-details"></a><span data-ttu-id="bd660-105">Ukázka podrobnosti</span><span class="sxs-lookup"><span data-stu-id="bd660-105">Sample Details</span></span>  
 <span data-ttu-id="bd660-106">Služba WCF poskytuje kolekci zákazníků způsobem resource orientované nebo rozhraní REST.</span><span class="sxs-lookup"><span data-stu-id="bd660-106">The WCF service zákazníka.</span><span class="sxs-lookup"><span data-stu-id="bd660-110">When a new customer is added to the collection, the service assigns it a unique URI and stores the URI as part of the customer’s data-stu-id="bd660-111">Also, it communicates the URI to the client using the Location HTTP header of the response.</span></span>  
  
 <span data-ttu-id="bd660-112">Soubor App.config Konfiguruje službu WCF s výchozím <xref:System.ServiceModel.Description.WebHttpEndpoint> , který má <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> nastavenou na `true`.</span><span class="sxs-lookup"><span data-stu-id="bd660-112">The App.config file configures the WCF service with a default <xref:System.ServiceModel.Description.WebHttpEndpoint> that has the <xref:System.ServiceModel.Description.WebHttpEndpoint.HelpEnabled%2A> property set to `true`.</span></span> <span data-ttu-id="bd660-113">V důsledku toho WCF vytvoří automatickou založený na jazyce HTML stránku nápovědy na `http://localhost:8000/Customers/help` , který poskytuje informace o tom, jak vytvořit HTTP žádostí o služby a jak získat přístup k služby odpověď HTTP – třeba příklad, jak se podrobnosti o zákazníkovi vyjádřena v XML a JSON.</span><span class="sxs-lookup"><span data-stu-id="bd660-113">As a result, the WCF creates an automatic HTML-based help page at `http://localhost:8000/Customers/help` that provides information about how to construct HTTP requests to the service and how to access the service’s HTTP response – for instance, an example of how the customer details is represented in XML and JSON.</span></span>  
  
 <span data-ttu-id="bd660-114">Vystavení kolekcí zákazníků (a obecněji prostředek) tímto způsobem umožňuje pracovat s ním jednotným způsobem pomocí identifikátorů URI a HTTP klientovi `GET`, `PUT`, `DELETE` a `POST`.</span><span class="sxs-lookup"><span data-stu-id="bd660-114">Exposing the customer collection (and more generally, any resource) in this manner allows the client to interact with it ukázkové) ukazují, jak použít tyto třídy komunikovat se službou WCF.</span><span class="sxs-lookup"><span data-stu-id="bd660-118">Other samples in the SDK (such as the [Basic HTTP  <span data-ttu-id="bd660-119">Ukázka obsahuje služba v místním prostředí a klient i spuštění v rámci konzolové aplikace.</span><span class="sxs-lookup"><span data-stu-id="bd660-119">The sample consists a self-hosted service and a client that both run within a console application.</span></span> <span data-ttu-id="bd660-120">Konzolová aplikace běží, klient vytvářejí požadavky na službu a zapíše relevantní informace z odpovědi v okně konzoly.</span><span class="sxs-lookup"><span data-stu-id="bd660-120">As the console application runs, the client makes requests to the service and writes the pertinent information from the responses to the console window.</span></span>
