# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Zadanou transakci pro zadanou operaci byla dokončena z důvodu nezpracované výjimce provádění.  
  
## <a name="description"></a>Popis  
 Sledovat, když dojde k chybě při pokusu o dokončení aktuální transakci. K tomu dojde před odpověď nebo selhání odesílání volajícímu.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte trasovaná zprávu pro zprávu výjimky a všechny položky, kterého lze provést akci.
