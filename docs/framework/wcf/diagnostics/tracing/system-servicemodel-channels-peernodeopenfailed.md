# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Pokus o otevření připojení sousedním se nezdařil.  
  
## <a name="description"></a>Popis  
 Trasování nastane po pokusu o navázání připojení sousedním. Možné důvody může zahrnovat reagovat překladače, na nesprávný vazby nastavení nebo selhání přechodný síťový.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zajistěte, aby oba uzly platné adresy a jsou správně nakonfigurována.
