# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Popis  
 Zavřít operaci ServiceHost vypršení časového limitu. To označuje, že některé relacemi kanály nebyly správně vypnutí klientem a může byla přerušena. Upravte výchozí doba upravit zavřít časový limit.
