# <a name="systemservicemodelcommunicationobjectabortfailed"></a>System.ServiceModel.CommunicationObjectAbortFailed
Přerušení komunikace objektu se nezdařilo.  
  
## <a name="description"></a>Popis  
 Přerušit, protože došlo k výjimce během provádění metody jeho přerušení komunikace objektu se nezdařilo. Jinou blízkou trasování může obsahovat podrobnosti o výjimce.
