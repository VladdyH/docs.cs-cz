# <a name="systemservicemodelchannelspeerneighbornotfound"></a>System.ServiceModel.Channels.PeerNeighborNotFound
Operace na sousedním nelze provést, protože tento sousedním nebyl nalezen v seznamu sousedním.  
  
## <a name="description"></a>Popis  
 K této chybě dojde, když propojíte nebo zavřením sousedním.
