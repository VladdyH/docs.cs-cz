# <a name="systemservicemodelserviceoperationmissingreplycontext"></a>System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## <a name="description"></a>Popis  
 ReplyContext má hodnotu null pro dvoucestné operaci. Data trasování se zobrazí následující zprávu: "odpověď operace nemá žádný kontext IRequestContext pro odpověď."  
  
 Trasování je užitečné pro ladění během vývoje.
