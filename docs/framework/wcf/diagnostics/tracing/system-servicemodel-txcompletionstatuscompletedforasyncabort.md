# <a name="systemservicemodeltxcompletionstatuscompletedforasyncabort"></a>System.ServiceModel.TxCompletionStatusCompletedForAsyncAbort
Zadanou transakci pro zadanou operaci byla dokončena z důvodu asynchronní přerušení.  
  
## <a name="description"></a>Popis  
 Aktuální transakce byla přerušena kvůli jiné účastnické volíte přerušení, ke kterým dochází vypršení časových limitů nebo jiné chybě uvnitř účastník transakce.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Pokud je tato přerušení neočekávaná, zkontrolujte všechny protokoly systému k určení skutečné důvod přerušení.
