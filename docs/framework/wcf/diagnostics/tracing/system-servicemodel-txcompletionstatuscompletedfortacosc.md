# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
Konkrétní transakce byla dokončena, když relace byla ukončena.  
  
## <a name="description"></a>Popis  
 Trasovat při zavření aktuální aktivní relace a `TransactionAutoCompleteOnSessionClose` je nastaven na `true`.
