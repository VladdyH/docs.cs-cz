# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
Instance služby byla uvolněna při dokončení transakce '{0}, protože atribut ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute byla nastavena na hodnotu true.  
  
## <a name="description"></a>Popis  
 Sledovat, když aktuální instance služby je vydání nebo zlikvidován kvůli dokončení aktuální aktivní transakci a ReleaseServiceInstanceOnTransactionComplete je nastaven na `true`.
