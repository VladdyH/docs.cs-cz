# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
Objekt komunikace se nepovedlo uzavřít.  
  
## <a name="description"></a>Popis  
 Objekt komunikace se nepodařilo ukončit, protože došlo k výjimce během zpracování jeho Close – metoda. Jinou blízkou trasování může obsahovat podrobnosti o výjimce.
