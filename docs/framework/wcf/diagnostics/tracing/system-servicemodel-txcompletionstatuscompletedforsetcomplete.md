# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Zadanou transakci pro zadanou operaci byla dokončena z důvodu volání SetTransactionComplete.  
  
## <a name="description"></a>Popis  
 Trasovat, pokud uživatel explicitně rozhodli dokončit aktuální transakce prostřednictvím OperationContext.
