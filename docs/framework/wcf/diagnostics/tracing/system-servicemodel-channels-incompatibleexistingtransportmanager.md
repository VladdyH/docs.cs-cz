# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Pro zadaný identifikátor URI byl nalezen stávající nekompatibilní správce přenosu.  
  
## <a name="description"></a>Popis  
 Informační trasování označuje, že pro naslouchací proces otevíráte, ale není kompatibilní stávající správce přenosu nebyl nalezen odpovídající registraci. Naslouchací proces se pokusí použít jiný správce přenosu a způsobí výjimku, pokud neúspěšná.
