# <a name="systemservicemodelactivationwebhostserviceclosefailed"></a>System.ServiceModel.Activation.WebHostServiceCloseFailed
Nastane, když služba nelze zavřít, řádně a byl přerušen.  
  
## <a name="description"></a>Popis  
 Tento kód chyby se zobrazí pouze v souboru protokolu. Obvykle označuje chybě programování, například při pokusu o uzavření služby po přerušení již byla volána.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte zdrojovému kódu aplikace.
