# <a name="systemservicemodelserviceoperationmissingreply"></a>System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## <a name="description"></a>Popis  
 Požadavek nebo odpověď operace nemá žádná zpráva s odpovědí.  
  
 To je obousměrný operace; ale nepřijde žádná odpověď k odeslání do klienta. Trasování můžete použít pro ladění během vývoje.
