# <a name="systemservicemodelchannelsfailedpipeconnect"></a>System.ServiceModel.Channels.FailedPipeConnect
Pokus o připojení ke koncovému bodu pojmenovaný kanál se nezdařil. Další pokus se uskuteční během zadaného časového limitu.  
  
## <a name="description"></a>Popis  
 Informační trasování označuje selhání připojení ke koncovému bodu pojmenovaný kanál. Toto může nastat, pokud koncový bod pojmenovaný kanál nebyl nalezen nebo je zaneprázdněný. Další pokusy, jsou odděleny krátkou dobu, dokud jeden neuspěje nebo OpenTimeout vyprší.
