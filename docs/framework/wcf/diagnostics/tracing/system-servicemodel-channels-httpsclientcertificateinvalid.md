# <a name="systemservicemodelchannelshttpsclientcertificateinvalid"></a>System.ServiceModel.Channels.HttpsClientCertificateInvalid
Klientský certifikát je neplatný.  
  
## <a name="description"></a>Popis  
 Trasování označuje, že certifikát poskytnutý klientem byl nalezen neplatný podle naslouchací proces HTTPS. Naslouchací proces HTTPS k pokusu o ověření pravosti klienta s použitím tohoto certifikátu. Certifikát může být neplatné, pokud jeho certifikační autorita není rozpoznáno serverem, který je hostitelem služby.
