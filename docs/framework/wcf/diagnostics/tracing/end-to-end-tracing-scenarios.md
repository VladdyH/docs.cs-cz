# <a name="end-to-end-tracing-scenarios"></a>Scénáře komplexního trasování
Tento oddíl obsahuje témata, které popisují různé scénáře pro použití trasování.  
  
 Trasování aktivit je implementována pro odpovědi a odpověď, OneWay a duplexní režim scénáře s využitím následující  
  
-   Synchronní nebo asynchronní  
  
-   BasicHttpBinding – netNamedPipeBinding netTcpBinding, wsHttpBinding nebo – netMsmqBinding  
  
-   Služby modelu COM  
  
-   Šíření id aktivity uživatele
