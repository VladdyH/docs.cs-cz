# <a name="end-to-end-tracing"></a>Komplexní trasování
Koncová trasování (e2e) umožňuje vývojářům sledovat spuštění kódu v infrastruktuře Windows Communication Foundation (WCF), zjistěte, proč se nezdařilo kódové cestě, nebo zadejte podrobného trasování pro analýzu výkon a plánování kapacity. Windows Communication Foundation (WCF) poskytuje tři korelace mechanismy pro usnadnění diagnostiky příčiny chybu: aktivit, přenosy a šíření.  
  
 V tématu [scénáře trasování začátku do konce](../../../../../docs/framework/wcf/diagnostics/tracing/end-to-end-tracing-scenarios.md) seznam trasování začátku do konce scénáře a jejich odpovídající aktivitu a trasování návrhu.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [Aktivita](../../../../../docs/framework/wcf/diagnostics/tracing/activity.md): popisuje aktivity trasování v modelu trasování Windows Communication Foundation (WCF).  
  
 [Přenos](../../../../../docs/framework/wcf/diagnostics/tracing/transfer.md): Popisuje přenosu v modelu trasování Windows Communication Foundation (WCF), a pomocí přenosu ke korelaci aktivity v rámci koncové body.  
  
 [Šíření](../../../../../docs/framework/wcf/diagnostics/tracing/propagation.md): Popisuje rozšíření aktivity v systému Windows Communication Foundation (WCF) trasování modelu a pomocí šíření vazbu mezi aktivitami v koncových bodů.
