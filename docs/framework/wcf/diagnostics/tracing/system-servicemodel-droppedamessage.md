# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Popis  
 Byla přijata zpráva odpovědi, ale nejsou žádné nevyřízené žádosti čekající na tuto zprávu. Zpráva Probíhá vyřazování modulem runtime Windows Communication Foundation (WCF).  
  
 Trasování můžete použít pro ladění ztrátě zpráv.
