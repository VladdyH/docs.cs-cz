# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
Objekt komunikace se nepodařilo otevřít.  
  
## <a name="description"></a>Popis  
 Objekt komunikace se nepodařilo otevřít, protože došlo k výjimce během zpracování jeho Open – metoda. Jinou blízkou trasování může obsahovat podrobnosti o výjimce.
