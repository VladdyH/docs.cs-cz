# <a name="systemservicemodelmanualflowthrottlelimitreached"></a>System.ServiceModel.ManualFlowThrottleLimitReached
System.ServiceModel.ManualFlowThrottleLimitReached  
  
## <a name="description"></a>Popis  
 Systém dosažen limit nastavený pro ManualFlowControlLimit omezení. Hodnota omezení lze změnit úpravou vlastnosti ManualFlowControlLimit na hostiteli služby nebo objekt InstanceContext, podle vhodnosti.  
  
 Trasování jsou vydávány po limit ruční toku řízení původně zmenšení na hodnotu 0. Následné změny 0 nejsou trasovat. Tok řízení limit pro daný kontext instance trasovaný jednou pro každý kontext.
