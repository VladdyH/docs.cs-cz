# <a name="systemservicemodelchannelspeernodeauthenticationtimeout"></a>System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
Bezpečnostní ověření typu handshake se nezdařilo v určeném časovém limitu.  
  
## <a name="description"></a>Popis  
 Při pokusu o vytvoření zabezpečeného sousedním připojení dojde k trasování.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zajistěte, aby všechny okolí mají příslušná zabezpečovací přihlašovací údaje a reagovat na všechny zprávy za méně než 1 minutu.
