# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished
Stav stavového stroje pro zápis účastníka přešla do konečného stavu.  
  
## <a name="description"></a>Popis  
 Sledovat po dokončení zpracování 2pc podřízené účastnické zařazení. Výsledek pro zařazení může být potvrzené nebo přerušené. Také trasovaný Pokud kterýkoli účastník hlasy během Příprava jen pro čtení.
