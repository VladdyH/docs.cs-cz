# <a name="systemservicemodeltxcompletionstatusabortedonsessionclose"></a>System.ServiceModel.TxCompletionStatusAbortedOnSessionClose
Zadaný transakce byla přerušena, protože nebyla při zavření relace a TransactionAutoCompleteOnSessionClose OperationBehaviorAttribute byl nastaven na hodnotu false.  
  
## <a name="description"></a>Popis  
 Trasovat, pokud se aktuální aktivní relace bylo ukončeno a transakce nebyla dokončena a TransactionAutoCompleteOnSessionClose je nastaven na `false`.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Trasování označuje potenciální chybu aplikace, která by se měly prozkoumat.
