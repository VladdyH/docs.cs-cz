# <a name="systemservicemodeltxasyncabort"></a>System.ServiceModel.TxAsyncAbort
Zadaný transakce byla přerušena asynchronně.  
  
## <a name="description"></a>Popis  
 Aktuální transakce byla přerušena kvůli jiné účastnické volíte přerušení, ke kterým dochází vypršení časových limitů nebo jiné chybě uvnitř účastník transakce.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Pokud je tato přerušení neočekávaná chcete zjistit, proč skutečné pro přerušení, zkontrolujte všechny protokoly systému.
