# <a name="debugging-on-the-client"></a>Ladění na klientovi
Aby bylo snazší pro uživatele pro psaní aplikací klienta pro vaše [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] služby, můžete přidat [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) služby chování do konfiguračního souboru služby. Toto chování lze použít k publikování stránky nápovědy a vrátí informace o spravovaných výjimce v podrobnostech chyb SOAP vrácen do klienta.
