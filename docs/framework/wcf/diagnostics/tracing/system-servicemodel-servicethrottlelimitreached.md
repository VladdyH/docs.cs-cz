# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Popis  
 Bylo dosaženo jedním z omezení MaxConcurrentCalls/instance nebo relace a zpráva byla zařazena.  
  
 Trasování můžete upravit hodnoty omezení podle očekávaných a aktuálních produkční zatížení.
