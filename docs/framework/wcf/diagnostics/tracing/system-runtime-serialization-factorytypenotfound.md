# <a name="systemruntimeserializationfactorytypenotfound"></a>System.Runtime.Serialization.FactoryTypeNotFound
System.Runtime.Serialization.FactoryTypeNotFound  
  
## <a name="description"></a>Popis  
 Deserializátor nemá žádné informace o typ, který má použít při deserializaci. Zvažte přidání typ určený jako typ objektu pro vytváření v souboru XML do seznamu ze známých typů - například pomocí KnownTypeAttribute atribut.
