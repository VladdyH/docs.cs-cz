# <a name="systemservicemodelchannelshttpchannelrequestaborted"></a>System.ServiceModel.Channels.HttpChannelRequestAborted
Se nepodařilo odeslat zprávu požadavku přes protokol HTTP.  
  
## <a name="description"></a>Popis  
 Úroveň trasování upozornění označuje, že se přerušil kanál požadavku HTTP. Další informace v této trasování bude odhalit příčinu nebo zprávy předaný metodě přerušení.
