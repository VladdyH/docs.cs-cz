# <a name="microsofttransactionstransactionbridgeregisterparticipant"></a>Microsoft.Transactions.TransactionBridge.RegisterParticipant
Služba protokolu WS-Atomic Transactions úspěšně zaregistrovala účastníka pro kontrolní protokol.  
  
## <a name="description"></a>Popis  
 Sledovat, když místní správce transakcí má úspěšně potvrzeny žádost o registraci od účastníka 2pc protokolu.
