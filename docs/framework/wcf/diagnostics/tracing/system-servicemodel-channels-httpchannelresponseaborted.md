# <a name="systemservicemodelchannelshttpchannelresponseaborted"></a>System.ServiceModel.Channels.HttpChannelResponseAborted
Nepodařilo se odeslat zprávu odpovědi přes protokol HTTP.  
  
## <a name="description"></a>Popis  
 Úroveň trasování upozornění označuje, že se přerušil kanál odpovědi HTTP. Další informace v této trasování bude odhalit příčinu nebo zprávy předaný metodě přerušení.
