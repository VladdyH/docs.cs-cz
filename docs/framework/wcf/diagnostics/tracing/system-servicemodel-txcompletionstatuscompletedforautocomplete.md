# <a name="systemservicemodeltxcompletionstatuscompletedforautocomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForAutocomplete
Zadanou transakci pro zadanou operaci byla dokončena z důvodu `OperationBehaviorAttribute.TransactionAutoComplete` člen je nastavená na `true`.  
  
## <a name="description"></a>Popis  
 Sledovat po infrastruktury automaticky dokončení transakce jménem uživatele.
