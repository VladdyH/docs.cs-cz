# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Sousedním druhé strany nebyla přijata.  
  
## <a name="description"></a>Popis  
 Trasování nastane při zpracování připojení dalšímu uživateli. Možné důvody patří nastavení neodpovídající zabezpečení, časový limit překročen, sousedním, které nemůžou přijímat připojení nebo přechodný síťový problémy.  
  
## <a name="troubleshooting"></a>Poradce při potížích  
 Zkontrolujte nastavení zabezpečení neodpovídající, byl překročen časový limit, sousedním nemůže přijímat připojení nebo přechodný síťový problémy.
