# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Je požadován klientský certifikát. V požadavku nebyl nalezen žádný certifikát.  
  
## <a name="description"></a>Popis  
 Trasování označuje, že naslouchací proces HTTPS přijala požadavek HTTPS, který nebyl přidružený certifikát klienta. Vzhledem k tomu, že naslouchací proces byl nakonfigurován tak, aby vyžadovala certifikáty klientů pro všechny požadavky HTTPS, naslouchací proces se nezdařilo ověření pravosti klienta.
