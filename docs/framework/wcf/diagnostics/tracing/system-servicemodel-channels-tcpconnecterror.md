# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
TCP připojit operace se nezdařila.  
  
## <a name="description"></a>Popis  
 Úroveň trasování upozornění označuje Nepodařilo se připojit k koncový bod TCP. Toto může nastat, pokud vzdálený koncový bod nereaguje na danou adresu IP a portu. Trasování můžete ignorovat, pokud následné pokusí připojit k jiné platnou IP adresy (například adresy IPv4 nebo IPv6, nebo jiné IP adresy představující daný název hostitele) úspěšné. Výjimka v rámci trasování může odhalit další informace o této chybě.
