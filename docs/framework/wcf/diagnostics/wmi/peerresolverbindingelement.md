# <a name="peerresolverbindingelement"></a>PeerResolverBindingElement
PeerResolverBindingElement  
  
## <a name="syntax"></a>Syntaxe  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## <a name="methods"></a>Metody  
 Třída PeerResolverBindingElement Třída nedefinuje žádné metody.  
  
## <a name="properties"></a>Vlastnosti  
 Třída PeerResolverBindingElement třída má následující vlastnost:  
  
### <a name="referralpolicy"></a>ReferralPolicy  
 Datový typ: řetězec  
  
 Přístup k typu: jen pro čtení  
  
 Určuje, jak jsou odkazy sdíleny mezi partnerské uzly.
