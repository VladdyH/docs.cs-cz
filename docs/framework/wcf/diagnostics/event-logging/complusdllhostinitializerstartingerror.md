# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError
ID: 131  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě při spouštění inicializátoru DllHost. Události jsou uvedeny AppID, výjimky, název procesu a ID procesu.
