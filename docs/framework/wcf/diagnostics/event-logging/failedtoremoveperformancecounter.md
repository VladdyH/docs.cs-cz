# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
ID: 119  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že nebyla odebrána čítače výkonu. Událost uvádí název kategorie, název čítače, název instance, výjimky, název procesu a ID procesu.
