# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
ID: 126  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že nebyl zaregistrován zprostředkovatele rozhraní WMI. Události obsahuje objekt WMI, chyba, název procesu a ID procesu.
