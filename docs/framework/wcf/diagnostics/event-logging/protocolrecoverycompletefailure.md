# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
ID: 145  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba protokolu WS-Atomic Transactions se nepodařilo dokončit spuštění a obnovení. V důsledku toho WS-Atomic Transactions funkce jsou zakázány. Událost uvádí ID protokol, název protokolu, výjimky, název procesu a ID procesu.
