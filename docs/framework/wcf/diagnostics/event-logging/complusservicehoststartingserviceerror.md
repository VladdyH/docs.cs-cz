# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
ID: 130  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě při spouštění služby COM +. Událost uvádí AppID, ClsID, náhradní, výjimky, název procesu a ID procesu.
