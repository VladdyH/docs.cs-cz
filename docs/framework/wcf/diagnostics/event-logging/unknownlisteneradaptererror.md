# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
ID: 161  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost

 Tato událost označuje, že došlo k neočekávané chybě v adaptéru naslouchání při zpracování oznámení služby Aktivace procesů systému Windows (WAS). Ukončení procesu. Událost uvádí protokol, výjimky, název procesu a ID procesu.
