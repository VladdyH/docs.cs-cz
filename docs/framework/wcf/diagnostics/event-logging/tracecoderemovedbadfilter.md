# <a name="tracecoderemovedbadfilter"></a>TraceCodeRemovedBadFilter
ID: 114  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že byla odebrána zprávy o chybném protokolování filtr, který byl přístup k textu zprávy. Události jsou uvedeny XPath, název procesu a ID procesu.
