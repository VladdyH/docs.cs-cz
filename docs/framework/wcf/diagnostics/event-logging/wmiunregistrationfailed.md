# <a name="wmiunregistrationfailed"></a>WmiUnregistrationFailed
ID: 127  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že nebyl zrušení registrace zprostředkovatele rozhraní WMI. Události obsahuje objekt WMI, chyba, název procesu a ID procesu.
