# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
ID: 168  
  
 Závažnost: Chyba  
  
 Kategorie: SecurityAudit  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba autorizace se nezdařila. Události jsou uvedené služby, akce, identita klienta, kontext ověřování, ID aktivity a ServiceAuthorizationManager.
