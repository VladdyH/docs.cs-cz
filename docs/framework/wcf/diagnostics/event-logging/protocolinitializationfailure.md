# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
ID: 142  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba protokolu WS-Atomic Transactions se nezdařila. V důsledku toho WS-Atomic Transactions funkce jsou zakázány. Událost uvádí ID protokol, název protokolu, výjimky, název procesu a ID procesu. .
