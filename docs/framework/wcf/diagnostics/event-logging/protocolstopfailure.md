# <a name="protocolstopfailure"></a>ProtocolStopFailure
ID: 147  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba protokolu WS-Atomic Transactions se nepovedlo zastavit. Událost uvádí ID protokol, název protokolu, výjimky, název procesu a ID procesu.
