# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
ID: 112  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě při pokusu o naslouchat na dané adrese URL. Je tento pracovní proces ukončen. Uvádí události výjimky, název procesu a ID procesu.
