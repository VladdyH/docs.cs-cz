# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
ID: 144  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že protokol WS-Atomic Transactions služby MSDTC nepodařilo na začátku obnovení. V důsledku toho WS-Atomic Transactions funkce jsou zakázány. Událost uvádí ID protokol, název protokolu, výjimky, název procesu a ID procesu.
