# <a name="fatalunexpectedstatemachineevent"></a>FatalUnexpectedStateMachineEvent
ID: 137  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že pokus o stavu počítače zpracování neočekávané události. Událost byla považována za kritické. Události jsou uvedené ID transakce, stav počítače, aktuální stav, historie, události název, události podrobnosti, název procesu a ID procesu.
