# <a name="bindingerror"></a>BindingError
ID: 159  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě při analýze konkrétní vazbu konkrétní lokality proto protokolu je pro lokalitu dočasně zakázána. Události jsou uvedeny vazby, zdroj, výjimky, název procesu a ID procesu.
