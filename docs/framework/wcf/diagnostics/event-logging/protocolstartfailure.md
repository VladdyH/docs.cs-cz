# <a name="protocolstartfailure"></a>ProtocolStartFailure
ID: 143  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba protokolu WS-Atomic Transactions se nepodařilo spustit. V důsledku toho WS-Atomic Transactions funkce jsou zakázány. Událost uvádí ID protokol, název protokolu, výjimky, název procesu a ID procesu. .
