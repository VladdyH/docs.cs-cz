# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
ID: 165  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě při odesílání duplicitní soketu. Tento popisovač je nyní úniku v procesu. Události jsou uvedeny zdroje, výjimky, název procesu a ID procesu.
