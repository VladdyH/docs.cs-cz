# <a name="nonfatalunexpectedstatemachineevent"></a>NonFatalUnexpectedStateMachineEvent
ID: 148  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že pokus o stavu počítače zpracování neočekávané události. Událost není považováno za kritické. Události jsou uvedené ID transakce, stav počítače, aktuální stav, historie, události název, události podrobnosti, název procesu a ID procesu.
