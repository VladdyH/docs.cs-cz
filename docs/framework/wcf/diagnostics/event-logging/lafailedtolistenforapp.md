# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
ID: 160  
  
 Závažnost: Chyba  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že došlo k chybě ve službě Aktivace zadaný zadaného protokolu, při pokusu o naslouchat pro určenou lokalitu, že proto protokolu je pro lokalitu dočasně zakázána... Události jsou uvedené adresy URL, stav, výjimky, název procesu a ID procesu.
