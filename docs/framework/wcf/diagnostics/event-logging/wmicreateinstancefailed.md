# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
ID: 123  
  
 Závažnost: Chyba  
  
 Kategorie: ServiceModel

 Tato událost označuje, že WMI CreateInstance požadavek nebyl zpracován. Událost uvádí název třídy, výjimky, název procesu a ID procesu.
