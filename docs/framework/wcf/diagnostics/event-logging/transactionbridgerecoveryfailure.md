# <a name="transactionbridgerecoveryfailure"></a>TransactionBridgeRecoveryFailure
ID: 146  
  
 Závažnost: Chyba  
  
 Kategorie: TransactionBridge  
  
## <a name="description"></a>Popis  
 Tato událost označuje, že služba MSDTC TransactionBridge došlo k chybě během obnovení. Toto je závažná, takže služba MSDTC byla ukončena. Uvádí události výjimky, název procesu a ID procesu. .
