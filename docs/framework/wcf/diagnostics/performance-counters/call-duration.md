# <a name="call-duration"></a>Doba volání
Název čítače: Doba volání  
  
## <a name="description"></a>Popis  
 Průměrná doba volání k této operaci. Průměrná doba se počítá podle této rovnice: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Při použití na asynchronní službu WCF čítač Doba trvání volání vždy vrátí hodnotu -1.
