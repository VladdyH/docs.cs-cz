# <a name="peer-channel"></a>Rovnocenný kanál
Toto téma uvádí všechny výjimky generované [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] rovnocenného kanálu.  
  
## <a name="exception-list"></a>Seznam výjimek  
  
|Kód prostředku|Řetězec prostředku|  
|-------------------|---------------------|  
|InsufficientCredentials|Zadané pověření nejsou dostatečná k provedení požadované operace. Zadejte platnou hodnotu pro zadanou operaci|  
|InvalidResolverMode|Zadaná hodnota třídy PeerResolverMode je neplatný. Zadejte třídu PeerResolveMode.Auto, výchozí nebo PNRP.|  
|PeerNodeAborted|Třídu PeerNode nelze otevřít, protože operace byla ukončena.|  
|PeerNullRegistrationInfo|Informace o registraci nemůže mít hodnotu null. Ujistěte se, že platný objekt RegistrationInfo vyvolání operace registrace.|  
|PeerNullResolveInfo|Informace nelze vyřešit `null`. Zkontrolujte, že operace řešení je vyvolána s platným objektem ResolveInfo.|
