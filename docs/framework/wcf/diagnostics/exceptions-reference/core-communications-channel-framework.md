## <a name="exception-list"></a>Seznam výjimek  
  
|Kód prostředku|Řetězec prostředku|  
|-------------------|---------------------|  
|ReceiveRequestTimedOut|Přijatý požadavek na adresu zadanou místní vypršel po zadanou dobu. Čas přidělený tato operace mohla být část delší časový limit.|

|UnableToDemuxChannel|Žádný kanál je k dispozici pro příjem zprávy spolu s určenou akcí.|
