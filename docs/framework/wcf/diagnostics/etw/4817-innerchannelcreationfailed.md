|||  
|-|-|  
|ID|4817|  
|Klíčová slova|Zjišťování|  
|úroveň|Upozornění|

## <a name="description"></a>Popis  
 Tato událost je vygenerované při DiscoveryClientChannel Nepodařilo se vytvořit kanál s zjištěných koncový bod. DiscoveryClientChannel se nyní pokusí použít další dostupné zjištěných koncový bod.  
  
## <a name="message"></a>Zpráva  
 DiscoveryClientChannel Nepodařilo se vytvořit kanál s zjištěných koncový bod s EndpointAddress = '%1' a prostřednictvím = '%2'. DiscoveryClientChannel se nyní pokusí použít další dostupné zjištěných koncový bod.
