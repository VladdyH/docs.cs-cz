## <a name="description"></a>Popis  
 Tato událost je vygenerované při zjišťování zpráva byla vyřazena pomocí objekty DiscoveryClient, protože nemá vlastnost DiscoveryMessageSequence.  
  
## <a name="message"></a>Zpráva  
 Zprávu %1 s messageId = '%2' byla vynechána pomocí objekty DiscoveryClient, protože nemá vlastnost DiscoveryMessageSequence.
