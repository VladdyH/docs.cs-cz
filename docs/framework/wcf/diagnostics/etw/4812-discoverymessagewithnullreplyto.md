|||  
|-|-|  
|ID|4812|  
|Klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerované při zjišťování zpráva byla vyřazena pomocí objekty DiscoveryClient, protože nemá ReplyTo adresu.  
  
## <a name="message"></a>Zpráva  
 Zprávu požadavku zjišťování s messageId = '%1' byla vyřazena, protože neobsahovala adresu ReplyTo.
