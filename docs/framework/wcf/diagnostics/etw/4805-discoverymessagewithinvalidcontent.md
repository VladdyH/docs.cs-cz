## <a name="description"></a>Popis  
 Tato událost je vygenerované při zjišťování zpráva obsahuje neplatný obsah.  
  
## <a name="message"></a>Zpráva  
 Zprávu %1 s messageId = '%2' byla vynechána, protože mělo neplatný obsah.
