# <a name="3803---routingserviceprocessingfailure"></a>3803 – RoutingServiceProcessingFailure
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|3803|  
|Klíčová slova|RoutingServices|  
|úroveň|Chyba|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerované při spuštění služby směrování se nezdařilo při zpracování zprávy v koncovém bodě.  
  
## <a name="message"></a>Zpráva  
 Služba směrování nemůže při zpracování zprávy v koncovém bodě s adresou '%1'.
