|||  
|-|-|  
|ID|4821|  
|Klíčová slova|Zjišťování|  
|úroveň|Informace o|

## <a name="description"></a>Popis  
 Tato událost je vygenerované při SynchronizationContext byla nastavena na hodnotu null ve DiscoveryClientChannel před zahájením operace hledání.  
  
## <a name="message"></a>Zpráva  
 SynchronizationContext byla nastavena na hodnotu null ve DiscoveryClientChannel před zahájením operace hledání.
