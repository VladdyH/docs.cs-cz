# <a name="4818---innerchannelopenfailed"></a>4818 – InnerChannelOpenFailed
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4818|  
|Klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerované při DiscoveryClientChannel se nepodařilo otevřít kanál s zjištěných koncový bod. DiscoveryClientChannel se nyní pokusí použít další dostupné zjištěných koncový bod.  
  
## <a name="message"></a>Zpráva  
 DiscoveryClientChannel se nepodařilo otevřít kanál s zjištěných koncový bod s EndpointAddress = '%1' a prostřednictvím = '%2'. DiscoveryClientChannel se nyní pokusí použít další dostupné zjištěných koncový bod.
