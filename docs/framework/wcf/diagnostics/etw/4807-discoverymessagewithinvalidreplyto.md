# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 – DiscoveryMessageWithInvalidReplyTo
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|4807|  
|Klíčová slova|Zjišťování|  
|úroveň|Upozornění|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Tato událost je vygenerované při zjišťování zpráva byla vyřazena, protože mělo neplatnou adresu ReplyTo.  
  
## <a name="message"></a>Zpráva  
 Zprávu požadavku zjišťování s messageId = '%1' byla vynechána, protože mělo neplatnou adresu ReplyTo.
