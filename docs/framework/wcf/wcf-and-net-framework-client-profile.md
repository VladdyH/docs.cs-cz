# <a name="wcf-and-net-framework-client-profile"></a>WCF a profil klienta .NET Framework

## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Funkce WCF nepodporuje rozhraní .net Framework Client Profile  
 Jsou podporovány následující funkce Windows Communication Foundation [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]:  
  
-   Všechny služby WCF je podporována s výjimkou Cardspace a publikování na webu.  
  
-   Vzdálená komunikace TCP/IP kanály jsou podporovány.  
  
-   ASMX (Web Services) nejsou podporovány.
