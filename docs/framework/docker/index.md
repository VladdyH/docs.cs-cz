# <a name="deploying-net-framework-applications-with-docker"></a>Nasazení aplikací rozhraní .NET Framework s Docker

Můžete nasadit aplikace rozhraní .NET Framework s Docker, použití Windows kontejnerů. Další požadavky pro používání [Windows kontejnery](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview) a postup [začít pracovat s Docker pro systém Windows](https://docs.docker.com/docker-for-windows/). 

Můžete začít pomocí [konzolovou aplikaci s Docker](console.md).
Pro spuštění webové aplikace v nástroji Docker, můžete si přečíst [aplikací ASP.NET MVC v Docker](/aspnet/mvc/overview/deployment/docker-aspnetmvc).
