 <span data-ttu-id="abd32-207">Tato vazba umožňuje spolehlivě komunikovat prostřednictvím jednoho nebo více zprostředkovatelů protokolu SOAP.</span><span class="sxs-lookup"><span data-stu-id="abd32-207">This binding can be used to communicate reliably through one or more SOAP intermediaries.</span></span>

## <a name="example"></a><span data-ttu-id="abd32-209">Příklad</span><span class="sxs-lookup"><span data-stu-id="abd32-209">Example</span></span>  
  
```xml  
<configuration>  
<system.ServiceModel>  
<bindings>  
<wsDualHttpBinding>  
    <binding   
        closeTimeout="00:00:10"  
        openTimeout="00:00:20"   
        receiveTimeout="00:00:30"  
        sendTimeout="00:00:40"  
        bypassProxyOnLocal="false"
```  
