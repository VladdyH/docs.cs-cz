# <a name="clr-user-defined-functions"></a>Uživatelem definované funkce CLR
Uživatelem definované funkce jsou rutiny, které přijímají parametry, provádět výpočty nebo jiné akce a vrátí výsledek. Uživatelem definované funkce můžete psát v libovolné rozhraní Microsoft .NET Framework programovací jazyk, jako je například Microsoft Visual Basic .NET nebo Microsoft Visual C#.  
  
 Další informace najdete v tématu [CLR User-Defined funkce](/sql/relational-databases/clr-integration-database-objects-user-defined-functions/clr-user-defined-functions).
