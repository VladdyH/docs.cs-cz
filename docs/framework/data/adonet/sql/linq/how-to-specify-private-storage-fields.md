# <a name="how-to-specify-private-storage-fields"></a>Postupy: Zadejte privátní úložiště polí
Použití [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> vlastnost <xref:System.Data.Linq.Mapping.DataAttribute> atribut k určení názvu na základní pole úložiště.  
  
 Příklady kódu najdete v tématu <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
### <a name="to-specify-the-name-of-an-underlying-storage-field"></a>Zadat název na základní pole úložiště  
  
1.  Přidat <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> vlastnost, která má <xref:System.Data.Linq.Mapping.ColumnAttribute> atribut.  
  
2.  Přiřadit název pole jako hodnotu <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> vlastnost.
