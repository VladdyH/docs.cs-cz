# <a name="aggregate-queries"></a>Agregační dotazy
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]podporuje `Average`, `Count`, `Max`, `Min`, a `Sum` agregovat operátory. Poznámka: následující vlastnosti agregační operátorů v [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]:  
  
-   Agregační dotazy jsou spustit okamžitě.  
  
     Další informace najdete v tématu [Úvod do dotazů LINQ (C#)](~/docs/csharp/programming-guide/concepts/linq/introduction-to-linq-queries.md).  
  
-   Agregační dotazy obvykle vracet číslo místo kolekce.  
  
     Další informace najdete v tématu [agregační operace](http://msdn.microsoft.com/library/36d97c83-5de5-457d-971d-10a69365e7c4).  
  
-   Nelze volat agregace proti anonymní typy.  
  
 Příklady v následujících tématech jsou odvozeny od ukázková databáze Northwind. Další informace najdete v tématu [stažení ukázkové databáze](../../../../../../docs/framework/data/adonet/sql/linq/downloading-sample-databases.md).
