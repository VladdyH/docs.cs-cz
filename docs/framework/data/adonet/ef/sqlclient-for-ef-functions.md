# <a name="sqlclient-for-entity-framework-functions"></a>SqlClient pro funkce Entity Framework
Zprostředkovatel dat .NET Framework pro SQL Server (SqlClient) pro rozhraní Entity Framework poskytuje sadu funkcí k provedení matematické a výpočty agregace, jakož i funkce k provedení `System.DateTime` a `string` operace. Tyto funkce jsou v `SQLServer` oboru názvů.  
  
 Seznam funkcí, které by měly spolupracovat s kteréhokoli poskytovatele služeb, najdete v tématu [kanonické funkce](../../../../../docs/framework/data/adonet/ef/language-reference/canonical-functions.md).  
  
 Informace o jak kanonické funkce mapu, aby funkce SQL Server najdete v tématu [koncepční kanonický modelu pro mapování funkce SQL Server](../../../../../docs/framework/data/adonet/ef/conceptual-model-canonical-to-sql-server-functions-mapping.md).
