 <span data-ttu-id="29603-192">Při rozpoznávání agregace, [!INCLUDE[esql](../../../../../../includes/esql-md.md)] poprvé pokusí o považovat za agregace se na základě kolekce.</span><span class="sxs-lookup"><span data-stu-id="29603-192">When resolving an aggregate, [!INCLUDE[esql](../../../../../../includes/esql-md.md)] first tries to treat it as a collection-based aggregate.</span></span> <span data-ttu-id="29603-193">Pokud to nepomůže, [!INCLUDE[esql](../../../../../../includes/esql-md.md)] transformuje agregační vstup na odkaz k agregaci vnoření a pokusí přeložit nové výrazu, jak je znázorněno v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="29603-193">If that fails, [!INCLUDE[esql](../../../../../../includes/esql-md.md)] transforms the aggregate input into a reference to the nest aggregate and tries to resolve this new expression, as illustrated in the following example.</span></span>
