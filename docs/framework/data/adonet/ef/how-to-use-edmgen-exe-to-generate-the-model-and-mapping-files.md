### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="74ab4-116">Generování modelu školní pro použití EdmGen.exe projektu Visual Basic</span><span class="sxs-lookup"><span data-stu-id="74ab4-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1.  <span data-ttu-id="74ab4-117">Vytvořte databázi školy.</span><span class="sxs-lookup"><span data-stu-id="74ab4-117">Create the School database.</span></span> <span data-ttu-id="74ab4-118">Další informace najdete v tématu [vytváření ukázkovou databázi školy](http://msdn.microsoft.com/library/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span><span class="sxs-lookup"><span data-stu-id="74ab4-118">For more information, see [Creating the School Sample Database](http://msdn.microsoft.com/library/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span></span>  
  
2.  <span data-ttu-id="74ab4-119">Na příkazovém řádku spusťte následující příkaz bez zalomení řádků:</span><span class="sxs-lookup"><span data-stu-id="74ab4-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
