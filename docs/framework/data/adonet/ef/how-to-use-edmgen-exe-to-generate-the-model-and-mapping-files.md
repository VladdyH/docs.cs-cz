### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="f6cd1-116">Generování modelu školní pro použití EdmGen.exe projektu Visual Basic</span><span class="sxs-lookup"><span data-stu-id="f6cd1-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="f6cd1-119">Na příkazovém řádku spusťte následující příkaz bez zalomení řádků:</span><span class="sxs-lookup"><span data-stu-id="f6cd1-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```

    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
