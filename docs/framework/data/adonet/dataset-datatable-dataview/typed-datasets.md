# <a name="typed-datasets"></a>Typové datové sady
Společně s pozdní vázané přístup k hodnotám prostřednictvím slabě typovaná proměnné <xref:System.Data.DataSet> poskytuje přístup k datům prostřednictvím silného typu jedná. Tabulky a sloupce, které jsou součástí **datovou sadu** lze přistupovat pomocí uživatelsky přívětivých názvů a silného typu proměnné.  
  
 Představuje zadaný **datovou sadu** je třída, která je odvozena z **datovou sadu**. Jako takový dědí všechny metody, události a vlastnosti **datovou sadu**. Kromě toho typové **datovou sadu** poskytuje silného typu metody, události a vlastnosti. To znamená, že dostanete tabulky a sloupce, podle názvu, namísto použití metody založené na kolekcích. Kromě zajištění dostatečného lepší čitelnost kódu, představuje zadaný **datovou sadu** také umožňuje kód sady Visual Studio .NET editor při psaní automaticky dokončit řádky.  
  
 Kromě toho silného **datovou sadu** poskytuje přístup k hodnotám jako správného typu v době kompilace. S silného typu **datovou sadu**, chyby neshoda typu jsou zachyceny, pokud kód je kompilovat spíše než v dobu běhu.
