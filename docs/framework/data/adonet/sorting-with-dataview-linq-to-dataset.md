 <span data-ttu-id="eeb1f-112">Založené na výrazech řazení nabízí výkonný a komplexní řazení než založené na řetězcích řazení jednodušší.</span><span class="sxs-lookup"><span data-stu-id="eeb1f-112">Expression-based sorting offers more powerful and complex sorting than the simpler string-based sorting.</span></span> <span data-ttu-id="eeb1f-113">Všimněte si, že na základě řetězců a založené na výrazech řazení se vzájemně vylučují.</span><span class="sxs-lookup"><span data-stu-id="eeb1f-113">Note that string-based and expression-based sorting are mutually exclusive.</span></span> <span data-ttu-id="eeb1f-114">Pokud řetězec na základě <xref:System.Data.DataView.Sort%2A> se nastaví po <xref:System.Data.DataView> je vytvořena z dotazu, založené na výrazu filtru odvodit z dotazu není zaškrtnuté a nelze jej obnovit.</span><span class="sxs-lookup"><span data-stu-id="eeb1f-114">If the string-based <xref:System.Data.DataView.Sort%2A> is set after a <xref:System.Data.DataView> is created from a query, the expression-based filter inferred from the query is cleared and cannot be reset.</span></span>  
  
 <span data-ttu-id="eeb1f-115">Index <xref:System.Data.DataView> vychází i v případě <xref:System.Data.DataView> je vytvořena a při řazení nebo filtrování je upravit informace.</span><span class="sxs-lookup"><span data-stu-id="eeb1f-115">The index for a <xref:System.Data.DataView> is built both when the <xref:System.Data.DataView> is created and when any of the sorting or filtering information is modified.</span></span> <span data-ttu-id="eeb1f-116">Získáte nejlepší výkon zadáním kritérií pro řazení [!INCLUDE[linq_dataset](../../../../includes/linq-dataset-md.md)] dotaz, který <xref:System.Data.DataView> je vytvořený z a není úprava řazení informace později.</span><span class="sxs-lookup"><span data-stu-id="eeb1f-116">You get the best performance by supplying sorting criteria in the [!INCLUDE[linq_dataset](../../../../includes/linq-dataset-md.md)] query that the <xref:System.Data.DataView> is created from and not modifying the sorting information, later.</span></span> <span data-ttu-id="eeb1f-117">Další informace najdete v tématu [zobrazení dat výkonu](../../../../docs/framework/data/adonet/dataview-performance.md).</span><span class="sxs-lookup"><span data-stu-id="eeb1f-117">For more information, see [DataView Performance](../../../../docs/framework/data/adonet/dataview-performance.md).</span></span>
