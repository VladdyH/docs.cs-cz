# <a name="install-the-net-framework-for-developers"></a>Nainstalujte rozhraní .NET Framework pro vývojáře

Rozhraní .NET je nedílnou součástí mnoho aplikace běžící v systému Windows a poskytuje běžné funkce pro tyto aplikace spouštět. Pro vývojáře rozhraní .NET Framework poskytuje komplexní a konzistentní programovací model pro vytváření aplikací, které mají vizuálně omráčení koncových uživatelů a hladkou a zabezpečenou komunikaci.  

> [!NOTE]
> Toto téma je určeno pro **vývojáři** kdo buď chcete nainstalovat rozhraní .NET Framework na své vlastní systému nebo kteří chtějí nainstalovat pomocí svých aplikací. Pro **uživatelé** zajímá instalace rozhraní .NET Framework, najdete v jednotlivých tématech, která popisuje instalaci rozhraní .NET Framework na konkrétní operační systémy, jako například [nainstalovat rozhraní .NET Framework na Windows 10 a Windows Server 2016](on-windows-10.md).  

Tento článek obsahuje odkazy pro instalaci z rozhraní .NET Framework 4.5 na všechny verze rozhraní .NET Framework [!INCLUDE[net_current](../../../includes/net-current-version.md)] ve vašem počítači. Pokud jste vývojář, můžete také pomocí těchto odkazů můžete stáhnout a znovu distribuovat rozhraní .NET Framework s vaší aplikací. Informace o nasazení verzi rozhraní .NET Framework s vaší aplikací najdete v tématu [rozhraní .NET Framework – Průvodce nasazením pro vývojáře](../deployment/deployment-guide-for-developers.md).

## <a name="installation-choices"></a>Možnosti instalace

Nainstalujte vývojář cílení pack vyvíjet na nejnovější verzi rozhraní .NET Framework v sadě Visual Studio nebo jiné vývojové prostředí nebo stáhnout rozhraní .NET Framework redistributable pro distribuci pomocí aplikace nebo ovládací prvek.  
  
### <a name="to-install-the-net-framework-developer-pack-or-targeting-pack"></a>K instalaci rozhraní .NET Framework Developer Pack nebo cílení Pack

A *cílení pack* umožní aplikaci cílení na konkrétní verzi rozhraní .NET Framework při vývoji v sadě Visual Studio a některé vývojové prostředí. A *developer pack* obsahuje konkrétní verzi rozhraní .NET Framework a jeho doprovodné SDK společně s jeho odpovídající cílení pack.

Sada vývojáře pro rozhraní .NET Framework 4.5.1 nebo 4.5.2, sada cílení pro rozhraní .NET Framework 4.6 a sada pro vývojáře pro rozhraní .NET Framework 4.6.1 4.6.2, 4.7 4.7.1 nebo 4.7.2 poskytuje referenční sestavení, na konkrétní rozhraní .NET Framework verze jazykové sady a soubory IntelliSense pro použití v integrovaném vývojovém prostředí, třeba v sadě Visual Studio.  Pokud používáte Visual Studio, sada pro vývojáře nebo cílení pack nainstalovaná verze rozhraní .NET Framework také přidá do cílové volby při vytvoření nového projektu.  Vyberte jednu z následujících možností:  

- [Rozhraní Microsoft .NET Framework 4.7.2 Developer Pack](http://go.microsoft.com/fwlink/?LinkId=863261)

- [Rozhraní Microsoft .NET Framework 4.7.1 Developer Pack](http://go.microsoft.com/fwlink/?LinkId=852105)

- [Rozhraní Microsoft .NET Framework 4.7 Developer Pack](http://go.microsoft.com/fwlink/?LinkId=825319)

Pokud cílíte na konkrétní verzi rozhraní .NET Framework, vaše aplikace je sestavena pomocí referenční sestavení, které jsou součástí této verze developer pack. V době běhu se přeloží sestavení z globální mezipaměti sestavení a referenční sestavení nepoužívají.

Při vytváření aplikace z Visual Studio nebo pomocí nástroje MSBuild z příkazového řádku, MSBuild může zobrazit chyba MSB3644, "Referenční sestavení pro architekturu"*verze framework*"nebyly nalezeny." Chcete-li vyřešit chyby, stáhněte si sada pro vývojáře nebo sada cílení pro tuto verzi rozhraní .NET Framework.   

### <a name="to-install-or-download-the-net-framework-redistributable"></a>Pro instalaci nebo stáhnout rozhraní .NET Framework redistributable

Instalační programy stáhnout součásti rozhraní .NET Framework pro aplikace nebo ovládací prvek, který se zaměřuje na tyto verze rozhraní .NET Framework. Tyto komponenty musí být nainstalován na každém počítači, kde je spuštěna aplikace nebo ovládací prvek. Tyto instalační programy jsou redistributable, takže je možné je zadat v instalačním programu pro vaši aplikaci.  
  
Stránce pro stažení je k dispozici v několika jazycích, ale většina stahování jsou uvedena pouze v angličtině. Pro podporu dalších jazyků musíte nainstalovat jazykovou sadu.  
  
K dispozici jsou dva typy redistributable instalačních programů:  
  
- **Instalační program webové** (webový zaváděcí nástroj), soubory ke stažení požadované součásti a jazykové sady, která odpovídá operačnímu systému počítače instalaci z webu. Tento balíček je mnohem menší než offline instalačního programu, ale vyžaduje konzistentní připojení k Internetu. Si můžete stáhnout [samostatné jazykové sady](#standalone_language_packs) k instalaci podpory dalších jazyků.  
  
- **Offline instalační program** (samostatný redistributable) obsahuje všechny požadované součásti pro instalaci rozhraní .NET Framework, ale neobsahuje jazykových sad. Tento soubor ke stažení je větší než webovou Instalační službu. Offline instalační program nevyžaduje připojení k Internetu. Po spuštění offline instalačního programu si můžete stáhnout [samostatné jazykové sady](#standalone_language_packs) k instalaci podpory jazyka. Použití offline instalačního programu, pokud spoléháte nemůže mít konzistentní připojení k Internetu.  
  
Web a offline instalační programy jsou určeny pro počítače x86 x 86 i na x64 (viz [požadavky na systém](~/docs/framework/get-started/system-requirements.md)), ale nepodporují počítačích s procesorem Itanium.

## <a name="to-install-language-packs"></a>Chcete-li nainstalovat jazykové sady

 Jazykové sady jsou spustitelné soubory, které obsahují lokalizované prostředky (například přeložený chybové zprávy a textu uživatelského rozhraní) podporovaných jazyků. Pokud nenainstalujete jazykové sady, rozhraní .NET Framework chybové zprávy a další text se zobrazí v angličtině.  Všimněte si, že webový instalační program automaticky nainstaluje jazyková sada, která odpovídá operačnímu systému, ale další jazykových sad můžete stáhnout do počítače. Offline instalační programy neobsahují všechny jazykové sady. 
  
> [!IMPORTANT]
> Jazykové sady neobsahují součásti rozhraní .NET Framework, které jsou nutné ke spuštění aplikace, takže je nutné spustit web nebo offline instalační program před instalací jazykové sady. Pokud jste již nainstalovali jazykovou sadu, odinstalujte ji, nainstalujte rozhraní .NET Framework a poté znovu nainstalujte jazykové sady.  
  
1.  Otevřete stránku language pack stahování pro verzi rozhraní .NET Framework, který jste nainstalovali:  

    - [Rozhraní .NET framework 4.7.2 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=863258) 
  
    - [Rozhraní .NET framework 4.7.1 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=852090) 

    - [Rozhraní .NET framework 4.7 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=825306) 

    - [Rozhraní .NET framework 4.6.2 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=780604)  
  
    - [Rozhraní .NET framework 4.6.1 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=671747)  
  
    - [Rozhraní .NET framework 4.6 jazykové sady](http://go.microsoft.com/fwlink/?LinkID=528314)  
  
    - [Rozhraní .NET framework 4.5.2 jazykové sady](http://go.microsoft.com/fwlink/?LinkId=397701)  
  
    - [Rozhraní .NET framework 4.5.1 jazykové sady](http://go.microsoft.com/fwlink/?LinkId=322101)  
  
    - [Rozhraní .NET framework 4.5 jazykové sady](http://go.microsoft.com/fwlink/p/?LinkId=245451)  
  
2.  V seznamu jazyk vyberte jazyk, který chcete stáhnout a počkejte několik sekund pro stránku načíst znovu v daném jazyce.  
  
3.  Zvolte **Stáhnout**.
