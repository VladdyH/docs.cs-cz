# <a name="connection-string-and-connection-string-name"></a>Připojovací řetězec a název připojovacího řetězce
**Připojovací řetězec** vlastnost určuje připojovací řetězec, který úložiště Instance pracovního postupu SQL měla použít pro připojení k databázi trvalost. Tento parametr je volitelný parametr. **Název připojovacího řetězce** vlastnost určuje název pojmenovaného připojovací řetězec, který úložiště Instance pracovního postupu SQL měla použít pro připojení k databázi trvalost. Tento parametr je volitelný parametr. Musí zadat hodnotu pro vlastnost název připojovacího řetězce nebo vlastnost připojovací řetězec, pokud nechcete, aby úložiště Instance SQL pracovního postupu, který chcete použít výchozí připojovací řetězec s názvem **DefaultSqlWorkflowInstanceStoreConnectionString** .
