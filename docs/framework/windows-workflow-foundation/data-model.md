# <a name="windows-workflow-foundation-data-model"></a>Windows Workflow Foundation datový Model
Datový model Windows Workflow Foundation tvoří tři koncepty: proměnné, argumentů a výrazy. Proměnné představují úložiště dat a argumenty představují tok dat do a z aktivity. Argumenty jsou svázané s (přiřazenou hodnotu) pomocí výrazů, které mohou odkazovat proměnné.
