# <a name="1223---switchcasenotfound"></a>1223 - SwitchCaseNotFound
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1223|  
|Klíčová slova|WFActivities|  
|úroveň|Informace o|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že případu aktivity s odpovídající výsledek výrazu nebyl nalezen v aktivitě přepínače.  
  
## <a name="message"></a>Zpráva  
 Aktivita přepínači '%1' nelze nalézt aktivitu případu odpovídající výsledek výrazu.
