# <a name="1025---bookmarkscopeinitialized"></a>1025 - BookmarkScopeInitialized
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1025|  
|Klíčová slova|WFRuntime|  
|úroveň|Verbose|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že byl inicializován BookmarkScope.  
  
## <a name="message"></a>Zpráva  
 BookmarkScope, který měl TemporaryId: '%1' byl inicializován s Id: '%2'.  
  
## <a name="details"></a>Podrobnosti  
  
|Název položky dat|Datová položka – Typ|Popis|  
|--------------------|--------------------|-----------------|  
|TemporaryId|xs:String|Dočasné id záložky.|  
|InitializedId|xs:String|Inicializovaného id záložky.|  
|Domény aplikace|xs:String|Řetězec vrácený AppDomain.CurrentDomain.FriendlyName.|
