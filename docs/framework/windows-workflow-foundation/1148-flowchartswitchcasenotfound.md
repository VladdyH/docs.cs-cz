# <a name="1148---flowchartswitchcasenotfound"></a>1148 - FlowchartSwitchCaseNotFound
## <a name="properties"></a>Vlastnosti  
  
|||  
|-|-|  
|ID|1148|  
|Klíčová slova|WFActivities|  
|úroveň|Informace o|  
|Kanál|Aplikaci Microsoft Windows Server – aplikace/Debug|  
  
## <a name="description"></a>Popis  
 Označuje, že ani odpovídající případu nebo výchozí případu v přepínači vývojový diagram nebyla nalezena. Vývojový diagram provádění se ukončí.  
  
## <a name="message"></a>Zpráva  
 Vývojový diagram ' %1'/FlowSwitch - může najít aktivitu případu ani výchozí případu odpovídající výsledek výrazu. Vývojový diagram provádění se ukončí.  
  
## <a name="details"></a>Podrobnosti  
  
|Název položky dat|Datová položka – Typ|Popis|  
|--------------------|--------------------|-----------------|  
|Vývojový diagram|xs:String|Zobrazovaný název vývojový diagram.|  
|Domény aplikace|xs:String|Řetězec vrácený AppDomain.CurrentDomain.FriendlyName.|
