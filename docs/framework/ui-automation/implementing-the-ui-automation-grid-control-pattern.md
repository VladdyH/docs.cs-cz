 <span data-ttu-id="32544-115">![Průzkumník Windows zobrazení zobrazující nepravidelné rozložení. ] (../../../docs/framework/ui-automation/media/uia-gridpattern-ragged-array.PNG "UIA_GridPattern_Ragged_Array")</span><span class="sxs-lookup"><span data-stu-id="32544-115">![Windows Explorer view showing ragged layout.](../../../docs/framework/ui-automation/media/uia-gridpattern-ragged-array.PNG "UIA_GridPattern_Ragged_Array")</span></span>  
<span data-ttu-id="32544-116">Příklad ovládacího prvku mřížky s prázdnou souřadnice</span><span class="sxs-lookup"><span data-stu-id="32544-116">Example of a Grid Control with Empty Coordinates</span></span>  
  
-   <span data-ttu-id="32544-117">Mřížka s jednou položkou je stále vyžadují k implementaci <xref:System.Windows.Automation.Provider.IGridProvider> Pokud logicky se považuje za mřížka.</span><span class="sxs-lookup"><span data-stu-id="32544-117">A grid with a single item is still required to implement <xref:System.Windows.Automation.Provider.IGridProvider> if it is logically considered to be a grid.</span></span> <span data-ttu-id="32544-118">Počet podřízené položky v mřížce je důležité.</span><span class="sxs-lookup"><span data-stu-id="32544-118">The number of child items in the grid is immaterial.</span></span>  
  
-   <span data-ttu-id="32544-119">Skryté řádky a sloupce, v závislosti na implementaci zprostředkovatele, mohou být načteny v [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] stromu a proto budou promítnuta <xref:System.Windows.Automation.GridPattern.GridPatternInformation.RowCount%2A> a <xref:System.Windows.Automation.GridPattern.GridPatternInformation.ColumnCount%2A> vlastnosti.</span><span class="sxs-lookup"><span data-stu-id="32544-119">Hidden rows and columns, depending on the provider implementation, may be loaded in the [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] tree and therefore will be reflected in the <xref:System.Windows.Automation.GridPattern.GridPatternInformation.RowCount%2A> and <xref:System.Windows.Automation.GridPattern.GridPatternInformation.ColumnCount%2A> properties.</span></span> <span data-ttu-id="32544-120">Pokud dosud nebyly byl načteny skryté řádky a sloupce, by neměly být započítány.</span><span class="sxs-lookup"><span data-stu-id="32544-120">If the hidden rows and columns have not yet been loaded, they should not be counted.</span></span>  
  
-   <span data-ttu-id="32544-121"><xref:System.Windows.Automation.Provider.IGridProvider> neumožňuje active zpracování mřížka; <xref:System.Windows.Automation.Provider.ITransformProvider> Pokud chcete tuto funkci povolit, musí být implementována.</span><span class="sxs-lookup"><span data-stu-id="32544-121"><xref:System.Windows.Automation.Provider.IGridProvider> does not enable active manipulation of a grid; <xref:System.Windows.Automation.Provider.ITransformProvider> must be implemented to enable this functionality.</span></span>  
  
-   <span data-ttu-id="32544-122">Použití <xref:System.Windows.Automation.StructureChangedEventHandler> naslouchat strukturálních nebo rozložení mřížky například buněk, které jste přidali, odebrat nebo sloučit změny.</span><span class="sxs-lookup"><span data-stu-id="32544-122">Use a <xref:System.Windows.Automation.StructureChangedEventHandler> to listen for structural or layout changes to the grid such as cells that have been added, removed, or merged.</span></span>  
  
-   <span data-ttu-id="32544-123">Použití <xref:System.Windows.Automation.AutomationFocusChangedEventHandler> sledovat traversal prostřednictvím položky nebo buněk mřížka.</span><span class="sxs-lookup"><span data-stu-id="32544-123">Use a <xref:System.Windows.Automation.AutomationFocusChangedEventHandler> to track traversal through the items or cells of a grid.</span></span>
