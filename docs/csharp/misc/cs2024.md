# <a name="compiler-error-cs2024"></a>Compiler Error CS2024
Neplatný soubor části zarovnání číslo '#'  
  
 Byla předána neplatná hodnota [/filealign](../../csharp/language-reference/compiler-options/filealign-compiler-option.md) – možnost kompilátoru.  
  
 Následující ukázka generuje CS2024:  
  
```csharp  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
