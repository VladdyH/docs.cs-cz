<span data-ttu-id="dd9cf-103">Operátor 'operátor' je nejednoznačný na operand typu "typ"</span><span class="sxs-lookup"><span data-stu-id="dd9cf-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="dd9cf-104">Kompilátor má více než jeden dostupný převod a nebude vědět, který zvolit před použitím operátor.</span><span class="sxs-lookup"><span data-stu-id="dd9cf-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="dd9cf-105">Další informace najdete v tématu [Šablonované uživatele definované převody](/cpp/misc/templated-user-defined-conversions) a [operátory převodu](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="dd9cf-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="dd9cf-106">Následující ukázka generuje CS0035:</span><span class="sxs-lookup"><span data-stu-id="dd9cf-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
