## <a name="example"></a><span data-ttu-id="caadc-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="caadc-105">Example</span></span>  
 <span data-ttu-id="caadc-106">Následující kód generuje CS1715.</span><span class="sxs-lookup"><span data-stu-id="caadc-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
