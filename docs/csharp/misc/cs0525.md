# <a name="compiler-error-cs0525"></a>Compiler Error CS0525
Rozhraní nemohou obsahovat pole  
  
 [Rozhraní](../../csharp/language-reference/keywords/interface.md) může obsahovat metody a vlastnosti, ale není pole.  
  
 Následující ukázka generuje CS0525:  
  
```csharp  
// CS0525.cs  
namespace x  
{  
   public interface clx  
   {  
      public int i;   // CS0525  
   }  
}  
```
