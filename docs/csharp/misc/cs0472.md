# <a name="compiler-warning-level-2-cs0472"></a>CS0472 kompilátoru upozornění (úroveň 2)
Výsledkem výrazu je vždy 'value1' vzhledem k tomu, že se hodnota typu 'hodnota2' se nikdy rovná null typu 'hodnota3.  
  
 Kompilátor měli upozornit, pokud použijete operátor s konstantní hodnotu null.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0472.  
  
```csharp  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
