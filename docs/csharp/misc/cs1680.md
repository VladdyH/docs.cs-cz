# <a name="compiler-error-cs1680"></a>CS1680 chyby kompilátoru
Neplatný parametr aliasu odkazu: 'alias =' – chybí název souboru.  
  
 K této chybě dojde, pokud použijete `alias` funkci s **/reference** – možnost kompilátoru bez zadání platný název souboru.  
  
 Následující ukázka generuje CS1680.  
  
```  
// CS1680.cs  
// compile with: /reference:alias=  
// CS1680 expected  
// To resolve, specify the name of a file with an assembly manifest  
class MyClass {}  
```
