# <a name="compiler-error-cs1555"></a>CS1555 chyby kompilátoru
Nebyla nalezena třída zadané pro metodu Main  
  
 Třída zadaná k [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru, ale název třídy nebyl nalezen ve zdrojovém kódu.
