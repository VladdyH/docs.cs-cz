<span data-ttu-id="21634-103">Struktury nemůže obsahovat explicitní bezparametrové konstruktory</span><span class="sxs-lookup"><span data-stu-id="21634-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="21634-104">Každý [struktura](../../csharp/language-reference/keywords/struct.md) již má výchozí konstruktor, který inicializuje objekt na nulu.</span><span class="sxs-lookup"><span data-stu-id="21634-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="21634-105">Konstruktory, které můžete vytvořit pro struktury proto musí mít jeden nebo více parametrů.</span><span class="sxs-lookup"><span data-stu-id="21634-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="21634-106">Následující ukázka generuje CS0568:</span><span class="sxs-lookup"><span data-stu-id="21634-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
