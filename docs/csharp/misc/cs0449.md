 <span data-ttu-id="3dc2d-104">Omezení u parametru typu obecný typ nebo metoda se musí vyskytovat v určitém pořadí: `class` nebo `struct` musí být první, pokud je k dispozici, pak žádná omezení rozhraní a nakonec žádná omezení konstruktor.</span><span class="sxs-lookup"><span data-stu-id="3dc2d-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="3dc2d-105">Tato chyba je způsobená `class` nebo `struct` omezení, která nejsou uvedena první.</span><span class="sxs-lookup"><span data-stu-id="3dc2d-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="3dc2d-106">Pokud chcete tuto chybu vyřešit, změnit jeho pořadí klauzulích omezení.</span><span class="sxs-lookup"><span data-stu-id="3dc2d-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3dc2d-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="3dc2d-107">Example</span></span>  
 <span data-ttu-id="3dc2d-108">Následující ukázka generuje CS0449.</span><span class="sxs-lookup"><span data-stu-id="3dc2d-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
