# <a name="compiler-error-cs1650"></a><span data-ttu-id="7fc41-102">CS1650 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="7fc41-102">Compiler Error CS1650</span></span>
<span data-ttu-id="7fc41-103">Nelze přiřadit pole statického pole "identifikátor" (s výjimkou statického konstruktoru nebo inicializátoru proměnné)</span><span class="sxs-lookup"><span data-stu-id="7fc41-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="7fc41-104">K této chybě dojde, když se pokusí změnit členem pole, který je jen pro čtení a statické kde není povoleno upravovat.</span><span class="sxs-lookup"><span data-stu-id="7fc41-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="7fc41-105">Pokud chcete tuto chybu vyřešit, omezení přiřazení pole jen pro čtení do konstruktoru nebo inicializátor proměnné nebo odebrat `readonly` – klíčové slovo z deklarace pole.</span><span class="sxs-lookup"><span data-stu-id="7fc41-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
