# <a name="compiler-error-cs0589"></a>CS0589 chyby kompilátoru
Vnitřní chyba kompilátoru: fáze 'analyzovat.  
  
 Pokuste se zjistit, pokud kompilátor selhává kvůli jeho nemůže analyzovat neočekávanou syntaxi. Pokud není tento případ, zkuste [kontaktujte nás](/visualstudio/ide/talk-to-us).
