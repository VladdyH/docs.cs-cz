# <a name="compiler-error-cs1551"></a>CS1551 chyby kompilátoru
Indexery musí mít minimálně jeden parametr  
  
 [Indexer](../../csharp/programming-guide/indexers/index.md) , aby byla deklarována trvá žádné argumenty.  
  
 Následující ukázka generuje CS1551:  
  
```  
// CS1551.cs  
public class MyClass  
{  
   int intI;  
  
   int this[]   // CS1551  
   // try the following line instead  
   // int this[int i]  
   {  
      get  
      {  
         return intI;  
      }  
      set  
      {  
         intI = value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
