 <span data-ttu-id="8aad7-104">Proměnné byla nalezena, přičemž byla očekávána konstanta.</span><span class="sxs-lookup"><span data-stu-id="8aad7-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="8aad7-105">Další informace najdete v tématu [přepínač](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="8aad7-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="8aad7-106">Následující ukázka generuje CS0150:</span><span class="sxs-lookup"><span data-stu-id="8aad7-106">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```

    }
