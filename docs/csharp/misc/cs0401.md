# <a name="compiler-error-cs0401"></a>CS0401 chyby kompilátoru
Omezení new() musí být poslední zadané omezení.  
  
 Pokud používáte více omezení, zobrazí seznam všech omezení před omezení new().  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0401.  
  
```  
// CS0401.cs  
// compile with: /target:library  
using System;  
 class C<T> where T : new(), IDisposable {}  // CS0401  
  
class D<T> where T : IDisposable  
{  
   static void F<U>() where U : new(), IDisposable{}   // CS0401  
}  
```
