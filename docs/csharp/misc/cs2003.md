# <a name="compiler-error-cs2003"></a>CS2003 chyby kompilátoru
Soubor odpovědí 'file' zahrnuta vícekrát  
  
 Kompilátor byl předán soubor odezvy více než jednou. Soubor odpovědí můžete předat pouze kompilátoru jednou za výstupní soubor.  
  
 Další informace o souborech odpovědí, najdete v části [@ (určení souboru odezvy)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md).
