# <a name="compiler-error-cs0148"></a>CS0148 chyby kompilátoru
Delegát 'delegáta' nemá platný konstruktor  
  
 Importovat a použít spravovaný program (ten, který používá rozhraní .NET Framework common language runtime), byl vytvořen s jinou kompilátoru. Tento kompilátoru povolené nedůvěryhodný [delegovat](../../csharp/language-reference/keywords/delegate.md) konstruktor. Další informace najdete v tématu [delegáti](../../csharp/programming-guide/delegates/index.md).
