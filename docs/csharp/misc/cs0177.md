# <a name="compiler-error-cs0177"></a>CS0177 chyby kompilátoru
Výstupní parametr "parametr" musí být přiřazená k předtím, než aktuální metoda opustí ovládací prvek  
  
 Označené jako parametr [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) – klíčové slovo nebyla přiřazena hodnota v těle metoda. Další informace najdete v tématu [předávání parametrů](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 Následující ukázka generuje CS0177:  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
