# <a name="compiler-error-cs2015"></a>CS2015 chyby kompilátoru
'file' je binární soubor, místo do textového souboru  
  
 Soubor byl předán kompilátoru, která byla binární soubor. Kompilátor očekává souboru zdrojového kódu.
