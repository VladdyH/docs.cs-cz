# <a name="compiler-error-cs1585"></a>CS1585 chyby kompilátoru
Člen modifikátor '– klíčové slovo' musí předcházet typ člena a název  
  
 Specifikace přístup v podpis metody neuskutečnilo ve správném umístění.  
  
 Následující ukázka generuje CS1585:  
  
```  
// CS1585.cs  
public class Class1  
{  
   public void static Main(string[] args)   // CS1585  
   // try the following line instead  
   // public static void Main(string[] args)  
   {  
   }  
}  
```
