## <a name="example"></a><span data-ttu-id="0a4f0-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="0a4f0-107">Example</span></span>  
 <span data-ttu-id="0a4f0-108">Následující kód vygeneruje CS0744:</span><span class="sxs-lookup"><span data-stu-id="0a4f0-108">The following code generates CS0744:</span></span>

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};

    }  
}
