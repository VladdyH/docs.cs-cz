# <a name="compiler-error-cs0744"></a>Compiler Error CS0744
Očekávaný kontextové klíčové slovo "je rovno"  
  
 Tento vzor `join` je klauzule `join`... `in`... `on`... `equals`, jak je uvedeno v následujícím příkladu:  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Přidat `equals` – klíčové slovo chcete `join` klauzule.  
  
## <a name="example"></a>Příklad  
 Následující kód generuje CS0744:  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
