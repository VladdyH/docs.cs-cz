# <a name="compiler-error-cs1033"></a>CS1033 chyby kompilátoru
Zdrojový soubor došlo k překročení 16,707,565 čar reprezentovat PDB; budou nesprávné informace o ladění  
  
 Soubor zdrojového kódu překročil maximální povolený počet řádků, které může zpracovat kompilátoru. Pokud chcete tuto chybu vyřešit, vytvořte dva nebo více souborů zdrojového kódu z původního souboru. Maximální počet řádků, které je 268,435,454 řádky. Pokud používáte **/debug**, pomocí více než 16,707,566 bude mít za následek poškozená ladicí informace o.
