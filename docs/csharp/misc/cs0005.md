# <a name="compiler-error-cs0005"></a>CS0005 chyby kompilátoru
– Možnost kompilátoru 'compiler_option' musí být následováno argument  
  
 Některé možnosti kompilátoru vyžadují parametry. Pokud jste nepředávejte argumenty, které vyžadují možnost kompilátoru, vygeneruje se CS0005.  
  
 Další informace najdete v tématu [možnosti kompilátoru C#](../../csharp/language-reference/compiler-options/index.md).
