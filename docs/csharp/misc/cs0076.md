# <a name="compiler-error-cs0076"></a>CS0076 chyby kompilátoru
Název enumerátor 'value__' je rezervována a nelze ji použít  
  
 Položky v [výčtu](../../csharp/language-reference/keywords/enum.md) nemůže mít identifikátor s názvem **value__**.
