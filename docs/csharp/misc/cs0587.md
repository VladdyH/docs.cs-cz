# <a name="compiler-error-cs0587"></a>CS0587 chyby kompilátoru
Vnitřní chyba kompilátoru: fáze 'fáze.  
  
 Pokuste se zjistit, pokud kompilátor selhává kvůli jeho nemůže analyzovat neočekávanou syntaxi. Pokud tomu tak není, zkuste [kontaktujte nás](/visualstudio/ide/talk-to-us).
