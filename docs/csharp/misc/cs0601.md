# <a name="compiler-error-cs0601"></a><span data-ttu-id="93d68-102">CS0601 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="93d68-102">Compiler Error CS0601</span></span>
<span data-ttu-id="93d68-103">Atribut DllImport musí být zadán v metoda označena "statická" a "externí.</span><span class="sxs-lookup"><span data-stu-id="93d68-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="93d68-104">`DllImport` Atribut byla použita na metodu, která nemá správný přístup klíčová slova.</span><span class="sxs-lookup"><span data-stu-id="93d68-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="93d68-105">Následující ukázka generuje CS0601:</span><span class="sxs-lookup"><span data-stu-id="93d68-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
