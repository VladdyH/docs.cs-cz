## <a name="example"></a><span data-ttu-id="83d3d-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="83d3d-108">Example</span></span>  
 <span data-ttu-id="83d3d-109">Následující ukázka generuje CS0559.</span><span class="sxs-lookup"><span data-stu-id="83d3d-109">The following sample generates CS0559.</span></span>  
  
```csharp  
// CS0559_b.cs  
// compile with: /target:library  
public struct S  
{  
   public static S operator ++(S? s) { return new S(); }   // CS0559  
   public static S operator --(S? s) { return new S(); }   // CS0559  
}  
  
public struct T  
{  
// OK  
   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
}  
```
