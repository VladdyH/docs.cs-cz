# <a name="compiler-error-cs0551"></a>CS0551 chyby kompilátoru
Implementace explicitního rozhraní 'implementace' chybí přistupujícího objektu 'přistupujícího objektu.  
  
 Všechny přístupové objekty, které definuje rozhraní musí implementovat třídu, která explicitně implementuje rozhraní vlastnost.  
  
 Další informace najdete v tématu [pomocí vlastnosti](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0551.  
  
```  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
