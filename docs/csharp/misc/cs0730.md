# <a name="compiler-error-cs0730"></a>CS0730 chyby kompilátoru
Nelze předat zadejte "typ", protože je vnořené typ "typ"  
  
 Tato chyba se vygeneruje, když se pokusíte předat vnořené třídy.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0730. Obsahuje dva zdrojové soubory. Napřed zkompilovat soubor knihovny `CS0730a.cs`a kompilaci souboru `CS0730.cs` odkazující na soubor knihovny.  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
