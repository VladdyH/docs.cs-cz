# <a name="compiler-error-cs1002"></a>CS1002 chyby kompilátoru
; očekávání  
  
 Kompilátor zjištěna chybí středník. Středník v požadované na konci každý příkaz v jazyce C#. Příkaz může zahrnovat víc než jeden řádek.  
  
 Následující ukázka generuje CS1002:  
  
```csharp  
// CS1002.cs  
namespace x  
{  
   abstract public class clx  
   {  
      int i   // CS1002, missing semicolon  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
