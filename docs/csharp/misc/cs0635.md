# <a name="compiler-error-cs0635"></a>CS0635 chyby kompilátoru
'atribut': System.Interop.UnmanagedType.CustomMarshaller vyžaduje pojmenované argumenty ComType a zařazování  
  
 **ComType** a **zařazování** argumenty se musí zadat při zařazování formát je **System.Interop.UnmanagedType.CustomMarshaller**.
