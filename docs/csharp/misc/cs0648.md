# <a name="compiler-error-cs0648"></a>CS0648 chyby kompilátoru
"typ" je typu nepodporuje jazyk  
  
 Metadata, která byla vygenerována z jiný jazyk, které by mohly mít C++ obsahovala typ, který nebyl označen jako spravovaný. Informace o tomto typu je zahrnut v metadatech, ale typ není k dispozici pro programy, které jsou napsané v C#.
