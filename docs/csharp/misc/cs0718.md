# <a name="compiler-error-cs0718"></a>Compiler Error CS0718
'type': statické typy nelze použít jako argumenty typů  
  
 Protože nelze vytvořit instanci typu statické, nelze použít jako obecný argument. Pokud chcete tuto chybu vyřešit, odeberte z obecné argument statického typu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0718:  
  
```csharp  
// CS0718.cs  
public static class SC  
{  
    public static void F()  
    {  
    }  
}  
  
public class G<T>  
{  
}  
  
public class CMain  
{  
    public static void Main()  
    {  
        G<SC> gsc = new G<SC>();  // CS0718  
    }  
}  
```
