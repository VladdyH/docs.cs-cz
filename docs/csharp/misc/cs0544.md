 <span data-ttu-id="2a004-104">Došlo pokusu o přepsání nonproperty datový typ jako [vlastnost](../../csharp/programming-guide/classes-and-structs/properties.md), což není povolené.</span><span class="sxs-lookup"><span data-stu-id="2a004-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>
