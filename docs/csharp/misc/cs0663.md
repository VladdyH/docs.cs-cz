# <a name="compiler-error-cs0663"></a>CS0663 chyby kompilátoru
Nelze definovat přetížené metody, které se liší pouze na ref a out.  
  
 Metody, které se liší pouze v jejich používání [ref](../../csharp/language-reference/keywords/ref.md) a [out](../../csharp/language-reference/keywords/out.md) na parametr nejsou povoleny.  
  
 Následující ukázka generuje CS0663:  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
