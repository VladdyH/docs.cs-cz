# <a name="compiler-error-cs1922"></a><span data-ttu-id="a7c02-102">CS1922 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="a7c02-102">Compiler Error CS1922</span></span>

 <span data-ttu-id="a7c02-104">Chcete-li použít inicializátoru kolekce s typem, musí implementovat typ `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="a7c02-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="a7c02-105">Této chybě může dojít, pokud používáte omylem syntaxe inicializátoru kolekce, pokud jste chtěli použít inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="a7c02-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

-   <span data-ttu-id="a7c02-107">Pokud typ nepředstavuje kolekci, použijte místo syntaxe inicializátoru kolekce syntaxe inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="a7c02-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

## <a name="example"></a><span data-ttu-id="a7c02-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="a7c02-110">Example</span></span>  
 <span data-ttu-id="a7c02-111">Následující kód vytvoří CS1922:</span><span class="sxs-lookup"><span data-stu-id="a7c02-111">The following code produces CS1922:</span></span>  
  
```csharp  
// cs1922.cs  
public class Test  
{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922  
  
        // Object initalizer.  
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK  
    }  
}  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public string memberB { get; set; }  
}  
```
