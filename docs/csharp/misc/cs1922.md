# <a name="compiler-error-cs1922"></a><span data-ttu-id="80d2d-102">CS1922 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="80d2d-102">Compiler Error CS1922</span></span>
<span data-ttu-id="80d2d-103">Inicializátor kolekce vyžaduje typ 'type' System.Collections.IEnumerable implementace.</span><span class="sxs-lookup"><span data-stu-id="80d2d-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>  
  
 <span data-ttu-id="80d2d-104">Chcete-li použít inicializátor kolekce s typem, musí implementovat typ `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="80d2d-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="80d2d-105">K této chybě může dojít, pokud používáte omylem syntaxe inicializátoru kolekce, pokud jste chtěli použít inicializátor objektu.</span><span class="sxs-lookup"><span data-stu-id="80d2d-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="80d2d-106">Oprava této chyby</span><span class="sxs-lookup"><span data-stu-id="80d2d-106">To correct this error</span></span>  
  
-   <span data-ttu-id="80d2d-107">Pokud typ nepředstavuje kolekci, použijte namísto syntaxe inicializátoru kolekce syntaxe inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="80d2d-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>  
  
-   <span data-ttu-id="80d2d-108">Pokud typ představuje kolekci, upravte ho implementovat `IEnumerable` před inicializátory kolekce můžete použít třeba inicializovat objekty daného typu.</span><span class="sxs-lookup"><span data-stu-id="80d2d-108">If the type does represent a collection, modify it to implement `IEnumerable` before you can use collection initializers to initialize objects of that type.</span></span>  
  
-   <span data-ttu-id="80d2d-109">Pokud typ představuje kolekci a nemáte přístup ke zdrojovému kódu, stačí inicializace jeho prvky pomocí jejich konstruktory tříd nebo jiné metody inicializace.</span><span class="sxs-lookup"><span data-stu-id="80d2d-109">If the type represents a collection and you do not have access to the source code, just initialize its elements by using their class constructors or other initialization methods.</span></span>  
  
## <a name="example"></a><span data-ttu-id="80d2d-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="80d2d-110">Example</span></span>  
 <span data-ttu-id="80d2d-111">Následující kód vytvoří CS1922:</span><span class="sxs-lookup"><span data-stu-id="80d2d-111">The following code produces CS1922:</span></span>  
  
```csharp  
// cs1922.cs  
public class Test  
{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922  
  
        // Object initalizer.  
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK  
    }  
}  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public string memberB { get; set; }  
}  
```
