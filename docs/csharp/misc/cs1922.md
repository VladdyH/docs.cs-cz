 <span data-ttu-id="0b86a-104">Chcete-li použít inicializátoru kolekce s typem, musí implementovat typ `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="0b86a-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="0b86a-105">Této chybě může dojít, pokud používáte omylem syntaxe inicializátoru kolekce, pokud jste chtěli použít inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="0b86a-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>

-   <span data-ttu-id="0b86a-107">Pokud typ nepředstavuje kolekci, použijte místo syntaxe inicializátoru kolekce syntaxe inicializátoru objektu.</span><span class="sxs-lookup"><span data-stu-id="0b86a-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

## <a name="example"></a><span data-ttu-id="0b86a-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="0b86a-110">Example</span></span>  
 <span data-ttu-id="0b86a-111">Následující kód vytvoří CS1922:</span><span class="sxs-lookup"><span data-stu-id="0b86a-111">The following code produces CS1922:</span></span>  
  
```csharp  
// cs1922.cs  
public class Test  
{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922  
  
        // Object initalizer.  
        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK  
    }

{  
    public int memberA { get; set; }  
    public string memberB { get; set; }  
}  
```
