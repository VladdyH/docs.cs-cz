 <span data-ttu-id="d3640-104">K této chybě dojde, pokud se pokoušíte předat členem [foreach](../../csharp/language-reference/keywords/foreach-in.md) proměnnou, [pomocí](../../csharp/language-reference/keywords/using-statement.md) proměnnou, nebo [pevné](../../csharp/language-reference/keywords/fixed-statement.md) proměnnou funkce ref nebo out argument.</span><span class="sxs-lookup"><span data-stu-id="d3640-104">This error occurs if you are attempting to pass a member of a [foreach](../../csharp/language-reference/keywords/foreach-in.md) variable, a [using](../../csharp/language-reference/keywords/using-statement.md) variable, or a [fixed](../../csharp/language-reference/keywords/fixed-statement.md) variable to a function as a ref or out argument.</span></span> <span data-ttu-id="d3640-105">Protože tyto proměnné jsou považovány za v těchto kontexty jen pro čtení, což není povolené.</span><span class="sxs-lookup"><span data-stu-id="d3640-105">Because these variables are considered read-only in these contexts, this is not allowed.</span></span>  
  
 <span data-ttu-id="d3640-106">Následující ukázka generuje CS1655:</span><span class="sxs-lookup"><span data-stu-id="d3640-106">The following sample generates CS1655:</span></span>  
  
```csharp  
// CS1655.cs  
struct S   
{  
   public int i;  
}  
  
class CMain  
{  
  static void f(ref int iref)  
  {  
  }

     }  
  }  
}  
```
