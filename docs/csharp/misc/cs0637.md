# <a name="compiler-error-cs0637"></a><span data-ttu-id="41639-102">CS0637 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="41639-102">Compiler Error CS0637</span></span>

<span data-ttu-id="41639-103">Atribut FieldOffset není povolen pro statickou nebo const polí.</span><span class="sxs-lookup"><span data-stu-id="41639-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="41639-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) nelze atribut použít na pole označené parametrem [statické](../../csharp/language-reference/keywords/static.md) nebo [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="41639-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="41639-105">Následující ukázka generuje CS0637:</span><span class="sxs-lookup"><span data-stu-id="41639-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
