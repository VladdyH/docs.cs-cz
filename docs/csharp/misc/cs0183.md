# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="5e0ad-102">CS0183 kompilátoru upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="5e0ad-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="5e0ad-103">Daném výrazu je vždy typu zadaný ("typ")</span><span class="sxs-lookup"><span data-stu-id="5e0ad-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="5e0ad-104">Pokud je výsledkem podmíněného příkaz vždy **true**, pak nepotřebujete podmíněného příkazu.</span><span class="sxs-lookup"><span data-stu-id="5e0ad-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="5e0ad-105">Toto upozornění se zobrazí při pokusu o vyhodnocení typ pomocí **je** operátor.</span><span class="sxs-lookup"><span data-stu-id="5e0ad-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="5e0ad-106">Pokud vyhodnocení typ hodnoty, je kontrola zbytečné.</span><span class="sxs-lookup"><span data-stu-id="5e0ad-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="5e0ad-107">Následující ukázka generuje CS0183:</span><span class="sxs-lookup"><span data-stu-id="5e0ad-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
