# <a name="compiler-warning-level-1-cs0183"></a>CS0183 kompilátoru upozornění (úroveň 1)
Daném výrazu je vždy typu zadaný ("typ")  
  
 Pokud je výsledkem podmíněného příkaz vždy **true**, pak nepotřebujete podmíněného příkazu. Toto upozornění se zobrazí při pokusu o vyhodnocení typ pomocí **je** operátor. Pokud vyhodnocení typ hodnoty, je kontrola zbytečné.  
  
 Následující ukázka generuje CS0183:  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
