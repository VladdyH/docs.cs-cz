# <a name="compiler-error-cs1918"></a><span data-ttu-id="a5e45-102">Compiler Error CS1918</span><span class="sxs-lookup"><span data-stu-id="a5e45-102">Compiler Error CS1918</span></span>
<span data-ttu-id="a5e45-103">Členové vlastnosti "název" typ "typ" nelze přiřadit pomocí inicializátoru objektu protože se jedná o hodnotu zadat.</span><span class="sxs-lookup"><span data-stu-id="a5e45-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="a5e45-104">K této chybě dojde při pokusu o použití inicializátoru objektu k chybě při inicializaci vlastnosti struktura typu, který je sám vlastnost třídy, která je inicializován.</span><span class="sxs-lookup"><span data-stu-id="a5e45-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>

## <a name="example"></a><span data-ttu-id="a5e45-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="a5e45-108">Example</span></span>  
 <span data-ttu-id="a5e45-109">Následující příklad generuje CS1918:</span><span class="sxs-lookup"><span data-stu-id="a5e45-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
