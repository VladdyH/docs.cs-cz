# <a name="compiler-error-cs1676"></a>Compiler Error CS1676
Parametr "číslo" je třeba deklarovat s – klíčové slovo '– klíčové slovo'  
  
 K této chybě dojde, když – modifikátor parametrů typu anonymní metoda se liší od kterého v deklaraci, které jsou přetypování metody delegáta.  
  
 Následující ukázka generuje CS1676:  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
