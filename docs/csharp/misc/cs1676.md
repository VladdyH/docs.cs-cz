# <a name="compiler-error-cs1676"></a><span data-ttu-id="33582-102">CS1676 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="33582-102">Compiler Error CS1676</span></span>
<span data-ttu-id="33582-103">Parametr "číslo" je třeba deklarovat s – klíčové slovo '– klíčové slovo'</span><span class="sxs-lookup"><span data-stu-id="33582-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="33582-104">K této chybě dojde, když – modifikátor parametrů typu anonymní metoda se liší od kterého v deklaraci, které jsou přetypování metody delegáta.</span><span class="sxs-lookup"><span data-stu-id="33582-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="33582-105">Následující ukázka generuje CS1676:</span><span class="sxs-lookup"><span data-stu-id="33582-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
