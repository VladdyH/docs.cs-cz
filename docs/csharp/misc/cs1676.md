 <span data-ttu-id="f79ff-104">K této chybě dojde, když – modifikátor parametrů typu anonymní metoda se liší od kterého v deklaraci, které jsou přetypování metody delegáta.</span><span class="sxs-lookup"><span data-stu-id="f79ff-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>

```csharp  
// CS1676.cs

{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
