# <a name="compiler-warning-level-1-cs1200"></a>CS1200 kompilátoru upozornění (úroveň 1)
Funkce "Neplatná funkce" je zastaralý. Místo toho použijte "platný funkce" '.  
  
 Funkce, kterou se pokoušíte použít, je nyní zastaralý. Aktualizujte kód a místo ní používejte funkci platný.
