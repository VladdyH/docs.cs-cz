# <a name="compiler-warning-level-1-cs3010"></a>CS3010 kompilátoru upozornění (úroveň 1)
"člen": kompatibilní se specifikací CLS rozhraní musí mít pouze kompatibilní se specifikací CLS členy  
  
 V sestavení označené jako `[assembly:CLCSompliant(true)]`, rozhraní obsahuje člena označené jako `[CLCSompliant(false)]`. Odeberte jeden z atributů dodržování předpisů specifikace CLS (Common Language). Další informace o souladu se specifikací CLS najdete v tématu [ \<PAVE přes > kompatibilní se specifikací CLS psaní kódu](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) a [jazyková nezávislost a jazykově nezávislé komponenty](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3010:  
  
```  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}  
  
public class C : I  
{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
