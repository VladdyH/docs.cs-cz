# <a name="compiler-error-cs0070"></a><span data-ttu-id="46490-102">CS0070 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="46490-102">Compiler Error CS0070</span></span>
<span data-ttu-id="46490-103">Událost 'událost' se může vyskytovat pouze na levé straně += nebo-= (s výjimkou při použít v rámci typu "typ")</span><span class="sxs-lookup"><span data-stu-id="46490-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="46490-104">Mimo třídu, která je definována v [událostí](../../csharp/language-reference/keywords/event.md) můžete přidat pouze nebo odečtena odkazy.</span><span class="sxs-lookup"><span data-stu-id="46490-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="46490-105">Další informace najdete v tématu [události](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="46490-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="46490-106">Následující ukázka generuje CS0070:</span><span class="sxs-lookup"><span data-stu-id="46490-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
