 <span data-ttu-id="1e3f0-104">V seznamu dědičnosti [rozhraní](../../csharp/language-reference/keywords/interface.md) obsahuje přímý nebo nepřímý odkaz na sebe sama.</span><span class="sxs-lookup"><span data-stu-id="1e3f0-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="1e3f0-105">Rozhraní nelze dědění ze sebe samotné.</span><span class="sxs-lookup"><span data-stu-id="1e3f0-105">An interface cannot inherit from itself.</span></span>
