# <a name="compiler-error-cs0567"></a>Compiler Error CS0567
Rozhraní nemohou obsahovat operátory  
  
 Operátory nejsou povoleny v [rozhraní](../../csharp/language-reference/keywords/interface.md) definice.  
  
 Následující ukázka generuje CS0567:  
  
```csharp  
// CS0567.cs  
interface IA  
{  
   int operator +(int aa, int bb);   // CS0567  
}  
  
class Sample  
{  
   public static void Main()   
   {  
   }  
}  
```
