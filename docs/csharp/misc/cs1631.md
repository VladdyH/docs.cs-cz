 <span data-ttu-id="6d35d-104">Příkaz yield není povoleno z textu klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="6d35d-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="6d35d-105">Chcete-li se vyhnout této chybě, přesuňte příkaz yield mimo tělo klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="6d35d-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="6d35d-106">Následující ukázka generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="6d35d-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
