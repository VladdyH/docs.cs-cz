# <a name="compiler-error-cs1631"></a><span data-ttu-id="0af04-102">CS1631 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="0af04-102">Compiler Error CS1631</span></span>
<span data-ttu-id="0af04-103">Nelze yield hodnotu v těle klauzule catch</span><span class="sxs-lookup"><span data-stu-id="0af04-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="0af04-104">Příkaz yield není povoleno z textu klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="0af04-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="0af04-105">Chcete-li se vyhnout této chybě, přesuňte příkaz yield mimo tělo klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="0af04-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="0af04-106">Následující ukázka generuje CS1631:</span><span class="sxs-lookup"><span data-stu-id="0af04-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
