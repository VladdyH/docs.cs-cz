# <a name="compiler-error-cs1631"></a>Compiler Error CS1631
Nelze yield hodnotu v těle klauzule catch  
  
 Příkaz yield není povoleno z textu klauzuli catch. Chcete-li se vyhnout této chybě, přesuňte příkaz yield mimo tělo klauzuli catch.  
  
 Následující ukázka generuje CS1631:  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
