# <a name="compiler-error-cs1057"></a>Compiler Error CS1057
"člen": statické třídy nemohou obsahovat chráněné členy "  
  
 Tato chyba je generován deklarace chráněného člena uvnitř statická třída.  
  
## <a name="example"></a>Příklad  
 Následující příklad vytvoří CS1057.  
  
```csharp  
// CS1057.cs  
using System;  
  
static class Class1  
{  
    protected static int x;   // CS1057  
    public static void Main()  
    {  
    }  
}  
```
