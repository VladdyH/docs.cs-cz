# <a name="compiler-error-cs0619"></a><span data-ttu-id="81344-102">CS0619 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="81344-102">Compiler Error CS0619</span></span>
<span data-ttu-id="81344-103">'member' je zastaralý: 'text'</span><span class="sxs-lookup"><span data-stu-id="81344-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="81344-104">Člen třídy byl označen atributem [zastaralé](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) tak, aby chybu bude vydána, když se odkazuje člen třídy atributu.</span><span class="sxs-lookup"><span data-stu-id="81344-104">A class member was marked with the [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="81344-105">Následující ukázka generuje CS0619:</span><span class="sxs-lookup"><span data-stu-id="81344-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
