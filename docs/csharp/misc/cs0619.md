# <a name="compiler-error-cs0619"></a>CS0619 chyby kompilátoru
'member' je zastaralý: 'text'  
  
 Člen třídy byl označen atributem [zastaralé](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) tak, aby chybu bude vydána, když se odkazuje člen třídy atributu.  
  
 Následující ukázka generuje CS0619:  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
