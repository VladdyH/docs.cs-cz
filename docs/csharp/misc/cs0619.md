# <a name="compiler-error-cs0619"></a><span data-ttu-id="5244d-102">CS0619 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="5244d-102">Compiler Error CS0619</span></span>
<span data-ttu-id="5244d-103">"člen" je zastaralá: 'text'</span><span class="sxs-lookup"><span data-stu-id="5244d-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="5244d-104">Byl označené jako člena třídy [zastaralé](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atribut, tak, aby k chybě dojde při odkazování na člena třídy.</span><span class="sxs-lookup"><span data-stu-id="5244d-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="5244d-105">Následující ukázka generuje CS0619:</span><span class="sxs-lookup"><span data-stu-id="5244d-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
