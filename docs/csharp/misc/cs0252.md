# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="51a8f-102">CS0252 kompilátoru upozornění (úroveň 2)</span><span class="sxs-lookup"><span data-stu-id="51a8f-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="51a8f-103">Možné neúmyslnému porovnání; Chcete-li porovnat hodnoty, přetypujte na levé straně na typ "typ"</span><span class="sxs-lookup"><span data-stu-id="51a8f-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="51a8f-104">Kompilátor provádí porovnání odkaz.</span><span class="sxs-lookup"><span data-stu-id="51a8f-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="51a8f-105">Pokud chcete porovnat hodnotu řetězce, přetypovat na levé straně výraz, který se `type`.</span><span class="sxs-lookup"><span data-stu-id="51a8f-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="51a8f-106">Následující ukázka generuje CS0252:</span><span class="sxs-lookup"><span data-stu-id="51a8f-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
