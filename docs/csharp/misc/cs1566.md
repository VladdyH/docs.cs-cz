# <a name="compiler-error-cs1566"></a>CS1566 chyby kompilátoru
Chyba při čtení souboru 'file' prostředků – "důvod"  
  
 Kompilátor měl problémy s názvem souboru, který je předán [/Resource](../../csharp/language-reference/compiler-options/resource-compiler-option.md) – možnost kompilátoru.
