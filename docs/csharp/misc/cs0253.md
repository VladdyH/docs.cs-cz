 <span data-ttu-id="79776-104">Kompilátor provádí porovnání odkaz.</span><span class="sxs-lookup"><span data-stu-id="79776-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="79776-105">Pokud chcete porovnat hodnotu řetězce, přetypovat na pravé straně výraz, který se `type`.</span><span class="sxs-lookup"><span data-stu-id="79776-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="79776-106">Následující ukázka generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="79776-106">The following sample generates CS0253:</span></span>  
  
```csharp 
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
