# <a name="compiler-warning-level-2-cs0253"></a><span data-ttu-id="c7558-102">CS0253 kompilátoru upozornění (úroveň 2)</span><span class="sxs-lookup"><span data-stu-id="c7558-102">Compiler Warning (level 2) CS0253</span></span>
<span data-ttu-id="c7558-103">Možné neúmyslnému porovnání; Pokud chcete porovnat hodnoty, přetypujte pravé straně na typ "typ"</span><span class="sxs-lookup"><span data-stu-id="c7558-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="c7558-104">Kompilátor provádí porovnání odkaz.</span><span class="sxs-lookup"><span data-stu-id="c7558-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="c7558-105">Pokud chcete porovnat hodnotu řetězce, přetypovat na pravé straně výraz, který se `type`.</span><span class="sxs-lookup"><span data-stu-id="c7558-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="c7558-106">Následující ukázka generuje CS0253:</span><span class="sxs-lookup"><span data-stu-id="c7558-106">The following sample generates CS0253:</span></span>  
  
```  
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
