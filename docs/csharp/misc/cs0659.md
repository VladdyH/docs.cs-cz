# <a name="compiler-warning-level-3-cs0659"></a>Kompilátor CS0659 upozornění (úroveň 3)
'class' přepisuje funkci Object.Equals (object o), ale nepřepisuje funkci Object.GetHashCode()  
  
 Kompilátor zjistil přepsání **rovná** funkce, ale ne přepsání pro **GetHashCode**. Přepsání **rovná** znamená, že chcete také přepsat **GetHashCode**.  
  
 Další informace naleznete v tématu  
  
-   <xref:System.Collections.Hashtable>.  
  
-   [Operátory rovnosti](../../standard/design-guidelines/equality-operators.md)  
  
-   [NIB: Implementace metody Equals](https://msdn.microsoft.com/library/30f28aaf-8b9e-46cd-a746-58a12473af2c)  
  
-   <xref:System.Object.GetHashCode%2A>  
  
 Následující ukázka generuje CS0659:  
  
```csharp  
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test     
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
