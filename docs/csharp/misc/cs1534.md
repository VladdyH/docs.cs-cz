# <a name="compiler-error-cs1534"></a>CS1534 chyby kompilátoru
Přetížené binární operátor 'operátor' přebírá dva parametry  
  
 Definice binární [přetížitelné operátor](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) vyžaduje dva parametry.  
  
 Následující ukázka generuje CS1534:  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
