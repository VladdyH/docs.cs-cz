# <a name="compiler-error-cs1534"></a>CS1534 chyby kompilátoru
Přetěžovaný binární operátor 'operator' používá dva parametry  
  
 Definice binární soubor [operátor](../../csharp/language-reference/keywords/operator.md) musí mít dva parametry.  
  
 Následující ukázka generuje CS1534:  
  
```csharp  
// CS1534.cs  
class MyClass  
{  
   public static MyClass operator - (MyClass MC1, MyClass MC2, MyClass MC3)   // CS1534  
   // try the following line instead  
   // public static MyClass operator - (MyClass MC1, MyClass MC2)  
   {  
      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
