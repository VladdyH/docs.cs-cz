# <a name="compiler-error-cs1588"></a>CS1588 chyby kompilátoru
Nelze určit společný adresář modulu runtime jazyka – "důvod"  
  
 Kompilátor nelze načíst adresář, který je nainstalován modul common language runtime rozhraní .NET Framework v. Tato chyba označuje, zda je nainstalován nesprávně modul common language runtime.
