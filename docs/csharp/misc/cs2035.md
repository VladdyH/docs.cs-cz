# <a name="compiler-error-cs2035"></a>CS2035 chyby kompilátoru
Chyba syntaxe příkazového řádku: chybí ':\<číslo > "pro možnost"compiler_option.  
  
 Některé možnosti kompilátoru vyžadují hodnotu.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS2035.  
  
```  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
