# <a name="compiler-error-cs1514"></a>CS1514 chyby kompilátoru
{očekávání  
  
 Kompilátor očekává otevření složené závorky (`{`), nebyla nalezena.  
  
 Následující ukázka generuje CS1514:  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
