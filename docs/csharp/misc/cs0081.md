# <a name="compiler-error-cs0081"></a><span data-ttu-id="1fe1f-102">Compiler Error CS0081</span><span class="sxs-lookup"><span data-stu-id="1fe1f-102">Compiler Error CS0081</span></span>
<span data-ttu-id="1fe1f-103">Deklarace typu parametru musí být identifikátor není typu</span><span class="sxs-lookup"><span data-stu-id="1fe1f-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="1fe1f-104">Po deklarování obecná metoda nebo typu, zadejte parametr typu jako identifikátor, například "T" nebo "inputType".</span><span class="sxs-lookup"><span data-stu-id="1fe1f-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="1fe1f-105">Pokud kód klienta volá metodu, poskytuje typu, který nahradí každý její výskyt identifikátoru v těle metody nebo třída.</span><span class="sxs-lookup"><span data-stu-id="1fe1f-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="1fe1f-106">Další informace najdete v tématu [obecné parametry typu](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="1fe1f-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
