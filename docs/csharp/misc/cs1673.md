# <a name="compiler-error-cs1673"></a><span data-ttu-id="e51a3-102">CS1673 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="e51a3-102">Compiler Error CS1673</span></span>
<span data-ttu-id="e51a3-103">Anonymní metody, lambda – výrazy a výrazy dotazů uvnitř struktury nelze přístupu členů instance 'this'.</span><span class="sxs-lookup"><span data-stu-id="e51a3-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="e51a3-104">Zvažte kopírování 'this' místní proměnné mimo anonymní metoda, výrazu lambda nebo výrazu dotazu a místo toho použít místní.</span><span class="sxs-lookup"><span data-stu-id="e51a3-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="e51a3-105">Následující ukázka generuje CS1673:</span><span class="sxs-lookup"><span data-stu-id="e51a3-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
