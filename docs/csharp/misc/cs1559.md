# <a name="compiler-error-cs1559"></a>CS1559 chyby kompilátoru
Pro metodu Main nelze použít 'objekt', protože jeho importu  
  
 Byla zadána neplatná třída k [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru; třídy nelze použít jako umístění [hlavní](../../csharp/programming-guide/main-and-command-args/index.md) metoda.
