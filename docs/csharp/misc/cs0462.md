<span data-ttu-id="fe32a-103">Zděděné členy "Člen1" a "člen2" mějte stejným podpisem typu "typ", aby nebylo možné přepsat</span><span class="sxs-lookup"><span data-stu-id="fe32a-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="fe32a-104">Tato chyba nastane se zavedením obecných typů.</span><span class="sxs-lookup"><span data-stu-id="fe32a-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="fe32a-105">Za normálních okolností nemůže mít dvě verze metody ve třídě se stejným podpisem.</span><span class="sxs-lookup"><span data-stu-id="fe32a-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="fe32a-106">Ale s obecnými typy, můžete zadat Obecná metoda, která může být duplicitní jinou metodu, pokud je vytvořen s konkrétním typem.</span><span class="sxs-lookup"><span data-stu-id="fe32a-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fe32a-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="fe32a-107">Example</span></span>  
 <span data-ttu-id="fe32a-108">Při `C<int>` je vytvořena, dvě verze metody `F` vytvářejí se stejným podpisem, takže přepsání v třídě `D` nelze rozhodnout, které z nich má přepsání vztahovat.</span><span class="sxs-lookup"><span data-stu-id="fe32a-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="fe32a-109">Následující ukázka generuje CS0462.</span><span class="sxs-lookup"><span data-stu-id="fe32a-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
