# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="f3254-102">Kompilátor CS0612 upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="f3254-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="f3254-103">'member' je zastaralý</span><span class="sxs-lookup"><span data-stu-id="f3254-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="f3254-104">Návrhář tříd označené se členem [zastaralé](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atribut.</span><span class="sxs-lookup"><span data-stu-id="f3254-104">The class designer marked a member with the [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="f3254-105">To znamená, že nemusí být člen podporován v budoucí verzi třídy.</span><span class="sxs-lookup"><span data-stu-id="f3254-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="f3254-106">Následující příklad ukazuje, jak přístup k zastaralý člen generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="f3254-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
