# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="b2c01-102">CS0612 kompilátoru upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="b2c01-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="b2c01-103">"člen" je zastaralá</span><span class="sxs-lookup"><span data-stu-id="b2c01-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="b2c01-104">Návrhář tříd označena člena s [zastaralé](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atribut.</span><span class="sxs-lookup"><span data-stu-id="b2c01-104">The class designer marked a member with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="b2c01-105">To znamená, že člen nemusí být podporován v budoucí verzi třídy.</span><span class="sxs-lookup"><span data-stu-id="b2c01-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="b2c01-106">Následující příklad ukazuje, jak přístup k člena zastaralé generuje CS0612:</span><span class="sxs-lookup"><span data-stu-id="b2c01-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
