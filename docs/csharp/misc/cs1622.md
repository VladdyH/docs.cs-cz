# <a name="compiler-error-cs1622"></a>CS1622 chyby kompilátoru
Nelze vrátit hodnotu z iterace. Použijte příkaz return yield vrácení hodnoty, nebo yield přerušení na konec iterace.  
  
 Iterátor je speciální funkce, která vrátí hodnotu prostřednictvím příkaz yield spíše než příkaz return. Další informace najdete v tématu **iterátory**.  
  
 Následující ukázka generuje CS1622:  
  
```  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
