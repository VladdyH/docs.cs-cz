# <a name="compiler-error-cs1542"></a>CS1542 chyby kompilátoru
'knihovny dll' nelze přidat do tohoto sestavení, protože už je sestavení; použít ' / R se místo něj  
  
 Soubor, který byl odkazovaný s [/addmodule](../../csharp/language-reference/compiler-options/addmodule-compiler-option.md) – možnost kompilátoru nebyl vytvořen s [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md); použít [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) účelem odkazu na soubor v této kompilaci.
