# <a name="compiler-error-cs0186"></a>Compiler Error CS0186
Použití hodnoty null není platná v tomto kontextu  
  
 Následující ukázka generuje CS0186:  
  
```csharp  
// CS0186.cs  
using System;  
using System.Collections;  
  
class MyClass   
{  
   static void Main()   
   {  
      // Each of the following lines generates CS0186:  
      foreach (int i in null) {}   // CS0186  
      foreach (int i in (IEnumerable) null) { };   // CS0186  
   }  
}  
```
