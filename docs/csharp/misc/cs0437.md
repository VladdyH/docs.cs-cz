# <a name="compiler-warning-level-2-cs0437"></a>CS0437 kompilátoru upozornění (úroveň 2)
Typ 'type' v 'assembly2' je v konfliktu s importované oboru názvů atribut namespace' v 'fassembly1'. Použití s typem definovaným v 'assembly'.  
  
 Když k typu v zdrojový soubor, file_2, je v konfliktu s importovaný obor názvů v souboru _1 se objeví toto upozornění. Kompilátor používá typ ve zdrojovém souboru.  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS0437_a.cs  
// compile with: /target:library  
namespace Util   
{  
   public class A {  
      public void Test() {  
         System.Console.WriteLine("CS0437_a.cs");  
      }  
   }  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0437.  
  
```csharp  
// CS0437_b.cs  
// compile with: /reference:CS0437_a.dll /W:2  
// CS0437 expected  
class Util   
{  
   public class A {   
      public void Test() {  
         System.Console.WriteLine("CS0437_b.cs");  
      }  
   }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Util.A x = new Util.A();  
      x.Test();  
   }  
}  
```
