## <a name="example"></a><span data-ttu-id="22139-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="22139-107">Example</span></span>  
 <span data-ttu-id="22139-108">Následující kód vytvoří CS1912, protože `memberA` je inicializován dvakrát:</span><span class="sxs-lookup"><span data-stu-id="22139-108">The following code generates CS1912 because `memberA` is initialized two times:</span></span>  
  
```  
// cs1912.cs  
using System.Linq;  
  
public class TestClass  
{  
    public int memberA { get; set; }  
    public int memberB { get; set; }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        TestClass tc = new TestClass() { memberA = 5, memberA = 6, memberB = 2}; // CS1912  
    }  
}  
```
