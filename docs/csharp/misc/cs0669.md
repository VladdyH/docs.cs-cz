# <a name="compiler-error-cs0669"></a><span data-ttu-id="a3813-102">CS0669 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="a3813-102">Compiler Error CS0669</span></span>

<span data-ttu-id="a3813-103">Třída s atributem ComImport nemůže mít uživatelem definované konstruktor.</span><span class="sxs-lookup"><span data-stu-id="a3813-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="a3813-104">Poskytuje vrstvě zprostředkovatele komunikace s objekty COM v modulu common language runtime v konstruktoru pro [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) třídy.</span><span class="sxs-lookup"><span data-stu-id="a3813-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="a3813-105">V důsledku toho objektu COM slouží jako spravovaného objektu v modulu runtime.</span><span class="sxs-lookup"><span data-stu-id="a3813-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="a3813-106">Následující ukázka generuje CS0669:</span><span class="sxs-lookup"><span data-stu-id="a3813-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
