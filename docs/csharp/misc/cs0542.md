 <span data-ttu-id="a432e-104">Členové třídě nebo struktuře nemůže mít stejný název jako třídě nebo struktuře, pokud člen je konstruktor.</span><span class="sxs-lookup"><span data-stu-id="a432e-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="a432e-105">Následující ukázka generuje CS0542:</span><span class="sxs-lookup"><span data-stu-id="a432e-105">The following sample generates CS0542:</span></span>  
  
```csharp

{  
    public int C;  
}  
```  
  
 <span data-ttu-id="a432e-106">Tato chyba může být způsobena, když vložíte nechtěně návratový typ v konstruktoru, takže v platnosti je do běžné metody.</span><span class="sxs-lookup"><span data-stu-id="a432e-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="a432e-107">Následující příklad vytvoří CS0542, protože `F` je metoda, není konstruktoru, protože má návratový typ:</span><span class="sxs-lookup"><span data-stu-id="a432e-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp

{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
