# <a name="compiler-error-cs0021"></a><span data-ttu-id="18087-102">CS0021 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="18087-102">Compiler Error CS0021</span></span>

 <span data-ttu-id="18087-104">Byl proveden pokus o přístup k hodnotu prostřednictvím indexeru na datový typ, který nepodporuje [indexery](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="18087-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="18087-105">Při pokusu použít indexer v sestavení C++, může dojít k CS0021.</span><span class="sxs-lookup"><span data-stu-id="18087-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="18087-106">V takovém případě uspořádání třídami C++ pomocí `DefaultMember` atribut, kompilátor jazyka C# zná, je výchozím nastavením, které indexer.</span><span class="sxs-lookup"><span data-stu-id="18087-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="18087-107">Následující ukázka generuje CS0021.</span><span class="sxs-lookup"><span data-stu-id="18087-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="18087-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="18087-108">Example</span></span>  
 <span data-ttu-id="18087-109">Tento soubor zkompiluje na soubor .dll – s `DefaultMember` komentované atribut – Pokud chcete generovat chyba.</span><span class="sxs-lookup"><span data-stu-id="18087-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="18087-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="18087-110">Example</span></span>  
 <span data-ttu-id="18087-111">Zde je souboru C#, která volá souboru .dll.</span><span class="sxs-lookup"><span data-stu-id="18087-111">The following is the C# file that calls the .dll file.</span></span> <span data-ttu-id="18087-112">Tento soubor se pokusí o přístup ke třídě prostřednictvím indexeru, ale protože žádné člen byla deklarována jako výchozí indexeru, který se má použít, je generována chyba.</span><span class="sxs-lookup"><span data-stu-id="18087-112">This file attempts to access the class via an indexer, but because no member was declared as the default indexer to be used, the error is generated.</span></span>

```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
