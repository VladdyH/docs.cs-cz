# <a name="compiler-error-cs1038"></a>CS1038 chyby kompilátoru
\#endregion – direktiva  
  
 A [#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) – direktiva nemá odpovídající [#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) – direktiva.  
  
 Následující ukázka generuje CS1038:  
  
```  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
