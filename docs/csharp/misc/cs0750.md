<span data-ttu-id="4679d-103">Částečné metody nemůže mít modifikátory přístupu nebo virtuální, abstraktní, přepis, nový, zapečetěné nebo externí modifikátory.</span><span class="sxs-lookup"><span data-stu-id="4679d-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="4679d-104">Kvůli jejich zvláštní chování částečné metody se vztahují omezení, modifikátory, který se může přijmout.</span><span class="sxs-lookup"><span data-stu-id="4679d-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

## <a name="example"></a><span data-ttu-id="4679d-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="4679d-107">Example</span></span>  
 <span data-ttu-id="4679d-108">Následující příklad generuje CS0750:</span><span class="sxs-lookup"><span data-stu-id="4679d-108">The following example generates CS0750:</span></span>  
  
```  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
