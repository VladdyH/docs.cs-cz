# <a name="compiler-error-cs0750"></a>CS0750 chyby kompilátoru
Částečné metody nemůže mít modifikátory přístupu nebo virtuální, abstraktní, přepis, nový, zapečetěné nebo externí modifikátory.  
  
 Kvůli jejich zvláštní chování částečné metody se vztahují omezení, modifikátory, který se může přijmout.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odebrání neoprávněným modifikátor deklarace metody.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS0750:  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
