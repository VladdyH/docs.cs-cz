# <a name="compiler-error-cs0682"></a>CS0682 chyby kompilátoru
'type1' nelze implementovat 'type2', protože není podporován jazyk  
  
 K této chybě dojde, když chcete implementovat rozhraní v jiném jazyce a kompilátor nepodporuje rozhraní.
