## <a name="example"></a><span data-ttu-id="ea8f1-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="ea8f1-105">Example</span></span>  
 <span data-ttu-id="ea8f1-106">Následující ukázka generuje CS1909.</span><span class="sxs-lookup"><span data-stu-id="ea8f1-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
