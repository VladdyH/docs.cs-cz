 <span data-ttu-id="df5c7-109">Následující ukázka generuje CS0672:</span><span class="sxs-lookup"><span data-stu-id="df5c7-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}

{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}

{  
   static public void Main()  
   {  
   }  
}  
```
