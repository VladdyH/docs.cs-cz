 <span data-ttu-id="5b655-105">Kompilátor nalezen `override` metodě označené jako `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="5b655-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="5b655-106">Ale přepsání metoda nebyla samotné označené jako zastaralé.</span><span class="sxs-lookup"><span data-stu-id="5b655-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="5b655-107">Přepsání metody stále vygeneruje [CS0612](../../csharp/misc/cs0612.md), pokud je volána.</span><span class="sxs-lookup"><span data-stu-id="5b655-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="5b655-108">Zkontrolujte vaše metoda deklarace a explicitně označuje, zda má být označen metody (a všechny jeho přepsání) `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="5b655-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="5b655-109">Následující ukázka generuje CS0672:</span><span class="sxs-lookup"><span data-stu-id="5b655-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}

{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}

{  
   static public void Main()  
   {  
   }  
}  
```
