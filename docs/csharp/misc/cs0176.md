# <a name="compiler-error-cs0176"></a>CS0176 chyby kompilátoru
Statický člen "člen" nelze získat přístup s odkazem na instanci; Místo toho kvalifikaci s názvem typu  
  
 Jenom název třídy můžete použít k vyfiltrování [statické](../../csharp/language-reference/keywords/static.md) proměnné; Název instance nesmí být kvalifikátor. Další informace najdete v tématu [statické třídy a statické členy třídy](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).  
  
 Následující ukázka generuje CS0176:  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
