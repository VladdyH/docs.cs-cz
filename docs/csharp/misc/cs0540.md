## <a name="example"></a><span data-ttu-id="77e9a-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="77e9a-106">Example</span></span>

```csharp

{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}

{  
   void I.m() {}

}  
```
