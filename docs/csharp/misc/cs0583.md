# <a name="compiler-error-cs0583"></a>CS0583 chyby kompilátoru
Vnitřní chyba kompilátoru. V kompilátoru došlo k vnitřní chybě. Chcete-li vyřešit tento problém, zkuste zjednodušit nebo změna programu téměř na místech uvedených. Umístění v horní části seznamu, jsou blíže bodu, kdy došlo k vnitřní chybě. Chyby jako např. to můžete pomocí možnosti/errorreport ohlášena společnosti Microsoft.  
  
 Vytvořte soubor sestavy chyb s ' / bugreport' možnost a odeslání zprávy o přiřazené kontaktujte hlášení problémů.  
  
 Použití [/bugreport](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) – možnost kompilátoru k zachycení problém.
