# <a name="compiler-warning-level-1-cs3002"></a>CS3002 kompilátoru upozornění (úroveň 1)
Návratový typ "metody" není kompatibilní se specifikací CLS  
  
 A [veřejné](../../csharp/language-reference/keywords/public.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo `protected internal` metoda musí vrátit hodnotu s typem je kompatibilní s specifikace CLS (Common Language). Další informace o souladu se specifikací CLS najdete v tématu [kompatibilní se specifikací CLS psaní kódu](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) a [jazyková nezávislost a jazykově nezávislé komponenty](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3002:  
  
```  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
