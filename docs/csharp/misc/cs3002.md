# <a name="compiler-warning-level-1-cs3002"></a>Kompilátor CS3002 upozornění (úroveň 1)
Návratový typ 'metody' není kompatibilní se Specifikací CLS  
  
 A [veřejné](../../csharp/language-reference/keywords/public.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo `protected internal` metoda musí vracet hodnotu, jejíž typ je kompatibilní s specifikace CLS (Common Language). Další informace o dodržování specifikace CLS naleznete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](~/docs/standard/language-independence.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3002:  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
