 <span data-ttu-id="cd347-104">Při použití `goto case`, musí být implicitní převod z hodnoty případu goto typu přepínače.</span><span class="sxs-lookup"><span data-stu-id="cd347-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cd347-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="cd347-105">Example</span></span>  
 <span data-ttu-id="cd347-106">Následující ukázka generuje CS0469.</span><span class="sxs-lookup"><span data-stu-id="cd347-106">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
