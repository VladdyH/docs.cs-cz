# <a name="compiler-error-cs1627"></a><span data-ttu-id="596b6-102">Compiler Error CS1627</span><span class="sxs-lookup"><span data-stu-id="596b6-102">Compiler Error CS1627</span></span>
<span data-ttu-id="596b6-103">Byl očekáván po yield vrátit výraz</span><span class="sxs-lookup"><span data-stu-id="596b6-103">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="596b6-104">K této chybě dojde, pokud `yield` se používá bez výrazu.</span><span class="sxs-lookup"><span data-stu-id="596b6-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="596b6-105">Chcete-li se vyhnout této chybě, vložte odpovídající výraz v příkazu.</span><span class="sxs-lookup"><span data-stu-id="596b6-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="596b6-106">Následující ukázka generuje CS1627:</span><span class="sxs-lookup"><span data-stu-id="596b6-106">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
