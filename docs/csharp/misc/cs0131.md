## <a name="example"></a><span data-ttu-id="9c0f1-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="9c0f1-108">Example</span></span>

```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```
