# <a name="compiler-error-cs0459"></a><span data-ttu-id="55634-102">CS0459 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="55634-102">Compiler Error CS0459</span></span>
<span data-ttu-id="55634-103">Nelze převést na adresu místního proměnnou jen pro čtení</span><span class="sxs-lookup"><span data-stu-id="55634-103">Cannot take the address of a read-only local variable</span></span>  
  
 <span data-ttu-id="55634-104">Existují tři běžné scénáře v jazyce C#, které generují jen pro čtení lokální proměnné: `foreach`, `using`, a `fixed`.</span><span class="sxs-lookup"><span data-stu-id="55634-104">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="55634-105">V každé z těchto případů nejsou povoleny k zápisu do místní proměnné jen pro čtení a využít její adresy.</span><span class="sxs-lookup"><span data-stu-id="55634-105">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="55634-106">Tato chyba se vygeneruje, když kompilátor rozpoznává, že se pokoušíte provést adresu místní proměnné jen pro čtení.</span><span class="sxs-lookup"><span data-stu-id="55634-106">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55634-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="55634-107">Example</span></span>  
 <span data-ttu-id="55634-108">Následující příklad generuje CS0459, když je proveden pokus o trvat adresu jen pro čtení místní proměnné v `foreach` smyčky a v `fixed` příkaz bloku.</span><span class="sxs-lookup"><span data-stu-id="55634-108">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block.</span></span>  
  
```csharp  
// CS0459.cs  
// compile with: /unsafe  
  
class A  
{  
    public unsafe void M1()  
    {  
        int[] ints = new int[] { 1, 2, 3 };  
        foreach (int i in ints)  
        {  
            int *j = &i;  // CS0459  
        }  
  
        fixed (int *i = &_i)  
        {  
            int **j = &i;  // CS0459  
        }  
    }  
  
    private int _i = 0;  
}  
```
