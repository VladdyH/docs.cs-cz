# <a name="compiler-error-cs0673"></a>Compiler Error CS0673
System.Void nelze používat ze C# – použít konstrukci System.Void jazyka GET pro objekt typu void.  
  
 **System.Void** nelze použít v jazyce C#.  
  
 Následující ukázka generuje CS0673:  
  
```csharp  
// CS0673.cs  
class MyClass  
{  
   public static void Main()  
   {  
      System.Type t = typeof(System.Void);   // CS0673  
      // try the following line instead  
      // System.Type t = typeof(void);  
   }  
}  
```
