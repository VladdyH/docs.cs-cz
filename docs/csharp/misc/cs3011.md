# <a name="compiler-warning-level-1-cs3011"></a>CS3011 kompilátoru upozornění (úroveň 1)
"člen": pouze kompatibilní se specifikací CLS členy mohou být abstraktní  
  
 Člena třídy nejde nastavit na obojí [abstraktní](../../csharp/language-reference/keywords/abstract.md) a je nekompatibilní s specifikace CLS (Common Language). Specifikace CLS Určuje, že všichni členové třídy se provádí. Další informace o souladu se specifikací CLS najdete v tématu [kompatibilní se specifikací CLS psaní kódu](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) a [jazyková nezávislost a jazykově nezávislé komponenty](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3011:  
  
```  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
