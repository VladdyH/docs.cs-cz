# <a name="compiler-error-cs1055"></a>CS1055 chyby kompilátoru
Přidat nebo odebrat přistupující objekt očekávání  
  
 Pokud vaše [událostí](../../csharp/language-reference/keywords/event.md) není deklarovaný jako pole, musíte definovat, jak **přidat** a **odebrat** přístupové funkce.  
  
 Následující ukázka generuje CS1055:  
  
```  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
