# <a name="compiler-warning-level-3-cs0168"></a>CS0168 kompilátoru upozornění (úroveň 3)
Proměnná 'var' je deklarovaná, ale nikdy nepoužívali

Kompilátor vystaví tři úrovně upozornění, když jste deklarovat proměnnou, ale nepoužívejte ho.
  
Následující ukázka generuje upozornění CS0168:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
