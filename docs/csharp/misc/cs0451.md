# <a name="compiler-error-cs0451"></a><span data-ttu-id="6c03a-102">CS0451 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="6c03a-102">Compiler Error CS0451</span></span>
<span data-ttu-id="6c03a-103">Omezení 'new()' nelze použít s omezením struct.</span><span class="sxs-lookup"><span data-stu-id="6c03a-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="6c03a-104">Při zadávání omezení na typ obecný `new()` omezení lze použít pouze s – třída typu omezení, omezení typu rozhraní, odkaz na typ omezení a omezení parametr typu, ale ne se hodnota typu omezení.</span><span class="sxs-lookup"><span data-stu-id="6c03a-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6c03a-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="6c03a-105">Example</span></span>  
 <span data-ttu-id="6c03a-106">Následující ukázka generuje CS0451.</span><span class="sxs-lookup"><span data-stu-id="6c03a-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4   
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
