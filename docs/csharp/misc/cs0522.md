# <a name="compiler-error-cs0522"></a><span data-ttu-id="5c16e-102">Compiler Error CS0522</span><span class="sxs-lookup"><span data-stu-id="5c16e-102">Compiler Error CS0522</span></span>

 <span data-ttu-id="5c16e-104">A [struktura](../../csharp/language-reference/keywords/struct.md) nelze volat konstruktor základní třída; odeberte volání konstruktoru základní třídy.</span><span class="sxs-lookup"><span data-stu-id="5c16e-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="5c16e-105">Následující ukázka generuje CS0522:</span><span class="sxs-lookup"><span data-stu-id="5c16e-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
