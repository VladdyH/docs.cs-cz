## <a name="example"></a><span data-ttu-id="1cdbb-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="1cdbb-106">Example</span></span>  
 <span data-ttu-id="1cdbb-107">Následující příklad generuje CS0082:</span><span class="sxs-lookup"><span data-stu-id="1cdbb-107">The following example generates CS0082:</span></span>  
  
```  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
