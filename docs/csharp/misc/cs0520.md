# <a name="compiler-error-cs0520"></a>CS0520 chyby kompilátoru
Předdefinované typu 'name' je deklarovaná nesprávně  
  
 Jeden nebo více souborů, které vyžaduje kompilátor nebyly nalezeny. Znovu nainstalujte modul common language runtime rozhraní .NET Framework.
