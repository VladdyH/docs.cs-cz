## <a name="example"></a><span data-ttu-id="4bdd5-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="4bdd5-108">Example</span></span>  
 <span data-ttu-id="4bdd5-109">Následující příklad generuje CS3021:</span><span class="sxs-lookup"><span data-stu-id="4bdd5-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
