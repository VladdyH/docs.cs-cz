# <a name="compiler-error-cs1714"></a>CS1714 chyby kompilátoru
Základní třídu nebo rozhraní Názevtypu1 nebylo možné přeložit, nebo je neplatný  
  
 Implementaci Názevtypu1 z třídy nebo rozhraní, že buď nebylo možné přeložit, například kompilátor se nepodařilo najít nebo je neplatný. Řešení je určit, které z těchto dvou případech je, a buď umístění typu, zadejte více správně nebo opravte případné chyby kompilátoru v základní třídě.
