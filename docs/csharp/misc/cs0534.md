# <a name="compiler-error-cs0534"></a><span data-ttu-id="db4c5-102">CS0534 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="db4c5-102">Compiler Error CS0534</span></span>
<span data-ttu-id="db4c5-103">'function1' neimplementuje zděděného členu abstraktní "funkce2"</span><span class="sxs-lookup"><span data-stu-id="db4c5-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="db4c5-104">Třída je vyžadují k implementaci všech [abstraktní](../../csharp/language-reference/keywords/abstract.md) členy základní třídy, pokud třída je také abstraktní.</span><span class="sxs-lookup"><span data-stu-id="db4c5-104">A class is required to implement all the [abstract](../../csharp/language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="db4c5-105">Následující ukázka generuje CS0534:</span><span class="sxs-lookup"><span data-stu-id="db4c5-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
