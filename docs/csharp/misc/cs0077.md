# <a name="compiler-error-cs0077"></a><span data-ttu-id="8e61b-102">CS0077 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="8e61b-102">Compiler Error CS0077</span></span>
<span data-ttu-id="8e61b-103">As operátor musí být použit s na odkaz na typ nebo typ s možnou hodnotou Null ('int' je typ hodnot neumožňující hodnotu Null).</span><span class="sxs-lookup"><span data-stu-id="8e61b-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="8e61b-104">[Jako](../../csharp/language-reference/keywords/as.md) byl předán operátor [typ hodnoty](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="8e61b-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="8e61b-105">Protože `as` může vrátit [null](../../csharp/language-reference/keywords/null.md), lze pouze předat [odkazové typy](../../csharp/language-reference/keywords/reference-types.md) nebo typ s možnou hodnotou Null.</span><span class="sxs-lookup"><span data-stu-id="8e61b-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="8e61b-106">Další informace o typech s povolenou hodnotou Null, najdete v části [typy s možnou hodnotou Null](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="8e61b-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="8e61b-107">Následující ukázka generuje CS0077:</span><span class="sxs-lookup"><span data-stu-id="8e61b-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
