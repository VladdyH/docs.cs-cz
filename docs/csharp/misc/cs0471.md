# <a name="compiler-error-cs0471"></a>CS0471 chyby kompilátoru
Metoda "název" není obecná metoda. Pokud jste chtěli seznam výrazů, použijte v závorkách < výraz.  
  
 Tato chyba se vygeneruje, když kód obsahuje seznam výraz bez závorek.  
  
## <a name="example"></a>Příklad  
 Následující příklad vytvoří CS0471.  
  
```  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
