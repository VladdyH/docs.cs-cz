# <a name="compiler-error-cs1606"></a>CS1606 chyby kompilátoru
Podepisování sestavení se nezdařilo; výstup nemusí být podepsán - důvodu  
  
 Sestavení bylo vytvořeno, ale při pokusu o kompilátoru dokončit registraci ho, došlo k chybě.
