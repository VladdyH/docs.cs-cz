# <a name="compiler-error-cs0053"></a>Compiler Error CS0053
Nekonzistentní dostupnost: vlastnost typu "typ" je méně přístupný než vlastnost 'vlastnost'  
  
 Veřejný konstruktor musí vrátit objekt veřejně přístupná. Další informace najdete v tématu [modifikátory přístupu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Následující ukázka generuje CS0053:  
  
```csharp  
// CS0053.cs  
class MyClass //defaults to private accessibility  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2  
{  
   public MyClass myProperty   // CS0053  
   {  
      get  
      {  
         return new MyClass();  
      }  
      set  
      {  
      }  
   }  
}  
  
public class MyClass3  
{  
   public static void Main()  
   {  
   }  
}  
```
