# <a name="compiler-error-cs0569"></a>CS0569 chyby kompilátoru
'method2': "method1" nelze přepsat, protože není podporován jazyk  
  
 K této chybě dojde, pokud odvozujete od základní třídu, která byla zapsána v jiném jazyce, a když kompilátor nepodporuje metody, která se pokoušíte přepsat.
