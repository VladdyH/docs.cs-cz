# <a name="compiler-error-cs0513"></a>Compiler Error CS0513
je abstraktní 'function', ale je obsažený v neabstraktní třídy 'class'.  
  
 Metodu nelze [abstraktní](../../csharp/language-reference/keywords/abstract.md) člena neabstraktní třídy.  
  
 Následující ukázka generuje CS0513:  
  
```csharp  
// CS0513.cs  
namespace x  
{  
   public class clx  
   {  
      abstract public void f();   // CS0513, abstract member of nonabstract class  
   }  
}  
```
