# <a name="compiler-warning-level-1-cs0602"></a>CS0602 kompilátoru upozornění (úroveň 1)
Funkce 'old_feature' je zastaralý. Místo toho použijte 'new_feature.  
  
 Jazykové funkce použitá v kódu (*old_feature*) je stále podporováno, ale tato podpora může být odebrán v budoucí verzi. Místo toho používejte doporučené syntaxe (*new_feature*).
