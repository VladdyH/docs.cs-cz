# <a name="compiler-error-cs0055"></a>CS0055 chyby kompilátoru
Nekonzistentní dostupnost: typ parametru "typ" je méně přístupný než indexer 'indexer.  
  
 Veřejný konstruktor musí vrátit objekt veřejně přístupná. Další informace najdete v tématu [modifikátory přístupu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Následující ukázka generuje CS0055:  
  
```  
// CS0055.cs  
class MyClass //defaults to private accessibility  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2  
{  
   public int this[MyClass myClass]   // CS0055  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class MyClass3  
{  
   public static void Main()  
   {  
   }  
}  
```
