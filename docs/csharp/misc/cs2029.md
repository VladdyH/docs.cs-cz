# <a name="compiler-warning-level-1-cs2029"></a>CS2029 kompilátoru upozornění (úroveň 1)
Neplatná hodnota pro '/ definovat'; "identifikátor" není platný identifikátor.  
  
 Toto upozornění se zobrazí, pokud hodnota, která se používá v [/ define](../../csharp/language-reference/compiler-options/define-compiler-option.md) možnost obsahuje některé neplatné znaky.  
  
 Toto upozornění nelze potlačit pomocí [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) možnost.
