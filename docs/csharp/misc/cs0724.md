# <a name="compiler-error-cs0724"></a>Compiler Error CS0724
není nutné atribut CLSCompliant protože sestavení nemá atribut CLSCompliant  
  
 Následující příklad generuje CS0724 z důvodu `throw` příkaz uvnitř `finally` klauzule bloku.  
  
## <a name="example"></a>Příklad  
 Následující příklad vytvoří CS0724.  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
