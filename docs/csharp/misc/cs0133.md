# <a name="compiler-error-cs0133"></a>CS0133 chyby kompilátoru
Musí být konstantní výraz přiřazeny "proměnné"  
  
 A [const](../../csharp/language-reference/keywords/const.md) proměnnou nelze zpracovat jako svou hodnotu výraz, který není konstantní. Další informace najdete v tématu [konstanty](../../csharp/programming-guide/classes-and-structs/constants.md).  
  
 Následující ukázka generuje CS0133:  
  
```csharp  
// CS0133.cs  
public class MyClass  
{  
   public const int i = c;   // CS0133, c is not constant  
   public static int c = i;  
   // try the following line instead  
   // public const int i = 6;  
  
   public static void Main()  
   {  
   }  
}  
```
