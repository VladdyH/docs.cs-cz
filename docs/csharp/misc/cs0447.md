# <a name="compiler-error-cs0447"></a><span data-ttu-id="4ac26-102">CS0447 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="4ac26-102">Compiler Error CS0447</span></span>
<span data-ttu-id="4ac26-103">Atributy nelze použít u argumentů typu.</span><span class="sxs-lookup"><span data-stu-id="4ac26-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="4ac26-104">K této chybě dojde, když použijete atribut pro argument typu, k níž dojde v příkazu volání.</span><span class="sxs-lookup"><span data-stu-id="4ac26-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="4ac26-105">Se dá použít atribut pro parametr typu v příkazu deklarace třída nebo metoda například následující:</span><span class="sxs-lookup"><span data-stu-id="4ac26-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

 <span data-ttu-id="4ac26-106">Následující řádek kódu způsobí vygenerování této chyby.</span><span class="sxs-lookup"><span data-stu-id="4ac26-106">The following line of code will generate this error.</span></span> <span data-ttu-id="4ac26-107">Předpokládá se, že třída `C`, definované v předchozí řádek kódu, má statickou metodu s názvem `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="4ac26-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

