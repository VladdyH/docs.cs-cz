# <a name="compiler-warning-level-1-cs3000"></a>CS3000 kompilátoru upozornění (úroveň 1)
Metody s proměnné argumenty nejsou kompatibilní se specifikací CLS  
  
 Argumenty použité v metodě vystavit funkce, které nejsou v specifikace CLS (Common Language). Další informace o souladu se specifikací CLS najdete v tématu [kompatibilní se specifikací CLS psaní kódu](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).  
  
 Následující příklad generuje upozornění CS3000.  
  
```  
// CS3000.cs  
// compile with: /target:library  
// CS3000 expected  
[assembly:System.CLSCompliant(true)]  
  
public class Test  
{  
   public void AddABunchOfInts( __arglist ) {}   // CS3000  
}  
```
