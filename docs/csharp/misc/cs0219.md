# <a name="compiler-warning-level-3-cs0219"></a>CS0219 kompilátoru upozornění (úroveň 3)
Proměnná "proměnné" je přiřazena, ale jeho hodnota se nikdy nepoužívá.  
  
 Kompilátor vystaví tři úrovně upozornění, když deklarovat a přiřaďte proměnnou, ale nepoužívejte ho.  
  
 Následující ukázka generuje CS0219:  
  
```csharp  
// CS0219.cs  
// compile with: /W:3  
public class MyClass  
{  
   public static void Main()  
   {  
      int a = 0;   // CS0219  
   }  
}  
```
