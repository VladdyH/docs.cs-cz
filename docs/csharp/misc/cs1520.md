# <a name="compiler-error-cs1520"></a><span data-ttu-id="90d21-102">CS1520 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="90d21-102">Compiler Error CS1520</span></span>
<span data-ttu-id="90d21-103">Metoda musí mít návratový typ.</span><span class="sxs-lookup"><span data-stu-id="90d21-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="90d21-104">Metoda, která je deklarována v třídy, struktury nebo rozhraní musí mít explicitního návratového typu.</span><span class="sxs-lookup"><span data-stu-id="90d21-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="90d21-105">V následujícím příkladu Čtvereček metoda má návratovou hodnotu [řetězec](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="90d21-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="90d21-106">Následující ukázka generuje CS1520:</span><span class="sxs-lookup"><span data-stu-id="90d21-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="90d21-107">Tato chyba můžou taky nastat při případ název konstruktoru se liší od deklarace třídy nebo struktury, stejně jako v následujícím příkladu.</span><span class="sxs-lookup"><span data-stu-id="90d21-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="90d21-108">Protože název je přesně stejný jako název třídy, kompilátor interpretuje jako metodu pravidelné není konstruktor a Chyba:</span><span class="sxs-lookup"><span data-stu-id="90d21-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
