# <a name="compiler-error-cs1520"></a><span data-ttu-id="6be83-102">CS1520 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="6be83-102">Compiler Error CS1520</span></span>
<span data-ttu-id="6be83-103">Metoda musí mít typ vrácené hodnoty.</span><span class="sxs-lookup"><span data-stu-id="6be83-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="6be83-104">Metoda, která je deklarovaná ve třídě, struktura nebo rozhraní musí mít explicitní návratový typ.</span><span class="sxs-lookup"><span data-stu-id="6be83-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="6be83-105">V následujícím příkladu má metodu odmocnina návratová hodnota [řetězec](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="6be83-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="6be83-106">Následující ukázka generuje CS1520:</span><span class="sxs-lookup"><span data-stu-id="6be83-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="6be83-107">Alternativně tato chyba může být při zjistil v případě název konstruktor se liší od třídě nebo struktuře deklaraci, která jako následující příklad.</span><span class="sxs-lookup"><span data-stu-id="6be83-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="6be83-108">Vzhledem k tomu, že je název není stejný jako název třídy, kompilátor interpretuje jako regulární metodu, není konstruktoru a k chybě:</span><span class="sxs-lookup"><span data-stu-id="6be83-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
