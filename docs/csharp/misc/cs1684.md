## <a name="example"></a><span data-ttu-id="2aa07-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="2aa07-108">Example</span></span>  
  
```csharp

   public void Test() {}  
}

```  
  
## <a name="example"></a><span data-ttu-id="2aa07-109">Příklad</span><span class="sxs-lookup"><span data-stu-id="2aa07-109">Example</span></span>

```csharp  
   public static A GetA() { return new A(); }

}  
```
