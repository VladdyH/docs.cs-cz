# <a name="compiler-error-cs0723"></a>CS0723 chyby kompilátoru
Nelze deklarovat proměnnou statického typu "typ"  
  
 Nelze vytvořit instance statické typy.  
  
 Následující ukázka generuje CS0723:  
  
```  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
