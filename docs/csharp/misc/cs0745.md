 <span data-ttu-id="90807-104">Tento vzor `group` je klauzule `group...by` následuje volitelný `into`, jak je znázorněno v následujícím příkladu:</span><span class="sxs-lookup"><span data-stu-id="90807-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="90807-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="90807-108">Example</span></span>  
 <span data-ttu-id="90807-109">Následující kód generuje CS0745:</span><span class="sxs-lookup"><span data-stu-id="90807-109">The following code generates CS0745:</span></span>

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };
