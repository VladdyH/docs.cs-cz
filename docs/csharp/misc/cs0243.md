 <span data-ttu-id="37c5a-106">Kompilátor nikdy váže k přepsání metody; vytvoří vazbu jenom základní metody a modul common language runtime volá přepsání, podle potřeby.</span><span class="sxs-lookup"><span data-stu-id="37c5a-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="37c5a-107">Následující ukázka generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="37c5a-107">The following sample generates CS0243:</span></span>  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
