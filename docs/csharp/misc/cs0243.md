 <span data-ttu-id="5c281-104">[Podmíněného](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atributu není povolená na metodu, která je označena [přepsat](../../csharp/language-reference/keywords/override.md) – klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="5c281-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="5c281-105">Další informace najdete v tématu [zároveň budete vědět, při použití přepsání a nová klíčová slova](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="5c281-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="5c281-106">Kompilátor nikdy váže k přepsání metody; vytvoří vazbu jenom základní metody a modul common language runtime volá přepsání, podle potřeby.</span><span class="sxs-lookup"><span data-stu-id="5c281-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="5c281-107">Následující ukázka generuje CS0243:</span><span class="sxs-lookup"><span data-stu-id="5c281-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
