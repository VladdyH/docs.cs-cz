# <a name="compiler-error-cs0127"></a>CS0127 chyby kompilátoru
Vzhledem k tomu, že 'function' Vrátí void, návratový – klíčové slovo nesmí být následováno výrazem objektu  
  
 Metoda s [void](../../csharp/language-reference/keywords/void.md) vrátí typ nemůže vrátit hodnotu. Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Následující ukázka generuje CS0127:  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
