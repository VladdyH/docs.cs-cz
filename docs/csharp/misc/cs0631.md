# <a name="compiler-error-cs0631"></a>CS0631 chyby kompilátoru
REF a out nejsou platné v tomto kontextu  
  
 Deklarace pro [indexer](../../csharp/programming-guide/indexers/index.md) nesmí obsahovat použití [ref](../../csharp/language-reference/keywords/ref.md) nebo [out](../../csharp/language-reference/keywords/out.md) parametry.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0631:  
  
```  
// CS0631.cs  
public class MyClass  
{  
    public int this[ref int i]   // CS0631  
    // try the following line instead  
    // public int this[int i]  
    {  
        get  
        {  
            return 0;  
        }  
    }  
}  
  
public class MyClass2  
{  
    public static void Main()  
    {  
    }  
}  
```
