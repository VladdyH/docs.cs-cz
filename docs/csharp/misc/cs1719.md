# <a name="compiler-error-cs1719"></a>CS1719 chyby kompilátoru
Chyba při čtení souboru název – "důvod" zdrojového souboru Win32  
  
 Pokus o čtení zdrojového souboru Win32 se nezdařila z důvodu zadané v chybě, obvykle něco jako "soubor nebyl nalezen" nebo "přístup byl odepřen". Tato chyba se vyřeší za účelem opravy problému popsaného z důvodu.
