# <a name="compiler-error-cs0167"></a>CS0167 chyby kompilátoru
Delegát 'delegáta' chybí metodu Invoke  
  
 Importovat a použít spravovaný program (ten, který používá rozhraní .NET Framework common language runtime), byl vytvořen s jinou kompilátoru. Tento kompilátoru povolené nedůvěryhodný [delegovat](../../csharp/language-reference/keywords/delegate.md). Proto `Invoke` metoda nebyl k dispozici. Další informace najdete v tématu [delegáti](../../csharp/programming-guide/delegates/index.md).
