# <a name="compiler-warning-level-1-cs3027"></a>CS3027 kompilátoru upozornění (úroveň 1)
'type_1' není kompatibilní se specifikací CLS, protože základní rozhraní 'type_2' není kompatibilní se specifikací CLS  
  
 Kompatibilní s typem specifikací CLS nemůže být základní typ pro typ, který je kompatibilní se specifikací CLS.  
  
## <a name="example"></a>Příklad  
 Následující příklad obsahuje rozhraní s metodu, která používá se v jeho podpis, provedení kompatibilní typ jiný CLS kompatibilní typ specifikací CLS.  
  
```  
// CS3027.cs  
// compile with: /target:library  
public interface IBase  
{  
   void IMethod(uint i);  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS3027.  
  
```  
// CS3027_b.cs  
// compile with: /reference:CS3027.dll /target:library /W:1  
[assembly:System.CLSCompliant(true)]  
public interface IDerived : IBase {}  
```
