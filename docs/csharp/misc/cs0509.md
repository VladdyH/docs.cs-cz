# <a name="compiler-error-cs0509"></a>CS0509 chyby kompilátoru
"třída1": nelze odvodit z zapečetěné typu "třída2"  
  
 A [zapečetěné](../../csharp/language-reference/keywords/sealed.md) třída nemůže fungovat jako [základní](../../csharp/language-reference/keywords/base.md) třídy. Ve výchozím nastavení jsou zapečetěné struktury.  
  
 Následující ukázka generuje CS0509:  
  
```csharp  
// CS0509.cs  
// compile with: /target:library  
sealed public class clx {}  
public class cly : clx {}   // CS0509  
```
