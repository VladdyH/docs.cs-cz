# <a name="compiler-error-cs0690"></a>CS0690 chyby kompilátoru
Vstupní soubor 'file' obsahuje neplatná metadata.  
  
 Budete moci otevřít soubor metadat, ale kvůli potížím lokalizace byla poškozena metadata. Toto je podobná chyba [CS0009](../../csharp/misc/cs0009.md)kromě toho, že budete moci otevřít soubor metadat.  
  
 (Spustitelný proces) PE je spustitelný soubor.
