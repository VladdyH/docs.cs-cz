 <span data-ttu-id="d4ce2-104">Deklarace všechny metody, který používá `override`, **virtuální**, nebo **abstraktní** – klíčové slovo nelze použít také **statické** – klíčové slovo.</span><span class="sxs-lookup"><span data-stu-id="d4ce2-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="d4ce2-105">Další informace najdete v tématu [metody](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="d4ce2-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="d4ce2-106">Následující ukázka generuje CS0112:</span><span class="sxs-lookup"><span data-stu-id="d4ce2-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
