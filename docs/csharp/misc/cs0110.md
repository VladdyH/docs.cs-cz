# <a name="compiler-error-cs0110"></a>CS0110 chyby kompilátoru
Vyhodnocení konstantní hodnoty 'Deklarace const' zahrnuje cyklickou definici  
  
 Deklarace [const](../../csharp/language-reference/keywords/const.md) proměnné (`a`) nemůže odkazovat na jiné proměnné const (`b`), který také odkazuje na (`a`).  
  
 Následující ukázka generuje CS0110:  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
