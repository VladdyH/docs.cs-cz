# <a name="compiler-error-cs0110"></a><span data-ttu-id="562c5-102">CS0110 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="562c5-102">Compiler Error CS0110</span></span>
<span data-ttu-id="562c5-103">Vyhodnocení konstantní hodnota "const prohlášení" zahrnuje cyklickou definici</span><span class="sxs-lookup"><span data-stu-id="562c5-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="562c5-104">Deklaraci [const](../../csharp/language-reference/keywords/const.md) proměnné (`a`) nemůže odkazovat na jiné proměnné const (`b`), odkazuje také (`a`).</span><span class="sxs-lookup"><span data-stu-id="562c5-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="562c5-105">Následující ukázka generuje CS0110:</span><span class="sxs-lookup"><span data-stu-id="562c5-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
