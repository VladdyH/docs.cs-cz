## <a name="example"></a><span data-ttu-id="b0f87-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="b0f87-108">Example</span></span>  
 <span data-ttu-id="b0f87-109">Následující ukázka generuje CS3018.</span><span class="sxs-lookup"><span data-stu-id="b0f87-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
