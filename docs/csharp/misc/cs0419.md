 <span data-ttu-id="f0ace-105">V dokumentaci XML komentáře v kódu nebylo možné přeložit odkaz.</span><span class="sxs-lookup"><span data-stu-id="f0ace-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="f0ace-106">Tato situace může nastat, pokud metoda je přetížena, nebo pokud se nacházejí dva různé identifikátory se stejným názvem.</span><span class="sxs-lookup"><span data-stu-id="f0ace-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="f0ace-107">K vyřešení upozornění, pomocí kvalifikovaný název kvůli zajištění jednoznačnosti odkaz nebo použít konkrétní přetížení v závorkách.</span><span class="sxs-lookup"><span data-stu-id="f0ace-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="f0ace-108">Následující ukázka generuje CS0419.</span><span class="sxs-lookup"><span data-stu-id="f0ace-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
