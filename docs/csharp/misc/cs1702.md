# <a name="compiler-warning-level-3-cs1702"></a>CS1702 kompilátoru upozornění (úroveň 3)
Za předpokladu, že odkaz na sestavení "Název sestavení #1" odpovídá "Název sestavení #2", budete možná muset zadat zásady modulu runtime  
  
 Odkazy na dvě sestavení mají odlišné sestavení a revize čísel, tak nebude automaticky sjednocení. Musíte zadat zásady běhu vynutit sjednocení pomocí direktivy v souboru config aplikace.
