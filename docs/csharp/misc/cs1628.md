# <a name="compiler-error-cs1628"></a>CS1628 chyby kompilátoru
Nelze použít ref nebo out parametr "parametr" uvnitř anonymní metoda, výrazu lambda nebo výrazu dotazu  
  
 K této chybě dojde, pokud používáte ref nebo out parametr uvnitř bloku anonymní metody. Chcete-li se vyhnout této chybě, použijte místní proměnné nebo některých jiných konstrukce.  
  
 Následující ukázka generuje CS1628:  
  
```  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
