# <a name="compiler-error-cs1623"></a>CS1623 chyby kompilátoru
Iterátory nemůže mít ref nebo out parametry  
  
 K této chybě dojde, pokud je metoda iterator trvá `ref` nebo `out` parametr. Chcete-li se vyhnout této chybě, odeberte `ref` nebo `out` – klíčové slovo z podpis metody.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1623:  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
