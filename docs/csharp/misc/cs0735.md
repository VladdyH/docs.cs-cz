# <a name="compiler-error-cs0735"></a>CS0735 chyby kompilátoru
Neplatný typ zadaný jako argument atributu TypeForwardedTo  
  
 Následující ukázka generuje CS0735.  
  
```csharp  
// CS735.cs  
// compile with: /target:library  
using System.Runtime.CompilerServices;  
[assembly:TypeForwardedTo(typeof(int[]))]   // CS0735  
[assembly:TypeForwardedTo(typeof(string))]   // OK  
```
