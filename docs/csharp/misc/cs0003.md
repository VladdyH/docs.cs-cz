# <a name="compiler-error-cs0003"></a>CS0003 chyby kompilátoru
Nedostatek paměti  
  
 Kompilátor nemohla přidělit dostatek virtuální paměti pro dokončení kompilace. Ukončete všechny nepotřebné aplikace a znovu zkompilovat.  
  
 Můžete také zvětšete velikost stránkovacího souboru a ujistěte se, že máte volného místa na disku.  
  
 Tato chyba může také zobrazit jako výsledek s neodpovídající verze [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] a kompilátor jazyka C# nebo jeden nebo více poškozené soubory, které podporují kompilátor jazyka C#; znovu nainstalujte Visual Studio.
