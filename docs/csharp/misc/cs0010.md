# <a name="compiler-error-cs0010"></a>CS0010 chyby kompilátoru
Neočekávané závažná chyba – "Chyba".  
  
 Tato chyba se vygeneruje, když dojde ke zcela neočekávané dojde k zastavení kompilace.  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Znovu zkompiluje jiného projektu. Pokud se zobrazí ke stejné chybě, zkuste opakovat instalaci sady Visual Studio. Pokud tato chyba se zobrazí, odešlete zprávu o chybách společnosti Microsoft.
