# <a name="compiler-error-cs1630"></a>CS1630 chyby kompilátoru
Neplatná možnost "možnost" pro/errorreport; musí být řádku, odeslání, fronty nebo none  
  
 Tato možnost příkazového řádku [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) musí následovat **řádku**, **odeslat**, **fronty**, nebo **žádné**, zadání, jakou akci chcete přijatá když dojde k chybě vnitřní kompilátoru.
