# <a name="compiler-error-cs1557"></a>CS1557 chyby kompilátoru
Pro metodu Main nelze použít 'class', protože je v jiné výstupní soubor  
  
 [/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) – možnost kompilátoru zadaná pro jednu výstupní soubor v kompilaci více výstupní soubor. Ale třída nebyla nalezena ve zdrojovém kódu pro / Main kompilaci. nenašel se v souboru zdrojového kódu pro jednu výstupní soubory v kompilace.
