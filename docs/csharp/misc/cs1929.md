 <span data-ttu-id="67118-104">Tato chyba je generována při pokusu o vyvolání metody rozšíření z třídy, která není rozšíření.</span><span class="sxs-lookup"><span data-stu-id="67118-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="67118-105">V tomto příkladu je definována rozšiřující metoda odvozené třídy `A`, ale ne pro základní třídu `B`.</span><span class="sxs-lookup"><span data-stu-id="67118-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>

## <a name="example"></a><span data-ttu-id="67118-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="67118-108">Example</span></span>  
 <span data-ttu-id="67118-109">Následující kód vygeneruje CS1928 a CS1929:</span><span class="sxs-lookup"><span data-stu-id="67118-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
