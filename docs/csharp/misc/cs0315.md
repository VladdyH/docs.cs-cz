## <a name="example"></a><span data-ttu-id="e4f09-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="e4f09-108">Example</span></span>  
 <span data-ttu-id="e4f09-109">Následující příklad generuje CS0315:</span><span class="sxs-lookup"><span data-stu-id="e4f09-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{         
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
