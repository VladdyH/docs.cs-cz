# <a name="compiler-error-cs1554"></a>Compiler Error CS1554
Deklarace není platná. použít '\<typ > op – operátor (...' místo  
  
 Návratový typ pro uživatelem definované [operátor](../../csharp/language-reference/keywords/operator.md) musí být před operátor – klíčové slovo.  
  
 Následující ukázka generuje CS1554:  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
