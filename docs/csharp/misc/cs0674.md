# <a name="compiler-error-cs0674"></a>CS0674 chyby kompilátoru
Nepoužívejte 'System.ParamArrayAttribute'. Místo toho použijte klíčové slovo 'parametry'.  
  
 Kompilátor jazyka C# neumožňuje použít <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; použít [parametry](../../csharp/language-reference/keywords/params.md) místo.  
  
 Následující ukázka generuje CS0674:  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
