# <a name="compiler-error-cs0535"></a>Compiler Error CS0535
'class' neimplementuje rozhraní člen "člen"  
  
 A [třída](../../csharp/language-reference/keywords/class.md) odvozené od [rozhraní](../../csharp/language-reference/keywords/interface.md), ale třída neimplementovala jeden nebo více členů rozhraní. Třída musí implementovat rozhraní, ze kterých se odvozuje jinak deklarovat všechny členy `abstract`.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0535.  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0535.  
  
```csharp  
// CS0535_b.cs  
using System;  
class C : IDisposable {}   // CS0535  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
