## <a name="example"></a><span data-ttu-id="13979-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="13979-106">Example</span></span>  
 <span data-ttu-id="13979-107">Následující ukázka generuje CS0535.</span><span class="sxs-lookup"><span data-stu-id="13979-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
