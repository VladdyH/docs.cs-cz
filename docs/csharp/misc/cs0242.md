# <a name="compiler-error-cs0242"></a>Compiler Error CS0242
V operaci je definovaná na neplatné ukazatele  
  
 Zvyšování neplatný ukazatel není povoleno. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0242:  
  
```csharp  
// CS0242.cs  
// compile with: /unsafe  
class TestClass  
{  
   public unsafe void Test()  
   {  
      void * p = null;  
      p++;   // CS0242, incrementing a void pointer not allowed  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
