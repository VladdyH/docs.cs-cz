# <a name="compiler-warning-level-1-cs0809"></a>Kompilátor CS0809 upozornění (úroveň 1)
Zastaralý člen memberA přepisuje nezastaralý člen memberB.  
  
 Obvykle by neměl člena, který je označen jako zastaralý přepsání člena, který není označen jako zastaralý. Toto upozornění je generováno v [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] , ale ne v [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].  
  
## <a name="to-correct-this-error"></a>Oprava této chyby  
  
1.  Odeberte `Obsolete` atribut z přepsáním členu, nebo ho přidejte do členu základní třídy.  
  
## <a name="example"></a>Příklad  
  
```csharp  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}  
public class C : Base  
{  
    [System.Obsolete()]  
    public override void Test1() // CS0809  
    {  
    }  
}  
```
