# <a name="compiler-error-cs1562"></a>CS1562 chyby kompilátoru
Výstup bez zdroje musí mít zadán přepínač/out.  
  
 Kompilace může vytvoření výstupního souboru, ale jako vstup, ze kterého může být název souboru výstupního souboru implicitní došlo bez souboru se zdrojovým kódem. Pokoušíte například zkompilovat soubor nebo prostředků pouze metadata.  
  
 Použití [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) – možnost kompilátoru k zadání názvu výstupního souboru.
