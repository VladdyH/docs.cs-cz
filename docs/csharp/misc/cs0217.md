# <a name="compiler-error-cs0217"></a><span data-ttu-id="8f977-102">Compiler Error CS0217</span><span class="sxs-lookup"><span data-stu-id="8f977-102">Compiler Error CS0217</span></span>
<span data-ttu-id="8f977-103">Chcete-li být použitelná jako operátor zkráceného vyhodnocení logického operátoru (operátor) uživatelem definované musí mít stejné návratový typ jako typ její 2 parametry.</span><span class="sxs-lookup"><span data-stu-id="8f977-103">In order to be applicable as a short circuit operator a user-defined logical operator ('operator') must have the same return type as the type of its 2 parameters.</span></span>  
  
 <span data-ttu-id="8f977-104">Pokud definujete operátor pro typ definovaný uživatelem a potom se pokusíte použít operátor jako krátká smyčka – operátor, operátor uživatelem definované musí mít parametry a návratové hodnoty stejného typu.</span><span class="sxs-lookup"><span data-stu-id="8f977-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have parameters and return values of the same type.</span></span> <span data-ttu-id="8f977-105">Další informace o krátká smyčka – operátory najdete v tématu [& & – operátor](../../csharp/language-reference/operators/conditional-and-operator.md) a [ &#124; &#124; operátor](../../csharp/language-reference/operators/conditional-or-operator.md).</span><span class="sxs-lookup"><span data-stu-id="8f977-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="8f977-106">Následující ukázka generuje CS0217:</span><span class="sxs-lookup"><span data-stu-id="8f977-106">The following sample generates CS0217:</span></span>  
  
```csharp  
// CS0217.cs  
using System;  
  
public class MyClass  
{  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static int operator & (MyClass f1, MyClass f2)   // CS0217  
   // try the following line instead  
   // public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;  
   }  
}  
```
