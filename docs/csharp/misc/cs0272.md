## <a name="example"></a><span data-ttu-id="60f8e-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="60f8e-107">Example</span></span>

```

    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}

{  
    static void Main()  
    {  
        MyClass c = new MyClass();

        // To resolve, remove the previous line

    }  
}  
```
