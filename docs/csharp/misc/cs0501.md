# <a name="compiler-error-cs0501"></a>CS0501 chyby kompilátoru
'– členská funkce, musí deklarovat text, protože není označena jako abstraktní, externí nebo částečné  
  
 Neabstraktní metody musí mít implementace.  
  
 Následující ukázka generuje CS0501:  
  
```  
// CS0501.cs  
// compile with: /target:library  
public class clx  
{  
   public void f();   // CS0501 declared but not defined  
   public void g() {}   // OK  
}  
```
