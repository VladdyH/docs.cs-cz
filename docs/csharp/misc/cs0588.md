# <a name="compiler-error-cs0588"></a>CS0588 chyby kompilátoru
Vnitřní chyba kompilátoru: fáze 'LEX.  
  
 Pokuste se zjistit, zda kompilátor selhává kvůli jeho nemůže analyzovat neočekávanou syntaxi. Pokud tomu tak není, zkuste [kontaktujte nás](/visualstudio/ide/talk-to-us).
