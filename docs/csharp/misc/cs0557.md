# <a name="compiler-error-cs0557"></a>Compiler Error CS0557
Duplicitní převod uživatelem definované v typu 'class'.  
  
 Rutiny převodu duplicitní nejsou povoleny v třídě.  
  
 Následující příklad generuje CS0557:  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
