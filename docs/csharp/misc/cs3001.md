# <a name="compiler-warning-level-1-cs3001"></a>CS3001 kompilátoru upozornění (úroveň 1)
Argument typu "typ" není kompatibilní se specifikací CLS  
  
 A [veřejné](../../csharp/language-reference/keywords/public.md), [chráněné](../../csharp/language-reference/keywords/protected.md), nebo `protected internal` metoda musí přijmout parametr, jehož typ je v souladu s specifikace CLS (Common Language). Další informace o souladu se specifikací CLS najdete v tématu [kompatibilní se specifikací CLS psaní kódu](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) a [jazyková nezávislost a jazykově nezávislé komponenty](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS3001:  
  
```csharp  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
