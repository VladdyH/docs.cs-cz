# <a name="compiler-error-cs1621"></a><span data-ttu-id="d2303-102">CS1621 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="d2303-102">Compiler Error CS1621</span></span>
<span data-ttu-id="d2303-103">Příkaz yield nejde použít uvnitř anonymní metody nebo výrazu lambda</span><span class="sxs-lookup"><span data-stu-id="d2303-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="d2303-104">Příkaz yield nemůže být v bloku anonymní metody v iterátoru.</span><span class="sxs-lookup"><span data-stu-id="d2303-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d2303-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="d2303-105">Example</span></span>  
 <span data-ttu-id="d2303-106">Následující ukázka generuje CS1621:</span><span class="sxs-lookup"><span data-stu-id="d2303-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
