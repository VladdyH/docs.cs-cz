# <a name="compiler-error-cs1621"></a>CS1621 chyby kompilátoru
Příkaz yield nelze použít v rámci anonymní metody nebo výrazu lambda  
  
 Příkaz yield nemůže být v bloku anonymní metody v iterace.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1621:  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
