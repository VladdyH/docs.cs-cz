# <a name="compiler-error-cs1605"></a>Compiler Error CS1605
Nelze předat 'var' ref nebo out argument protože je jen pro čtení  
  
 Proměnné předány jako [ref](../../csharp/language-reference/keywords/ref.md) nebo [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametr musí být upravena volané metody. Proto není možné předat jako parametr jen pro čtení `ref` nebo `out`.
