# <a name="compiler-warning-level-1-cs2023"></a>CS2023 kompilátoru upozornění (úroveň 1)
/ Noconfig – možnost je ignorována, protože byl zadán v souboru odpovědí  
  
 [/Noconfig](../../csharp/language-reference/compiler-options/noconfig-compiler-option.md) – možnost kompilátoru byl zadán v souboru odpovědí, který není povolen.  
  
 Toto upozornění nelze potlačit pomocí [/nowarn](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) možnost.
