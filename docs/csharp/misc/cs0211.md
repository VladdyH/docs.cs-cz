# <a name="compiler-error-cs0211"></a>Compiler Error CS0211
Nelze převést na adresu daného výrazu.  
  
 Můžete provést na adresu pole, místní proměnné a indirekce ukazatele, ale nemůžete provést, například na adresu součet dvou lokální proměnné. Další informace najdete v tématu [nezabezpečený kód a ukazatele](../../csharp/programming-guide/unsafe-code-pointers/index.md).  
  
 Následující ukázka generuje CS0211:  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
