 <span data-ttu-id="b52fb-104">Je nutné použít **deklarací StructLayout(LayoutKind.Explicit)** atributu u struktura deklaraci, pokud obsahuje všechny členy označené jako **FieldOffset** atribut.</span><span class="sxs-lookup"><span data-stu-id="b52fb-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="b52fb-105">Další informace najdete v tématu [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="b52fb-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>

{  
   public static void Main ()

   }  
}
