# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="d7ff1-102">CS0684 kompilátoru upozornění (úroveň 1)</span><span class="sxs-lookup"><span data-stu-id="d7ff1-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="d7ff1-103">rozhraní, rozhraní, označené jako CoClassAttribute neoznačené 'ComImportAttribute.</span><span class="sxs-lookup"><span data-stu-id="d7ff1-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="d7ff1-104">Pokud zadáte **CoClassAttribute** na rozhraní, je nutné také zadat **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="d7ff1-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="d7ff1-105">Následující ukázka generuje CS0684:</span><span class="sxs-lookup"><span data-stu-id="d7ff1-105">The following sample generates CS0684:</span></span>  
  
```  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
