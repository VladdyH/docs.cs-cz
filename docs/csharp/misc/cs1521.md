# <a name="compiler-error-cs1521"></a>CS1521 chyby kompilátoru
Neplatná základní typ  
  
 A [základní](../../csharp/language-reference/keywords/base.md) třída specifikace špatně byl vytvořen.  
  
 Následující ukázka generuje CS1521:  
  
```  
// CS1521.cs  
class CMyClass  
{  
   public static void Main()  
   {  
   }  
}  
  
class CMyClass1 : CMyClass     // OK  
{  
}  
  
class CMyClass2 : CMyClass[]   // CS1521  
{  
}  
  
class CMyClass3 : CMyClass*    // CS1521  
{  
}  
```
