# <a name="compiler-error-cs1629"></a>CS1629 chyby kompilátoru
Iterátory nesmí obsahovat nezabezpečený kód  
  
 Specifikace jazyka C# neumožňuje nezabezpečený kód v iterátory.  
  
 Následující ukázka generuje CS1629:  
  
```  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
