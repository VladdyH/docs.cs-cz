# <a name="compiler-error-cs2001"></a>CS2001 chyby kompilátoru
Zdrojový soubor 'file' nebyla nalezena.  
  
 Název zdrojového souboru byl předán pro kompilátor, ale nebyl nalezen. Zkontrolujte, zda název souboru a umístění souboru.
