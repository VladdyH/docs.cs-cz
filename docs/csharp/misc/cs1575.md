# <a name="compiler-error-cs1575"></a><span data-ttu-id="6d6a6-102">CS1575 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="6d6a6-102">Compiler Error CS1575</span></span>
<span data-ttu-id="6d6a6-103">Výraz stackalloc vyžaduje po typu]</span><span class="sxs-lookup"><span data-stu-id="6d6a6-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="6d6a6-104">Velikost požadované přidělení, s [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), musí být zadán v hranatých závorkách.</span><span class="sxs-lookup"><span data-stu-id="6d6a6-104">The size of the requested allocation, with [stackalloc](../../csharp/language-reference/keywords/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="6d6a6-105">Následující ukázka generuje CS1575:</span><span class="sxs-lookup"><span data-stu-id="6d6a6-105">The following sample generates CS1575:</span></span>  
  
```  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
