 <span data-ttu-id="33d55-104">Jedná se o chybu pro parametr modifikátory například `this`, `ref`, `in`, a `out` se objeví v definici parametru více než jednou.</span><span class="sxs-lookup"><span data-stu-id="33d55-104">It is an error for parameter modifiers such as `this`, `ref`, `in`, and `out` to appear more than one time in a parameter definition.</span></span>  
  
## <a name="example"></a><span data-ttu-id="33d55-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="33d55-105">Example</span></span>  
 <span data-ttu-id="33d55-106">Následující příklad generuje CS1107:</span><span class="sxs-lookup"><span data-stu-id="33d55-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
