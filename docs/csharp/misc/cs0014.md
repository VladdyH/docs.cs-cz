# <a name="compiler-error-cs0014"></a>CS0014 chyby kompilátoru
Požadovaný soubor 'file' nebyla nalezena.  
  
 Soubor vyžaduje kompilátor, ale není v systému. Ujistěte se, že je cesta správná. Pokud je soubor [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] systémový soubor a pak může nutné opravit instalaci nebo jiného odebrat a znovu nainstalovat [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] úplně.
