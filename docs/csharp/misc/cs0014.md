# <a name="compiler-error-cs0014"></a>CS0014 chyby kompilátoru
Požadovaný soubor 'file' nebyla nalezena.  
  
 Soubor vyžaduje kompilátor, ale není v systému. Ujistěte se, že je cesta správná. Pokud je soubor systémový soubor Visual Studio, pak musíte opravte instalaci aplikace nebo s jiným odstranit a znovu nainstalovat Visual Studio úplně.
