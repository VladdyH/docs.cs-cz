# <a name="compiler-warning-level-1-cs1697"></a>CS1697 kompilátoru upozornění (úroveň 1)
Různé kontrolní součet hodnoty zadané pro "název souboru.  
  
 Zadali jste více než jeden kontrolního součtu daného souboru. Ladicí program používá hodnotu kontrolního součtu k určit, který soubor k ladění, pokud existuje více než jeden soubor v projektu se stejným názvem. Většina uživatelů nebude k této chybě dojde, ale pokud píšete aplikaci, která generuje kód, může spustit do ní. Chcete-li tuto chybu vyřešit, zkontrolujte vygenerování kontrolního součtu jenom jednou pro všechny soubory daného kódu.
