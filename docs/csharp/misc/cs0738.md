## <a name="example"></a><span data-ttu-id="91af3-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="91af3-108">Example</span></span>  
 <span data-ttu-id="91af3-109">Následující kód vygeneruje CS0738, protože metoda třídy vrací `void` a interface člena se stejným názvem vrátí `int`:</span><span class="sxs-lookup"><span data-stu-id="91af3-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

    int TestMethod();
