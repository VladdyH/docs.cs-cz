 <span data-ttu-id="d1529-104">[Podmíněného](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) nelze atribut použít v implementaci rozhraní.</span><span class="sxs-lookup"><span data-stu-id="d1529-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="d1529-105">Následující ukázka generuje CS0629:</span><span class="sxs-lookup"><span data-stu-id="d1529-105">The following sample generates CS0629:</span></span>  
  
```  
// CS0629.cs

{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
