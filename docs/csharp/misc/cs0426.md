 <span data-ttu-id="a3a2e-104">Tato chyba nastane, když odkazujete typu vnořené v rámci jiného typu, ale není žádná taková vnořené, že typ existuje.</span><span class="sxs-lookup"><span data-stu-id="a3a2e-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="a3a2e-105">Tato situace může nastat, pokud překlep název vnořené typu.</span><span class="sxs-lookup"><span data-stu-id="a3a2e-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="a3a2e-106">Zkontrolujte, zda názvů používaných a ověřte, zda očekávaný člen nadřazených typů.</span><span class="sxs-lookup"><span data-stu-id="a3a2e-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="a3a2e-107">Následující ukázka generuje CS0426, protože třída C nemá žádné vnořené typy A:</span><span class="sxs-lookup"><span data-stu-id="a3a2e-107">The following sample generates CS0426 because class C has no nested type A:</span></span>

{  
    // No nested types are declared.     
}
