 <span data-ttu-id="9a06b-104">Tato chyba nastane odkazovat na typ, vnořeny v jiném typu, ale není žádná taková vnořený typ existuje.</span><span class="sxs-lookup"><span data-stu-id="9a06b-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="9a06b-105">Tato situace může nastat, pokud napíšete název vnořeného typu.</span><span class="sxs-lookup"><span data-stu-id="9a06b-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="9a06b-106">Kontrola pravopisu názvů používaných a ověřte, zda má nadřazený typ. očekávaný člen.</span><span class="sxs-lookup"><span data-stu-id="9a06b-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="9a06b-107">Následující ukázka generuje CS0426, protože třída C nemá žádný vnořený typ A:</span><span class="sxs-lookup"><span data-stu-id="9a06b-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
