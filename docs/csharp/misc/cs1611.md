# <a name="compiler-error-cs1611"></a>CS1611 chyby kompilátoru
Nelze deklarovat parametr parametry ref nebo out  
  
 Klíčová slova [ref](../../csharp/language-reference/keywords/ref.md) nebo [out](../../csharp/language-reference/keywords/out.md) nelze použít s [parametry](../../csharp/language-reference/keywords/params.md) – klíčové slovo.  
  
 Následující ukázka generuje CS1611:  
  
```  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
