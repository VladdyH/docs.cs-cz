# <a name="compiler-error-cs0218"></a><span data-ttu-id="933cd-102">CS0218 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="933cd-102">Compiler Error CS0218</span></span>
<span data-ttu-id="933cd-103">Typ ("typ") musí obsahovat deklarace operátor true a false.</span><span class="sxs-lookup"><span data-stu-id="933cd-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
 <span data-ttu-id="933cd-104">Pokud definujete operátor pro typ definovaný uživatelem a potom se pokusíte použít operátor jako krátká smyčka – operátor, operátor uživatelem definované musí mít operátor true a false definované operátor.</span><span class="sxs-lookup"><span data-stu-id="933cd-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have operator true and operator false defined.</span></span> <span data-ttu-id="933cd-105">Další informace o krátká smyčka – operátory najdete v tématu [& & – operátor](../../csharp/language-reference/operators/conditional-and-operator.md) a [ &#124; &#124; operátor](../../csharp/language-reference/operators/conditional-or-operator.md).</span><span class="sxs-lookup"><span data-stu-id="933cd-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="933cd-106">Následující ukázka generuje CS0218:</span><span class="sxs-lookup"><span data-stu-id="933cd-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
