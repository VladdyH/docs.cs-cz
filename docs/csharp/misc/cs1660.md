# <a name="compiler-error-cs1660"></a><span data-ttu-id="cc578-102">CS1660 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="cc578-102">Compiler Error CS1660</span></span>
<span data-ttu-id="cc578-103">Blok anonymní metody na typ "typ", protože není typem delegáta nelze převést.</span><span class="sxs-lookup"><span data-stu-id="cc578-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="cc578-104">K této chybě dojde, pokud se pokusíte se ji přiřadit nebo jinak bloku anonymní metody převést na typ, který není typu delegáta.</span><span class="sxs-lookup"><span data-stu-id="cc578-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="cc578-105">Následující ukázka generuje CS1660:</span><span class="sxs-lookup"><span data-stu-id="cc578-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
