## <a name="example"></a><span data-ttu-id="feb5c-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="feb5c-106">Example</span></span>

```csharp  
// CS0704.cs

{  
    public class I { }  
}

{  
    T.I f;  // CS0704 – member lookup on type parameter

    // B.I f;  
}

{  
    public static void Main() {}  
}
```
