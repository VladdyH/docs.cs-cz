# <a name="compiler-error-cs1713"></a>CS1713 chyby kompilátoru
Došlo k neočekávané chybě vytváření názvu metadat pro typ NázevTypu1 – "Důvod"  
  
 Tato chyba je způsobená často vnitřní chyby kompilátoru. Provedením menšími změnami kódu, například zkrátit délku názvu a pak nutnosti rekompilace, může tuto chybu vyřešit.
