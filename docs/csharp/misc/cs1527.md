# <a name="compiler-error-cs1527"></a><span data-ttu-id="f3b12-102">Compiler Error CS1527</span><span class="sxs-lookup"><span data-stu-id="f3b12-102">Compiler Error CS1527</span></span>
<span data-ttu-id="f3b12-103">Elementy definovaný v oboru názvů nelze deklarovat explicitně jako soukromý, chráněný, chráněné interní nebo privátní chráněný.</span><span class="sxs-lookup"><span data-stu-id="f3b12-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="f3b12-104">Typ deklarace v oboru názvů může mít buď [veřejné](../../csharp/language-reference/keywords/public.md) nebo [interní](../../csharp/language-reference/keywords/internal.md) přístup.</span><span class="sxs-lookup"><span data-stu-id="f3b12-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="f3b12-105">Pokud není zadaný žádný usnadnění, **interní** je výchozí.</span><span class="sxs-lookup"><span data-stu-id="f3b12-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="f3b12-106">Následující ukázka generuje CS1527:</span><span class="sxs-lookup"><span data-stu-id="f3b12-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="f3b12-107">Následující příklad generuje CS1527, protože pokud žádný obor názvů explicitně deklarován v programovém kódu, všechny deklarace typu nacházejí implicitně v rámci oboru názvů globální.</span><span class="sxs-lookup"><span data-stu-id="f3b12-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4{}  
private struct S1{}  
```
