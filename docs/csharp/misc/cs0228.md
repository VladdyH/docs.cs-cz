# <a name="compiler-error-cs0228"></a>CS0228 chyby kompilátoru
"typ" neobsahuje definici "člen", nebo není přístupná  
  
 Můžete nahradit verzi systému **System.Hashtable**, **System.String**, nebo **System.Array**, a nahrazení neobsahuje `member`.  
  
 Jinak musíte opravit nebo znovu nainstalovat Visual Studio.
