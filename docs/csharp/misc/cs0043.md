# <a name="compiler-error-cs0043"></a>CS0043 chyby kompilátoru
Soubor PDB 'file' má formát nesprávné nebo jsou zastaralé. Odstraňte ji a znovu vytvořit.  
  
 Odstraňte soubor .pdb kompilace a pak ji znovu zkompilovat.
