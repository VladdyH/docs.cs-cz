 <span data-ttu-id="1197c-106">Následující ukázka generuje CS1620:</span><span class="sxs-lookup"><span data-stu-id="1197c-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;

    }  
}
```
