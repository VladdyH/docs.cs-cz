# <a name="compiler-error-cs1626"></a><span data-ttu-id="eb22d-102">CS1626 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="eb22d-102">Compiler Error CS1626</span></span>
<span data-ttu-id="eb22d-103">Nelze yield hodnotu v těle bloku try s klauzulí catch</span><span class="sxs-lookup"><span data-stu-id="eb22d-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="eb22d-104">Příkaz yield není povolena v bloku try, pokud existuje klauzule catch přidružené bloku try.</span><span class="sxs-lookup"><span data-stu-id="eb22d-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="eb22d-105">Chcete-li se vyhnout této chybě, přesuňte příkaz yield mimo klauzuli catch.</span><span class="sxs-lookup"><span data-stu-id="eb22d-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="eb22d-106">Následující ukázka generuje CS1626:</span><span class="sxs-lookup"><span data-stu-id="eb22d-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
