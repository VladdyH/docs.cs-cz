# <a name="compiler-error-cs0517"></a>CS0517 chyby kompilátoru
'class' nemá žádný základní třídy a nelze volat konstruktor základní třídy  
  
 CS0517 může dojít pouze v případě, že modul common language runtime rozhraní .NET Framework zkompiluje zdrojový kód pro třídu objektu, který je jediná třída, která nemá žádné základní třídy.
