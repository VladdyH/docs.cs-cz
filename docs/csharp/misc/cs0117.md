# <a name="compiler-error-cs0117"></a>CS0117 chyby kompilátoru
"typ" neobsahuje definici "identifikátor"  
  
-   V některých situacích se tato chyba nastane, když se odkazuje na člena, který je pro datový typ neexistuje.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0117:  
  
```csharp  
// CS0117.cs  
public class BaseClass { }  
  
    public class base021 : BaseClass  
    {  
        public void TestInt()  
        {  
            int i = base.someMember; //CS0117  
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
