# <a name="compiler-error-cs0056"></a>CS0056 chyby kompilátoru
Nekonzistentní dostupnost: vrácené hodnoty typu "typ" než – operátor 'operátor'  
  
 Veřejný konstruktor musí vrátit objekt veřejně přístupná. Další informace najdete v tématu [modifikátory přístupu](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
 Následující ukázka generuje CS0056:  
  
```csharp  
// CS0056.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class A  
{  
   public static implicit operator MyClass(A a)   // CS0056  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
