 <span data-ttu-id="e8581-108">CS0012 povede k následujícímu pořadí kompilace:</span><span class="sxs-lookup"><span data-stu-id="e8581-108">The following sequence of compilations will result in CS0012:</span></span>

  
 <span data-ttu-id="e8581-109">Potom:</span><span class="sxs-lookup"><span data-stu-id="e8581-109">Then:</span></span>  
  
```csharp

public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="e8581-110">Potom:</span><span class="sxs-lookup"><span data-stu-id="e8581-110">Then:</span></span>  
  
```csharp

   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
