# <a name="compiler-error-cs0040"></a>CS0040 chyby kompilátoru
Neočekávaná chyba při vytváření souboru informace ladění – "důvod"  
  
 Této chybě může dojít při použití [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) – možnost kompilátoru a určuje, že byl kompilátor nelze zapisovat do souboru pdb. Možná řešení k této chybě zahrnovat opětovné instalace sady Visual Studio, zajistíte, že kompilátor oprávnění k zápisu pro soubor nebo adresář nebo není kompilujete s/Debug.
