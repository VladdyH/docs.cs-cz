 <span data-ttu-id="a4071-108">Následující ukázka generuje CS0241.</span><span class="sxs-lookup"><span data-stu-id="a4071-108">The following sample generates CS0241.</span></span> <span data-ttu-id="a4071-109">Kromě toho ukázku ukazuje, jak k simulaci s přetížení metody s výchozí argumenty.</span><span class="sxs-lookup"><span data-stu-id="a4071-109">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {   
      B x = new B();  
      x.Test();  
   }  
}  
```
