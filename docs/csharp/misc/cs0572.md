# <a name="compiler-error-cs0572"></a><span data-ttu-id="bf86a-102">CS0572 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="bf86a-102">Compiler Error CS0572</span></span>
<span data-ttu-id="bf86a-103">'type': nemůže odkazovat na typ pomocí výrazu. Místo toho zkuste 'path_to_type.</span><span class="sxs-lookup"><span data-stu-id="bf86a-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="bf86a-104">Došlo pokusu o přístup člena třídy prostřednictvím identifikátor, což není povoleno v této situaci.</span><span class="sxs-lookup"><span data-stu-id="bf86a-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="bf86a-105">Následující ukázka generuje CS0572:</span><span class="sxs-lookup"><span data-stu-id="bf86a-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
