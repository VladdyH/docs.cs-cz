## <a name="example"></a><span data-ttu-id="0b3ae-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="0b3ae-108">Example</span></span>  
 <span data-ttu-id="0b3ae-109">Následující kód vygeneruje CS0314:</span><span class="sxs-lookup"><span data-stu-id="0b3ae-109">The following code generates CS0314:</span></span>

// cs0314.cs

{ }

{ }

{ }
