# <a name="compiler-error-cs0004"></a>CS0004 chyby kompilátoru
Upozornění považováno za chybu.  
  
 Kompilovat s `/warnaserror` – možnost kompilátoru, což způsobí, že kompilátoru generovat upozornění jako chyby.
