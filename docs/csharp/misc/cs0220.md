 <span data-ttu-id="f03b4-104">Operace byla zjištěna ve [zaškrtnutí](../../csharp/language-reference/keywords/checked.md), který je výchozí a vedla ke ztrátě dat.</span><span class="sxs-lookup"><span data-stu-id="f03b4-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="f03b4-105">Buď opravte vstupů pro přiřazení nebo použijte [nezaškrtnuté](../../csharp/language-reference/keywords/unchecked.md) chcete tuto chybu vyřešit.</span><span class="sxs-lookup"><span data-stu-id="f03b4-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="f03b4-106">Další informace najdete v tématu [zaškrtnuto a nezaškrtnuto](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="f03b4-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="f03b4-107">Následující ukázka generuje CS0220:</span><span class="sxs-lookup"><span data-stu-id="f03b4-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
