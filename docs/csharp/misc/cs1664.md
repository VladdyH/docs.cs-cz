# <a name="compiler-error-cs1664"></a>CS1664 chyby kompilátoru
Vyrovnávací paměti pevné velikosti délky "délkou" a typu "typ" je příliš velký.  
  
 Maximální velikost vyrovnávací paměti pevné velikosti (jako délka násobí hodnotou velikost element) je 2 ^ 31 = 268435455.
