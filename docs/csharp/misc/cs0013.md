# <a name="compiler-error-cs0013"></a>CS0013 chyby kompilátoru
Neočekávaná chyba při zápisu do souboru 'ModelStore.dll' – metadata, již není logické místo vytvořit další uživatele řetězce.  
  
 Modul common language runtime rozhraní .NET Framework není emitování metadat. Zkontrolujte, zda je cesta správná a zda disk není plný. Pokud potíže potrvají, budete muset opravy a nové instalace sady Visual Studio nebo rozhraní .NET Framework.
