# <a name="compiler-error-cs0013"></a>CS0013 chyby kompilátoru
Neočekávaná chyba při zápisu do souboru 'ModelStore.dll' – metadata, již není logické místo vytvořit další uživatele řetězce.  
  
 Modul common language runtime rozhraní .NET Framework není emitování metadat. Zkontrolujte, zda je cesta správná a zda disk není plný. Pokud potíže potrvají, bude pravděpodobně nutné přeinstalovat nebo opravit [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] nebo rozhraní .NET Framework.
