# <a name="compiler-error-cs0515"></a>Compiler Error CS0515
'function': Modifikátory přístupu nejsou povoleny na statické konstruktory  
  
 Statický konstruktor nemůže mít [– modifikátor přístupu](../../csharp/language-reference/keywords/modifiers.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0515:  
  
```csharp  
// CS0515.cs  
public class Clx  
{  
    public static void Main()  
    {  
    }  
}  
  
public class Clz  
{  
    public static Clz()   // CS0515, remove public keyword  
    {  
    }  
}  
```
