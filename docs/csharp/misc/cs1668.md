# <a name="compiler-warning-level-2-cs1668"></a>Kompilátor CS1668 upozornění (úroveň 2)
Neplatná vyhledávací cesta zadaná v cestě řetězec--chybovou zprávu systému cesta  
  
 Zadaná cesta k [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) do příkazového řádku nebyla platná nebo není platná cesta v proměnné prostředí LIB. Zkontrolujte cestu k ověření, že existuje a je přístupný. Chybová zpráva do jednoduchých uvozovek je chyba vrácená z operačního systému.
