# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="c1597-102">CS0251 kompilátoru upozornění (úroveň 2)</span><span class="sxs-lookup"><span data-stu-id="c1597-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="c1597-103">Indexování pole s indexem záporné (pole indexů vždy spustit na nule)</span><span class="sxs-lookup"><span data-stu-id="c1597-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="c1597-104">Do pole nepoužívejte záporné číslo indexu.</span><span class="sxs-lookup"><span data-stu-id="c1597-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="c1597-105">Následující ukázka generuje CS0251:</span><span class="sxs-lookup"><span data-stu-id="c1597-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
