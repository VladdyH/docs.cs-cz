# <a name="compiler-error-cs1647"></a>Compiler Error CS1647
Výraz je příliš dlouhý nebo komplexní zkompilovat téměř 'kódu.  
  
 Došlo k přetečení zásobníku v kompilátoru zpracování vašeho kódu. Pokud chcete tuto chybu vyřešit, Zjednodušte vašeho kódu. Pokud váš kód je platný, obraťte se na [podporu](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
