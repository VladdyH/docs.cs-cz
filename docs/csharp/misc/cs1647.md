# <a name="compiler-error-cs1647"></a>CS1647 chyby kompilátoru
Výraz je moc dlouhý nebo složitý ke kompilaci téměř "kód"  
  
 V kompilátoru zpracování kódu došlo k přetečení zásobníku. Chcete-li vyřešit tuto chybu, Zjednodušte váš kód. Pokud váš kód je platný, obraťte se na [podporu produktů](https://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
