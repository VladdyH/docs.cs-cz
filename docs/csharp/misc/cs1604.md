# <a name="compiler-error-cs1604"></a>CS1604 chyby kompilátoru
Nelze přiřadit "proměnné", protože je jen pro čtení  
  
 Přiřazení byl proveden proměnnou jen pro čtení. Chcete-li se vyhnout této chybě, nezadávejte přiřadit nebo pokusí změnit na tuto proměnnou v tomto kontextu.
