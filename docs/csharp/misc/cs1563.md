# <a name="compiler-error-cs1563"></a>CS1563 chyby kompilátoru
Výstup 'výstupního souboru, nemá žádné zdrojové soubory  
  
 [/Out](../../csharp/language-reference/compiler-options/out-compiler-option.md) – možnost kompilátoru byla zadána, ale podle žádné soubory zdrojového kódu. Byste neměli používat buď **/out**, nebo musíte zadat soubory zdrojového kódu následující **/out**.
