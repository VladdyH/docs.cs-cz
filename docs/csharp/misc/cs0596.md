# <a name="compiler-error-cs0596"></a>CS0596 chyby kompilátoru
Atribut Guid musí být zadán s atributem ComImport  
  
 **Guid** atributu musí být při použití **ComImport** atribut.  
  
 Následující ukázka generuje CS0596:  
  
```csharp  
// CS0596.cs  
using System.Runtime.InteropServices;  
  
namespace x  
{  
   [ComImport]   // CS0596  
   // try the following line to resolve this CS0596  
   // [ComImport, Guid("00000000-0000-0000-0000-000000000001")]  
   public class a  
   {  
   }  
  
   public class b  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
