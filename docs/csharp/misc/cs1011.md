# <a name="compiler-error-cs1011"></a>CS1011 chyby kompilátoru
Prázdný znak literálu  
  
 A [char](../../csharp/language-reference/keywords/char.md) byl deklarován a inicializována na hodnotu null. Inicializace `char` musíte zadat znak.  
  
 Následující ukázka generuje CS1011:  
  
```csharp  
// CS1011.cs  
class Sample  
{  
   public char CharField = '';   // CS1011  
}  
```
