# <a name="compiler-error-cs0042"></a>CS0042 chyby kompilátoru
Neočekávaná chyba při vytváření ladicí informace o souboru 'file' – "důvod"  
  
 Kompilátor nebylo možné vytvořit ladicí informace; *důvod* obsahuje další informace o chybový stav.
