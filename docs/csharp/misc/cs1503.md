# <a name="compiler-error-cs1503"></a>CS1503 chyby kompilátoru
Argument "číslo" nelze převést z TypeA TypeB  
  
 Typ s jedním argumentem v metodě, se neshoduje typ, který byl předán, když byla vytvořena instance třídy. Tato chyba se obvykle zobrazují společně s CS1502. V tématu [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) informace o tom, jak tuto chybu vyřešit.
