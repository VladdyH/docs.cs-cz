 <span data-ttu-id="6cf62-104">Toto upozornění se může objevit v několik scénářů, ve kterých můžete ověřit kompilátor nikdy odkazuje proměnné:</span><span class="sxs-lookup"><span data-stu-id="6cf62-104">This warning can occur in several scenarios in which the compiler can verify that a variable is never referenced:</span></span>

-   <span data-ttu-id="6cf62-108">Soukromý nebo interní statické pole je přiřazen pouze v inicializátoru konstantní hodnotu.</span><span class="sxs-lookup"><span data-stu-id="6cf62-108">A private or internal static field is assigned a constant value only in the initializer.</span></span> <span data-ttu-id="6cf62-109">Zvažte změnu pole const.</span><span class="sxs-lookup"><span data-stu-id="6cf62-109">Consider changing the field to a const.</span></span>  
  
-   <span data-ttu-id="6cf62-110">Soukromý nebo interní pole je přiřazen konstantní hodnoty a použít pouze ve bloků, které jsou vyloučený #ifdef direktivy.</span><span class="sxs-lookup"><span data-stu-id="6cf62-110">A private or internal field is assigned constant values and only used in blocks that are excluded by #ifdef directives.</span></span> <span data-ttu-id="6cf62-111">Zvažte vložení pole uvnitř bloku #ifdef.</span><span class="sxs-lookup"><span data-stu-id="6cf62-111">Consider putting the field inside the #ifdef block.</span></span>  
  
-   <span data-ttu-id="6cf62-112">Soukromý nebo interní pole je přiřazen konstantní hodnoty na více místech, ale jinak nejsou dostupné.</span><span class="sxs-lookup"><span data-stu-id="6cf62-112">A private or internal field is assigned constant values in multiple locations but not otherwise accessed.</span></span> <span data-ttu-id="6cf62-113">Pokud není nutné pole, zvažte odebrání ho.</span><span class="sxs-lookup"><span data-stu-id="6cf62-113">If you do not need the field, consider removing it.</span></span> <span data-ttu-id="6cf62-114">Jinak použijte ho některé vhodným způsobem.</span><span class="sxs-lookup"><span data-stu-id="6cf62-114">Otherwise, use it in some appropriate way.</span></span>

 <span data-ttu-id="6cf62-116">Následující příklad ukazuje jeden ze způsobů, ve které se budou generovat CS0414:</span><span class="sxs-lookup"><span data-stu-id="6cf62-116">The following sample shows one way in which CS0414 will be generated:</span></span>  
  
```csharp

   private int i = 1;  // CS0414
```  
