# <a name="compiler-error-cs1517"></a>Compiler Error CS1517
Neplatný výraz preprocesoru  
  
 Kompilátor zjistil neplatný výraz preprocesoru.  
  
 Další informace najdete v tématu [direktivy pro preprocesor](../../csharp/language-reference/preprocessor-directives/index.md).  
  
 Následující příklad ukazuje některé platné a neplatné preprocesoru výrazů:  
  
```csharp  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
