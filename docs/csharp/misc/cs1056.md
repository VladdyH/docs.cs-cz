# <a name="compiler-error-cs1056"></a>CS1056 chyby kompilátoru
Neočekávaný znak "znak.  
  
 Kompilátor jazyka C# byl zjištěn neočekávaný znak a nemůže určit token právě zpracovává. Například pokud kompilátor zaznamená znak Euro uprostřed zpracování identifikátor, se nepodařilo klasifikovat identifikátor, protože by byl znak Euro jen platný uvnitř řetězec a kompilátor ví, že nezpracovává řetězec.
