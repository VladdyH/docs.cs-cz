# <a name="compiler-error-cs0538"></a><span data-ttu-id="7c18c-102">CS0538 chyby kompilátoru</span><span class="sxs-lookup"><span data-stu-id="7c18c-102">Compiler Error CS0538</span></span>
<span data-ttu-id="7c18c-103">"název" v explicitní deklarace rozhraní není rozhraní</span><span class="sxs-lookup"><span data-stu-id="7c18c-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="7c18c-104">Byl proveden pokus o explicitně deklarovat [rozhraní](../../csharp/language-reference/keywords/interface.md), ale nebyla zadaná rozhraní.</span><span class="sxs-lookup"><span data-stu-id="7c18c-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="7c18c-105">Následující ukázka generuje CS0538:</span><span class="sxs-lookup"><span data-stu-id="7c18c-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
