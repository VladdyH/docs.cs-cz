# <a name="compiler-error-cs0683"></a>CS0683 chyby kompilátoru
"explicitmethod: explicitní implementace metody nemůže implementovat 'metody', protože se jedná o přistupující objekt  
  
 Následující ukázka generuje CS0683:  
  
```csharp  
// CS0683.cs  
interface IExample  
{  
   int Test { get; }  
}  
  
class CExample : IExample  
{  
   int IExample.get_Test() { return 0; } // CS0683  
   int IExample.Test { get{ return 0; } } // correct  
}  
```
