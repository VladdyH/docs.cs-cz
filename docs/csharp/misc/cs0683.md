# <a name="compiler-error-cs0683"></a>Compiler Error CS0683
"'explicitmethod' explicitní implementace metody"metoda, protože se jedná přístupového objektu  
  
 Následující ukázka generuje CS0683:  
  
```csharp  
// CS0683.cs  
interface IExample  
{  
   int Test { get; }  
}  
  
class CExample : IExample  
{  
   int IExample.get_Test() { return 0; } // CS0683  
   int IExample.Test { get{ return 0; } } // correct  
}  
```
