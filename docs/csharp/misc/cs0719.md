# <a name="compiler-error-cs0719"></a>CS0719 chyby kompilátoru
'type': elementy pole nemůže být statické typu  
  
 Pole typu statické nemá smysl vzhledem k tomu, že jsou elementy pole instance, ale není možné vytvořit instanci statické typy.  
  
 Následující ukázka generuje CS0719:  
  
```  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
