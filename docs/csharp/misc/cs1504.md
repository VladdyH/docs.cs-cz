# <a name="compiler-error-cs1504"></a>CS1504 chyby kompilátoru
Nelze otevřít zdrojový soubor 'file' ('důvodu')  
  
 Zdrojový soubor nelze otevřít nebo číst kompilátoru. Soubor může být uzamčen jiná aplikace, nebo může být jiný problém operačního systému. Součástí zprávy je i operační systém důvod proč kompilátor nelze otevřít nebo čtení tohoto souboru.
