# <a name="compiler-error-cs1639"></a>CS1639 chyby kompilátoru
Spravované třída typu coclass obálku – třída podpis 'podpis' pro rozhraní, rozhraní, není platnou třídu název podpis  
  
 K této chybě dojde, pokud metadata pro rozhraní je neplatný. To by mělo nikdy nastat, pokud rozhraní byla generována pomocí jazyka C# nebo podporované jazykové rozhraní .NET. Zkontrolujte u dodavatele rozhraní nebo ověřte, že bylo úspěšně vygenerováno sestavení rozhraní.
