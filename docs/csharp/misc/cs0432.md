# <a name="compiler-error-cs0432"></a>CS0432 chyby kompilátoru
Alias nebyl nalezen identifikátor  
  
 K této chybě dojde, pokud používáte "::" napravo od identifikátor, který není alias. Chcete-li vyřešit chyby, použijte "." místo.  
  
 Následující příklad generuje CS0432:  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
