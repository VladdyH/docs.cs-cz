 <span data-ttu-id="8f87c-104">K této chybě dojde, pokud používáte "::" napravo od identifikátor, který není alias.</span><span class="sxs-lookup"><span data-stu-id="8f87c-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="8f87c-105">Chcete-li vyřešit chyby, použijte "." místo.</span><span class="sxs-lookup"><span data-stu-id="8f87c-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="8f87c-106">Následující příklad generuje CS0432:</span><span class="sxs-lookup"><span data-stu-id="8f87c-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
