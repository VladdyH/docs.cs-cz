# <a name="compiler-error-cs0145"></a>CS0145 chyby kompilátoru
Const pole vyžaduje hodnotu, musíte zadat  
  
 Je třeba inicializovat [const](../../csharp/language-reference/keywords/const.md) proměnné. Další informace najdete v tématu [konstanty](../../csharp/programming-guide/classes-and-structs/constants.md).  
  
 Následující ukázka generuje CS0145:  
  
```  
// CS0145.cs  
class MyClass  
{  
   const int i;   // CS0145  
   // try the following line instead  
   // const int i = 0;  
  
   public static void Main()  
   {  
   }  
}  
```
