# <a name="compiler-error-cs0140"></a>CS0140 chyby kompilátoru
Popisek "štítek" je duplicitní  
  
 Štítek se stejným názvem zobrazovaly dvakrát. Další informace najdete v tématu [goto](../../csharp/language-reference/keywords/goto.md).  
  
 Následující ukázka generuje CS0140:  
  
```  
// CS0140.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         label1: int i = 0;  
         label1: int j = 0;   // CS0140, comment this line to resolve  
         goto label1;  
      }  
   }  
}  
```
