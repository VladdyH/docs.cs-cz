# <a name="compiler-error-cs2018"></a>CS2018 chyby kompilátoru
Nelze najít soubor cscmsgs.dll zprávy  
  
 Nebyl nalezen soubor .dll, který obsahuje kompilátoru chybové zprávy a upozornění. Tento soubor musí být ve stejném adresáři jako další podpůrné soubory kompilátoru.
