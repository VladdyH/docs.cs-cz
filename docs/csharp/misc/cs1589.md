# <a name="compiler-warning-level-1-cs1589"></a>CS1589 kompilátoru upozornění (úroveň 1)
Nemůže obsahovat fragment XML fragment souboru 'souboru' – důvodu  
  
 Syntaxe (*fragment*) z [ \<zahrnují >](../../csharp/programming-guide/xmldoc/include.md) značky, které odkazovalo na soubor (`file`), se pro zadaný správný ***důvod***.  
  
 Chybná řádku bude uložena v umístění generovaný soubor XML.  
  
 Následující ukázka generuje CS1589:  
  
```csharp  
// CS1589.cs  
// compile with: /W:1 /doc:CS1589_out.xml  
  
/// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/' />   // CS1589  
// try the following line instead  
// /// <include file='CS1589.doc' path='MyDocs/MyMembers[@name="test"]/*' />  
class Test  
{  
   public static void Main()  
   {  
   }  
}  
```
