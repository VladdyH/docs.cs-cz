# <a name="compiler-error-cs0174"></a>CS0174 chyby kompilátoru
Základní třída je vyžadována pro odkaz na "base"  
  
 K této chybě dojde, pouze když modul common language runtime rozhraní .NET Framework zkompiluje zdrojový kód `System.Object` třída, která je jediná třída, která nemá žádné základní třídy.
