# <a name="compiler-warning-level-1-cs3026"></a>CS3026 kompilátoru upozornění (úroveň 1)
Kompatibilní se specifikací CLS pole 'pole' nemůže být volatile  
  
 Volatile proměnné by neměla být kompatibilní se specifikací CLS.  
  
## <a name="example"></a>Příklad  
 Následující příklad vytvoří CS3026.  
  
```csharp  
// CS3026.cs  
[assembly:System.CLSCompliant(true)]  
public class Test  
{  
public volatile int v0 =0;   // CS3026  
      // To resolve remove the CLS-CComplient attribute.  
public static void Main() { }  
}  
```
