# <a name="compiler-error-cs2008"></a>CS2008 chyby kompilátoru
Vstup nebyl určen.  
  
 Byla vyvolána kompilátor a byly zadány – možnosti kompilátoru, ale bylo předáno žádné soubory zdrojového kódu.
