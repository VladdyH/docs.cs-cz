# <a name="compiler-error-cs1513"></a>Compiler Error CS1513
} očekávání  
  
 Kompilátor očekávaný ukončovací složené závorky (`}`), nebyla nalezena.  
  
 Následující ukázka generuje CS1513:  
  
```csharp  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
