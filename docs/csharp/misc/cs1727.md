# <a name="compiler-error-cs1727"></a>CS1727 chyby kompilátoru
Nelze odeslat zprávu o chybách automaticky bez autorizace. Navštivte prosím "k autorizaci odeslání zprávy o chybách.  
  
 Web uvedený text chyby vysvětluje, jak povolit automatické zasílání zpráv o chybách [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)] nástroje příkazového řádku.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS1727.  
  
```csharp  
// CS1727.cs  
// compile with: /errorreport:send  
// CS1727 expected  
class Test  
{  
    static void Main(){}  
}  
```
