# <a name="compiler-error-cs0550"></a>CS0550 chyby kompilátoru
"objekt" přidá přistupující objekt nebyl nalezen ve členovi rozhraní 'vlastnost'  
  
 Implementace v odvozené třídě vlastnost obsahuje přistupující objekt, který nebyl zadán v základní rozhraní.  
  
 Další informace najdete v tématu [pomocí vlastnosti](../../csharp/programming-guide/classes-and-structs/using-properties.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0550.  
  
```  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
