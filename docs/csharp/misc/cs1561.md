# <a name="compiler-error-cs1561"></a>CS1561 chyby kompilátoru
Název výstupního souboru je příliš dlouhý nebo neplatný  
  
 Název výstupního souboru nesmí být delší než 256 znaků a nesmí obsahovat neplatné znaky, jako je například Euro symbol, otazník nebo zpětné lomítko, mimo jiné.
