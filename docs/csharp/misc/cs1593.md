# <a name="compiler-error-cs1593"></a>CS1593 chyby kompilátoru
Delegát "del" nevyžaduje argumenty "číslo"  
  
 Počet argumentů předaných [delegovat](../../csharp/language-reference/keywords/delegate.md) volání Nesouhlasím s počet parametrů v deklaraci delegáta.  
  
 Následující ukázka generuje CS1593:  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
