# <a name="compiler-error-cs1593"></a><span data-ttu-id="75760-102">Compiler Error CS1593</span><span class="sxs-lookup"><span data-stu-id="75760-102">Compiler Error CS1593</span></span>
<span data-ttu-id="75760-103">Delegát "del" nevyžaduje argumenty "číslo"</span><span class="sxs-lookup"><span data-stu-id="75760-103">Delegate 'del' does not take 'number' arguments</span></span>  
  
 <span data-ttu-id="75760-104">Počet argumentů předaných [delegovat](../../csharp/language-reference/keywords/delegate.md) volání Nesouhlasím s počet parametrů v deklaraci delegáta.</span><span class="sxs-lookup"><span data-stu-id="75760-104">The number of arguments passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the number of parameters in the delegate declaration.</span></span>  
  
 <span data-ttu-id="75760-105">Následující ukázka generuje CS1593:</span><span class="sxs-lookup"><span data-stu-id="75760-105">The following sample generates CS1593:</span></span>  
  
```csharp  
// CS1593.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello(8, 9);   // CS1593  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
