# <a name="compiler-warning-level-2-cs0162"></a>CS0162 kompilátoru upozornění (úroveň 2)
Nedosažitelný kód zjistil  
  
 Kompilátor zjištěna kód, který bude nikdy spuštěn.  
  
 Následující ukázka generuje CS0162:  
  
```  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
