 <span data-ttu-id="4e61f-104">Hodnota typu omezení jsou implicitně zapečetěné tak těchto omezení nelze použít jako druhý parametr typu omezení.</span><span class="sxs-lookup"><span data-stu-id="4e61f-104">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="4e61f-105">Je to proto, že typy hodnot nelze přepsat.</span><span class="sxs-lookup"><span data-stu-id="4e61f-105">This is because value types cannot be overridden.</span></span> <span data-ttu-id="4e61f-106">Pokud chcete tuto chybu vyřešit, uveďte omezení typu hodnoty přímo na druhý parametr typu místo to tak nepřímo prostřednictvím první parametr typu.</span><span class="sxs-lookup"><span data-stu-id="4e61f-106">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>
