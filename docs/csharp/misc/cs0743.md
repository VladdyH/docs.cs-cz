<span data-ttu-id="ef21c-103">Očekávalo se kontextové klíčové slovo 'na'</span><span class="sxs-lookup"><span data-stu-id="ef21c-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="ef21c-104">Vzor pro `join` je klauzule `join`... `in`... `on`... `equals`, jak je znázorněno v tomto příkladu:</span><span class="sxs-lookup"><span data-stu-id="ef21c-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="ef21c-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="ef21c-107">Example</span></span>  
 <span data-ttu-id="ef21c-108">Následující kód vygeneruje CS0743:</span><span class="sxs-lookup"><span data-stu-id="ef21c-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
