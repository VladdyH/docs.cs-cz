# <a name="compiler-warning-level-4-cs0422"></a>CS0422 kompilátoru upozornění (úroveň 4)
/ Přírůstkové možnost již není podporována.  
  
 Přírůstkové kompilace (**/incr** nebo **/ přírůstkové**) není podporována v aplikaci Visual C# 2005.
