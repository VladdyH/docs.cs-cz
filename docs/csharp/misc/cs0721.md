# <a name="compiler-error-cs0721"></a>CS0721 chyby kompilátoru
'type': statické typy nelze použít jako parametry  
  
 Statický typ není smysluplný jako parametr. Vzhledem k tomu může být vytvořeny žádné instance statické typů, žádná instance může někdy předat jako parametr.  
  
 Následující ukázka generuje CS0721:  
  
```csharp  
// CS0721.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public void F(SC sc)  // CS0721  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
