# <a name="compiler-error-cs0025"></a>CS0025 chyby kompilátoru
Standardní knihovna souboru 'file' nebyla nalezena.  
  
 Soubor, který je nutný pro kompilátor nebyl nalezen. Zkontrolujte, zda je cesta správná a zda soubor existuje.  
  
 Pokud je soubor systémový soubor Visual Studio, musíte opravit instalaci sady Visual Studio, nebo znovu nainstalovat úplně.
