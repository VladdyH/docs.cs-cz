# <a name="compiler-error-cs0185"></a>CS0185 chyby kompilátoru
'type' není typu odkaz, jak vyžaduje příkaz lock  
  
 [Zámek](../../csharp/language-reference/keywords/lock-statement.md) příkazu můžete vyhodnotit jenom typy odkazů. Další informace najdete v tématu [vlákna synchronizace](../../csharp/programming-guide/concepts/threading/thread-synchronization.md) a [odkazové typy](../../csharp/language-reference/keywords/reference-types.md).  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0185:  
  
```csharp  
// CS0185.cs  
public class MainClass  
{  
    public static void Main ()  
    {  
        lock (1)   // CS0185  
        // try the following lines instead  
        // MainClass x = new MainClass();  
        // lock(x)  
        {  
        }  
    }  
}  
```
