# <a name="compiler-error-cs1023"></a>CS1023 chyby kompilátoru
Vložené příkaz nemůže být deklarace nebo příkaz s popiskem  
  
 Vložené příkaz, jako je například následující příkazy **Pokud** příkaz, může obsahovat deklarace ani příkaz s popiskem.  
  
 Následující ukázka generuje CS1023 dvakrát:  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
