# <a name="compiler-error-cs1583"></a>CS1583 chyby kompilátoru
'file' není platný soubor prostředků Win32  
  
 K této chybě dojde, když zadáte název souboru s [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) možností kompilátoru, který není platný nebo není správně formátovaný zdrojového souboru Win32. V sadě Visual Studio je zadán název souboru v podokně aplikace v Návrháři projektu.
