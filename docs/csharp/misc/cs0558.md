# <a name="compiler-error-cs0558"></a>Compiler Error CS0558
Uživatelem definované operátor 'operátor' musí být deklarován statické a veřejné  
  
 Obě **statické** a **veřejné** přístup [modifikátory](../../csharp/language-reference/keywords/modifiers.md) musí být zadán v uživatelem definované operátory.  
  
 Následující ukázka generuje CS0558:  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
