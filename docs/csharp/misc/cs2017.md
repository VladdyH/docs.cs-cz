# <a name="compiler-error-cs2017"></a>CS2017 chyby kompilátoru
/ Main nelze zadat, pokud vytváření modulu nebo knihovny  
  
 Hlavní vstupní bod nelze zadat, když vytváříte [/target: library](../../csharp/language-reference/compiler-options/target-library-compiler-option.md).  
  
 Následující ukázka generuje CS2017:  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
