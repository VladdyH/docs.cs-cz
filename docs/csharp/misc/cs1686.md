## <a name="example"></a><span data-ttu-id="56f9d-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="56f9d-105">Example</span></span>  
 <span data-ttu-id="56f9d-106">Následující ukázka generuje CS1686.</span><span class="sxs-lookup"><span data-stu-id="56f9d-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs

class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
