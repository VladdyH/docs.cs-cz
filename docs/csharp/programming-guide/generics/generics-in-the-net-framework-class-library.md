# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Obecné typy v knihovně tříd rozhraní .NET Framework (Průvodce programováním v C#)
Knihovna tříd rozhraní .NET Framework verze 2.0 poskytuje nový obor názvů, <xref:System.Collections.Generic>, který zahrnuje několik obecnou kolekci připravené k použití třídy a přidružené rozhraní. Jiných oborech názvů, jako například <xref:System>, také zadejte nové obecná rozhraní, jako třeba <xref:System.IComparable%601>. Tyto třídy a rozhraní jsou efektivnější a bezpečnost typů než negenerická kolekce tříd uvedených v dřívějších verzích rozhraní .NET Framework. Před navrhování a implementace vaše vlastní třídy vlastní kolekce, zvažte, jestli můžete použít nebo odvození třídy z jednoho ze tříd uvedených v knihovně tříd rozhraní .NET Framework.
