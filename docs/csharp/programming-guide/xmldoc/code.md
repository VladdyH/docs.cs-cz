# <a name="ltcodegt-c-programming-guide"></a>&lt;kód&gt; (C# Průvodce programováním)
## <a name="syntax"></a>Syntaxe  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parametry  
 `content`  
 Text, který je označený jako kód.  
  
## <a name="remarks"></a>Poznámky  
 \<Kód > Značka poskytuje způsob, jak vyjádřit jako kód více řádků. Použití [ \<c >](../../../csharp/programming-guide/xmldoc/code-inline.md) indikující, že text v rámci popis by měl být označen jako kód.  
  
 Kompilovat s [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) pro zpracování dokumentačních komentářů do souboru.
