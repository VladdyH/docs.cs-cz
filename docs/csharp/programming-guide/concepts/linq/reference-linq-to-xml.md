Toto téma obsahuje odkazy na LINQ to XML referenční témata.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 Referenční dokumentace pro technologie LINQ to XML třídy, najdete v části <xref:System.Xml.Linq>.  
  
 Referenční dokumentace pro rozšiřující metody, které vám pomůžou ověřit XML stromy proti soubor XSD, najdete v části <xref:System.Xml.Schema.Extensions?displayProperty=nameWithType>.  
  
 Referenční dokumentace pro rozšiřující metody, které umožňují vyhodnotit dotazů XPath na strom XML, najdete v části <xref:System.Xml.XPath.Extensions?displayProperty=nameWithType>.
