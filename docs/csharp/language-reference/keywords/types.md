# <a name="types-c-reference"></a>Typy (Referenční dokumentace jazyka C#)
Zadáním systému jazyka C# obsahuje následujících kategorií:  
  
-   [Typy hodnot](../../../csharp/language-reference/keywords/value-types.md)  
  
-   [Odkazové typy](../../../csharp/language-reference/keywords/reference-types.md)  
  
-   [Typy ukazatelů](../../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)  
  
 Proměnné, které jsou typy hodnot ukládání dat, a ty, které jsou odkazové typy ukládání odkazů na skutečná data. Odkazové typy jsou také označovány jako objekty. Typy ukazatelů lze použít pouze v [unsafe](../../../csharp/language-reference/keywords/unsafe.md) režimu.  
  
 Je možné převést typ hodnoty na typ odkazu a zpět na typ hodnoty, s použitím [zabalení a rozbalení](../../../csharp/programming-guide/types/boxing-and-unboxing.md). S výjimkou zabalené hodnoty typu nelze převést typ odkazu na typ hodnoty.  
  
 Tato část také obsahuje [void](../../../csharp/language-reference/keywords/void.md).  
  
 Typy hodnot se také s možnou hodnotou Null, což znamená, že se můžete uložit do další stavu bez hodnoty. Další informace najdete v tématu [typy s možnou hodnotou Null](../../../csharp/programming-guide/nullable-types/index.md).
