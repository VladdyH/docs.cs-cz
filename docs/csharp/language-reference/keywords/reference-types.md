# <a name="reference-types-c-reference"></a>Typy odkazů (Referenční dokumentace jazyka C#)
V jazyce C# existují dva druhy typů: typy odkazu a typy hodnoty. Proměnné typů odkazu ukládají odkazy na data (objekty), zatímco proměnné typů hodnoty data přímo obsahují příslušná data. V případě typů odkazu mohou dvě proměnné odkazovat na stejný objekt. Operace v rámci jedné proměnné tedy mohou ovlivňovat objekt odkazovaný jinou proměnnou. S typy hodnot, každá proměnná má svou vlastní kopii dat a není možné pro operace na jednu proměnnou ovlivnit dalších (s výjimkou v případě ref a out proměnné parametrů najdete v části [ref](../../../csharp/language-reference/keywords/ref.md) a [vnější parametr Modifikátor](../../../csharp/language-reference/keywords/out-parameter-modifier.md)).  
  
 Pro deklaraci typů odkazu se používají následující klíčová slova:  
  
-   [– Třída](../../../csharp/language-reference/keywords/class.md)  
  
-   [rozhraní](../../../csharp/language-reference/keywords/interface.md)  
  
-   [Delegát](../../../csharp/language-reference/keywords/delegate.md)  
  
 Jazyk C# nabízí také následující vestavěné typy odkazu:  
  
-   [dynamické](../../../csharp/language-reference/keywords/dynamic.md)  
  
-   [objekt](../../../csharp/language-reference/keywords/object.md)  
  
-   [řetězec](../../../csharp/language-reference/keywords/string.md)
