# <a name="operator-keywords-c-reference"></a>Klíčová slova operátorů (Referenční dokumentace jazyka C#)
Používat různé akce, třeba vytvořit objekty, kontrolu za běhu typ objektu, získávání velikosti, typu a provádět jiné akce. Tato část představuje následující klíčová slova:  
  
-   [jako](../../../csharp/language-reference/keywords/as.md) převede objekt na kompatibilní typ.  
  
-   [operátor await](../../../csharp/language-reference/keywords/await.md) asynchronní metody pozastaví až do dokončení očekávané úlohy.  
  
-   [je](../../../csharp/language-reference/keywords/is.md) kontroly za běhu typ objektu.  
  
-   [new](../../../csharp/language-reference/keywords/new.md)  
  
    -   [Operátor new](../../../csharp/language-reference/keywords/new-operator.md) vytvoří objekty.  
  
    -   [New – modifikátor](../../../csharp/language-reference/keywords/new-modifier.md) skrývá zděděný člen.  
  
    -   [New – omezení](../../../csharp/language-reference/keywords/new-constraint.md) kvalifikuje parametr typu.  
  
-   [nameof](nameof.md) získá jednoduchého (nekvalifikovaného) řetězcového názvu proměnné, typ nebo člen.
 
-   [operátor sizeof:](../../../csharp/language-reference/keywords/sizeof.md) získá velikost typu.  
  
-   [typeof](../../../csharp/language-reference/keywords/typeof.md) obdrželi **System.Type** pro typ objektu.  
  
-   [true](../../../csharp/language-reference/keywords/true.md)  
  
    -   [True – operátor](../../../csharp/language-reference/keywords/true-operator.md) vrátí logickou hodnotu true nastavena hodnota true označuje, v opačném případě vrátí hodnotu false.  
  
    -   [true literál](../../../csharp/language-reference/keywords/true-literal.md) představuje logickou hodnotu true.  
  
-   [false](../../../csharp/language-reference/keywords/false.md)  
  
    -   [false – – operátor](../../../csharp/language-reference/keywords/false-operator.md) vrátí logickou hodnotu true označíte, false a v opačném případě vrátí hodnotu false.  
  
    -   [false – literál](../../../csharp/language-reference/keywords/false-literal.md) představuje logickou hodnotu false.  
  
-   [stackalloc](../../../csharp/language-reference/keywords/stackalloc.md) přiděluje blok paměti v zásobníku.  
  
 Následující klíčová slova, které je možné použít jako operátory a jako příkazy, jsou popsané v [příkazy](../../../csharp/language-reference/keywords/statement-keywords.md) části:  
  
-   [checked](../../../csharp/language-reference/keywords/checked.md) kontextu určuje zaškrtnuto.  
  
-   [unchecked](../../../csharp/language-reference/keywords/unchecked.md) určuje nezkontrolovaném kontextu.
