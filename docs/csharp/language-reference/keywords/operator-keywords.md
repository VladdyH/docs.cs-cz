# <a name="operator-keywords-c-reference"></a>Klíčová slova operátorů (Referenční dokumentace jazyka C#)
Umožňuje provádět různé akce, například vytváření objektů a kontrola běhového typu objektu, získávání velikost typu a další akce. Tato část obsahuje následující klíčová slova:  
  
-   [jako](../../../csharp/language-reference/keywords/as.md) převede objekt typu kompatibilní.  
  
-   [await](../../../csharp/language-reference/keywords/await.md) asynchronní metody pozastaví, dokud se nedokončí awaited úloh.  
  
-   [je](../../../csharp/language-reference/keywords/is.md) kontroluje běhového typu objektu.  
  
-   [new](../../../csharp/language-reference/keywords/new.md)  
  
    -   [Operátor new](../../../csharp/language-reference/keywords/new-operator.md) vytvoří objekty.  
  
    -   [New – modifikátor](../../../csharp/language-reference/keywords/new-modifier.md) skryje zděděného členu.  
  
    -   [New – omezení](../../../csharp/language-reference/keywords/new-constraint.md) vyfiltrování parametr typu.  
  
-   [nameof](nameof.md) získá jednoduchým řetězcem (neúplné) název proměnné, typ nebo člen.
 
-   [sizeof –](../../../csharp/language-reference/keywords/sizeof.md) získá velikost typu.  
  
-   [typeof](../../../csharp/language-reference/keywords/typeof.md) obdrželi **System.Type** objektu pro typ.  
  
-   [true](../../../csharp/language-reference/keywords/true.md)  
  
    -   [True – operátor](../../../csharp/language-reference/keywords/true-operator.md) vrátí logickou hodnotu true udávajících hodnotu true, v opačném případě vrátí hodnotu false.  
  
    -   [true – literál](../../../csharp/language-reference/keywords/true-literal.md) představuje logickou hodnotu true.  
  
-   [false](../../../csharp/language-reference/keywords/false.md)  
  
    -   [false – – operátor](../../../csharp/language-reference/keywords/false-operator.md) vrátí hodnotu true logickou hodnotu false označíte a v opačném případě vrátí hodnotu false.  
  
    -   [false – literál](../../../csharp/language-reference/keywords/false-literal.md) představuje logickou hodnotu false.  
  
-   [stackalloc](../../../csharp/language-reference/keywords/stackalloc.md) přiděluje blok paměti v zásobníku.  
  
 Následující klíčová slova, které se dají použít jako operátory a jako příkazy, jsou popsané v [příkazy](../../../csharp/language-reference/keywords/statement-keywords.md) části:  
  
-   [zaškrtnutí](../../../csharp/language-reference/keywords/checked.md) určuje zaškrtnutí kontextu.  
  
-   [nezaškrtnuté](../../../csharp/language-reference/keywords/unchecked.md) určuje nezaškrtnuté kontextu.
