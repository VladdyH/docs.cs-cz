# <a name="method-parameters-c-reference"></a>Parametry metody (Referenční dokumentace jazyka C#)
Pokud je deklarovaný parametr pro metody bez [ref](../../../csharp/language-reference/keywords/ref.md) nebo [out](../../../csharp/language-reference/keywords/out.md), parametr může mít hodnotu s ním spojená. Tuto hodnotu lze změnit v metodě, ale změněné hodnoty nebudou uchována, pokud ovládací prvek předává zpět do volání procedury. S použitím klíčového slova parametru metody, můžete toto chování změnit.  
  
 Tato část popisuje klíčová slova, které můžete použít při deklarování parametry metody:  
  
-   [Parametry](../../../csharp/language-reference/keywords/params.md)  
  
-   [REF](../../../csharp/language-reference/keywords/ref.md)  
  
-   [na více systémů](../../../csharp/language-reference/keywords/out.md)
