# <a name="using-c-reference"></a>using (Referenční dokumentace jazyka C#)
`using` – Klíčové slovo má dvě hlavní využití:  
  
-   Jako direktivu, pokud se používá k vytvoření zástupce pro obor názvů nebo import typy definované v jiných oborech názvů. V tématu [using – direktiva](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Jako příkaz, pokud definuje obor na konci, které bude objekt zlikvidován. V tématu [pomocí příkazu](../../../csharp/language-reference/keywords/using-statement.md).  
  
Kromě toho [pomocí statické](using-static.md) umožňuje definovat typ jejichž statické členy dostanete bez určení názvu typu.
