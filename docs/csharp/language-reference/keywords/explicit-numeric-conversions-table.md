## <a name="remarks"></a><span data-ttu-id="849f1-132">Poznámky</span><span class="sxs-lookup"><span data-stu-id="849f1-132">Remarks</span></span>  
  
-   <span data-ttu-id="849f1-133">Explicitní číselný převod může způsobit ztrátu přesnosti nebo výsledek v aktivační výjimky.</span><span class="sxs-lookup"><span data-stu-id="849f1-133">The explicit numeric conversion may cause loss of precision or result in throwing exceptions.</span></span>  
  
-   <span data-ttu-id="849f1-134">Při převodu `decimal` integrální typ, tato hodnota se zaokrouhlí směrem k nule na nejbližší hodnotu integrálu.</span><span class="sxs-lookup"><span data-stu-id="849f1-134">When you convert a `decimal` value to an integral type, this value is rounded towards zero to the nearest integral value.</span></span> <span data-ttu-id="849f1-135">Pokud výsledná integrální hodnota je mimo rozsah cílového typu, <xref:System.OverflowException> je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="849f1-135">If the resulting integral value is outside the range of the destination type, an <xref:System.OverflowException> is thrown.</span></span>  
  
-   <span data-ttu-id="849f1-136">Při převodu z `double` nebo `float` integrální typ, hodnota je oříznuta.</span><span class="sxs-lookup"><span data-stu-id="849f1-136">When you convert from a `double` or `float` value to an integral type, the value is truncated.</span></span> <span data-ttu-id="849f1-137">Pokud výsledná integrální hodnota je mimo rozsah hodnoty cílové, výsledek bude záviset na přetečení kontrola kontextu.</span><span class="sxs-lookup"><span data-stu-id="849f1-137">If the resulting integral value is outside the range of the destination value, the result depends on the overflow checking context.</span></span> <span data-ttu-id="849f1-138">V kontextu zaškrtnuté `OverflowException` je vyvolána v kontextu není zaškrtnuto, výsledkem je, hodnotu nespecifikované cílového typu.</span><span class="sxs-lookup"><span data-stu-id="849f1-138">In a checked context, an `OverflowException` is thrown, while in an unchecked context, the result is an unspecified value of the destination type.</span></span>  
  
-   <span data-ttu-id="849f1-139">Při převodu `double` k `float`, `double` hodnota se zaokrouhlí na nejbližší `float` hodnotu.</span><span class="sxs-lookup"><span data-stu-id="849f1-139">When you convert `double` to `float`, the `double` value is rounded to the nearest `float` value.</span></span> <span data-ttu-id="849f1-140">Pokud `double` hodnota je příliš malá nebo příliš velká na typ cílového, výsledkem bude nula nebo nekonečna.</span><span class="sxs-lookup"><span data-stu-id="849f1-140">If the `double` value is too small or too large to fit into the destination type, the result will be zero or infinity.</span></span>  
  
-   <span data-ttu-id="849f1-141">Při převodu `float` nebo `double` k `decimal`, zdrojové hodnoty jsou převedeny na `decimal` reprezentace a zaokrouhlí na nejbližší číslo po 28th desetinné místo v případě potřeby.</span><span class="sxs-lookup"><span data-stu-id="849f1-141">When you convert `float` or `double` to `decimal`, the source value is converted to `decimal` representation and rounded to the nearest number after the 28th decimal place if required.</span></span> <span data-ttu-id="849f1-142">V závislosti na hodnotě zdrojové hodnoty může dojít mezi následující výsledky:</span><span class="sxs-lookup"><span data-stu-id="849f1-142">Depending on the value of the source value, one of the following results may occur:</span></span>  
  
    -   <span data-ttu-id="849f1-143">Pokud je příliš malá, aby být reprezentován jako zdrojové hodnoty `decimal`, výsledek klesne na nulu.</span><span class="sxs-lookup"><span data-stu-id="849f1-143">If the source value is too small to be represented as a `decimal`, the result becomes zero.</span></span>  
  
    -   <span data-ttu-id="849f1-144">Pokud je zdroj hodnota typu NaN (není číslo), infinity, nebo příliš velký a nelze je jako `decimal`, `OverflowException` je vyvolána výjimka.</span><span class="sxs-lookup"><span data-stu-id="849f1-144">If the source value is NaN (not a number), infinity, or too large to be represented as a `decimal`, an `OverflowException` is thrown.</span></span>  
  
-   <span data-ttu-id="849f1-145">Při převodu `decimal` k `float` nebo `double`, `decimal` hodnota se zaokrouhlí na nejbližší `double` nebo `float` hodnotu.</span><span class="sxs-lookup"><span data-stu-id="849f1-145">When you convert `decimal` to `float` or `double`, the `decimal` value is rounded to the nearest `double` or `float` value.</span></span>
