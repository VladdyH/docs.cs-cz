# <a name="new-c-reference"></a>new (Referenční dokumentace jazyka C#)
V jazyce C# `new` – klíčové slovo lze použít jako operátor, modifikátor nebo omezení.  
  
 [new – operátor](../../../csharp/language-reference/keywords/new-operator.md)  
 Použít k vytvoření objektů a vyvolávání konstruktorů.  
  
 [new – modifikátor](../../../csharp/language-reference/keywords/new-modifier.md)  
 Používají ke skrytí zděděného členu z členu základní třídy.  
  
 [new – omezení](../../../csharp/language-reference/keywords/new-constraint.md)  
 Slouží k omezení typů, které mohou sloužit jako argumenty pro parametr typu v obecné deklaraci.
