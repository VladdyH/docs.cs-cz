# <a name="compiler-error-cs0001"></a>Chyba kompilátoru CS0001
Vnitřní chyba kompilátoru  
  
 Pokuste se zjistit, zda kompilátor selhává kvůli jeho nemůže analyzovat neočekávanou syntaxi. Pokud tato chyba se zobrazí opakovaně, kontaktujte prosím společnosti Microsoft.
