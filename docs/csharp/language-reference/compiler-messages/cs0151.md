# <a name="compiler-error-cs0151"></a><span data-ttu-id="818af-102">Chyba kompilátoru CS0151</span><span class="sxs-lookup"><span data-stu-id="818af-102">Compiler Error CS0151</span></span>
<span data-ttu-id="818af-103">Hodnotu typu integrální očekávání</span><span class="sxs-lookup"><span data-stu-id="818af-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="818af-104">Proměnná byla použita v situaci, kde celočíselný datový typ nebyla nutná.</span><span class="sxs-lookup"><span data-stu-id="818af-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="818af-105">Další informace najdete v tématu [typy](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="818af-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="818af-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="818af-106">Example</span></span>  
 <span data-ttu-id="818af-107">Této chybě může dojít, když není žádný převod nebo pokud je k dispozici implicitní převody mít za následek dvojznačný situaci.</span><span class="sxs-lookup"><span data-stu-id="818af-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="818af-108">Následující ukázka generuje CS0151.</span><span class="sxs-lookup"><span data-stu-id="818af-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```
