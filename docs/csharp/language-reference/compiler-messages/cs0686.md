## <a name="example"></a><span data-ttu-id="a8309-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="a8309-110">Example</span></span>  
 <span data-ttu-id="a8309-111">Následující ukázka generuje CS0686:</span><span class="sxs-lookup"><span data-stu-id="a8309-111">The following sample generates CS0686:</span></span>  
  
```csharp  
// CS0686.cs  
interface I  
{  
    int get_P();  
}  
  
class C : I  
{  
    public int P  
    {  
        get { return 1; }  // CS0686  
    }  
}  
// But the following is valid:  
class D : I  
{  
    int I.get_P() { return 1; }  
    public static void Main() {}  
}  
```
