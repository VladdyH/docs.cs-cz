## <a name="example"></a><span data-ttu-id="99428-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="99428-107">Example</span></span>  
 <span data-ttu-id="99428-108">Následující příklad ukazuje jeden ze způsobů, ve kterém je generována CS1721 a potom zobrazí dva možné způsoby, aby nedošlo k chybě.</span><span class="sxs-lookup"><span data-stu-id="99428-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
