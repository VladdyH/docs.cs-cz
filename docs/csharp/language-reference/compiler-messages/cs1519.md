# <a name="compiler-error-cs1519"></a><span data-ttu-id="367da-102">Chyba kompilátoru CS1519</span><span class="sxs-lookup"><span data-stu-id="367da-102">Compiler Error CS1519</span></span>
<span data-ttu-id="367da-103">Neplatný token "token" v třída, struktura nebo deklarace členů rozhraní</span><span class="sxs-lookup"><span data-stu-id="367da-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="367da-104">Tato chyba je generována pokaždé, když token se vyskytuje v umístění, kde nepatří.</span><span class="sxs-lookup"><span data-stu-id="367da-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="367da-105">A *tokenu* je klíčové slovo; identifikátor (název – třída, struktura, metodu a tak dále); řetězec, znak nebo číselný literál, jako je například 108, "Hello", nebo "A"; nebo operátor nebo oddělovač, jako `==` nebo `;`.</span><span class="sxs-lookup"><span data-stu-id="367da-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="367da-106">Všechny [třída](../../../csharp/language-reference/keywords/class.md), struktura nebo deklarace členů rozhraní, která obsahuje neplatné modifikátory před typ způsobí vygenerování této chyby.</span><span class="sxs-lookup"><span data-stu-id="367da-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="367da-107">Chcete-li opravit chyby, odeberte neplatné modifikátory.</span><span class="sxs-lookup"><span data-stu-id="367da-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="367da-108">Následující ukázka generuje CS1519 pět místech, protože tokeny jsou umístěny v umístění, kde nejsou platné:</span><span class="sxs-lookup"><span data-stu-id="367da-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
