# <a name="compiler-error-cs1029"></a>Chyba kompilátoru CS1029
\#Chyba: 'text'  
  
 Zobrazí text chyby definované [#error](../../../csharp/language-reference/preprocessor-directives/preprocessor-error.md) směrnice.  
  
 Následující příklad ukazuje, jak vytvořit uživatelem definované chybové:  
  
```csharp  
// CS1029.cs  
class Sample  
{  
   static void Main()  
   {  
      #error Let's give an error here   // CS1029  
   }  
}  
```
Kompilace vytvoří následující výstup:

```console
example.cs(9,8): error CS1029: #error: 'DEBUG is defined'
```
