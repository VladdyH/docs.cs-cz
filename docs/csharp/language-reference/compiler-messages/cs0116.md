## <a name="example"></a><span data-ttu-id="5ebca-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="5ebca-107">Example</span></span>  
 <span data-ttu-id="5ebca-108">Následující ukázka generuje CS0116:</span><span class="sxs-lookup"><span data-stu-id="5ebca-108">The following sample generates CS0116:</span></span>  
  
```csharp  
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
