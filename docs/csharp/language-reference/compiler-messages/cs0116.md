# <a name="compiler-error-cs0116"></a><span data-ttu-id="1ffaa-102">Chyba kompilátoru CS0116</span><span class="sxs-lookup"><span data-stu-id="1ffaa-102">Compiler Error CS0116</span></span>

 <span data-ttu-id="1ffaa-104">Uvnitř [obor názvů](../../../csharp/language-reference/keywords/namespace.md), kompilátor přijímá pouze třídy, struktury, sjednocení, výčty, rozhraní a delegátů.</span><span class="sxs-lookup"><span data-stu-id="1ffaa-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="1ffaa-105">Tato chyba je často generována z jazyka C/C++ na pozadí vývojáři, kteří zapomenout, že v jazyce C#, metody a proměnné musí být deklarovány a definovány v rámci třídy nebo struktury.</span><span class="sxs-lookup"><span data-stu-id="1ffaa-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="1ffaa-106">Další informace najdete v tématu [obecná struktura programu v jazyce C#](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="1ffaa-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1ffaa-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="1ffaa-107">Example</span></span>

 <span data-ttu-id="1ffaa-108">Následující ukázka generuje CS0116:</span><span class="sxs-lookup"><span data-stu-id="1ffaa-108">The following sample generates CS0116:</span></span>  

```csharp
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
