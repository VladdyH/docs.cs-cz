## <a name="example"></a><span data-ttu-id="64872-106">Příklad</span><span class="sxs-lookup"><span data-stu-id="64872-106">Example</span></span>  
 <span data-ttu-id="64872-107">Následující programu C++ používá atribut RequiredAttributeAttribute, která nemusí být využívány službou další jazyky.</span><span class="sxs-lookup"><span data-stu-id="64872-107">The following C++ program uses an attribute, RequiredAttributeAttribute, which may not be consumed by other languages.</span></span>  
  
```cpp
// CPP0570.cpp  
// compile with: /clr /LD  
  
using namespace System;  
using namespace System::Runtime::CompilerServices;  
  
namespace CS0570_Server {  
   [RequiredAttributeAttribute(Int32::typeid)]    
   public ref struct Scenario1 {  
      int intVar;  
   };  
  
   public ref struct CS0570Class {  
      Scenario1 ^ sc1_field;  
  
      property virtual Scenario1 ^ sc1_prop {  
         Scenario1 ^ get() { return sc1_field; }  
      }  
  
      Scenario1 ^ sc1_method() { return sc1_field; }  
   };  
};  
```
