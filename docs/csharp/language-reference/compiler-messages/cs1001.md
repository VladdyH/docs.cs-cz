# <a name="compiler-error-cs1001"></a><span data-ttu-id="d50a9-102">Chyba kompilátoru CS1001</span><span class="sxs-lookup"><span data-stu-id="d50a9-102">Compiler Error CS1001</span></span>
<span data-ttu-id="d50a9-103">Byl očekáván identifikátor.</span><span class="sxs-lookup"><span data-stu-id="d50a9-103">Identifier expected</span></span>  
  
 <span data-ttu-id="d50a9-104">Není poskytnut identifikátor.</span><span class="sxs-lookup"><span data-stu-id="d50a9-104">You did not supply an identifier.</span></span> <span data-ttu-id="d50a9-105">Identifikátor je název – třída, struktura, obor názvů, metoda, proměnné a tak dále, který zadáte.</span><span class="sxs-lookup"><span data-stu-id="d50a9-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="d50a9-106">Následující příklad deklaruje jednoduchá, ale neposkytuje název třídy:</span><span class="sxs-lookup"><span data-stu-id="d50a9-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="d50a9-107">Následující ukázka generuje CS1001, protože při deklarování výčtu, je nutné zadat členy:</span><span class="sxs-lookup"><span data-stu-id="d50a9-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="d50a9-108">Názvy parametrů jsou požadovány, i když kompilátor nepoužívá, například v definici rozhraní.</span><span class="sxs-lookup"><span data-stu-id="d50a9-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="d50a9-109">Tyto parametry jsou povinné, programátory, kteří spotřebovávají rozhraní věděli o něco co znamená parametry.</span><span class="sxs-lookup"><span data-stu-id="d50a9-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
