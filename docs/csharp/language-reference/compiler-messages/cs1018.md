# <a name="compiler-error-cs1018"></a>Chyba kompilátoru CS1018
– Klíčové slovo 'this' nebo 'základní očekávání  
  
 Kompilátor byla zjištěna deklarace neúplné konstruktor.  
  
## <a name="example"></a>Příklad  
 Následující příklad generuje CS1018 a navrhne několik způsobů, jak vyřešte chybu:  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's default constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
