# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="91e06-102">Upozornění kompilátoru (úroveň 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="91e06-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="91e06-103">Přístup k člena na "člen" může způsobit výjimku modulu runtime, protože se jedná o pole zařazování referenční třídy</span><span class="sxs-lookup"><span data-stu-id="91e06-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="91e06-104">Toto upozornění se zobrazí při pokusu o volání metoda, vlastnost nebo indexer člena třídy, která je odvozena z <xref:System.MarshalByRefObject>, a člen je typ hodnoty.</span><span class="sxs-lookup"><span data-stu-id="91e06-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="91e06-105">Objekty, které dědí od `MarshalByRefObject` jsou obvykle mají být zařazena pomocí odkazu v rámci domény aplikace.</span><span class="sxs-lookup"><span data-stu-id="91e06-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="91e06-106">Pokud žádný kód někdy pokusí přímý přístup k typu hodnoty členem takového objektu napříč domény aplikace, dojde k výjimce modulu runtime.</span><span class="sxs-lookup"><span data-stu-id="91e06-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="91e06-107">K vyřešení upozornění, nejprve zkopírujte člena do místní proměnné a volat metodu na tuto proměnnou.</span><span class="sxs-lookup"><span data-stu-id="91e06-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="91e06-108">Následující ukázka generuje CS1690:</span><span class="sxs-lookup"><span data-stu-id="91e06-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
