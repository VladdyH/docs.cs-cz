 <span data-ttu-id="31d05-104">Všechna pole v `struct` musí být přiděleny konstruktorem, než konstruktor může volat metodu v `struct`.</span><span class="sxs-lookup"><span data-stu-id="31d05-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="31d05-105">Pokud se zobrazí tato chyba při pokusu o inicializaci vlastnosti v konstruktoru struktury, chcete-li změnit parametr konstruktoru určete pole základní místo samotné vlastnosti je řešení.</span><span class="sxs-lookup"><span data-stu-id="31d05-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="31d05-106">Automaticky implementované vlastnosti musí být použity ve strukturách, protože nemají žádné pole základní a proto jej nelze inicializovat jakýmkoli způsobem z konstruktoru.</span><span class="sxs-lookup"><span data-stu-id="31d05-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>

## <a name="example"></a><span data-ttu-id="31d05-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="31d05-108">Example</span></span>  
 <span data-ttu-id="31d05-109">Následující ukázka generuje CS0188:</span><span class="sxs-lookup"><span data-stu-id="31d05-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
