# <a name="compiler-warning-level-2-cs0618"></a><span data-ttu-id="c449f-102">Upozornění kompilátoru (úroveň 2) CS0618</span><span class="sxs-lookup"><span data-stu-id="c449f-102">Compiler Warning (level 2) CS0618</span></span>
<span data-ttu-id="c449f-103">"člen" je zastaralá: 'text'</span><span class="sxs-lookup"><span data-stu-id="c449f-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="c449f-104">Byl označené jako člena třídy `Obsolete` atribut, tak, že bude vydáno upozornění, když se člen třídy.</span><span class="sxs-lookup"><span data-stu-id="c449f-104">A class member was marked with the `Obsolete` attribute, such that a warning will be issued when the class member is referenced.</span></span> <span data-ttu-id="c449f-105">Další informace najdete v tématu [běžné atributy](../../programming-guide/concepts/attributes/common-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="c449f-105">For more information, see [Common Attributes](../../programming-guide/concepts/attributes/common-attributes.md).</span></span>  
  
 <span data-ttu-id="c449f-106">Následující ukázka generuje CS0618:</span><span class="sxs-lookup"><span data-stu-id="c449f-106">The following sample generates CS0618:</span></span>  
  
```csharp  
// CS0618.cs  
// compile with: /W:2  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", false)]   // warn if referenced  
   public static void m2()  
   {  
   }  
  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m2();  // CS0618  
   }  
}  
```
