 <span data-ttu-id="22d6c-104">Obecný parametr méně přístupný než samotná obecná třída nemusí vynutit omezení.</span><span class="sxs-lookup"><span data-stu-id="22d6c-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="22d6c-105">V následujícím příkladu, zatímco obecné třída C\<T > je deklarovaná veřejné, omezení se pokusí vynutit T implementovat vnitřní rozhraní.</span><span class="sxs-lookup"><span data-stu-id="22d6c-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="22d6c-106">I když to bylo povoleno, jen klienty s přístupem k interní, bude možné vytvořit parametr pro třídu, takže ve skutečnosti třída může používat pouze klienti s přístupem k interní.</span><span class="sxs-lookup"><span data-stu-id="22d6c-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="22d6c-107">Chcete-li odstranit této chyby, zkontrolujte, zda že úroveň přístupu tohoto obecná třída není méně omezující než všech tříd nebo rozhraní, které se zobrazují v hranice.</span><span class="sxs-lookup"><span data-stu-id="22d6c-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>
