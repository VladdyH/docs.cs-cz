 <span data-ttu-id="b89b1-105">Proměnná byla deklarována se stejným názvem jako proměnné v základní třídě.</span><span class="sxs-lookup"><span data-stu-id="b89b1-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="b89b1-106">Ale [nové](../../../csharp/language-reference/keywords/new.md) – klíčové slovo nebyl použit.</span><span class="sxs-lookup"><span data-stu-id="b89b1-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="b89b1-107">Toto upozornění informuje, že je možné použít **nové**; proměnné je deklarován jako **nové** bylo v deklaraci použito.</span><span class="sxs-lookup"><span data-stu-id="b89b1-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="b89b1-108">Následující ukázka generuje CS0108:</span><span class="sxs-lookup"><span data-stu-id="b89b1-108">The following sample generates CS0108:</span></span>  
  
```csharp  
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
