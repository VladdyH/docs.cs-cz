# <a name="compiler-error-cs0563"></a>Chyba kompilátoru CS0563
Jeden z parametrů binárního operátoru musí obsahovat typ  
  
 Deklarace metody pro [přetížení operátoru](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md) musí splňovat určitá pravidla.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0563:  
  
```csharp  
// CS0563.cs  
public class iii  
{  
    public static implicit operator int(iii x)  
    {  
        return 0;  
    }  
    public static implicit operator iii(int x)  
    {  
        return null;  
    }  
    public static int operator +(int aa, int bb)   // CS0563   
    // Use the following line instead:  
    // public static int operator +(int aa, iii bb)      
    {  
        return 0;  
    }  
    public static void Main()  
    {  
    }  
}  
```
