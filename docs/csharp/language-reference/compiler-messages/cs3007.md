# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="e343c-102">Upozornění kompilátoru (úroveň 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="e343c-102">Compiler Warning (level 1) CS3007</span></span>
<span data-ttu-id="e343c-103">Přetížená metoda "method" lišící se pouze typy nepojmenovaného pole není kompatibilní se specifikací CLS</span><span class="sxs-lookup"><span data-stu-id="e343c-103">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="e343c-104">K této chybě dochází, pokud existuje přetížená metoda, která přebírá nepravidelné pole, a jediným rozdílem mezi podpisy metod je typ prvku tohoto pole.</span><span class="sxs-lookup"><span data-stu-id="e343c-104">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="e343c-105">Chcete-li se vyhnout této chybě, zvažte použití obdélníková pole spíše než Vícenásobná pole; použijte další parametr k rozlišení volání funkce; Přejmenujte jednu nebo více metod přetížené; nebo, pokud není nutné souladu se specifikací CLS, odeberte <xref:System.CLSCompliantAttribute> atribut.</span><span class="sxs-lookup"><span data-stu-id="e343c-105">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="e343c-106">Další informace o souladu se specifikací CLS najdete v tématu [jazyková nezávislost a jazykově nezávislé komponenty](../../../standard/language-independence-and-language-independent-components.md).</span><span class="sxs-lookup"><span data-stu-id="e343c-106">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e343c-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="e343c-107">Example</span></span>  
 <span data-ttu-id="e343c-108">Následující příklad generuje CS3007:</span><span class="sxs-lookup"><span data-stu-id="e343c-108">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or   
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
