# <a name="compiler-error-cs0507"></a><span data-ttu-id="ed45f-102">Chyba kompilátoru CS0507</span><span class="sxs-lookup"><span data-stu-id="ed45f-102">Compiler Error CS0507</span></span>
<span data-ttu-id="ed45f-103">'function1': při přepisování "přístup" zděděného člena "funkce2' nelze měnit modifikátory přístupu</span><span class="sxs-lookup"><span data-stu-id="ed45f-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="ed45f-104">Došlo k pokusu o změnu specifikace přístupu v přepsání metody.</span><span class="sxs-lookup"><span data-stu-id="ed45f-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ed45f-105">Příklad</span><span class="sxs-lookup"><span data-stu-id="ed45f-105">Example</span></span>  
 <span data-ttu-id="ed45f-106">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="ed45f-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ed45f-107">Příklad</span><span class="sxs-lookup"><span data-stu-id="ed45f-107">Example</span></span>  
 <span data-ttu-id="ed45f-108">CS0507 může také nastat, pokud třída pokusí přepsat metodu označen jako `protected internal` definovanou v odkazovaných metadatech.</span><span class="sxs-lookup"><span data-stu-id="ed45f-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="ed45f-109">V takovém případě by měl být metodu přepsání označen jako `protected`.</span><span class="sxs-lookup"><span data-stu-id="ed45f-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ed45f-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="ed45f-110">Example</span></span>  
 <span data-ttu-id="ed45f-111">Následující ukázka generuje CS0507.</span><span class="sxs-lookup"><span data-stu-id="ed45f-111">The following sample generates CS0507.</span></span>
