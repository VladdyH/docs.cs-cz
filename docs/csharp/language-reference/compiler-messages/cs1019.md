# <a name="compiler-error-cs1019"></a><span data-ttu-id="3b21e-102">Chyba kompilátoru CS1019</span><span class="sxs-lookup"><span data-stu-id="3b21e-102">Compiler Error CS1019</span></span>
<span data-ttu-id="3b21e-103">Očekával se přetěžovatelný unární operátor.</span><span class="sxs-lookup"><span data-stu-id="3b21e-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="3b21e-104">Něco, co vypadá jako Přetěžovaný unární operátor je deklarovaná, ale operátor, který nebyl nalezen nebo je v nesprávném umístění v podpisu.</span><span class="sxs-lookup"><span data-stu-id="3b21e-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="3b21e-105">A *unární operátor* je operátor, který funguje na jediného operandu.</span><span class="sxs-lookup"><span data-stu-id="3b21e-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="3b21e-106">Například `++` je unárního operátoru.</span><span class="sxs-lookup"><span data-stu-id="3b21e-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="3b21e-107">Některé unárními operátory můžete přetížit pomocí `operator` – klíčové slovo a zadáte jeden parametr typu, který pracuje operátor.</span><span class="sxs-lookup"><span data-stu-id="3b21e-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="3b21e-108">Například, pokud budete chtít přetížit operátor `++` pro uživatelsky definované třídy `Temperature` tak, aby můžete napsat `Temperature++`, můžete ji definovat tímto způsobem:</span><span class="sxs-lookup"><span data-stu-id="3b21e-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="3b21e-109">Když se zobrazí tato chyba, je něco, co vypadá jako Přetěžovaný unární operátor, s tím rozdílem, že operátor chybí nebo je v nesprávném umístění v signatuře deklarován.</span><span class="sxs-lookup"><span data-stu-id="3b21e-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="3b21e-110">Pokud odeberete `++` z podpisu v předchozím příkladu bude generovat CS1019.</span><span class="sxs-lookup"><span data-stu-id="3b21e-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="3b21e-111">Následující kód vygeneruje CS1019:</span><span class="sxs-lookup"><span data-stu-id="3b21e-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
