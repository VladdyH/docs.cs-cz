<span data-ttu-id="bac87-103">Chybí částečný modifikátor deklarace typu "typ"; existuje jiné částečné deklarace typu</span><span class="sxs-lookup"><span data-stu-id="bac87-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="bac87-104">Tato chyba znamená, že deklarujete více tříd, které mají stejný název.</span><span class="sxs-lookup"><span data-stu-id="bac87-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="bac87-105">Kromě toho v obsahuje minimálně jeden, ale ne všechny deklarací `partial` modifikátor.</span><span class="sxs-lookup"><span data-stu-id="bac87-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="bac87-106">Pokud chcete definovat třídu v několik částí, musí deklarovat každou část s použitím klíčového slova `partial`.</span><span class="sxs-lookup"><span data-stu-id="bac87-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="bac87-107">K této chybě dojde, pokud deklarování třídy a omylem poskytněte stejný název jako částečné třídu, která je deklarovaná jinde v stejného oboru názvů.</span><span class="sxs-lookup"><span data-stu-id="bac87-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="bac87-108">Následující ukázka generuje CS0260:</span><span class="sxs-lookup"><span data-stu-id="bac87-108">The following sample generates CS0260:</span></span>  
  
```csharp  
// CS0260.cs  
// You must mark both parts of the definition of class C   
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C    
{  
}  
  
partial class C  
{  
}  
```
