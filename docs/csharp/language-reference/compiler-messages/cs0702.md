# <a name="compiler-error-cs0702"></a>Chyba kompilátoru CS0702
Omezení nemůže být speciální třída "identifikátor"  
  
 Následující typy nesmí být použity jako omezení: `System.Object,``System.Array`, `System.Delegate`, `System.Enum`, nebo `System.ValueType`.  
  
## <a name="example"></a>Příklad  
 Následující ukázka generuje CS0702:  
  
```csharp  
// CS0702.cs  
class C<T> where T : System.Array  // CS0702  
{  
}  
```
