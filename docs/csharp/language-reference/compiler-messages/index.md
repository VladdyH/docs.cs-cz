# <a name="c-compiler-errors"></a>Chyby kompilátoru jazyka C#

Některé chyby kompilátoru jazyka C# mají odpovídající témata s vysvětlením, proč je generována chyba a, v některých případech k vyřešení chyby. Použijte jednu z následujících kroků, zda je k dispozici pro konkrétní chybovou zprávu nápovědy.  
  
- Najít číslo chyby (například CS0029) v [okno výstup](/visualstudio/ide/reference/output-window)a pak ho vyhledat na Microsoft Docs.  
  
- Zvolte číslo chyby (například CS0029) [okno výstup](/visualstudio/ide/reference/output-window)a pak stiskněte klávesu F1.  
  
- V indexu, zadejte číslo chyby v **vyhledejte** pole.  
  
 Pokud žádný z těchto kroků vede na informace o chyby, přejděte na konec této stránky a odeslat zpětnou vazbu, která obsahuje číslo nebo text chyby.  
  
 Informace o tom, jak nakonfigurovat chyby a upozornění možnosti v jazyce C# najdete v tématu [stránku sestavení, Návrhář projektu (C#)](/visualstudio/ide/reference/build-page-project-designer-csharp).
