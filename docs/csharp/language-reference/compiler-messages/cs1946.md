# <a name="compiler-error-cs1946"></a><span data-ttu-id="34159-102">Chyba kompilátoru CS1946</span><span class="sxs-lookup"><span data-stu-id="34159-102">Compiler Error CS1946</span></span>

## <a name="example"></a><span data-ttu-id="34159-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="34159-108">Example</span></span>

    }
