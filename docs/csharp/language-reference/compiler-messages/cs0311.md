## <a name="example"></a><span data-ttu-id="dcf76-110">Příklad</span><span class="sxs-lookup"><span data-stu-id="dcf76-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```
