## <a name="example"></a><span data-ttu-id="f5d56-112">Příklad</span><span class="sxs-lookup"><span data-stu-id="f5d56-112">Example</span></span>  
 <span data-ttu-id="f5d56-113">Následující kód generuje CS0826, protože pole, které jsou elementy nejsou všechny stejného typu a logiku odvození typu kompilátoru nenajde jediný optimální typ:</span><span class="sxs-lookup"><span data-stu-id="f5d56-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>  
  
```csharp  
// cs0826.cs  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
