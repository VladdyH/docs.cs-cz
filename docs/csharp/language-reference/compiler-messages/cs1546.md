## <a name="example"></a><span data-ttu-id="35758-108">Příklad</span><span class="sxs-lookup"><span data-stu-id="35758-108">Example</span></span>  
 <span data-ttu-id="35758-109">Tento příklad se skládá z sada soubor, který se kompiluje ve formátu .dll, a .cs souboru, který používá tento .dll.</span><span class="sxs-lookup"><span data-stu-id="35758-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="35758-110">Následující kód je pro soubor .dll a definuje vlastnost kód v souboru .cs přístup.</span><span class="sxs-lookup"><span data-stu-id="35758-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp

// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
