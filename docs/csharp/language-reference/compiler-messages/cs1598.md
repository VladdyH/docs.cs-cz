# <a name="compiler-warning-level-1-cs1598"></a>Upozornění kompilátoru (úroveň 1) CS1598
Analyzátor XML nelze načíst z následujícího důvodu: "důvod". Souborů dokumentace XML 'file' se nevygeneruje.  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) byla zadána možnost, ale nemůže najít a načíst msxml3.dll kompilátoru. Ujistěte se, že je soubor msxml3.dll instalována a registrována.
