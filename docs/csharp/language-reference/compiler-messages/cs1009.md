# <a name="compiler-error-cs1009"></a><span data-ttu-id="f1740-102">Chyba kompilátoru CS1009</span><span class="sxs-lookup"><span data-stu-id="f1740-102">Compiler Error CS1009</span></span>
<span data-ttu-id="f1740-103">Nerozpoznaný řídicí sekvence</span><span class="sxs-lookup"><span data-stu-id="f1740-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="f1740-104">Neočekávaný znak následuje zpětné lomítko (\\) v [řetězec](../../../csharp/language-reference/keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="f1740-104">An unexpected character follows a backslash (\\) in a [string](../../../csharp/language-reference/keywords/string.md).</span></span> <span data-ttu-id="f1740-105">Kompilátor očekává, že jeden platný řídicí znaky.</span><span class="sxs-lookup"><span data-stu-id="f1740-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="f1740-106">Další informace najdete v tématu [řídicí sekvence znaků](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="f1740-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="f1740-107">Následující ukázka generuje CS1009.</span><span class="sxs-lookup"><span data-stu-id="f1740-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="f1740-108">Obvyklou příčinou této chyby je použití zpětného lomítka v názvu souboru, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="f1740-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="f1740-109">Chcete-li tuto chybu vyřešit, použijte "\\\\" nebo @-quoted řetězcový literál, jak ukazuje následující příklad.</span><span class="sxs-lookup"><span data-stu-id="f1740-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
