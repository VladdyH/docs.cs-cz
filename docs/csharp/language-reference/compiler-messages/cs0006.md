# <a name="compiler-error-cs0006"></a>Chyba kompilátoru CS0006
Soubor metadat "název_dll" nebyl nalezen.  
  
 Program byl kompilován s explicitně předaným název souboru, který obsahoval metadata; Knihovna .dll však nebyl nalezen. Další informace najdete v tématu [/Reference (možnosti kompilátoru C#)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md).
