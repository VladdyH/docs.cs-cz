# <a name="compiler-error-cs0016"></a>Chyba kompilátoru CS0016
Nelze zapisovat do výstupního souboru 'file', 'důvod.  
  
 Kompilátor nemůže zapisovat do výstupního souboru. Zkontrolujte cestu k souboru, abyste měli jistotu, že objekt že existuje. Pokud dříve vytvořený soubor už v umístění, ujistěte se, že je zapisovatelný. proto, že žádný proces aktuálně nemá zámek na souboru. Například Ujistěte se, že spustitelný soubor není spuštěna při pokusu o sestavení.
