# <a name="compiler-error-cs0016"></a>Chyba kompilátoru CS0016
Nelze zapsat do výstupního souboru 'file' – "důvod"  
  
 Kompilátor nemůže zapisovat do výstupního souboru. Zkontrolujte cestu k souboru a ujistěte se, že existuje. Pokud již dříve vytvořený soubor v umístění, zajistěte, aby je lze zapisovat, a že žádný proces aktuálně zámek na soubor. Například Ujistěte se, že vaše spustitelný soubor není spuštěna při pokusu o vytvoření.
