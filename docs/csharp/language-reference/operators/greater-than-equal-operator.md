# <a name="gt-operator-c-reference"></a>&gt;= – Operátor (referenční dokumentace jazyka C#)
Všechny typy číselné a výčet definovat "větší než nebo rovno" relační operátor `>=` , který vrací `true` Pokud první operand je větší než nebo rovna hodnotě druhého, `false` jinak.  
  
## <a name="remarks"></a>Poznámky  
 Uživatelem definované typy může přetížit `>=` operátor. Další informace najdete v tématu [operátor](../../../csharp/language-reference/keywords/operator.md). Pokud `>=` je přetížena [ <= ](../../../csharp/language-reference/operators/less-than-equal-operator.md) také musí být přetížený. Operace u celočíselných typů jsou obecně povoleny na výčtu.
