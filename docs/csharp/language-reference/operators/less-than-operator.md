# <a name="lt-operator-c-reference"></a>&lt; Operátor (referenční dokumentace jazyka C#)
Všechny typy číselné a výčet definovat "menší než" relační operátor (`<`), který vrací `true` Pokud první operand je menší než druhý, `false` jinak.  
  
## <a name="remarks"></a>Poznámky  
 Uživatelem definované typy může přetížit `<` – operátor (najdete v části [operátor](../../../csharp/language-reference/keywords/operator.md)). Pokud `<` je přetížena [ > ](../../../csharp/language-reference/operators/greater-than-operator.md) také musí být přetížený. Při binární operátor je přetížena, odpovídající operátor přiřazení, pokud existuje, je také implicitně přetížená.
