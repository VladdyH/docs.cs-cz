# <a name="lt-operator-c-reference"></a>&lt;= – Operátor (referenční dokumentace jazyka C#)
Všechny typy číselné a výčet definice "menší než nebo rovno" relační operátora (`<=`), který vrací `true` Pokud první operand je menší než nebo rovna druhý, `false` jinak.  
  
## <a name="remarks"></a>Poznámky  
 Uživatelem definované typy může přetížit `<=` operátor. Další informace najdete v tématu [operátor](../../../csharp/language-reference/keywords/operator.md). Pokud `<=` je přetížena [ >= ](../../../csharp/language-reference/operators/greater-than-equal-operator.md) také musí být přetížený. Operace u celočíselných typů jsou obecně povoleny na výčtu.
