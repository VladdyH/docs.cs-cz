### <a name="aspnet-accessibility-improvements-in-net-framework-471"></a>Vylepšení přístupnosti ASP.NET v rozhraní .NET Framework 4.7.1

|   |   |
|---|---|
|Podrobnosti|Od verze rozhraní .NET Framework 4.7.1, ASP.NET je vylepšený fungování ovládacích prvků ASP.NET pomocí technologie usnadnění v sadě Visual Studio lepší podporu ASP.NET zákazníkům.  Mezi ně patří následující změny:<ul><li>Změny implementace vzorů usnadnění chybějící uživatelského rozhraní v ovládacích prvcích, jako jsou například dialogové okno Přidat pole v Průvodci zobrazení podrobností nebo dialogové okno Konfigurovat ListView ListView průvodce.</li><li>Změny ke zlepšení zobrazení v režimu vysokého kontrastu, jako je Pager pole editoru služby Data.</li><li>Změny ke zlepšení navigační klávesnice vyskytne pro ovládací prvky, například dialogové okno pole v Průvodci upravit pole stránkování ovládacího prvku DataPager, dialogové okno Konfigurovat ObjectContext nebo dialogové okno Konfigurovat Data Selction Průvodce konfigurace zdroje dat.</li></ul>|
|Návrh|<strong>Postup aktivování nebo zrušení těchto změn</strong>aby návrháři sady Visual Studio moct využívat tyto změny, musí běžet na rozhraní .NET Framework 4.7.1 nebo novější. Webová aplikace využívat tyto změny v některém z následujících způsobů:<ul><li>Nainstalujte Visual Studio 2017 15.3 nebo novější, který podporuje nové funkce pro usnadnění přístupu s přepínačem následující AppContext ve výchozím nastavení.</li><li>Vyjádření výslovného nesouhlasu chování starší verze usnadnění přidáním <code>Switch.UseLegacyAccessibilityFeatures</code> AppContext přepínač tak, aby <code>&lt;runtime&gt;</code> část v devenv.exe.config souboru a jeho nastavení na hodnotu <code>false</code>, jak ukazuje následující příklad.</li></ul><pre><code class="lang-xml">&lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-8&quot;?&gt;&#13;&#10;&lt;configuration&gt;&#13;&#10;&lt;runtime&gt;&#13;&#10;...&#13;&#10;&lt;!-- AppContextSwitchOverrides value attribute is in the form of &#39;key1=true/false;key2=true/false&#39;  --&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.UseLegacyAccessibilityFeatures=false&quot; /&gt;&#13;&#10;...&#13;&#10;&lt;/runtime&gt;&#13;&#10;&lt;/configuration&gt;&#13;&#10;</code></pre>Aplikace, které cílí na rozhraní .NET Framework 4.7.1 nebo novější a chcete zachovat starší verze usnadnění chování můžete vyjádřit výslovný souhlas s použitím funkce usnadnění starší verze explicitně nastavením tento přepínač AppContext na <code>true</code>.|
