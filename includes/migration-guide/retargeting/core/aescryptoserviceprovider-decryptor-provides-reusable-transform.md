### <a name="aescryptoserviceprovider-decryptor-provides-a-reusable-transform"></a>Znovu použitelné transformace poskytuje AesCryptoServiceProvider modul pro dešifrování.

|   |   |
|---|---|
|Podrobnosti|Počínaje aplikací, které cílí na rozhraní .NET Framework 4.6.2, <xref:System.Security.Cryptography.AesCryptoServiceProvider> modul pro dešifrování. nabízí opakovaně použitelné transformace. Po volání <xref:System.Security.Cryptography.CryptoAPITransform.TransformFinalBlock(System.Byte[],System.Int32,System.Int32)?displayProperty=name>, transformovat budou znovu inicializována a lze opětovně použít. Pro aplikace, které mají starší verze rozhraní .NET Framework, pokus o znovu použít, modul pro dešifrování volání <xref:System.Security.Cryptography.CryptoAPITransform.TransformBlock(System.Byte[],System.Int32,System.Int32,System.Byte[],System.Int32)?displayProperty=name> po volání <xref:System.Security.Cryptography.CryptoAPITransform.TransformFinalBlock(System.Byte[],System.Int32,System.Int32)?displayProperty=name> vyvolá <xref:System.Security.Cryptography.CryptographicException> nebo je výsledný poškozená data.|
|Návrh|Dopad této změny musí být minimální, protože se jedná o očekávané chování. Aplikace, které závisí na předchozí chování můžete vyjádření výslovného nesouhlasu s pomocí přidáním následující konfigurace nastavení na <code>&lt;runtime&gt;</code> oddílu konfiguračního souboru aplikace:<pre><code class="lang-xml">&lt;runtime&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.System.Security.Cryptography.AesCryptoServiceProvider.DontCorrectlyResetDecryptor=true&quot;/&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;</code></pre>Kromě toho aplikace, které cílí na předchozí verzi rozhraní .NET Framework, ale běží v rámci verze rozhraní .NET Framework od verze rozhraní .NET Framework 4.6.2 můžete vyjádřit výslovný souhlas k němu přidáním následující nastavení konfigurace na <code>&lt;runtime&gt;</code> části konfiguračního souboru aplikace:<pre><code class="lang-xml">&lt;runtime&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.System.Security.Cryptography.AesCryptoServiceProvider.DontCorrectlyResetDecryptor=false&quot;/&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;</code></pre>|
