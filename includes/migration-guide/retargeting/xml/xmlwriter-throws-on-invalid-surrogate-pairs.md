### <a name="xmlwriter-throws-on-invalid-surrogate-pairs"></a>Objekt XmlWriter vyvolá na neplatný náhradní dvojice

|   |   |
|---|---|
|Podrobnosti|Pro aplikace, které cílí na rozhraní .NET Framework 4.5.2 nebo dřívějších verzí zápis dvojici neplatný náhradní, že pomocí záložní výjimek vždy nevyvolá výjimku. Pro aplikace, které cílí na rozhraní .NET Framework 4.6, pokus o zápis neplatný náhradní pár vyvolá <xref:System.ArgumentException?displayProperty=name>.|
|Návrh|V případě potřeby může být toto rozdělení, kterým se lze vyhnout cílením rozhraní .NET Framework 4.5.2 nebo starším. Případně může být neplatný náhradní dvojice předběžného zpracování do platný kód xml před jejich zápis.|
